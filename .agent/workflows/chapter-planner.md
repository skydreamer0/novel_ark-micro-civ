# 章節節奏設計工作流 (Chapter Planner)

此工作流確保每一章節都符合「爽文」節奏，並同時推進戰力、文明與陰謀三條線。

## 執行步驟

1. **設定章節目標**：
   - 確定本章要完成的「三線進度」：
     - **個人戰力**：有新的裝備/數據提升嗎？
     - **文明成長**：內部小人有沒有新的演化/信仰反饋？
     - **陰謀線**：是否揭露了一點高層的秘密？

2. **三段式結構**：
   - **開頭 (The Hook)**：立即的危機或未解決的懸念。
   - **中段 (Conflict)**：主角利用「維修工」技能或「主機截流」優勢翻盤。
   - **結尾 (Cliffhanger)**：為下一章埋下伏筆。

3. **感官填充**：
   - 加入至少一個賽博感官細節（如：義體過熱的滋滋聲、虛擬顯示幕的亂碼）。

4. **產出大綱**：
   - 條列式列出本章的 3-5 個關鍵場景。
