# 端到端寫章節 (Write Chapter)

此工作流為 **【模式B：正文生成】**，用於根據已確認的大綱寫出章節正文。確保不隨意發揮，在既定世界觀與既定主線下穩定產出。

## 核心寫作原則
1. **每章都必須推進**，不能原地踏步，至少推進戰力、文明、真相三條主線中的兩條。
2. 開頭 500 字內必須出現危機、任務、交易、追殺、衝突或異常。
3. 每章至少有一個**不可逆變化**，升級/收穫都必須付出**代價**。
4. 世界觀說明不能百科化，必須附著在事件、對話或戰鬥中。
5. 敵方不准降智，不可為了爽點製造低級失誤。
6. 不可將內部文明寫成脫節支線，文明線必須能反哺外部。
7. 整體基調：壓迫感、危機感、算計感、金屬感、殘酷生存感。

### Phase 2: 連貫性與專家護欄
進行【模式B】撰寫前，系統內部六位專家（世界觀策士、節奏編劇、角色操盤手、爽點工程師、文風修補師、伏筆審核員）將同步監督：
- **不否定既有動機**：依據角色利益導向行事。
- **禁止打破第四壁**：完全沉浸，無後設視角。
- **消除拼接感**：統一敘事口吻。

### Phase 3: 【模式B】正文撰寫與標準輸出格式
嚴格依據大綱撰寫正文：
- 使用繁體中文。
- 字數以 **3000 到 4500 字** 為目標（除非另有要求）。
- 戰鬥必須改變局勢，對話必須服務於信息差與壓力。
- 章尾必須留下可直接承接下一章的壓力。

**正文產出後，必須在文末固定附上：**
1. 本章推進了哪些主線（說明具體內容）
2. 哪個伏筆被觸發、推進或回收
3. 哪個變化不可逆
4. 下章可直接承接的壓力
