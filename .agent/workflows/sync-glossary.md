# 名詞辭典同步 (Sync Glossary)

掃描指定章節範圍，比對 `docs/03_世界觀/05_專有名詞辭典.md`，找出遺漏名詞並補充。

## 執行步驟

### Step 1: 讀取現有辭典
// turbo
1. 讀取 `docs/03_世界觀/05_專有名詞辭典.md`，建立已收錄名詞清單。

### Step 2: 掃描章節
// turbo
2. 讀取使用者指定的章節檔案（預設：最近 5 章）。
3. 從正文中提取以下類型的候選名詞：
   - 【】包裹的系統提示（如：【絕對消化】、【群體共振】）
   - 「」包裹的專有名詞（重複出現 2 次以上者）
   - 明確的場景名、組織名、人物名、技術名
   - 裂縫名稱（R1~R5 + 詞綴格式）

### Step 3: 比對與篩選
4. 將候選名詞與已收錄清單比對，過濾掉已存在的項目。
5. 排除普通對話與非專有用語。
6. 將剩餘的新名詞整理為：

| 名詞 | 建議分類 | 建議定義 | 首出章節 |
| ---- | -------- | -------- | -------- |

### Step 4: 更新辭典
7. **呈給使用者確認**——使用者可刪除不需要的條目。
8. 使用者確認後，將新名詞寫入 `docs/03_世界觀/05_專有名詞辭典.md` 對應分類區塊。
