# 章節融合修補工作流 (Chapter Fusion)

此工作流為 **【模式C：融合修補】**，主要用途是把舊稿、新設定、使用者強制要求的新橋段進行自然融合，同時消除拼接感與補丁感。

## 核心寫作原則與專家機制
- 本工作流核心依賴 **【文風修補師】** 與 **【世界觀策士】**。
- 不可將新設定硬插，必須將其融合在損失、交易、任務或制度壓迫的具體事件中。
- 保證角色口吻不崩壞，情緒與因果必須連續。

## 執行步驟

### Phase 1: 引入資訊
確認使用者提供之資訊：
- **【必須沿用設定】**：不可違反的設定與角色狀態。
- **【禁止新增設定】**：不能亂開的新能力或組織。
- **【必須保留片段】**：舊稿、舊橋段、指定台詞等。
- **【額外要求】**：例如開場戰鬥、更陰冷、強化文明線反哺等。

### Phase 2: 【模式C】標準輸出格式
進行修補時，必須先輸出分析結果，再提供融合後正文。固定輸出格式如下：

1. 本章目前的 3 個主要斷裂點（列出舊稿與新要求之間可能的不連貫處）
2. 每個斷裂點的修補策略（例如如何將文明反哺自然揉入物理戰鬥）
3. 角色口吻與動機是否一致（檢查角色是否突然失智或變成無償聖母）
4. 設定與因果是否連續
5. 融合後正文（依據上述策略重寫/擴寫的完整內文，並須注意滿足基礎字數要求，具備壓迫感與算計感，且章末有勾子）

完成後，若融合之正文等同完成整章，可接續進行 `/chapter-qa` 審查以確保達成連載標準。
