# 世界觀擴張工作流 (World Builder)

此工作流用於在現有的「賽博母艦」框架下，定義新的細節（如：特定艙段、組織或技術）。

## 執行步驟

1. **讀取核心設定**：
   - 務必先讀取 `README.md` 確保理解「雙層煉化權限」與「文明主機」邏輯。

2. **定義新元素**：
   - 使用以下結構描述新物件：
     - **名稱**：(例如：第 13 號貧民窟「鐵鏽帶」)
     - **視覺特徵**：(陰冷、霓虹燈閃爍、機油味)
     - **壓迫機制**：(該地段受哪個公司控制？稅收如何？)
     - **機遇點**：(主角能在這裡拾取什麼外域殘骸？)

3. **衝突檢查**：
   - 檢查新設定是否破壞了「母艦宇宙正在衰亡」的大背景。

4. **更新文檔**：
   - 將確認的新設定增補至 `README.md` 的相關章節。
