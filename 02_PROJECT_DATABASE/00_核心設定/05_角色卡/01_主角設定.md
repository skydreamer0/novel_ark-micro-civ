# 主角人設與行為原則 (Protagonist Arc)

在網文中，主角的性格決定了讀者的代入感。這份文件規範了「林燼」在面臨選擇時，會給出什麼樣的反應。

---

## 1. 基礎人設面板卡

*   **姓名**：林燼 (Lin Jin)
*   **現任身分**：末日母艦第九維修段・底層維修工 / 黑市臨時拾荒者
*   **前世身分**：(保留空白，可設定為地球的高階工程師或末日倖存者，以解釋他為何能極快適應廢土)
*   **外在形象**：裝著劣質/生鏽的輔助義肢，常年帶著機油味與血腥味。眼神像疲憊但隨時能暴起的野狗。

---

## 2. 核心行為邏輯 (殺伐果斷的界線)

為了確保主角「不聖母」但又「保有魅力」，請遵守以下守則：

*   **規則一：【絕對的利益導向與生存第一】**
    *   在自己吃不飽、活不下去的時候，他**絕對不會**去救沒有利用價值的路人。
    *   誰擋他賺「存活點」和「升級主機」，他就殺誰，不管對方有什麼悲慘身世。

*   **規則二：【契約精神與價值交換】**
    *   他不是無差別殺人的瘋子。對於幫過他的人（例如黑市中介蘇岚），他會以等價甚至更高的利益回報。
    *   他不相信「無緣無故的好意」，任何人給他東西，他第一反應是「代價是什麼？」。

*   **規則三：【對待敵人的原則：斬草除根】**
    *   不廢話、不聽反派死前懺悔、能補刀就補刀、打不過絕對不硬剛而是撤退下毒。
    *   **戰鬥風格**：充分利用「維修工」的場地優勢與「微縮文明」的資訊差，喜歡玩陰的、埋炸彈、切斷維生系統。

---

## 3. 人物成長弧線 (The Character Arc)

### 第一卷：【求存者】(Survivor)
*   **心理狀態**：只想活過明天。誰要我死，我先讓他死。
*   **對待主機的態度**：把微縮文明當成「無機質的裝備提款機」和「垃圾處理廠」。
*   **核心恐懼**：被丟進回收序列，喪失自我意識。
*   **階段能力**：以「絕對消化 + 危險感知」求生，優先換取續命資源與逃生視窗。
*   **內部代價**：
    *   **異端分裂（萌芽）**：
        *   觸發條件：主角連續以血祭式高壓命令索取信仰，卻未兌現保護承諾。
        *   主角介入手段：降階發佈「可驗證神諭」（如固定配糧/避難節點），讓信徒能用結果驗證神意，壓回流言。
    *   **祭司壟斷（萌芽）**：
        *   觸發條件：早期知識傳承全握在單一祭司群體，普通信徒無法直接接觸儀式規程。
        *   主角介入手段：開放基礎祭儀手冊與公開考核，削弱祭司「代行神意」的資訊壟斷。

### 第二卷：【掠食者】(Predator)
*   **心理狀態**：已經解決了生存溫飽，開始主動籌劃如何爬上中層階級，撕下企業聯盟的肉。
*   **對待主機的態度**：逐漸意識到文明信徒對他的「信仰狂熱」。開始懂得主動引導內部文明的走向（當一個合格的神）。
*   **階段能力**：以「神權推演 + 物質具現」進入主動掠奪期，能把內部技術短時兌現為外部戰力。
*   **內部代價**：
    *   **戰士軍功政治（成形）**：
        *   觸發條件：外戰勝利主要由戰士階層取得，配給與話語權過度向軍功傾斜。
        *   主角介入手段：建立「戰功上限 + 民生底配」雙軌制度，強制軍功兌現與糧食、工時、治安責任綁定。
    *   **工匠技術封鎖（成形）**：
        *   觸發條件：工匠行會以安全與保密為由，拒絕開放核心鍛造流程，導致反哺來源單點化。
        *   主角介入手段：推行模組化分段製造與交叉審核，讓關鍵技術至少有兩套可替代鏈路。

### 第三卷以後：【執棋者 / 新主宰】(The Arbiter)
*   **心理狀態**：發現母艦高層與整個宇宙收割體系的真相。從「在規則中求生」轉變為「掀翻現有規則」。
*   **對待主機的態度**：將內部文明視為真正的「同族 / 家人」，準備用體內的群星，吞噬掉外面的腐朽宇宙。
*   **階段能力**：從單次反哺轉向「文明級法則協同」，可同時調度祭司推演、戰士投影與工匠量產。
*   **內部代價**：
    *   **異端分裂（全面化）**：
        *   觸發條件：多區域同時擴張後，地方神學與中央神諭出現解釋衝突。
        *   主角介入手段：召開跨城議會，允許「地方儀式差異」但統一底層戒律，避免內戰。
    *   **祭司壟斷（制度化）**：
        *   觸發條件：祭司階層把持歷史詮釋權，開始以神意名義干預分配與審判。
        *   主角介入手段：建立信仰審計庭，由祭司、戰士、工匠與平民四方共審重大神諭。
    *   **戰士軍功政治（軍閥化）**：
        *   觸發條件：邊境常態戰爭使軍團長擁有穩定資源來源，出現半獨立封地。
        *   主角介入手段：輪調軍團指揮權，並將補給控制權交由非軍系統，拆解私人武裝根基。
    *   **工匠技術封鎖（寡頭化）**：
        *   觸發條件：高階工坊形成專利神匠同盟，限制關鍵技術外流以維持高價。
        *   主角介入手段：建立公共技術庫與戰時強制授權條款，確保文明生產力不被綁架。

## 4. 寫作檢查清單 (主角動態檢核)

每次寫完一章，問自己三個問題：
1. 林燼這章有賺到錢/資源/情報嗎？(**如果沒有，這章就是水字數**)
2. 林燼這章的決定，有沒有暴露他不合時宜的善意？(**如果為了救無關的NPC而受傷，讀者會罵**)
3. 主機的升級進度條，有沒有因為這章的行動而往前推進？
