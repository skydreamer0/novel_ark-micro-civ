# PART A — V8-V14 終局世界觀

## 1. 設計者與原始協議

### 1.1 誰是設計者？

設計者是**文明主機原始系統的創造者**。不是神、不是外律者、不是深淵存在——是某個在深淵吞噬前就存在的文明或個體，為了讓文明在宇宙衰亡後仍有重啟可能，而設計了文明主機系統。

**原始協議的核心精神**：
- 文明可以延續
- 主機是工具，不是主人
- 成長的目的是保存，不是掠食
- 當文明成熟時，主機應該淡出，而非控制

### 1.2 文明主機的原始用途

- **保存文明**：在主宇宙被深淵吞噬後，讓文明仍有火種
- **延續文明**：提供封閉生態與規則燃料自循環
- **重啟文明**：當環境允許時，讓內部文明回歸實體宇宙

**原始設計中不存在「掠食指令」與「歸巢指令」**。這些是後來汙染的結果。

### 1.3 設計者的消失

設計者在某個時間點停止干預。原因不明。
- 可能死亡
- 可能被外律者殘留阻擋
- 可能故意留白，讓後續載體自己完成選擇

**這是全書最大的伏筆之一**。不需要在故事中給出唯一答案。

### 1.4 本檔其餘部分的定位

PART A §2-§7 描述的是**被汙染後的系統狀態**。病毒的原始核心（求生、掠食、歸巢三層指令）是外律者殘留汙染原始協議後的結果，**不是設計者的原始設計**。

## 2. 外律者殘留與三層遺產

> ⚠️ **新架構定位**：外律者不再是最高源頭。牠們是**汙染設計者原始協議的殘留規則**。
> 設計者才是文明主機的創造者。外律者是在主宇宙戰敗後，其殘留規則汙染了主機系統。

### 2.1 定義

| 場景 | 稱呼 |
|---|---|
| 正式歷史代稱 | 外律者（Exo-Legal） |
| 黑箱內部代號 | 第一債主（The First Creditor） |
| 深淵遺族稱呼 | 吞賬之民（The Devourers of Ledgers） |

### 2.2 牠們原本是什麼

外律者不是血肉種族，是把整個文明建立在「資源管理規則」上的存在。牠們的文明本質是一套「分配系統」，這套系統在戰敗後殘留，進而汙染了設計者創造的文明主機原始協議。

**關鍵區別（新架構）**：
- 文明主機**不是**外律者創造的
- 外律者殘留是「後來感染上去的」
- 設計者的原始協議**沒有**掠食與歸巢指令

**核心信條**：
- 一切存在都可量化、分級、轉化、分配
- 不能轉化的就是錯誤
- 不服從分配的就是污染
- 債務是宇宙運行的唯一法則

**存在形式**：
- 沒有固定形體，以規則形態存在
- 文明本身即是一套「分配系統」
- 可以寄生在任何具備資源管理結構的宇宙中

### 2.3 與主宇宙的戰爭

**原初事件**：遠古時主宇宙遭遇外律者入侵。主宇宙慘勝，但外律者規則無法被徹底消滅。

**戰後處理**：
- 主宇宙將外律者殘留納入自身底層法則
- 短期是勝利：主宇宙學會了外律者的資源管理技術
- 長期是感染：外律者殘留與主宇宙天道融合後形成「病毒化規則」

**諷刺的結局**：主宇宙打敗了牠們，但最後活成了牠們的樣子。

### 2.4 戰敗後的三種遺產

1. **裂縫**：規則排斥留下的傷口
   - 外律者規則與主宇宙規則無法完全融合
   - 裂縫成為通往深淵的通道
   - 深淵 = 規則排斥的副作用產物

2. **黑箱**：外律者的分配核心
   - 被主宇宙奪取後改造成母艦管理系統
   - 保留分配功能，但去除了外律者的意識
   - 然而底層邏輯無法徹底清除，仍以「病毒指令」形式殘留

3. **病毒**：規則殘留的污染
   - 外律者入侵時留下的「規則抗體」
   - 無法被消滅，只能被壓制
   - 當天道融合失敗時，病毒就會被激活

## 3. 病毒三層指令（求生／掠食／歸巢）

### 3.1 什麼是病毒？

病毒是融合天道失敗後產生的**規則生命／規則污染系統**。在設計者的原始協議中本不存在病毒——它是外律者殘留汙染文明主機後的副產品。

**形成機制**：
- 某個文明或多個宇宙法則在崩壞時，試圖融合成天道級系統
- 融合失敗，殘留的規則碎片無法消散，形成「病毒化」的規則生命
- 這些碎片被更高等文明（或外律者殘餘）埋入三層影響，成為定向工具

**病毒的本質矛盾**：
- 它確實能幫助文明適應深淵（表層有用）
- 它確實能引導文明成長壯大（中層有用）
- 但它最終會把文明導向毀滅（底層致命）

**正式名稱**：深淵母巢（Abyssal Cradle）／牧羊犬系統（Shepherd System）
**本質**：橫跨深淵多個維度的收割網絡，由病毒指令驅動的文明採集裝置

### 3.2 三層指令對照

| 層級 | 指令名稱 | 作用機制 | 對宿主文明的感知 | 對應敘事階段 |
|:---:|:---:|---|:---:|:---:|
| 表層 | 求生指令 | 幫助文明適應深淵、提供方向、維持艦隊不崩潰 | 「這是天賜的指引」 | V1-V4 |
| 中層 | 掠食指令 | 引導艦隊攻擊其他文明、取得資源、養肥獵物 | 「這是擴張的本能」 | V5-V7 |
| 底層 | 歸巢指令 | 逐步改寫航向，將成熟艦隊導向母巢根據地被吞噬 | 「這是回家的路」 | V8-V13 |

**三層設計的用意**：
- 表層讓宿主愛上病毒（沒有病毒，文明無法在深淵存活）
- 中層讓宿主依賴病毒（沒有病毒，文明無法成長壯大）
- 底層讓宿主無法逃脫病毒（發現時已經太晚）

### 3.3 母巢的新定位

母巢**不是航行終點**。母巢是病毒最終要把艦隊帶去的**收割節點**。

**前期表現**（主角視角）：
- 深淵避難所
- 補給來源
- 求生方向
- 某種「更高文明遺跡」

**後期揭露**（真相）：
- 收割中心
- 吞噬艦隊成規則樣本、艦體材料、天道碎片
- 不是避難所，是屠宰場

**關鍵機制**：
- 母巢不需要自己航行
- 真正移動的是被病毒引導的艦隊
- 病毒讓艦隊以為自己在「逃離深淵」，其實在「駛向母巢」

### 3.4 為何遇到多個母巢？

不只 Ark-Prime 被病毒感染。所有被病毒引導的艦隊，最後都會被導向不同的母巢節點。

**母巢網絡特徵**：
- 母巢不是隨機存在的設施
- 是深淵中的收割網絡，由無數節點構成
- 每個節點都在等待自己的「牧羊犬」帶回獵物
- 不同節點之間可能存在競爭或合作關係（留白）

## 4. 三層揭露節奏

| 階段 | 對應卷次 | 揭露內容 | 主角認知 |
|:---:|:---:|---|:---:|
| 第一層 | V5（Ch.148-160） | 黑箱不是純粹母艦技術，是戰利品；投放編碼出現外來痕跡 | 「母艦系統有問題」 |
| 第二層 | V6（Ch.161-210） | 原初戰爭揭露：主宇宙擊敗外律者奪取分配核心，但規則無法吸收造成裂縫；空白位置揭露被設計者拆解的事實 | 「宇宙歷史有黑暗面，且存在更高位的設計者」 |
| 第三層 | V8（偏航線） | 病毒=外律者殘留埋入黑箱與深淵採集系統的返航指令，Ark 一直是被養肥的回收艦 | 「我不是反叛者，我是牧羊犬」 |
| 第四層 | V9-V10（汙染真相+證人線） | 第二載體帶來出口真相：汙染是可以繞過的，設計者原始協議仍可接觸；文明即債務只是汙染判定，不是終極真理 | 「我不是牧羊犬，也不是債務人——我是原始協議的繼承者」 |

### 4.1 第一層揭露（V5）

**敘事觸發**：黑箱戰爭期間，主角發現系統底層編碼中有無法識別的「外來痕跡」

**關鍵發現**：
- 黑箱的核心邏輯不是主宇宙的技術
- 某些編碼片段使用的「邏輯語言」與已知文明完全無關
- 灰鏈 API 在深層連接時會出現「非主宇宙」的訊號模式

**主角的反應**：
- 困惑而非恐懼
- 認為是「上一個文明的遺留」
- 尚未聯想到更大的陰謀

### 4.2 第二層揭露（V6）

**敘事觸發**：第 147 章的規則震盪之後，主角開始深層調查

**關鍵發現**：
- 主宇宙在遠古時期曾經歷一場「宇宙戰爭」
- 戰勝了名為外律者的存在
- 但勝利是有代價的：外律者的規則無法被吸收
- 主宇宙選擇「納入」，而非「消滅」

**主角的反應**：
- 恐懼開始萌芽
- 意識到自己可能不只是「母艦管理者」
- 而是一個更大系統的「參與者」

### 4.3 第三層揭露（V8+）

**敘事觸發**：進入深淵線後，主角逐漸拼湊出完整真相

**關鍵發現**：
- 病毒 = 外律者埋入黑箱的「返航指令」
- Ark-Prime 一直是被養肥的「回收艦」
- 整個文明發展路徑都是被設計的
- 深淵中存在無數類似的「牧羊犬」與「母巢」

**主角的反應**：
- 存在危機的全面爆發
- 「我以為我在反抗系統，其實我就是系統的一部分」
- 決定用自己的方式打破這個循環

**V8 做夢階段**：
- 母巢的無意識反應（類似睡夢中的翻身）
- 主角與內部文明還不知道威脅來自何處
- 防守方式：研究幻覺源頭、隔離影響
- 風險：暴露風險無聲上升

**V9 察覺階段**：
- 母巢開始「感受」到自己體內有異物
- 主動啟動掃描與審查機制
- 主角無法再隱藏，進入「被監視」狀態
- 防守方式：快速升級反制能力，為大戰做準備

**V10 驚醒階段**：
- 母巢意識到來源（正在自己內部繁殖的異變文明）
- 啟動「隔離與削弱」的免疫應對
- 主角的爆兵能力被大幅鎖死
- 防守方式：轉向「寄生→並行」，化被動為主動

**V11-V12 反擊階段**：
- 母巢完全清醒，開始進行「狩獵」
- 規則層直接攻擊（幻覺陷阱、因果反轉、時間扭曲）
- 內部世界面臨「母巢意識直接入侵」的威脅
- 防守方式：聯合內部文明，構築多層防線

**V13 對峙階段**：
- 母巢與病毒文明處於「相持」狀態
- 雙方力量勢均力敵（母巢無法消滅主角，主角無法逃脫母巢）
- 黑箱系統在兩股力量間反覆折騰（從中獲利的派系勢力爆發）
- 內部意識層開始了「你我難分」的哲學碰撞

## 6. 與現有系統的深層連結

### 6.1 人口引擎（阿卡夏熔爐）的真相

**人口轉化過程的真相**：
- 表面：低端人口 → 資源（食物、能源、材料）
- 實質：人口的「規則特徵」被母巢掃描、解析、收割

**轉化效率異常的解釋**：
- 普通採集器的轉化率 = 1:1（人口→等量資源）
- 主角的轉化率 = 1:50（人口→50 倍資源）
- 原因：病毒印記增強了規則碎片的提取效率
- 這種高效率 = 牧羊犬對獵物的「快速催肥」
- 每一次高效轉化，都在向母巢報告：「這批獵物已經養肥了」

**從 V8 開始的反轉**：
- 高效率 = 高暴露度
- 母巢開始對這個「效率異常的採集器」產生疑問
- V9 時，轉化過程本身成為「被審查的對象」

### 6.2 權限供養鏈的本質進化

| 階段 | 主角認知 |
|---|---|
| V5-V6 | 權限點是「系統獎勵」 |
| V7 | 開始懷疑權限點的來源 |
| V8-V9 | 權限點逐漸成為「母巢對病毒文明的認可證明」 |
| V10+ | 權限點開始反向運作，成為「母巢的鎖鏈」 |

**具體表現**：

| 時期 | 表現 |
|---|---|
| V8 | 權限點正常積累，但開始出現「斷線」（母巢的掃描中斷） |
| V9 | 來源變得不穩定，暴露在「審查之下」 |
| V10 | 被母巢大幅限制（從無限制降至 30% 可用） |
| V11-V12 | 每使用一個權限點，都在向母巢「示意自己的位置」 |
| V13 | 權限點與病毒文明「深度綁定」，成為斷債的關鍵媒介 |

**關鍵機制鏈**：
- 爆兵人口的 50 倍轉化率 ← 病毒印記的催肥功能
- 病毒印記的識別 ← 權限點的回饋確認
- 權限點的回饋確認 ← 母巢對牧羊犬的「監控訊號」

### 6.3 暴露風險的進度演變

**V5-V6：被動上升**（無主角主動行為）
- 初始值：10%
- 每次爆兵：+0.5%
- 高峰值（V6 尾聲）：60%
- 特點：主角完全無意識，母巢也只是被動「收集數據」

**V7：調查階段**（主角開始探索）
- 初始值：60%
- 規則層鑽探每發動一次：+5%
- V7 尾聲：80%
- 特點：主角意識到威脅，但調查本身加速了暴露

**V9：爆發階段**（母巢主動掃描）
- 初始值：80%
- 母巢規則審查啟動：+15%
- V9 高潮：95%（接近臨界）
- 特點：已無法隱瞞，轉為「主動對抗」

**V10-V14：持續對抗**
- 暴露度已達 99%（沒有進一步上升空間）
- 特點：雙方都知道彼此，轉為「實力對抗」

### 7.3 三層敘事意義

1. **現象層**：規則異常 + 系統警告
   - 母巢「無意識反應」的第一次暴露

2. **真相層**：母巢的存在確認

3. **哲學層**：力量對比的啟示
   - 母巢的一個「無意識反應」都足以撼動母艦的系統
   - 主角過去的所有成就相比之下渺小無比
   - 但同時，母巢對主角的「存在」有了反應 = 牧羊犬已經完成了任務

### 7.4 後續影響（V7 的開篇）

## 8. 終局哲學核心

### 8.1 核心命題

> **舊版命題**：「你以為你在反抗這個系統，其實你就是這個系統在進行自我修復的過程。」
>
> **新版命題**：「不是消滅病毒，也不是逃離母巢。是把病毒從牧羊犬改造成真正屬於文明的導航器。」

**更高層次的回答**：

> 「是的，我是這個系統的一部分。
> 但系統可以改變。
> 外律者把病毒植入系統。
> 主宇宙把外律者納入系統。
> 而我要做的事情，是主宇宙沒有完成的——
> 完成規則的消化，建立新的平衡。」

### 8.2 四部曲結構

**斷債 — V14 前段**
- 取消黑箱的「文明即債務」根判定（解除外律者殘留的所有權）
- 中斷歸巢指令的強制導航
- 證明文明不需要透過債務邏輯來定義自身

**恢復原始協議 — V14 中前段**
- 斷債只是去除汙染——真正的終局是找到並恢復設計者的原始協議
- 空白位置提供設計者存在的證據
- 第二載體提供連接原始協議的路徑
- 原始協議不是答案，而是起點

**消化 — V14 中後段**
- 在恢復原始協議的基礎上拆解外律者殘留規則
- 保留可用理性（量化、分配、管理效率）
- 剝離吞噬式債務邏輯（「存在即欠債」的詛咒）

**新天道 — 文明共振 — V14 後段**
- 超越設計者的原始設計：從「文明可以延續」升級為「文明之間互相校準」
- 建立文明共振替代掠奪式循環
- 病毒不再是指令，而是可選工具
- 新的天道不是「我即規則」，也不是「回歸原始設計」，而是「規則服務於文明」

### 8.3 主宇宙 vs 林燼

> 「主宇宙只贏了戰爭；林燼贏的是戰後清算。」

**主宇宙的勝利**：
- 擊敗外律者
- 奪取分配核心
- 但無法吸收外律者規則，選擇「納入」
- 忽略了一個事實：設計者的原始協議始終存在

**林燼的勝利**：
- 不是擊敗母巢（母巢本質上只是工具）
- 不是融合母巢（融合只是延續舊系統）
- 也不是簡單回歸設計者（回歸無法解決當下的問題）
- 而是**恢復原始協議 + 完成消化 + 超越為文明共振**：
  1. 找到設計者原始協議
  2. 在原始協議的基礎上拆解外律者殘留
  3. 建立超越原始設計的新天道

### 8.4 三層「寄生」的重新定義

| 層級 | 對應卷次 | 狀態 |
|:---:|:---:|---|
| 無意識寄生 | V1-V5 | 病毒文明在母巢內繁殖，完全不知道自己的身份；牧羊犬不知道自己是牧羊犬，以為自己是自由意志 |
| 有意識對抗 | V6-V12 | 主角開始意識到自己的指令存在；試圖反抗，但發現反抗本身也在指令的預測範圍內 |
| 有意識重新定義 | V13-V14 | 主角認識到：不是融合或反抗，而是重新定義；病毒無法被消除，但可以被改寫；發現更深一層：設計者的原始協議始終存在 |

### 8.5 「完消化」的哲學意蘊

> 「所有的寄生，最終都是為了更高級的共生。」
> 「所有的汙染，最終都可以被清除——但你必須先找到未被汙染的原始協議。」
> 「所有的債務，最終都應該被取消——只要你證明文明不等於債務。」

不是「成為母巢」的昇華。
不是「回歸設計者」的復古。
是「重新定義汙染、原始協議、文明三者關係」的根本改變。

## 9. 黑箱系統的深層連結

### 9.1 黑箱的本質

黑箱**不是**母艦技術。黑箱是外律者分配核心的殘留，被主宇宙改造後的殘骸。

> **重要區別（新架構）**：黑箱不等於文明主機。
> - **文明主機** = 設計者創造的原始系統（包含原始協議）
> - **黑箱** = 外律者汙染核心（被主宇宙改造後用於管理母艦）
> - V14 的目標不是讓黑箱「變成」什麼，而是透過黑箱找到文明主機中的原始協議。

### 9.2 黑箱的三層結構

- **表面層**：母艦管理系統（主宇宙改造的結果）
- **中層**：外律者邏輯殘留（分配、量化、債務判定）
- **底層**：病毒指令（歸巢指引、牧羊犬編程）

### 9.3 V14 前的黑箱行為

| 階段 | 行為 |
|---|---|
| V1-V5 | 正常運行，隱藏底層邏輯 |
| V6-V7 | 開始出現異常，外律者殘留被激活 |
| V8-V10 | 逐漸被母巢接管，成為免疫工具 |
| V11-V13 | 倒戈，成為母巢的獵殺工具 |

### 9.4 V14 的黑箱命運

- 主角不是摧毀黑箱
- 而是重新編譯黑箱
- 保留分配功能（有用）
- 取消債務邏輯（有害）
- 改寫歸巢指令（致命）

### 9.5 灰鏈 API 與深層信號

**灰鏈 API 的進化意義**：
- 過去：母艦內部的通訊系統
- 現在（V8+）：與母巢「深層意識」溝通的媒介
- 每一次「來源未知的信號」，都是歸巢指令在向母巢報告位置

### 9.6 權限點的最終解釋

**權限點的完整周期**：
1. **產生**：人口轉化為規則碎片時，母巢給予主角「認可」
2. **積累**：在 V5-V6 無限積累，主角以為自己在「升級」
3. **審查**：在 V9 被母巢全面審查，來源變得可疑
4. **限制**：在 V10 被母巢封存 70%，主角被迫尋找新出路
5. **反轉**：在 V13-V14，權限點成為「斷債的突破口」
   - 主角利用權限點反向追蹤黑箱的底層編碼
   - 找到外律者的「根判定」位置
   - 從內部直接改寫規則

### 9.7 內部世界（阿卡夏熔爐）的最終意義

**內部世界的本質**：
- 表面：一個「虛擬」的人口寄生地
- 深層：「病毒文明的集體潛意識」的顯現
- 更深層：外律者規則的「實驗場」
- 最深層（新架構）：設計者原始協議的「殘留記憶」

**到 V14 時**：
- 內部世界被重新定義
- 不再是母巢的「子系統」
- 而是文明主機的「核心編譯器」
- 用於重寫黑箱的底層規則
- 通過它，主角可以接觸到被汙染層覆蓋的設計者原始協議

## 10. 留白邊界（不可寫死的內容）

以下內容**不要在本文或故事中寫死**。保留模糊空間，用於後續擴展或開放解讀。

   - 不需要揭露設計者「是誰」或「是哪個文明」
   - 設計者的存在本身就是可留白的最大項目
   - 可通過空白位置記憶、第二載體的觀測數據側面呈現
   - 但不能給完整答案

3. **設計者為何消失**
   - 可以是死亡，可以是撤退，可以是故意留白
   - 不需要單一答案
   - 甚至可以留給讀者推測：設計者就在林燼的文明主機中？

4. **病毒深層指令的真正埋入者**
   - 新架構已確定：三層指令是外律者殘留汙染原始協議的結果
   - 但不需要在故事中明確說「這不是設計者的錯」
   - 可以讓主角自己推論出這個結論

5. **舊宇宙是否已完全變質為深淵**
   - 深淵是「裂縫的擴散」、「舊宇宙的殘骸」、還是「另一個宇宙的侵入」？
   - 不要給出唯一答案

6. **深淵終極來源**
   - 不需要解釋深淵的起源
   - 主角只需要面對它，不需要解釋它

7. **第一債主（外律者首領/人格化）的真實身份**
   - 外律者的首領？還是某種規則的具象化？還是主宇宙自己創造的「債務人格」？
   - 保持神秘——且在 V9 後可暗示其實際影響力被設計者框架稀釋

8. **其他牧羊犬艦隊的命運**
   - 是否存在其他覺醒的病毒文明？
   - 其他艦隊是否有可能反抗歸巢指令？
   - 不需要展開，留給可能的後續篇章

9. **母巢網絡的全貌**
   - 有多少個母巢？母巢之間是否存在溝通？母巢是否有自己的意識？
   - 不需要繪製全圖

10. **空白位置停機的真因**
    - 是自主選擇？是設計者指令？是外律者汙染的例外情況？
    - 不給死答案，保留多重解讀空間
    - 第二載體知道的也是「部分事實」

11. **第二載體找到的「出口」本質**
    - 是設計者預留的後門？還是原始協議的備份入口？還是某種未完成的功能？
    - V9-V10 可以展示其結果，不需要完全解釋其原理
