## 1. 母艦即作業系統（核心隱喻）

末日母艦（Ark-Prime）是一個封閉的**物理作業系統**。所有物理資源——氧氣、空間、重力、規則密度——都是系統分配的硬體資源。底層居民不是「生活」，而是在**掛機運行**。

### 1.1 統一資源邏輯

| 概念 | 統一定義 | 故事中的表現 |
|---|---|---|
| 存活點 (S-Credits) | 低頻算力 (L-Compute) | 維持肉體運行的基礎貨幣 |
| 信念回流 | 高頻算力 (H-Compute) | 主機專用能量，執行規則覆寫 |
| 規則碎片 | API 指令集 | 直接調用母艦底層代碼的快捷鍵 |
| 文明階段 | OS 內核版本 | 主機版本號（V1 種火、V2 神殿、V3 聖城…） |
| 審計司 | 防火牆 / 防毒軟體 | 系統異常進程的清除機關 |
| 文明主機 | 寄生新 OS | 林燼體內的高維內核，可覆寫局部規則 |

### 1.2 力量雙線

- **外線（權限竊取）**：盜取管理員權限 → 身分提升、戰術模組、物理權限。
- **內線（代碼覆寫）**：內部文明用信仰（H-Compute）計算 → 強行覆寫母艦局部物理規則。

### 1.3 反哺機制

- **反哺**：內部文明將解析結果回傳給主角的過程。
- **覆寫**：反哺發生時，身邊物理規則被改寫。
- **代價**：
  - 反哺失真（信仰偏差導致 Bug）
  - 算力溢出（過度調用導致大腦過熱）
  - 系統報警（大規模覆寫吸引防火牆，100% 時觸發區域格式化）

### 1.4 異常進程處理

母艦對非法 / 異常進程的標準三段流程：

1. **收編**：將非法進程轉化為系統插件，剝奪主動權。
2. **去敏**：刪除進程中的異常代碼（抹除意志或記憶）。
3. **清洗**：徹底格式化某一區域的硬體資源。

> 系統提示的呈現格式（允許 / 禁止用詞）屬於寫作鐵則，見 `RULES.md`。

## 2. 母艦物理結構（三層 + 艙段地理）

### 2.1 整體規格

- 正式名稱：**Ark-Prime：流亡級宇宙母艦**（亦稱「文明殘存母艦」）。
- 全長約 **400 公里**，多圈層巨型艙段集合體。
- 由中央龍骨向兩端延伸附掛結構。
- 三層之間以壓差節點、通行閘、冷卻管道交錯連接，形成立體迷宮。

### 2.2 三層結構

| 層級 | 別稱 | 規則密度（相對基準） | 體積佔比 | 規則燃料配給 | 主要內容 |
|---|---|---|---|---|---|
| 上層 | 內核區 | 90%~100%（基準） | ~15% | 高 | 董事會、審計司總部、起源池原型、私有內宇宙接入站 |
| 中層 | 應用層 | 65%~80% | ~25% | 中 | 企業聯盟、技術行會殘址、公轉化區、航路調度中心 |
| 底層 | 數據冗餘區 | 25%~50%（浮動） | ~60% | < 15% | 附掛都市、第九段微光區以降各艙段 |

- **內核區**：包裹主反應爐，幾乎無熵化波動。董事會會議廳位於反應爐正上方；審計司總部橫跨三個子艙段，下設數據中樞、清理者營舍、收編評估中心；高層成員的私有內宇宙接入站亦在此層。
- **應用層**：工業與行政帶。技術行會總部舊址已被部分廢棄，殘留實驗設備被灰議會殘部占據。
- **數據冗餘區**：規則密度波動劇烈，每月須注入規則燃料。

### 2.3 艙段編號邏輯

艙段以數字自 **艦首向艦尾遞增**（1~24+）。數字越大離核心越遠、規則密度越低、環境越惡劣。**編號 17~24 為已封閉或半封閉的死艙區**。

### 2.5 重點艙段細節

**第九段（微光區）**——附掛都市主體，位於右舷外掛結構中段。因遠離主供能管線，照明與氧壓長期低標。內部劃分為棲息艙網格（廢棄貨櫃 + 管線支架拼湊）、維修工坊帶（廢金屬回收站 + 二手義體改裝舖 + 非法煉化點）、鋼簷橋（廢棄交通節點，灰鏈改造後成補給影線第一站）。

**第十一段——審計前哨**：審計分站大樓為銀灰色密封立方體，外觀無窗。白格咖啡艙是顧衡的半公開會面場所。起源池外圍副本（基因模板庫 + 批量孵化槽）藏在審計分站深處。

**第十三段——鐵鏽帶冷凝層**：長約 3 公里的高壓管廊段，內部維持低溫高氣壓以抑制微生物擴散。安流保全承包二號與四號閘門。過磅站徵收躍階氧稅、記錄污染讀數。

**第十五段——灰槽帶**：供能僅達基準值 40%，約三分之一區域處於斷電黑暗。灰議會舊址保留中層試圖回收的高階設備殘骸。第四卷末以四成名冊資料換得 6 個月自治權（範圍：第十五段全段，撤銷條件：中層聯合審計小組 2/3 同意 + 48 小時書面通知）。設有三道閘門，最外圍的第三閘門是與死艙交界的緩衝區。

**公轉化區**：橫跨中層與底層之間的閥門段。結構分四區——進料艙（污染讀數）、轉化爐陣列（分解為規則燃料、能量晶核、可回收材料）、二次黑箱（高層在封閉環境的隱性抽取，截留 30%~50%）、灰區（不受監控的邊緣，灰鏈與清譜會的灰色交易在此）。

**第十九段（回聲堡）**：十年前主動切斷與核心區的通訊供給。採「半熵化共生」策略——居民不再抵抗熵化，而是將身體與規則環境融合。外貌畸變、壽命縮短，但不再依賴外域資源。

**第二十一段（沉默艙）**：三年前封閉的死艙，官方記錄標記「熵化不可逆」。內部殘存古代航行紀錄黑盒子（母艦歷史真相關鍵拼圖）。原居民已畸變為熵行者群落，估計 200~500 隻。

**附掛都市**：並非獨立艙段，而是依附在第九段外層結構的居住擴建體。通過四條主要通道與第九段本體連接。

### 2.6 空間迷宮屬性

母艦經數萬年疊加改建，內部空間早已超越單純物理體積：

- **非法改建**：底層居民在管線夾層、通風井、廢棄貨艙中私自搭建，形成官方地圖上不存在的暗空間。
- **回收井路網**：垃圾垂直輸送管道，可作為走私通道。
- **死艙捷徑**：盲舵會掌握的舊維修索道與艙外磁力軌道，可繞過多個封鎖門直達深處艙段。
- **通風管迷宮**：貫穿多艙段，是黑市中轉與伏擊的常見場所。

### 2.7 地圖熵化

底層艙段因規則密度低，每月發生 3~5 次可觀測拓撲偏移，主要集中在廢棄管線與舊航道。

- **拓撲偏移**：A→B 距離每次測量誤差 ±40%。
- **艙段重疊**：低規則密度區可能出現空間摺疊。
- **導航失效**：傳統地圖在熵化區有效期通常 < 72 小時；死艙內 < 12 小時。

導航手段：

| 手段 | 原理 | 時效 | 精度 |
|---|---|---|---|
| 盲舵會舊航道圖 | 數十年壓差節點觀測經驗 | 短期，不保證準確 | 中低 |
| 信仰錨點 | 主角透過主機投放信標 | 24 小時 | 高（固定座標） |
| 壓差節點定位法 | 利用各艙段氧壓差異推算 | 即時 | 中 |
| 規則密度測繪儀 | 掃描三維梯度，建構地圖 | 中等 | 受規則風暴干擾 |
| 慣性導航系統 | 追蹤躍遷歷史 | 長期累積誤差大 | 低 |
| 主機視野 | 主角可感知半徑 100m 內規則密度（金→灰黑色階） | 即時 | 高 |
| 標準官方地圖 | 官方測繪 | < 72 小時 | 底層低 |
| 深層信標（理論） | 母巢巨網節點 | 未知 | 需接入裸結構網才能接收 |

## 3. 通行制度與層級移動

**核心原則：移動本身就是一種資源**。高層透過限制遷移維持勞動力鎖定，對每一次移動課以重稅。

### 3.1 垂直移動權限

| 通行方向 | 所需權限 | 管制節點 | 通行代價 |
|---|---|---|---|
| 底層 → 中層 | C 級以上通行證或企業擔保 | 第十三段冷凝層閘門 | 躍階氧稅 + 污染押金（約 200 存活點起） |
| 中層 → 內核 | A 級權限或審計司直批 | 第七封鎖線 | 生物特徵校驗 + 意識審查 |
| 底層 → 死艙 | 無官方管道 | 維護封門 / 廢棄壓差節點 | 高額領航費 + 死亡率極高（盲舵會） |
| 艙段間水平移動 | 身分識別 + 區域稅繳清 | 各艙段交界閘門 | 標準通行費 |

### 3.2 冷凝層通行細則（第十三段）

- **標準通行**：基本通行費 + 躍階氧稅，約 200 存活點/人。
- **加速通道**：雙倍費用跳過排隊，約 500 存活點。
- **白名單豁免**：企業與承包軍購買月費白名單，免抽檢與部分稅項。
- **黑箱繞行**：灰鏈與安流保全的底層交易——以貨物抽成換免檢通行，主角團跨艙段行動的主要方式。

## 4. 深淵環境

### 4.1 物理性質

深淵不是傳統意義的太空，而是規則密度趨近於零的**維度夾層**。

| 性質 | 數值 / 描述 |
|---|---|
| 規則密度 | 10~50 r/m³（表層）；< 5 r/m³（深層） |
| 溫度 | 接近絕對零度（母艦殼體維持 ~15°C） |
| 介質 | 非真空——充滿極稀薄「規則霧」（見 §7） |
| 可見光 | 無——深淵不發光，光學傳感器視野純黑 |
| 時間流速 | 與母艦內部一致（無顯著時間膨脹） |
| 空間拓撲 | 非歐幾里得——「近」「遠」由規則密度梯度而非物理距離決定 |
| 重力 | 無外部重力；艦內由旋轉艙段與人工重力系統提供 |

### 4.2 三層結構

```
表層（0~10,000 km）
  │  規則密度：30~50 r/m³
  │  特徵：裂縫密集區，母艦在此採集資源
  │  威脅：低~中（裂縫不穩定、熵行者滲漏）
  ▼
中層（10,000~100,000 km）
  │  規則密度：5~30 r/m³
  │  特徵：規則霧濃度降低，裂縫稀少
  │  威脅：中~高（空間畸變、導航偏移）
  ▼
深層（> 100,000 km）
    規則密度：< 5 r/m³，趨近於零
    特徵：絕對黑暗
    威脅：極高（規則崩解、母巢接觸風險）
    現狀：深淵母巢本體在此層；母艦從不主動進入
```

深層資訊全來自遠古航行紀錄與第二載體（0xF1_0xXE_0_0xLL）的廣播。

### 4.3 深淵中的威脅

**規則風暴**：規則密度突發劇烈波動，母艦殼體讀數數分鐘內 ±200%，導航儀器失效。持續數小時~數天。應對：關閉非必要系統，啟動殼體穩定陣列。代價：每次消耗 1,000~3,000 單位規則燃料。

**畸變區**：規則密度異常高的「斑點」，直徑數百~數千公里。成因不明（可能是遠古文明廢墟，也可能是母巢的規則排遺）。靠近時母艦內部出現短暫物理常數漂移（重力 ±5%、電子邏輯錯誤）。高價值資源點（裂縫密集），但高風險。

**規則真空泡**：規則密度完全歸零的球形區域，直徑數公尺~數公里。誤入會導致殼體規則密度迅速向外擴散——艦內各艙段急降，相當於「漏氣」。探測方式：航路調度中心監測規則霧流動方向預判位置。

**深淵生物（未確認）**：第二載體廣播暗示「不是我們理解的生物」。遠古航行紀錄曾記錄殼體撞擊事件，撞擊點留下規則密度異常烙印至今未消退。是否存在主動獵食者尚未定論。**設定鎖**：V7 前不正式現身。

### 4.4 深淵真相（V8+ 逐步揭露）

> **終極隱喻**：深淵不是海洋，母巢不是海洋中的島嶼。**深淵是皮膚，母巢是身體，母艦是一隻在皮膚表面爬行的寄生蟲。**

- 深淵非自然存在——是原初病毒文明崩潰後，其規則守護系統（深淵母巢）的「代謝廢棄物」累積形成。
- 母巢存在於深層，規則網絡延伸至表層，形成裂縫與畸變區。
- 每一條裂縫，都是母巢的「毛細孔」——規則密度透過這些孔洞向外擴散，同時也是母巢感知外界的觸角。

## 5. 躍遷引擎與導航

### 5.1 核心原理：規則密度梯度驅動

母艦的躍遷引擎不產生推力，而是利用**規則密度梯度差**讓母艦在深淵中「滑行」：

```
高規則密度區（母艦周圍 ~1000 r/m³）→ 低規則密度區（深淵 < 50 r/m³）
                          ↓
              母艦被「推」向低密度方向
                          ↓
             如氣球在高壓氣體中向低壓區移動
```

方向控制：透過局部調整殼體規則密度分布改變壓力梯度方向。

### 5.2 物理實現

| 組件 | 位置 | 功能 |
|---|---|---|
| 主反應爐 | 內核區核心 | 產生維持全艦規則密度的基礎能量 |
| 規則密度調製器 | 龍骨全長分布 | 控制殼體各部位規則密度，製造梯度方向 |
| 躍遷噴口（×4） | 艦尾四角 | 規則密度洩壓口——在低密度區「噴出」高密度規則燃料產生反推 |
| 航向穩定陣列 | 艦體外殼 | 16 組規則密度天線陣，微調局部梯度向量 |

每次躍遷消耗規則燃料儲備。躍遷越頻繁、距離越長，燃料消耗越大。

### 5.3 躍遷模式

| 模式 | 速度 | 燃料消耗 | 持續時間 | 用途 |
|---|---|---|---|---|
| 巡航 | ~0.3 c | 低（5 單位/天） | 無限 | 常規航行 |
| 加速躍遷 | ~3 c | 中（200 單位/次） | 最多 4 小時 | 短距移動、避威脅、燒規則垢 |
| 深層躍遷 | ~30 c | 高（5,000 單位/次） | 約 15 分鐘 | 跨域長距跳躍 |
| 緊急跳躍 | ~100 c | 極高（20,000 單位/次） | 約 5 分鐘 | 逃生——可能結構受損 |

> **設定鎖**：低規則密度區（< 30 r/m³）進行深層躍遷可能導致殼體撕裂；不要讓深層躍遷成為常規手段——每次都應帶來明顯代價。

### 5.4 航路調度中心

位於中層的核心導航設施，名義上隸屬技術行會，實際受審計司與董事會直接控制。職責：

- **規劃躍遷路徑**：計算規則密度梯度，最節省燃料的航行路線。
- **投放時窗管理**：母艦靠近裂縫密集區時，協調各艙段投放作業。
- **避障導航**：監測規則密度異常點，提前調整航向。
- **結構完整性監控**：每次躍遷後全結構掃描。

三級授權制度：

1. **一級**：日常巡航路徑（中層操作員可執行）。
2. **二級**：躍遷模式切換、投放時窗（需技術行會主管簽核）。
3. **三級**：緊急跳躍、航向重大變更（需審計司或董事會直批）。

### 5.5 深淵中的位置概念

XYZ 三維座標系在深淵中無效。位置由以下參數定義：

```
位置 = f(規則密度梯度向量, 規則霧流向, 距畸變區距離, 距母艦距離)
```

意味著：
- 同一個「位置」在不同時間可能有不同座標。
- 距離感模糊——A→B 躍遷耗時可能因梯度變化大幅波動。
- **迷航是真正的威脅**：若導航系統全面失效，母艦可能永遠找不回已知區域。

> **設定鎖**：不要讓母艦在深淵中停下來——停下意味著規則密度梯度消失，母艦會被困在該區域難以重新啟動躍遷。

## 6. 裂縫與外域

### 6.1 生成機制

裂縫並非母艦躍遷的副產品，而是熵潮侵蝕時空時撕開的破口。生成條件：

```
熵潮侵蝕時空 → 高規則密度（母艦殼體）與低規則密度（深淵）形成急遽梯度
         ↓
特定共振頻率下時空撕裂 → 裂縫形成
         ↓
裂縫內部暴露高規則密度殘留區（200%~800%）→ 資源可開採
         ↓
維持數分鐘~數小時後自行閉合
```

### 6.2 裂縫類型

| 類型 | 規則密度 | 維持時間 | 資源 | 風險 |
|---|---|---|---|---|
| 微型裂縫 | 200%~300% | 5~15 分鐘 | 能量晶核（少量） | 低 |
| 標準裂縫 | 300%~500% | 30 分鐘~2 小時 | 規則碎片、能量晶核 | 中 |
| 大型裂縫 | 500%~700% | 2~8 小時 | 高純度規則碎片、活性樣本 | 高 |
| 畸變裂縫 | 800%+ | 不固定 | 未知（可能遠古文明遺物） | 極高（不建議進入） |

### 6.3 投放帶與投放流程

母艦調整航向使特定艙段靠近裂縫密集區，這區域稱「投放帶」。

1. **航路調度中心預測**：根據規則霧流動與歷史數據，預測 24~72 小時內裂縫生成位置。
2. **投放時窗分配**：各隊伍依合約等級分配時間窗口；企業與承包軍購買優先時窗，底層隊伍分配剩餘窗口（通常裂縫快閉合時的低收益時段）。
3. **隊伍部署**：從指定艙段（通常是第十三段冷凝層附近投放口）進入外域。
4. **採集作業**：必須在維持時間內完成採集並返回，否則裂縫閉合後被困深淵。
5. **回收**：返回經公轉化區進行樣本登記與污染讀數。

### 6.4 外域樣本與規則碎片

| 名詞 | 定義 |
|---|---|
| 外域裂縫 | 母艦航道中短時出現的異質空間窗口 |
| 裂縫等級標記 | 統一記為 `R1~R5 + 詞綴`（如 `R2-鏡折`） |
| 規則碎片 | 可改寫局部效率或法則的小型權限載體 |
| 異域神經拓印體 | 外域中的高級異常樣本 |
| 夜潮侵蝕 | 外域環境事件 |
| 高純度生物殘核 | 裂縫中極高價值的生物遺骸，可用於醫療與氧氣轉化 |
| 極地導電菌膜 | 裂縫稀有資源，具高級工業與升級價值 |
| 三級污染 | 一級可逆 / 二級半可逆 / 三級不可逆 |

### 6.5 R-17 節點

外域裂縫投放帶的關鍵節點。位於第十三段外側，需經冷凝層二/四號閘門。林燼於第41章取得 **局部控制模組**——決定 R-17 短時間開關優先權的硬件密鑰。

## 7. 規則霧與清霧技術

### 7.1 規則霧定義

深淵中瀰漫的極稀薄規則密度殘粒，濃度 0.01~0.5 r/m³，本質是遠古文明崩潰後逸散的規則碎片。

母艦航行時，艦殼維持高規則密度，會像靜電吸附灰塵一樣吸附規則霧。霧附著後逐漸累積成 **規則垢**——一層錯亂的微規則膜。

### 7.2 規則垢的四類危害

1. **轉向失靈**：干擾殼體規則密度調製，無法準確改變航向。
2. **燃料消耗增加**：目前 Ark 主要靠加速躍遷燒垢，但加速躍遷本身需大量燃料。
3. **引來深淵生物**：規則垢對某些深淵生物類似食物、訊號或獵物標記。
4. **航跡暴露**：薄霧層能遮蔽艦體高規則密度訊號，清得過亮反而更醒目。

舊航民古諺：**「殼不能亮，亮了就有人看見。」**

### 7.3 清霧技術三代體系

正式名稱：**剝律清垢技術**。目的不是消滅深淵中的規則霧，而是處理已附著在艦殼上的規則垢。

**第一代：加速躍遷燒垢**
- 使用者：Ark 官方、高層航路調度中心。
- 原理：加速躍遷讓艦殼產生高強度規則摩擦燒掉霧垢。
- 缺點：燃料消耗高、不適合頻繁使用、容易暴露航跡、無法精準保留安全灰膜。
- 敘事定位：舊文明的高耗能解法，象徵 Ark 用燃料與權限硬穿深淵。

**第二代：刮霧艇／逆相脈衝清垢**
- 使用者：外圍艦隊、清障艦、部分先遣艦殘存者。
- 流程：偵測 → 小艇貼近艦殼 → 逆相脈衝鬆動 → 刮霧臂剝離 → 收入隔離罐。
- 相關職業：**刮殼人**（高死亡率）。
- 缺點：作業者需貼近外殼、霧尾可能引來深淵生物、無法大規模覆蓋全艦。

**第三代：霧食螅馴化清垢法**
- 使用者：舊航民、深淵移民、後期聖城。
- 聖城整理後名稱：**霧食螅馴化清垢法**。
- Ark 官方稱呼：「非授權深淵生物附殼行為」。

### 7.4 霧食螅（生物設定）

深淵表層常見微型群落生物，附著於高規則密度物體表面，以低密度游離規則殘粒與規則垢為食。

- **外觀**：灰白色絨毛、薄膜或潮濕的灰斑。大量聚集時形成緩慢流動的灰色苔衣。
- **舊航民俗稱**：灰螅、吃灰的、殼絨、霧毛。

四個核心特性：

1. **吃霧垢不吃艦殼**：正常馴化狀態下只啃食低密度、游離態規則垢。
2. **群體同步**：螅群整片灰膜一起移動，可被特定低頻規則節拍引導。
3. **過飢會失控**：霧垢/誘餌不足時開始啃咬艦殼外層的規則鍍膜。
4. **會被異常霧垢污染**：變異後可能啃殼、自燃、狂奔、形成假航跡，甚至發出吸引深淵生物的訊號。

### 7.5 舊航民「餵殼」流程

```
捕捉野生霧食螅 → 用規則垢殘渣餵養 → 篩掉會啃殼的危險個體
→ 培養溫馴螅群 → 清霧時釋放到艦殼外 → 用低頻節拍引導
→ 保留安全灰膜 → 用誘餌垢罐召回螅群 → 封存、分株、休眠或再次餵養
```

核心經驗（不能單靠圖紙學會）：哪種霧食螅能養、哪種灰色代表溫馴、哪種白斑代表過飢、哪種黑線代表吃到異常霧垢、清到什麼厚度必須停、什麼霧紋不能讓螅群靠近、螅群收不回來時該燒哪段、保哪段。

### 7.6 為何 Ark 沒有普及霧食螅清霧

1. Ark 規模過大，霧食螅適合小艦/局部艦殼，不適合 400 公里母艦。
2. Ark 不信任活體技術——黑箱偏好可審計、可量化技術。
3. 霧食螅會干擾監控（生物規則噪音遮蔽外殼讀數）。對高層是風險，對舊航民是隱蔽優勢。
4. 舊航民刻意保密。
5. 高層只要燃料還夠，更願意加速燒垢。

### 7.7 聖城清霧系統（V6+ 取得路徑）

**取得階段**：

1. **V6 接納移民**：救下被 Ark 採集艦/清障艦追捕的舊航民聚落。
2. **灰罐誤會**：舊航民帶入的「灰罐」（休眠霧食螅株）一開始被當污染物，差點銷毀，雙方爆發信任危機。
3. **第一次救命**：聖城小艇規則垢過厚轉向失靈，燃料不足無法加速燒垢；舊航民釋放少量霧食螅清掉局部霧垢。
4. **聖城理論化**：審計軌與後勤軌將舊航民經驗 + 共振井 + 信仰回流結合。
5. **共同改良**：形成正式系統。

**正式名稱**：**信仰共振控螅清霧系統**（簡稱「控螅清霧」）。

**組成**：霧食螅吃垢 + 舊航民霧紋判讀 + 共振井節拍控群 + 刮霧艇回收螅群 + 審計軌判讀異常霧垢 + 後勤軌養殖與隔離。

**職能分工**：

| 角色 | 職責 |
|---|---|
| 舊航民 | 霧紋判讀、螅群馴養、外殼作業、危險預警 |
| 聖城後勤軌 | 螅群繁殖、休眠罐、誘餌垢、隔離艙、飼料配比 |
| 聖城審計軌 | 判斷哪些霧垢能吃、監控螅群是否變異 |
| 聖城軍團軌 | 清霧作業時護航、回收失控螅群、處理深淵生物襲擊 |
| 主角／文明主機 | 提供共振節拍、H-Compute 支援、局部規則覆寫、危機時中斷清霧程序 |

### 7.8 清霧副產物

1. **灰渣**：低純度規則殘渣，可低效轉化為規則燃料（5%~12%）。不能取代外域規則碎片。
2. **誘餌垢**：處理過的規則垢殘渣，戰術用途——偽造航跡、引開追擊、誘導深淵生物攻擊敵方、收回螅群。
3. **異常垢核**：少數霧垢中凝出難以解析的異常節點。**設定鎖**：不解釋來源；不可餵食、不可轉化、不可長期保存（V14 伏筆）。

### 7.9 技術限制

- **不能清得太乾淨**：必須保留安全灰膜，否則航跡暴露、深淵生物易感知、艦殼直接承受深淵污染。
- **不能餵異常霧垢**：螅群變異會啃殼、發出吸引深淵生物的規則震盪、艦殼縫隙中繁殖失控。
- **螅群需回收**：殘留會形成生物污染——感測延遲、艙壁灰絨、黑箱標記為深淵污染。
- **清霧會產生規則聲音**（規則層震盪，深淵生物可能感知）——清霧是一次高風險外殼行動。
- **需要活飼料**：規則垢殘渣維持活性；長期沒霧垢吃會休眠、互食或退化。聖城不能無限養殖。

### 7.10 三方文明對規則霧的態度

| 文明 | 態度 | 解法 | 代表邏輯 |
|---|---|---|---|
| Ark | 污染、成本、障礙、可轉化資源 | 燃燒、剝離、封存、武器化 | 用燃料與權限壓過深淵 |
| 舊航民 | 危險但可共處；不能全清，只能養、看、避、借力 | 餵殼、養灰、留膜 | 不是戰勝深淵，而是不要被深淵看見 |
| 聖城 | 可研究、可共生、可制度化，但須設倫理與隔離邊界 | 共振控螅清霧系統 | 不是吞噬移民，而是吸納移民知識 |

> **核心命題**：舊文明靠燃燒規則穿過深淵。新文明學會和深淵裡的生命交換道路。

## 8. 熵潮與規則密度

### 8.1 熵潮定義

熵潮（Entropy Tide）並非熱力學第二定律的熵增，而是空間本身的**規則承載能力**持續衰減的過程。可理解為：宇宙的底層程式碼正在被不知名外力逐行刪除。**一切掠奪行為的根因**。

### 8.2 規則密度（Rule Density, RD）

- **單位**：`r/m³`（規則量/立方公尺）。
- **基準值**：母艦核心區約 1000 r/m³，定義為 100%。
- **安全下限**：< 30%（300 r/m³），出現可觀測物理常數漂移。
- **熵化臨界值**：< 15%（150 r/m³），進入不可逆崩壞階段。

### 8.3 衰減速率（年）

| 區域 | 衰減率 | 倍數 |
|---|---|---|
| 核心區 | 2.1%~3.5% | 1.0x |
| 中層 | 4.0%~5.5% | 1.6x |
| 底層邊緣 | 8%~12% | 2.6x |

> 底層環境惡化速度遠快於上層，且差距隨時間擴大——這是高層與底層命運分歧的物理基礎。

### 8.4 熵化階段

| 階段 | 規則密度 | 觀測特徵 |
|---|---|---|
| 穩定期 | ≥ 80% | 物理定律完全穩定，精密儀器正常 |
| 潛伏期 | 50%~80% | 設備效能微降（0.8%~2.1%/月），僅技術部門察覺 |
| 加速期 | 30%~50% | 金屬無外力裂痕、能源電池異常自放電、弱 AI 邏輯迴圈 |
| 臨界期 | 15%~30% | 物理常數可觀測漂移（重力、光速、電磁力）、有機體無徵兆細胞崩解 |
| 熵死 | < 15% | 空間塌陷、物質還原為無結構粒子、時間流凝滯、信息歸零 |

### 8.5 各艙段規則密度差異

| 艙段／區域 | 規則密度 | 變動頻率 | 備註 |
|---|---|---|---|
| 內核區 | 90%~100% | 極低 | 最穩定 |
| 中層工業帶 | 65%~80% | 低 | 每月校準一次 |
| 公轉化區 | 55%~70% | 中 | 頻繁轉化作業 |
| 第十三段冷凝層 | 40%~55% | 中 | 受裂縫影響波動 |
| 第九段（底層） | 35%~50% | 中~高 | 每月需注入燃料 |
| 第十五段灰槽帶 | 25%~35% | 高 | 已降級，部分區域拓撲偏移 |
| 死艙（封閉後） | 5%~20% | 極高 | 持續下行直至熵死歸零 |
| 外域裂縫內部 | 200%~800% | 劇烈 | 高密度殘留區，但極不穩定 |

### 8.6 規則密度對日常的影響

- **氧氣轉化效率**：每降 10% 規則密度，氧循環設備效率降 ~6%。
- **義體性能**：低密度區電子義體神經響應延遲增加。
- **醫療效果**：淨化針劑在低密度區半衰期縮短，所需劑量增加（底層慢性熵污染患者難根治的物理原因）。
- **生育能力**：< 40% 時自然受孕率降 ~70%。底層人口主要靠起源池批量投放補充。

### 8.7 測量方式

- **規則密度計**：手持式儀器，發射校準信號分析衰減。中層以上標配，底層需黑市取得。
- **生物感知**：長期暴露者會產生本能壓迫感（類似潛水員減壓症，但作用於認知層面）。
- **主機視野**：主角可感知半徑 100m 內規則密度分布（金→灰黑色階）。

### 8.8 規則燃料

將外域資源或信仰產出轉化為可注入母艦系統的規則密度補給物。

| 來源 | 轉化效率 | 純度 | 說明 |
|---|---|---|---|
| 外域能量晶核 | 70%~85% | 高 | 裂縫中直接開採的凝結態規則密度 |
| 外域規則碎片 | 50%~65% | 中 | 局部物理定律快照，需解析才能釋放 |
| 活性生物樣本 | 30%~45% | 低~中 | 需二次轉化 |
| 信仰回流（主角內宇宙） | 90%~95% | 極高 | 內部文明的認知共振結晶 |
| 高層內宇宙信仰壓榨 | 25%~40% | 低 | 恐懼驅動，品質劣、衰減快 |

用途：注入維生系統、校準精密設備、驅動躍遷引擎、供養內宇宙。

### 8.9 反熵引擎（理論）

能自行產生規則密度、無需依賴外域掠奪的理論系統。目前已知唯一可能載體是主角的文明主機。

達成條件（推測）：
- 文明主機達 V10+。
- 內部文明形成穩定信仰生態系統（多神系、多教派並存）。
- 內部文明產出 > 維持自身運轉消耗（淨正產出）。
- 主角承受反哺失真與信仰偏差的認知污染風險。

## 9. 死艙與熵行者

### 9.1 死艙判定流程

由審計司與董事會的資源調度委員會共同決定：

1. **觸發條件**（任一成立即啟動評估）：
   - 艙段人口維持率連續 90 天 < 62%。
   - 規則密度跌至 < 45%。
   - 投入補給 > 產出稅收的 1.8 倍。
   - 臨界期前置評估 < 180 天。
2. **評估期（7~14 天）**：審計司派遣評估小組終讀，資源調度委員會核算成本效益。
3. **末批產出期（30 天）**：
   - 居民被告知「戰略任務」，要求最大化產出。
   - 所有產出歸核心區，居民不再獲得維生配給。
   - 實為強制最後一次勞動。
4. **封閉執行**：清理者小隊封鎖通道、外部焊死密封門、切斷維生與通訊、記錄為「不可逆熵化」。

### 9.2 末批產出的殘酷邏輯

高層從不撤離死艙居民，純粹成本計算：

- 撤離一萬人需 ~3,000 單位規則燃料。
- 將這 3,000 單位投入核心區維護，可延長母艦壽命約 45 天。
- 末批產出 30 天約可回收 800~1,200 單位規則燃料當量——遠低於撤離成本，但聊勝於無。

### 9.3 熵行者

極低規則密度環境中存活下來、因不可逆污染而畸變的非人類實體。

**形成條件**：
- 規則密度 < 20% 持續 > 30 天。
- 有機體未在臨界期到來前死亡或撤離。
- 身體與低規則環境達成「恐怖平衡」——細胞不再依賴標準物理常數維持結構。

**生物特徵**：
- 失去完整人形，體表結晶狀或灰質化外殼，肢體拉伸或扭曲、關節反向彎曲。
- 不依賴視聽，透過規則密度梯度感知環境——低規則環境中比人類敏銳數倍。
- 對高規則密度區域有強烈趨性，會主動攻擊進入死艙的人類以吸收其規則密度。
- 單體威脅 ≈ 企業傭兵小隊（3~5 人）；群體行動形成「熵共振」，集體同步可獲得壓倒性力量。

**分佈**：
- 第二十一段（沉默艙）：確認群落，估 200~500 隻。
- 第十七段：推測更高階個體（存活最長、畸變最深）。
- 其他死艙邊界：偶有零星個體穿越封鎖漏洞出現在降級艙段外圍（「熵行者滲漏事件」）。

### 9.4 進入死艙的方法

1. **官方管道**：不存在。所有死艙密封門從外部焊死。
2. **盲舵會繞行**：掌握舊維修紀錄與壓差節點，部分死艙的維修通道並未完全封閉，可透過艙外維修索道或廢棄壓差節點繞行進入。
3. **所需代價**：高額領航費、死亡率極高。盲舵會拆分銷售分段時窗、門碼與轉向點，確保客戶無法一次性脫離掌控。

## 10. 社會階級與意識形態

### 10.1 四大階級

母艦社會的本質是以規則密度分配為基礎的封閉階級體系。**階級固化具有物理基礎**（這是與現實世界最大的差異）。

| 階級 | 對應層級 | 人口佔比 | 規則密度 | 核心生存模式 |
|---|---|---|---|---|
| 董事會成員 | 內核區 | < 0.01% | 90%~100% | 管理內宇宙養殖鏈，享受近乎永生的意識備份 |
| 管理階級 | 中層 | ~3% | 65%~80% | 企業高管、技術行會長老、審計官，制度性抽成維生 |
| 技術勞工 | 中~底交界 | ~15% | 40%~65% | 維修技師、契約傭兵、駕駛員，有上流機會但代價極大 |
| 底層勞動力 | 底層 | ~82% | 25%~50% | 維修工、拾荒者、複製人勞工，每日為生存配額掙扎 |

### 10.2 階級流動的假象

官方宣傳「奉航貢獻達標者可申請階級審查」，但實際存在三層隱形門檻：

1. **生物特徵登記**：起源池投放碼與基因模板永久記錄，底層出身者在生物識別階段就會被過濾。
2. **知識壁壘**：中層以上大量使用技術術語與制度黑話，底層即使取得通行權也無法融入。
3. **維護成本陷阱**：晉升後生活標準（氧壓、溫度、空間）遠高於底層；若收入不穩，可能因無法承擔維護費而破產——屆時連回底層的資格都沒有（系統定義為「進程異常」，直接進入去敏流程）。

### 10.3 投放批次制度

底層人口非自然繁衍，由起源池批量生成的基因模板複製人：

- **勞動力控制**：高層可調整投放批次的人數與技能傾向。
- **稀釋反抗**：新投放的「生肉」對母艦制度無記憶與情感連結，不易參與抵抗。
- **降低養育成本**：批量孵化約為自然生育成本的 1/20。

## 11. 恆航教與地下信仰

### 11.1 恆航教教義

非組織化宗教，而是滲透在官方敘事、教育與廣播中的隱性意識形態。三條核心教義：

- **聖航命題**：母艦航行是人類文明延續的唯一方式，所有航行相關行為（包括掠奪與犧牲）都是神聖的。
- **聖火崇拜**：主反應爐被稱為「聖火」，維持聖火燃燒是所有人的最高義務。
- **獻禮等級制**：越接近核心的反應爐貢獻者，在來世的「永恆航行」中位階越高——為階級固化提供神學辯護。

### 11.2 運作機制

無專職祭司，教義透過：

- **公共廣播系統**：每天三個時段播放「航行日誌」與「奉航典型」故事。
- **教育艙課程**：兒童強制接受「聖航基礎」課程。
- **官方文書用語**：稅單、通行證、契約使用恆航教詞彙（「為聖航獻禮」「願聖火不滅」）。
- **續焰團**：恆航教在底層的活動化分支。

### 11.3 裂痕

- **底層信任破產**：絕大多數對恆航教嗤之以鼻——「聖航」就是每天 20 存活點的生存稅。
- **中層工具化**：管理者需要恆航教作為執法的道德外衣，但私下並不相信。
- **高層知情者**：董事會知道真實航行原因是逃離熵潮，恆航教只是維持勞動力穩定的管理工具。

### 11.4 神話工程

官方將剝削包裝為「聖航貢獻」的系統性宣傳機制。雙層結構：

- **可驗證小善**：發放過期維生素、低價氧包、臨時庇護所——真實存在，建立制度可信度。
- **不可質疑大義**：所有徵收、強制勞動、人口投放包裝為「為聖航必要措施」，任何質疑被定義為「對航行的背叛」。

典型範例——戰時稅則宣布流程：
1. 廣播「邊境艙段出現異常熵化加速」警報（製造危機感）。
2. 「為保護聖航不受干擾，即刻啟動戰時稅則」（給予正當性）。
3. 「本次犧牲將被記錄在聖航功績冊中」（給予補償承諾）。
4. 24 小時內氧氣價格翻 4 倍，但名義上是「自願為聖航做的貢獻」。

### 11.5 底層三派地下信仰

| 派別 | 核心主張 | 信眾 | 行動 | 高層態度 | 真相伏筆 |
|---|---|---|---|---|---|
| 回歸派 | 母艦應折返母星 | 中老年居民 | 拖延工作、破壞通訊、散布「母星仍在」謠言 | 不視為威脅，可製造底層分裂 | 部分成員可能掌握母艦出發點真實座標——但那裡規則密度可能早已歸零 |
| 前行派 | 母艦前方存在「應許之地」 | 年輕人、拾荒者、退伍承包軍 | 組織「前行探索隊」（通常有去無回） | 暗中扶植——讓底層自願投入高風險 | 「應許宇域」很可能是高層刻意散播 |
| 靜寂信仰 | 熵化是宇宙回歸本源，不應抵抗 | 邊緣艙段殘民、三級污染畸變者 | 不參與反抗或掠奪，在熵化區「靜待歸一」 | 完全忽略 | 提供「不抵抗」哲學對照 |

三派矛盾：回歸派＝過去好（往回走）／前行派＝未來好（往前走）／靜寂＝哪裡都不用去。高層可利用三派分裂製造底層內耗。

### 11.6 焚數派（技術異端）

技術行會邊緣成員與駭客社群中的信息永生主義小圈子。

- **教義**：物質世界注定衰亡，真正值得保存的只有純粹信息。人類應逐步「上傳」意識至數據載體。外域裂縫中的「知識矩陣」是已滅亡文明嘗試信息化永生的遺產。
- **實際活動**：在技術行會舊址與灰色駭客社群中活動，蘇嵐的駭客圈與焚數派有交集。擁有少量自行改裝的「意識快照」設備（粗糙且高風險——可能人格分裂或記憶缺損）。對外域文明遺骸中的知識矩陣有極高研究興趣，是主角未來可能的技術合作對象。
- **核心矛盾**：「上傳」技術本身也需大量規則密度維持數據完整性——在熵潮環境下，純信息載體同樣會衰變。

## 12. 底層黑話與日常壓迫

### 12.1 黑話術語

底層居民為躲避監聽與建立內部信用，發展出獨特社會方言（Sociolects）。監理演算法無法解析，因大量使用隱喻、反語與情境依賴詞彙。

| 黑話 | 字面義 | 實際義 |
|---|---|---|
| 漏氣者 | 正在漏氣的管線 | 即將破產或被系統清理的人 |
| 生肉 | 未加工的生肉 | 未安裝義體、無戰鬥經驗的新投放複製人 |
| 火花錢 | 會冒火花的錢 | 能換取規則燃料的硬通貨（不是存活點這種「廢紙」） |
| 掛機 | 機器暫停運作 | 形容人極度疲憊下勉強維生 |
| 洗白 | 把東西洗乾淨 | 將黑市收益轉化為合法存活點 |
| 押條 | 抵押用的鐵條 | 污染風險押金的俗稱 |
| 暗艙 | 黑暗的房間 | 官方記錄上不存在的地下空間 |

### 12.2 信用符號

- **手勢標記**：特定組合表「安全」「有監控」「貨真價實」「陷阱」。
- **牆壁塗鴉**：粉筆或刮痕標記告知該區風險等級與可用資源。
- **交易暗語**：特定問答模式確認身份（如「今天氧壓如何？」回「比昨天低兩度」表熟人）。

### 12.3 標準底層維修工的每日帳單

- 基礎氧氣配額：10 存活點
- 空間佔用費（床位）：10
- 硬體抗輻射費：5
- 廢水處理與營養膏：5
- **合計：30 存活點/天**

帳戶 < 100：系統自動發黃卡警告，強制參加高風險任務或接受義體抵押。
帳戶 < 20：系統執行「進程掛起」——凍結氧氣供應，個體 15 分鐘內窒息死亡。

### 12.4 醫療作為階級特權

- **淨化訂閱制**：基礎淨化針劑非福利，依身份評級分配的加購服務。未訂閱即使有存活點也無法取得潔淨血漿。
- **延淨排序**：清譜會主導，未購「加標服務」者永遠排最後，押金被長期凍結。
- **生育稅**：高污染環境自然生育需支付天文數字的「潔淨場域費」。

### 12.5 資訊封鎖

- 底層終端設備只能訪問經審計司過濾的「民用網」，無法查閱母艦真實航行數據與歷史。
- 跨艙段資訊流通需經多層審查，任何被判定「可能動搖航行信心」者皆被攔截。
- 地下資訊網絡（灰鏈情報交換站）成底層了解真相的唯一管道，但常伴高昂代價或陷阱。

## 13. 經濟：存活點與稅制

### 13.1 存活點（Survival Credits）

母艦內部唯一通用貨幣，本質是以「維生資源配給權」為擔保的系統代幣。**不是真正的貨幣**——持有存活點不代表持有價值，只代表持有者被允許消耗一定量的維生資源。別稱：**標準氧時 (SOH)**。

### 13.2 發行機制

存活點由母艦核心系統（Prime）統一發行，無私人或機構鑄幣權。發行公式：

```
每日發行量 = ∑(各艙段維生基準消耗 × 艙段人口) × 安全係數(1.05)
```

安全係數的 0.05 溢價從未真正進入流通——被二次黑箱抽取截流，成為高層的隱性收入。

### 13.3 流通路徑

```
系統發行 → 各艙段生存配額池 → 每日生存配額（標準 30 點/天，
經層層抽成後底層實際遠低於此）→ 消費回饋 → 資源轉化結算（企業/幫派
賣資源給公轉化區）→ 二次分配（公轉化區 → 企業 → 雇員工資）
```

存活點從未真正「離開」系統——只是在各帳戶之間轉移，作為勞動力調度與資源分配的數位鎖鏈。

### 13.4 購買力參考

| 項目 | 費用 | 佔每日收入比 |
|---|---|---|
| 基礎生存稅（一天） | 30 | 100% |
| 一劑基礎淨化針 | 50~80 | 167%~267% |
| 低階抗輻射藥（黑市） | 40~60 | 133%~200% |
| 二手義體（最低階） | 500~1,000 | 17~33 天生存稅 |
| 灰鏈洗白手續 | 15%~30% 抽成 | 視金額 |
| 污染押金（標準） | 200 | 7 天生存稅 |

### 13.5 基礎稅項

| 稅項 | 費率 | 徵收方式 |
|---|---|---|
| 氧氣稅 | 10 點/天/人 | 系統自動扣除 |
| 空間佔用費 | 10 點/天/人 | 系統自動扣除 |
| 抗輻射維護費 | 5 點/天/人 | 系統自動扣除 |
| 廢水處理費 | 5 點/天/人 | 系統自動扣除 |
| 通行稅 | 視距離與艙段 | 通行閘現場 |

### 13.6 差異化稅率與風控分

針對高收益個體的精密稅務打擊工具：

- 系統監控每個個體存活點流入量與資產波動。
- 偵測異常增長（如外域任務獲高額收益）即自動調升該個體邊際稅率。
- **稅務風控分**計算因子：收入波動率、高風險交易次數、異常資產增長頻率（單日入帳 > 月均 300% 的次數）、跨艙段移動頻率。
- 加成範圍：基準 +5% 到最高 +150%。

> **制度陷阱**：越努力逃離底層的人，稅務負擔越重，越難真正累積財富。

### 13.7 戰時稅則

高層在緊急情況下單方面調漲稅率的制度授權。啟動條件名義上「艙段遭重大威脅」，實際可由董事會隨意宣布。措施：

- 生存物資稅率 24 小時內調漲 200%~400%。
- 醫療與淨化服務改配給制，非優先級暫停。
- 跨艙段通行費翻 3 倍。
- 所有污染押金凍結 72 小時（讓清譜會有操作空間）。
- 居民不得拒絕「戰時勞動徵召」，違者以「破壞航行安全」論處。

### 13.8 躍階氧稅

在第十三段冷凝層等高風險區，按停留時間倍增的氧氣稅模型：

- 前 1 小時：基本通行費。
- 1~4 小時：每小時 +50%。
- 4~8 小時：每小時 +100%。
- 8 小時以上：每小時 +200%，觸發安全審查。

> 限制底層隊伍長時間活動的可能性，確保無法輕易逾越跨艙段門檻。

### 13.9 保證金互保池

深空資源回收公司主導，強制所有外域任務隊伍參加：

- 多支隊伍將部分資產存入共同風險池。
- 某隊全滅或違約時，債務由池中所有隊伍分攤。
- 名義「風險分攤互助」，實際是將違約風險轉嫁底層。
- 拒絕加入者將被禁用正規裂縫投放通道。

## 14. 經濟：公轉化區、黑箱、清譜會

### 14.1 公轉化區轉化流程

1. **進料登記**：樣本過磅、掃描、記錄來源與時間。
2. **污染讀數**：清譜會派遣污值核算師進行污染評級——直接影響樣本估價。
3. **轉化處理**：樣本進入轉化爐陣列，分解為規則燃料、能量晶核、可回收材料。
4. **官方結算**：以低於市價 30%~50% 的「官方指導價」支付存活點。
5. **二次黑箱**：官方結算完成後，進入封閉環節——高層第二次、不記帳抽取，規模約 30%~50%。

### 14.2 二次黑箱運作

位於公轉化區最深處的密閉艙室，極少數高層人士有權進入：

- 轉化完成的規則燃料分為「入庫」與「非入庫」兩條管線。
- 「入庫」進入正式帳目，維持母艦基本運作。
- 「非入庫」直接泵送至內核區的高層內宇宙接入站。
- 對外帳面上公轉化區轉化效率永遠標記「約 60%」——但這是二次抽取後的數字。

### 14.3 清譜會（污染金融）

半官方污染金融勢力，壟斷：

- **污染評級**：決定一批貨物的污染等級——可將高價樣本打成超標廢貨，也可將污染貨標成低風險。
- **清艙估值**：對高污染貨與封艙資產的估價程序，可任意壓低。
- **淨化排序**：決定誰先獲得淨化治療——醫療資源分配的實際控制權。

### 14.4 污染三級制

| 等級 | 定義 | 樣本價格 | 持有者影響 |
|---|---|---|---|
| 一級（可逆） | 污染輕微，標準淨化可清除 | 7~8 折 | 無長期影響 |
| 二級（半可逆） | 污染滲入組織，淨化後有後遺症 | 3~5 折 | 永久規則烙印，壽命縮短 |
| 三級（不可逆） | 污染導致畸變，無法逆轉 | 樣本沒收，持有者隔離 | 強制清洗或處決 |

評級操作空間：

- 污值核算師使用手持讀數槍檢測，可透過調整校準參數使讀數偏離 10%~30%。
- 「清艙證明」由清譜會獨家簽發，無他方可質疑。
- 高價值樣本通常先評為二級，迫使採集者低價出售，再由清譜會以「內部淨化」名義重新評級轉售牟利。

### 14.5 清譜會盈利模型

1. **押金滯留利差**：凍結 10,000 人 × 200 點押金 × 48 小時，可進行短期高利貸操作。
2. **延淨插隊費**：有錢隊伍支付額外費用跳過排隊——不計入官方帳目。
3. **低買高賣**：將高污染樣本低價買入，內部淨化後正常價格轉售。
4. **末批產出資產清算抽成**：艙段封閉時負責資產清算，抽取 10%~20%。

### 14.6 延淨仲裁員

清譜會派駐在淨化站的現場代表，掌握排序調整權。「一句延後淨化就能逼垮整支隊伍」是真實的勢力威脅。

## 15. 經濟：灰鏈

主角建立的私人地下流通網路，功能涵蓋洗白、走私、資訊交換與資源調度。

### 15.1 洗白通道（核心業務）

1. **資金池**：在底層多艙段設離線終端，客戶存入的黑市資源/存活點先匯入分散資金池。
2. **虛擬交易**：利用蘇嵐掌握的稅流稽核 API 後門，偽造多筆合法交易記錄（「物品回收」「技術諮詢」），將黑市收益掛載其下。
3. **稅務校準**：避免觸發差異化稅率，大額資金拆分為數十筆小額，48~72 小時分批入帳。
4. **手續費**：15%~30%，視規模與風險。

### 15.2 補給影線

第九段 → 第十五段的私有補給路線：

- **鋼簷橋**（第九段）：補給影線第一站，黑市中轉樞紐。
- **H-12 臨時節點**（第八、九段交界）：廢棄中繼站，灰鏈在此設暗倉存放醫療物資與規則燃料。
- **灰雨市場**（第七段）：灰鏈「民間低價供應回路」起始點，以低於續焰團的價格出售氧包與基礎藥劑。

### 15.3 資訊交換站

- 底層各艙段設情報收集點，監控審計司巡邏、裂縫投放時間表、企業招標。
- 資訊以「信用點數」計價——提供有價值情報者可獲灰鏈內部信用評級提升，享更低手續費與優先交易權。
- 唐雀的預警網（第四卷建立）為灰鏈資訊網絡的技術升級版。

### 15.4 灰鏈地盤範圍

| 項目 | 內容 |
|---|---|
| 地盤範圍 | 第九段至第十五段走廊（核心），延伸至第七段灰雨市場 |
| 控制節點 | 鋼簷橋、H-12 臨時節點、燼站、灰槽帶自治區 |
| 勢力據點 | 第九段維修工坊、第十五段灰槽帶（6 個月自治權） |
| 戰術特徵 | 資訊差作戰、局部機動性、主機規則覆寫 |

## 16. 經濟：四段式資源鏈與內宇宙養殖

### 16.1 四段式資源鏈

母艦經濟的完整循環——從底層到高層的制度性資源抽取系統。

**第一段：壓榨觸發**
- 底層隊伍在生存壓力下被迫簽署高風險外域任務合約。
- 黃卡警報線（餘額 100 點）系統強制推送高風險任務推薦。
- 拒絕者 72 小時內標記為「低效進程」，優先進入去敏或清洗名單。

**第二段：外掠**
- 隊伍深入外域裂縫開採。
- 風險由底層全額承擔（死亡、污染、畸變）。
- 樣本所有權自採集完成那刻即不屬於採集者（合約條款）。

**第三段：制度抽取**
- 清譜會污染評級壓價（截留 20%~30%）。
- 公轉化區官方結算抽成（30%~40%）。
- 二次黑箱不記帳抽取（剩餘的 30%~50%）。
- 底層採集隊伍最終獲得 ~樣本原始價值的 5%~12%。

**第四段：養殖回灌**
- 抽取的規則燃料最終流入高層內宇宙。
- 維持信仰養殖場運轉。
- 內宇宙產出低純度信仰回流，經轉化後再次循環供應母艦維生系統。
- 回灌效率極低（15%~25%），但因不需對外購買，被高層視為「免費資源」。
- 任何內宇宙產出缺口由擴大底層掠奪補足——形成壓迫閉環。

### 16.2 黑箱斷供閥（第五卷核心揭密點）

**G7 黑箱節點**（位於中層深處）——由審計司與董事會共同控制的分配閥門系統：

- 輸入端：連接公轉化區規則燃料管線。
- 輸出端：分支至各艙段維生系統與高層內宇宙接入站。
- 控制邏輯：Prime 系統依即時配額結算動態調整各輸出端分配比例。

**觸發條件**：當規則燃料儲備 < 安全線（約 30 天存量），啟動以下邏輯：

1. **優先保護內核區**：核心區與高層內宇宙供應不減。
2. **削減中層配給**：企業與技術行會降至 70%。
3. **中斷底層補給**：底層降至最低維持線以下——讓底層承受斷供主要衝擊。
4. **選擇性斷供**：同層艙段中依「產出效率」與「戰略價值」決定優先斷供對象——不配合高層者最先斷糧。

**劇情意義**：底層生死不是由「資源不足」決定，而是由一個可精確控制的分配閥門決定。**高層從未真正資源不足——他們只是在計算誰值得救。**

### 16.3 高層內宇宙養殖鏈

高層的「內宇宙」並非主角那樣培育真正獨立的文明，而是**工業化信仰養殖場**。

**養殖流程**：

1. **人口投放**：從起源池批量生成複製人，投放至內宇宙封閉生態艙。
2. **神話建構**：透過植入記憶與環境設計，讓內宇宙居民形成對「高層即神」的原始信仰。
3. **恐懼驅動**：設計週期性災難（規則密度波動、人造瘟疫），驅使居民產生強烈求生信仰。
4. **規則抽取**：信仰回流經轉化設備萃取為規則燃料，泵送回外部。
5. **週期重置**：居民大規模死亡或信仰枯竭時，內宇宙被格式化重新投放。

**養殖規模**：
- 每個高層核心成員通常擁有 3~7 個獨立的內宇宙養殖單元。
- 單個單元容納 5,000~20,000 居民。
- 單元年產規則燃料約 50~120 單位。
- 總體：高層內宇宙年產量約等於母艦全年外域掠奪收入的 15%~20%。

**根本性缺陷**：

- **信仰純度低**：恐懼驅動的信仰品質差，同體積下僅為主機信仰的 1/3~1/4 規則密度。
- **衰減速度快**：工業化信仰半衰期 ~60 天，主角內宇宙信仰半衰期可達 180 天以上。
- **維護成本高**：需持續投入資源維持壓迫系統（看守、懲罰設施、重置程序）。
- **道德風險**：長期工業化養殖導致集體性精神崩潰，大規模信仰枯竭週期越來越短。

**對母艦政治的影響**：高層內宇宙淨產出持續下降（年衰減 3%~5%），董事會被迫不斷擴大外域掠奪規模。**這解釋了為何隨時間推移，底層的壓迫只會越來越重——不是因為高層越來越邪惡，而是因為他們的內宇宙效率越來越差。**

### 16.4 主角內宇宙 vs 高層內宇宙

主角文明主機採共生模式（內部文明自主發展）；高層內宇宙是工業化牧場。前者產出純度極高（90%+），後者低純度且衰減快（25%~40%）。但高層靠規模取勝——十個低效率牧場仍可壓過一個高效文明主機產出。

## 17. 勢力一覽（母艦內部）

> **寫作核心準則**：每個勢力存在的目的，都是為了卡主角的脖子（壟斷資源），並成為主角的經驗包。

### 17.1 鐵鏽幫（艙底幫派／前期主要敵人）

- **活動範圍**：第九段微光區。
- **壟斷資源**：底層黑市抗輻射藥劑；近域拾荒帶廉價偷渡通道（收 30% 抽成）。
- **首要衝突**：主角第一天繳不出保護費與氧氣稅，幫派打手準備抓去解剖賣器官。主角必須反殺獲取啟動資金。
- **戰鬥兵種**：劣質義體打手（二手工業機械臂，弱點後頸神經接口）、毒氣噴射兵（廢料酸液，範圍腐蝕）。
- **第五卷狀態**：勢力衰落，邊緣化。

### 17.2 深空資源回收公司（企業聯盟代表／第一卷中後期）

- **壟斷資源**：高階武器、防具、正規醫療設備；唯一擁有「外域裂縫」正規開採權。
- **首要衝突**：企業主管試圖以「終身奴隸合約」套牢主角，將隊友當消耗品推入裂縫送死。
- **戰鬥兵種**：外骨骼傭兵（電磁步槍 + 戰術協同 AI）、戰術無人機群（偵查與自爆，逼主角動用主機感知）。

### 17.3 母艦審計司（懸頭達摩克利斯之劍）

- **壟斷資源**：合法生存權（一鍵凍結配額）、資料庫。
- **首要衝突**：主角實力增長過快或展現「不屬於這階級」的規則能力，觸發【異常成長警報】。
- **戰鬥兵種**：審計官（規則級裝備如能短暫消除動能的力場，肉體完美無瑕，極度冷靜精準）、清理者小隊（無痛覺無感情的量產改造人特種部隊）。
- **地盤範圍**：全艦（密度遞減：內核區最高，底層最低）。
- **關鍵節點**：審計司總部（內核區）、第十一段審計分站、第七封鎖線。

### 17.4 安流保全（區域承包商／過渡區壓迫節點）

- **活動範圍**：第十三段鐵鏽帶冷凝層與通行閘。
- **收入來源**：冷凝層通行費、躍階氧稅抽成、污染風險押金滯留利差。
- **壟斷手段**：冷凝層二/四號閘門權限（以「設備檢修」名義臨時封鎖路線）；區域監控演算法白名單。
- **對主角態度**：初期視為可榨取的高收益對象；連續三次突破封鎖後升級為「秩序擾動源」並通報審計司。
- **首要衝突**：扣押污染押金逼主角 12 小時內補件；硬闖被掛高風險標記。
- **戰鬥兵種**：冷凝盾衛（重裝防暴盾 + 低溫泡沫噴槍，擅長封走廊拖時間）、閘控技師（門禁、斷電、氣壓閥切割戰場）。
- **與主角關係**：黑箱繞行交易（貨物抽成換免檢通行）。

### 17.5 清譜會（污染金融／制度型敵人／中期壓迫節點）

- **活動範圍**：公轉化區高污染貨場、降級艙段交界、封艙倒數區。
- **收入來源**：污染押金滯留利差、延淨插隊費、清艙估值、污染貨打包轉售、末批產出抽成。
- **壟斷手段**：董事會外包的污染評級模型與淨化排序權；清艙證明與高污染貨交接資格。
- **對主角態度**：初期視為高收益獵物（高估污染值再逼折價）；穩定逆轉污染收益後標記為「破壞清算秩序者」。
- **首要衝突**：將主角高價樣本重判為污染超標，凍結 48 小時押金並要求轉賣指定買家。
- **戰鬥兵種**：污值核算師（手持讀數槍 + 封條槍，把現場讀數變成不可推翻的「合規事實」）、白膜搬運隊（全封閉白膜防護裝，泡沫束網與低溫噴劑控制高污染貨與鬧事者）、延淨仲裁員（掌握淨化排序與押金凍結權）。

### 17.6 盲舵會（封鎖帶領港／高風險合作對象／死艙入口節點）

- **活動範圍**：降級艙段外圍、死艙封鎖門、維護封存通道。
- **收入來源**：過期門禁碼、舊航道導航、切艙工班租用費、封鎖突破固定佣金與探索抽成。
- **壟斷手段**：舊維修紀錄、壓差節點、艙外維修索道、死人名冊。從不出售完整路線，只拆分時窗、門碼與轉向點。
- **對主角態度**：起初視為敢進高風險區的肥客；若主角掌握黑盒子或舊航道真圖，視為必須綁上船或滅口的變數。
- **首要衝突**：承諾安全路線卻在第二道封鎖門前臨時加價，甚至把同一組門碼賣給競爭隊伍。
- **戰鬥兵種**：舊圖師（假訊標 + 信號擾動器扭曲導航判讀）、切艙工（靜音切割鉗 + 爆栓 + 磁吸攀具突破封門）、引路人（壓差節點熟手，纜索槍 + 短衝鋒武器邊撤邊打）。

### 17.7 續焰團（宗教外包／民心型敵人／長線滲透節點）

- **活動範圍**：第七段底層救濟站、灰雨市場周邊、降級艙段轉運前哨。
- **收入來源**：招募佣金、志願遠征分潤、與企業/承包軍的人口供應合約、救濟站配給差價、官方宣講補貼。
- **壟斷手段**：多個低價氧吧、施粥站、「奉航志願書」登記口（救濟與徵募綁定）；借恆航教語彙為高風險任務洗白。
- **招募流程**：
  1. **誘餌**：發放低價物資（氧包、營養膏、基礎淨化針），條件是聽完「聖航宣講」。
  2. **轉折**：宣講中將苦難歸因於「個人奉航貢獻不足」——簽署奉航志願書並參加幾趟「聖航任務」即可獲更好配給與階級審查資格。
  3. **鎖定**：奉航志願書格式化契約，內藏陷阱條款（任務期間傷亡不賠、自願放棄部分配額、同意「必要改造」）。
  4. **轉運**：簽約者集中送往轉運前哨，分配至各企業外域任務或降級艙段末批產出。
- **對主角態度**：初期想包裝主角為底層逆襲招牌見證；拒絕並拆穿後定性為「假火種」與煽動墮航的異端。
- **首要衝突**：在主角熟悉街區發放低價氧包，條件是簽三趟奉航志願書；主角當場拆穿契約真相引爆民心對撞與群眾失控。
- **戰鬥兵種**：引焰使（廣播陣列 + 興奮霧 + 話術引爆群眾情緒）、餘燼修士（救濟站經營，衝突時啟用拘束帶 + 鎮靜針 + 臨時收容籠）、奉航監督（押運簽約者，電擊短棍 + 追蹤扣環）。

## 18. 艦隊體系（Ark-Prime + 外圍 + 先遣 + 子文明）

### 18.1 核心原則

小說開始時，世界上不再存在完整艦隊文明，只剩一艘在深淵中航行太久的宇宙級母艦 Ark-Prime。它周圍仍保有少量護衛艦、採集艦與清障艦，但這些已不再是榮耀軍事編制，而是母艦的**外部消耗器官**。更遠處散落著早年失聯的先遣艦隊殘存者。後期，主角體內/聖城孕育出的子文明會逐步形成與 Ark-Prime 對照的新型宇宙艦隊。

### 18.2 五層艦隊結構

| 層級 | 名稱 | 定位 |
|---|---|---|
| 0 | Ark-Prime 宇宙級母艦 | 唯一核心，文明主體 |
| 1 | 內部艙段文明 | 底層、中層、內核區，各階級社會 |
| 2 | 外圍護衛艦隊 | 近衛、採集、清障、拖曳、巡獵 |
| 3 | 失聯先遣艦隊 | 早期派出、深淵中殘存或異化 |
| 4 | 子文明宇宙艦隊 | 主角後期建立的新文明外延 |

### 18.3 層 2：外圍護衛艦隊

| 類型 | 功能 | 當前狀態 |
|---|---|---|
| 近衛艦 | 保護母艦外殼、反制深淵威脅 | 數量少，優先保護內核區 |
| 採集艦 | 外出拖回裂縫樣本、殘骸、生命信號 | 高損耗，常年不滿編 |
| 清障艦 | 清除深淵生物、畸變區障礙、失控單位 | 半軍事半回收，常被派去送死 |

**核心差異——黑箱管得到但管不細**：

```
母艦內部：黑箱看得到人、動作、資源、路徑、反饋。
外圍艦隊：黑箱只看得到任務回報、艦體狀態、返航貨物、死亡名單、污染讀數。
```

常見上下其手手段：

- **私吞採集品**：高價值樣本先被切走才上報，說詞「拖曳過程中熵化損耗」「污染超標已就地焚毀」「深淵壓差導致貨櫃破裂」。
- **虛報損耗**：回報數字永遠小於實際採集量，理由可歸咎規則風暴、污染隔離、護衛戰鬥消耗。
- **私藏人口**：從世界殘民、先遣艦倖存者、深淵殖民者中挑選有用者（技術人員修船、醫療人員維持艦隊存活、深淵適應者當導航員、兒童養成下一代）。外圍艦隊逐漸長出自己的小社會。
- **走私與黑市航線**：未登記規則碎片、深淵生物樣本、污染抗性組織、舊世界晶片、先遣艦航線圖。流通路徑：外圍艦隊 → 企業承包船 → 安流押運 → 灰鏈/清譜會/技術行會殘部 → 底層黑市。

**黑箱容忍腐敗的理由**：外圍艦隊不可或缺（採集、清理深淵威脅、回收外部人口、探測裂縫、承擔污染風險）。原則：**容忍腐敗，但不容忍失控**。

```
偷一點可以、私藏一點可以、走私一點可以。
但不能切斷母艦供應、不能建立獨立艦隊政權、
不能掌握反制黑箱的方法、不能和先遣艦隊/深淵遺族結盟。
```

**外圍艦隊四派系**：

| 派系 | 主張 | 代表 |
|---|---|---|
| 忠航派 | 母艦第一，貪污但不背叛 | 近衛艦老艦長 |
| 生存派 | 艦隊要先活下去，母艦只是補給源 | 採集艦隊主流 |
| 黑市派 | 任務只是幌子，真正目標是外部走私 | 企業承包艦、安流押運 |
| 斷鏈派 | 母艦已是捕食者，應該脫離 | 清障艦污染倖存者、部分先遣接觸者 |

**與主線的接點**：

1. **傳聞期**：底層只知道外圍艦隊是高危外勤，有採集艦私藏活人、清障艦回來後全員沉默等傳聞。
2. **灰鏈接觸**：林燼的灰鏈碰到外圍走私貨——先遣艦信標殘片、未登記規則碎片、外部人口記憶晶片、能干擾投放編碼的深淵樣本。
3. **艦隊交易**：外圍艦隊主動找林燼——他能洗白資源、破解黑箱監控、提供替代性規則燃料。
4. **清障艦真相揭露**：清障艦清除過先遣艦殘存者、世界殘民、甚至母艦自己失控的採集艦——把母艦捕食者邏輯推到台前。

### 18.4 層 3：失聯先遣艦隊

| 類型 | 狀態 | 對主線作用 |
|---|---|---|
| 漂流先遣艦 | 還保留人類指揮系統 | 提供深淵情報 |
| 殖民先遣艦 | 在畸變區建立小文明 | 可合作，也可能被母艦吞噬 |
| 融合先遣艦 | 與艦體/AI/深淵素材半融合 | 恐怖、異質、真相副本 |
| 反叛先遣艦 | 知道母艦現在的捕食本質 | 後期揭露母艦道德困境 |

先遣艦隊不是主要敵人，也不是完整友軍，而是母艦過去拋出去的碎片。

### 18.5 層 4：子文明宇宙艦隊

| 階段 | 子文明艦隊形態 |

## 19. 深淵遺族四類

> 深淵遺族不是單一種族，而是深淵中各種殘存生命、殘存文明與異化群體的統稱。共同點：曾經或正在深淵中生存，並已被深淵環境不同程度改寫。

### 19.1 A 類：世界殘民

其他宇宙被深淵吞噬時，逃出或殘留的生還者。**證明深淵不只吞噬主角世界，而是已經毀滅過很多宇宙**。

- 可能保留原文明技術、語言、宗教與記憶。
- 對深淵有古老經驗。
- 對 Ark 可能好奇、恐懼或想合作。
- 有些文明只剩碎片。

**敘事功能**：世界觀擴展、情報來源、悲劇文明、異世界殘骸。

### 19.2 B 類：敗戰後裔

曾與主宇宙交戰、戰敗後被困入深淵的外來世界後代。**不是單純敵人，而是舊戰爭留下的後果**。

- 與主角方有歷史仇恨。
- 後代未必知道當年真相。
- 可能把主宇宙或 Ark 視為災厄象徵。
- 同樣是深淵受害者。

**敘事功能**：戰爭遺恨、敵我界線模糊、和解或衝突支線。

### 19.3 C 類：先遣艦隊殘存者

主宇宙早期危機時派出去偵查深淵、調查規則漏洞、封鎖污染區的先遣艦隊。

**背景**：在深淵正式吞噬主宇宙前，世界已出現異常（規則漏洞、局部空間腐爛、物理常數漂移、因果延遲、畸變區生成）。高層曾派多批先遣艦隊調查與修補，但深淵吞噬太快，艦隊來不及回歸。

**現狀**：沒有全滅；有些漂流求生，有些建立小型殖民區；有些與艦體、AI、深淵素材半融合；有些已退化或深淵適應化；**他們比 Ark 更熟悉深淵環境**。

**敘事功能**：主宇宙殘留、深淵情報來源、Ark 道德困境、母艦冷血邏輯的具體受害者。

### 19.4 D 類：深淵原生種

不是外界進入深淵的遺民，而是在深淵環境中自然誕生或演化出的生命。

- 沒有「外面世界」的概念。
- 不認為深淵是災難，而是故鄉。
- 生命形式最異質、最不可預測。
- 可能視 Ark 為入侵者、獵物、病原體或異常物。

**敘事功能**：未知生態、深淵本質、非人類視角、真正異質生命。

### 19.5 母艦對四類遺族的處理

| 類型 | 母艦判定 | 回收重點 |
|---|---|---|
| 世界殘民 | 異世界資源 | 技術、文明資料、規則樣本、可用人口 |
| 敗戰後裔 | 高危高價值目標 | 舊敵科技、外來規則抗性、戰爭資料 |
| 先遣艦隊殘存者 | 高價值回收目標 | 主宇宙舊科技、深淵航線、艦體資源 |
| 深淵原生種 | 深淵適應模板 | 捕捉、解剖、實驗、轉化、武器化 |

### 19.6 Ark 的外部形象與核心道德困境

從外界看，**Ark 不是救世方舟，而是披著方舟外殼的文明級捕食者**。它會吞噬遇到的一切，把對方轉化為燃料、零件、情報、兵力、樣本、維生資源、存續率。

母艦不是單純邪惡。它的殘酷行動都能用生存率證明：不這麼做，Ark 就會死；Ark 一死，文明火種就會滅絕。

> **真正的問題**：如果文明只能靠吃掉其他殘存者活下去，那它救下來的到底還是不是文明？
> **主角未來要挑戰的不是單一命令，而是母艦整套生存邏輯**：在深淵中，文明能不能不靠捕食同類與異類而存續？

## 20. 深淵集合體六類

> 「母巢」只是 Ark 對某類集合體的方便代稱，**不是通用分類**。深淵集合體不是同源物種，也不是同一系統的分支，而是不同世界在低規則密度環境下，為了保存最後存在而形成的異質殘骸。

### 20.1 趨同演化原則

在深淵裡，越龐大的文明越不可能完整保存。最後能留下來的，只會是它最省資源、最執著、最核心的功能：

> 軍事文明留下戰場、宗教文明留下回聲、生物文明留下巢構、工程文明留下墟殼、多重毀滅區留下縫合物、真正想重啟文明的才留下核種。

### 20.2 Ark 觀測分類

| 觀測名 | 本質 | 關鍵差異 |
|---|---|---|
| 巢構集合體 | 採集、孵育、轉化的殘存結構 | 像器官，不一定有意識 |
| 戰場集合體 | 未停機的戰爭系統 | 只剩敵我判定與毀滅流程 |
| 回聲集合體 | 集體記憶／意識殘響 | 會重播、感染、覆寫人格 |
| 墟殼集合體 | 規則耗盡的文明空殼 | 像宇宙級死艙，可回收但危險 |
| 縫合集合體 | 多世界殘骸混合物 | 非同源，深淵新生態溫床 |
| 核種集合體 | 文明重啟核心 | 稀有，可能是反熵或新文明火種 |

### 20.3 各類型詳述

**巢構集合體**：呈現採集、孵育、轉化、增殖的結構（故 Ark 稱為巢構）。不一定是生物，也不一定是母巢。可能是一個文明最後把自己改造成自動保存器官，也可能是一個世界的生態系被壓縮後只剩繁殖與捕食功能。**Ark 自己可以被外界視為一種巢構集合體，但 Ark 內部未必承認**。

**戰場集合體**：某個宇宙最後一場戰爭沒有結束。本質不是生產，而是敵我判定仍在運作。不是要活下去，而是要繼續打完一場早已無人的戰爭。

**回聲集合體**：某個文明的集體記憶被深淵壓縮成可感染結構。不一定想害人，可能只是一直在重播自己毀滅前的最後幾天。危險在於：進去以後，它會把你當成它的一部分來補全。

**墟殼集合體**：宇宙級死艙。已經沒有明確意志，也沒有完整生態，只剩低規則密度空殼、古代設備、半熄滅的法則碎片。對 Ark 來說最像礦場，但也可能是陷阱。

**縫合集合體**：最接近深淵本質的類型。不是一個世界的遺骸，而是多個世界殘骸互相嵌合後形成的怪東西。沒有歷史、沒有原文明、沒有單一意志。是深淵本身把廢棄宇宙揉在一起後長出的新生結構。

**核種集合體**：稀有。不是殘骸，而是某個文明在滅亡前留下的核心種子。**林燼的文明主機往這個方向靠，但不代表所有核種都同源**。有的核種是神明胚胎、有的是文明備份、有的是演算核心、有的是反熵火種。共同點不是來源，而是功能——在深淵中重新長出一個文明。

## 24. 不可違反的世界規則邊界

> 寫作鐵則（系統提示格式、敘事原則、主題與隱喻指導、派系寫作）見 `RULES.md`。本節只列**世界規則本身的物理／機制邊界**。

### 24.1 物理／世界觀

1. **不要讓母艦在深淵中「停下來」**——停下意味著規則密度梯度消失，母艦會被困在該區域。
2. **不要讓深淵生物在 V7 前正式現身**——可以暗示、可以撞擊、但不能看見或接觸。
3. **不要讓主角獲得完整的深淵地圖**——深淵的地圖永遠是局部、暫態、不可靠的。
4. **不要解釋規則霧的終極來源**——留給 V14 的終局真相。
5. **不要讓深層躍遷成為常規手段**——每次都應該帶來明顯代價。
6. **不要解釋異常垢核來源**——V14 伏筆，標記為「不可餵食、不可轉化、不可長期保存」。

### 24.2 技術／勢力機制

7. **霧食螅不能餵異常霧垢**、**不能清得太乾淨**、**清霧後必須回收螅群**——三條安全鐵則。
8. **黑箱原則：容忍腐敗，但不容忍失控**——外圍艦隊可以偷、藏、走私，但不能切斷供應、不能建立獨立政權、不能掌握反制黑箱的方法、不能和先遣艦隊/深淵遺族結盟。
9. **續焰團不是正式教會**——是恆航教在底層的市場化變體（人力供應鏈），高層默許但不直管。
