## 1. 主角 林燼

### 1.1 基礎資料

- **身份**：末日母艦第九維修段・底層維修工 / 文明主機持有者
- **前世**：穿越者（保留空白）
- **外在形象**：劣質生鏽義肢，常年帶機油味與血腥味；眼神像疲憊但隨時能暴起的野狗

### 1.2 核心行為邏輯

1. **絕對利益導向**：吃不飽時絕不救無價值路人。誰擋財路殺誰。
2. **契約精神**：對幫過他的人以等價或更高利益回報。任何人給東西，第一反應是「代價是什麼？」
3. **斬草除根**：不廢話、不聽懺悔、能補刀就補刀。打不過絕不硬剛，撤退下毒。

### 1.3 戰鬥風格

利用維修工場地優勢 + 主機資訊差，玩陰的、埋炸彈、切維生系統。

### 1.4 台詞風格

少、穩、帶壓迫。

- 「比你快。」
- 「座標發我。」
- 「過不去，也得過。」

### 1.5 主軸人物弧線

- **求存者（V1）**：只想活過明天。把主機當裝備提款機。
- **掠食者（V2）**：主動籌劃爬向中層。開始當合格的神。
- **執棋者（V3+）**：掀翻現有規則。視內部文明為同族。

## 5. 第五卷反派 祁闕（配額催收官）

### 5.1 基礎資料

| 項目 | 內容 |
|---|---|
| 身份 | 配額催收官（跨艙段級、中層編制） |
| 首次出現 | 第 140 章（預計） |
| 外在形象 | 短髮、標準配給面容（標誌性營養供應線條）、審計署制服改良版、動作精確度過高（接近機械） |
| 不可替代資產 | 掌握黑箱斷供閥的配額調度權限；六艙段跨域物資流向改單權 |
| 行為邏輯 | 系統效率極致化，每一步決策都能用「降低配額成本」或「優化資源配置」合理化。**不認為自己在做壞事——他是在遵循制度的數學邏輯** |
| 台詞風格 | 冷峻、術語堆砌、帶著行政命令的語調。沒有廢話，只有配額數字 |

### 5.2 台詞範例

- 「您的分類等級不支持此類配額請求。」
- 「刪除會優化系統效率。」
- 「這是制度評估，不是個人意見。」

### 5.3 常見誤判

1. 以為配額催收官只掌握後勤、實際掌握斷供權
2. 以為他有道德底線、實際他把道德當作「低效率變數」
3. 以為他懼怕威脅、實際威脅觸發他的「防異常流程」

### 5.4 崩潰表現

不是手停在半空——是配額調度邏輯陷入死循環、他的眼神從機械性轉為反覆掃描（尋找系統漏洞、尋找新的優化方案），最後發現自己成為了「不可優化的異常元素」。

### 5.5 與主角衝突

林燼通過反向投放製造的邏輯死鎖，讓配額體系無法判斷誰該活誰該死——**這對祁闕而言是比死亡更恐怖的事：制度失效**。

### 5.6 第五卷弧線

- **獵犬（140-144）**：無差別執行白名單清洗
- **困獸（145-148）**：被反向投放困在決策悖論
- **系統崩壞者（149-150）**：執行最後的「非人道式決策」來恢復秩序
