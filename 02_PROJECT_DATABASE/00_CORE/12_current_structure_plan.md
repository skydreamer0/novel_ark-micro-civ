## 二、全書主線架構

本書的主線不是單純升級打怪，而是「被收割者逐步取得規則改寫權」。

1. **底層求生**：林燼從欠稅死亡倒數中活下來，文明主機讓他能用內部文明反哺自身。
2. **制度反擊**：灰鏈、裂縫入口、稅流與投放名冊逐步揭露，主角開始反制母艦秩序。
3. **聖城成形**：內部文明從神殿信仰走向政治機構，祭司派、工匠派、軍務與資訊線分化。
4. **黑箱斷流**：林燼不再只偷資源，而是切斷高層補給線，逼出配額結算與母艦遠征真相。
5. **規則改寫**：第五卷後段的核心目標是把「底層人口優先保留權」寫入母艦調度邏輯。
6. **終局覺醒**：後續V8-V14走向深淵母巢、權限經濟、母艦意識與新天道的融合結局。

## 九、續寫第145章前的決策清單

1. 第145章要承接 60 分鐘保留時限，不能讓第144章的短時成功變成無成本豁免。
2. 追蹤「非母艦單位」補給請求時只露輪廓，不一次揭開深淵母巢或終局設計者。
3. 陌生更新簽名可作第二載體/同源主機線預熱，但不宜正式登場。
4. 祁闕要維持制度代理人的冷酷，不宜寫成普通武力型反派。
5. 祭司派與工匠派衝突要制度化，不要簡單消滅其中一方；它們是聖城後續雙引擎。
6. 林燼外洩的主機更新簽名必須帶來逆向追蹤代價。
7. V4不可預測風險已升到23%，後續可讓副作用從候選者、子編碼或陶青連結處冒頭。
