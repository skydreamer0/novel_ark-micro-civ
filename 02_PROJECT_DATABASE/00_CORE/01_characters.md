## 主角：林燼

### 基礎資料
- **身份**：末日母艦第九維修段・底層維修工 / 文明主機持有者
- **前世**：穿越者（保留空白）
- **外在形象**：劣質生鏽義肢，常年帶機油味與血腥味，眼神像疲憊但隨時能暴起的野狗

### 核心行為邏輯
1. **絕對利益導向**：吃不飽時絕不救無價值路人。誰擋財路殺誰。
2. **契約精神**：對幫過他的人以等價或更高利益回報。任何人給東西，第一反應是「代價是什麼？」
3. **斬草除根**：不廢話、不聽懺悔、能補刀就補刀。打不過絕不硬剛，撤退下毒。

### 戰鬥風格
利用維修工場地優勢 + 主機資訊差，玩陰的、埋炸彈、切維生系統。

### 台詞風格
少、穩、帶壓迫。
- 「比你快。」
- 「座標發我。」
- 「過不去，也得過。」

### 人物弧線
- **求存者（Vol.1）**：只想活過明天。把主機當裝備提款機。
- **掠食者（Vol.2）**：主動籌劃爬向中層。開始當合格的神。
- **執棋者（Vol.3+）**：掀翻現有規則。視內部文明為同族。

### 目前 V9 能力（第120章）
- 名冊資料操縱、權限偽裝、調度序列干擾、內世界反饋強化
- 暴露概率：使用一次 5% → 持續使用指數增長至 70%

## 第五卷主要反派

### 祁闕（配額催收官）

| 項目 | 內容 |
|---|---|
| **身份** | 配額催收官（跨艙段級、中層編制） |
| **首次出現** | 第140章（預計） |
| **外在形象** | 短髮、標準配給面容（標誌性營養供應線條）、審計署制服改良版、動作精確度過高（接近機械） |
| **不可替代資產** | 掌握黑箱斷供閥的配額調度權限；六艙段跨域物資流向改單權 |
| **行為邏輯** | 系統效率極致化，每一步決策都能用「降低配額成本」或「優化資源配置」合理化。不認為自己在做壞事——他是在遵循制度的數學邏輯 |
| **台詞風格** | 冷峻、術語堆砌、帶著行政命令的語調。沒有廢話，只有配額數字 |
| **台詞範例** | 「您的分類等級不支持此類配額請求」「刪除會優化系統效率」「這是制度評估，不是個人意見」 |
| **常見誤判** | 1. 以為配額催收官只掌握後勤、實際掌握斷供權；2. 以為他有道德底線、實際他把道德當作「低效率變數」；3. 以為他懼怕威脅、實際威脅觸發他的「防異常流程」 |
| **崩潰表現** | 不是手停在半空——是配額調度邏輯陷入死循環、他的眼神從機械性轉為反覆掃描（尋找系統漏洞、尋找新的優化方案），最後發現自己成為了「不可優化的異常元素」 |
| **與主角衝突** | 林燼通過反向投放製造的邏輯死鎖，讓配額體系無法判斷誰該活誰該死——這對祁闕而言是比死亡更恐怖的事：制度失效 |
| **第五卷弧線** | 獵犬(140-144)：無差別執行白名單清洗 → 困獸(145-148)：被反向投放困在決策悖論 → 系統崩壞者(149-150)：執行最後的「非人道式決策」來恢復秩序 |

## 配角詳細設定

### 唐雀（預警網建立者）

| 項目 | 內容 |
|---|---|
| **身份** | 民用網系統工程師（第四卷晉升為前線協調官） |
| **首次出現** | 第91章 |
| **不可替代資產** | 掌握灰槽帶全景監聽點佈局；民用網與審計網的跳轉中繼邏輯；400m預警網的容錯機制細節 |
| **致命弱點** | 監聽點本身容易暴露（連續兩次無回傳即自動重配，留下漏洞）；對自己的網路架構有執著，認為任何刪改都是「破壞」而非「進化」；與三名早期監聽點維護員工有私誼 |
| **台詞風格** | 嚴謹、帶著工程師的「還原邏輯」思維。焦慮時會重複數字 |
| **台詞範例** | 「誤報率7%，意味著每十四個告警有一個假。我們無法分辨」「這個節點的連接方式是我設計的，我不能讓它暴露」 |
| **與主角衝突** | 林燼要求扯斷預警網為掩護行動；唐雀害怕失去對監聽生態的控制，同時害怕監聽點被敵方利用 |
| **道德底線** | 可以犧牲監聽點，但不能犧牲依賴預警活著的底層平民。白名單生存制啟動後，她會面臨「繼續維護網給要被清洗的人」還是「關閉網避免自己暴露」的兩難 |
| **第五卷功能定位** | 民生探針去向追蹤者；預警網在白名單制下的「逆向濾波」角色；第131-135章的關鍵信息源 |

### 陶青（民生線管理者）

| 項目 | 內容 |
|---|---|
| **身份** | 醫療資源協調員（灰槽帶民生線負責人） |
| **首次出現** | 第91章 |
| **不可替代資產** | 掌握傷員分級與淨化符消耗的精確數據；與各艙段人脈醫護網絡；中階淨化符替代方案知識（熵污染緩解的非官方手段） |
| **致命弱點** | 對患者的因果性關懷（知道名字的人，死時會失眠）；淨化符庫存極限5天這個數字是死線；有一個感染者親屬在中層工作 |
| **台詞風格** | 冷靜但帶著疲憊的同情。說話時會不自覺清點物資 |
| **台詞範例** | 「我們有四十一個理由再撐五天」「淨化符不是藥，是制度允許活著的證明」「那個人叫詹勇，他有女兒」 |
| **與主角衝突** | 林燼的黑箱斷流計畫會讓中層物資調度癱瘓，包括淨化符供應。陶青必須選擇：是否允許為更大的反抗犧牲眼下的四十一個傷患 |
| **道德底線** | 不會主動傷害無辜，但可以被迫做出「集體捨棄」決定。如果白名單制判定某些患者「不值得活」，陶青會遵守（她無法違抗制度），但會記住每一個名字 |
| **第五卷關鍵抉擇** | 第124章：淨化符徹底耗盡時，是否執行「優先救治戰鬥力強者」的分級方案；第128-130章：是否將民生線資源轉向支援黑箱打擊行動 |

### 祁老者（祭司派領袖）

| 項目 | 內容 |
|---|---|
| **身份** | 地下信仰組織領袖（約70人規模的神祕派系） |
| **首次出現** | 第103章（作為背景群體） → 第112章（正式出場） |
| **組織規模** | 初始40人 → 第120章時70人（兩天內翻倍），其中至少2人出現在收割派外圍支援名單上 |
| **不可替代資產** | 掌握底層宗教信仰網絡（在高層看不見的地方執行精神控制）；與某些中層審查機構的秘密聯繫；地下裂痕的選址知識 |
| **致命弱點** | 相信「制度本身就是神聖的」，因此相信只要向制度靠攏，自己就能被利用而獲利。這是他最大的盲點；組織擴張速度超過他的掌控範圍，內部已有外部滲透者 |
| **台詞風格** | 宗教性、引經據典、帶著對「秩序」的變態崇拜 |
| **台詞範例** | 「母艦的呼吸就是神聖的節奏」「被選中的人不需要反抗，只需要獻祭」「配額就是真理的量化」 |
| **行為邏輯** | 1. 將制度神聖化，認為向高層獻忠誠、向配額屈膝都是「智慧」；2. 對信徒進行精神控制，鼓勵他們為了「更高的秩序」接受不人道的分配；3. 不主動反抗，而是尋求與權力結構的「共謀」（以為這樣能獲得豁免權） |
| **與主角衝突** | 林燼的反抗本質上是「否定制度的神聖性」。祁老者會被激怒（林燼褻瀆了他的信仰），同時也會威脅要向高層舉報（他仍然相信高層會保護「有用的工具」） |
| **黑化條件** | 當高層無視他的獻忠誠（如白名單制將祁老者的追隨者同樣劃入待清洗名單）時，他會經歷信仰瓦解。黑化不是反抗——而是從「親近權力的聖職」轉變為「複製權力的冷酷」，成為第二個祁闕 |
| **第五卷動向** | 第122-125章：試圖透過宗教動員為白名單制招募「自願犧牲者」；第126-130章：發現自己的信徒也被納入清洗名單，信仰開始動搖；第131-135章：尋求與林燼的交易（以組織忠誠換取豁免权），被拒絕後陷入精神崩壞 |
