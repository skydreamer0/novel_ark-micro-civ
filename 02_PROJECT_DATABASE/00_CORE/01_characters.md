## 主角：林燼

### 基礎資料
- **身份**：末日母艦第九維修段・底層維修工 / 文明主機持有者
- **前世**：穿越者（保留空白）
- **外在形象**：劣質生鏽義肢，常年帶機油味與血腥味，眼神像疲憊但隨時能暴起的野狗

### 核心行為邏輯
1. **絕對利益導向**：吃不飽時絕不救無價值路人。誰擋財路殺誰。
2. **契約精神**：對幫過他的人以等價或更高利益回報。任何人給東西，第一反應是「代價是什麼？」
3. **斬草除根**：不廢話、不聽懺悔、能補刀就補刀。打不過絕不硬剛，撤退下毒。

### 戰鬥風格
利用維修工場地優勢 + 主機資訊差，玩陰的、埋炸彈、切維生系統。

### 台詞風格
少、穩、帶壓迫。
- 「比你快。」
- 「座標發我。」
- 「過不去，也得過。」

### 人物弧線
- **求存者（Vol.1）**：只想活過明天。把主機當裝備提款機。
- **掠食者（Vol.2）**：主動籌劃爬向中層。開始當合格的神。
- **執棋者（Vol.3+）**：掀翻現有規則。視內部文明為同族。

對話風格：短句、粗口、不打官腔
關鍵轉折：第3章首次登場，與主角建立債務關係
卷末狀態：活下來了，但被審計司捏著把柄

##### V2 — 艙段野犬
核心焦慮：如何在戰場活下來又不欠人情債
決策模式：用暴力建立威懾，打服為止
對話風格：命令式、不耐煩、打斷別人
關鍵轉折：在關鍵戰鬥中證明自己不是普通砲灰
卷末狀態：從流動傭兵轉為有固定地盤的戰力

##### V3 — 守夜者
核心焦慮：抑制劑倒數計時（剩下不到48小時）
決策模式：被動防守、保存實力、伺機反咬
對話風格：多疑、試探，每句話都在蒐集情報
關鍵轉折：佔領並熟悉特定艙段，建立根據地
卷末狀態：有領地意識了，開始把該艙段當「自己的」

##### V4 — 戰線砥柱
核心焦慮：這道防線垮了，所有人一起死
決策模式：開始思考戰術組合，不再單打獨鬥
對話風格：簡潔、精準，廢話減少
關鍵轉折：防線指揮權非正式轉移到他手上
卷末狀態：防線穩住，但他的價值已經藏不住了

##### V5 — 聖城軍務司令
核心焦慮：四百多條人命壓在一個前傭兵肩上
決策模式：被迫學習指揮與資源調度
對話風格：指揮腔調成形，但底層粗礪痕跡仍在
關鍵轉折：正式任命；改用聖城自製替代品控制抑制劑成癮
卷末狀態：從肉盾升級為軍事領袖，但被看見就等於被盯上

##### V6 — 體系架構師
核心焦慮：大規模混編防線的結構性漏洞在哪裡
決策模式：系統化部署、分層防禦、預備隊調度
對話風格：教導式語氣出現，對菜鳥不耐煩但會教
關鍵轉折：設計並實行混編防線體系，不再只是執行者
卷末狀態：防線制度化，他的不可替代性上升

##### V7 — 政治緩衝器
核心焦慮：高層利益交換時自己的士兵變成籌碼
決策模式：政治妥協與軍事需求之間的走鋼索
對話風格：兩套語言——對上委婉迂迴，對下仍然直接
關鍵轉折：在高層政治鬥爭中成功保護部下，代價是欠了人情
卷末狀態：意識到打仗簡單，政治才要命

##### V8 — 核心執行者
核心焦慮：我不能倒，因為沒有備用方案
決策模式：危機決策、快速判斷、直覺精確化
對話風格：越來越少話，命令越來越短
關鍵轉折：成為軍事防線無可替代的核心節點
卷末狀態：所有人都依賴他，但沒有人能替代他

##### V9 — 戰術革新者
核心焦慮：老戰術對新型威脅完全無效
決策模式：實驗性戰術、敢於試錯、從失敗中快速迭代
對話風格：討論式姿態出現，願意聽懂的人建議
關鍵轉折：針對新威脅開發出有效戰術，打破經驗主義
卷末狀態：從戰將進化為戰術思想家

##### V10 — 防線之盾
核心焦慮：身體和心理還能撐多久
決策模式：保守防禦、以逸待勞、最小化傷亡
對話風格：語氣沉重、簡短、透著疲憊
關鍵轉折：身體或心理出現不可逆的疲態信號
卷末狀態：傷痕累累但防線仍在，代價是他的健康

##### V11 — 傳承者

決策模式：培養接班人、逐步放權、容忍新人犯錯
對話風格：嘮叨的長者語氣出現（反差感）
關鍵轉折：開始系統性培養接班人，正視自己的可替代性
卷末狀態：交出一部分權力，但核心職責仍在他身上

##### V12 — 決策者
核心焦慮：是否要犧牲少數來保全多數
決策模式：戰略層級判斷，不再只管戰術
對話風格：決定性語氣，一錘定音
關鍵轉折：被迫做出重大犧牲決策，承受代價
卷末狀態：做過最難的決定了，接下來只剩執行

##### V13 — 最終旗手
核心焦慮：這一戰可能是最後一戰
決策模式：破釜沉舟、不留後路
對話風格：最終戰前動員，簡短有力
關鍵轉折：最終決戰指揮，他的每一個決定決定結局
卷末狀態：打了，活下來或沒活下來

##### V14 — 句點
核心焦慮：仗打完了，然後呢
決策模式：從戰時狀態轉向重建或退役
對話風格：輕鬆、如釋重負，或不存在的沉默
關鍵轉折：找到戰後的位置或接受無法找到
卷末狀態：弧線閉合——犧牲、退役、晉升、或消失在人群中

#### 跨卷演化表

##### V1 — 觀察者
核心焦慮：混沌正在蔓延，必須在失控前量化它
決策模式：紀錄、分類、不急著介入
對話風格：臨床報告式的精確
關鍵轉折：注意到主角，開始建立檔案
卷末狀態：冷靜的觀察者，尚未暴露意圖

##### V2 — 獵人
核心焦慮：獵物比預期中聰明，秩序可能壓不住
決策模式：主動搜索、設陷阱、有限度介入
對話風格：冷嘲熱諷但控制精準
關鍵轉折：從觀察跨入直接行動，燒掉回頭路
卷末狀態：鎖定目標，但獵物已經知道被盯上

##### V3 — 被獵者
核心焦慮：獵物反咬，而且咬得很準
決策模式：被動防守、資訊封鎖、試圖重新掌握主動權
對話風格：句子變短，停頓變長，咖啡杯開始出現
關鍵轉折：被主角反制，第一次失手
卷末狀態：從獵人變獵物，立場反轉

##### V4 — 偏執者
核心焦慮：失去對局面的掌控，而且速度在加快
決策模式：越來越激進，在秩序框架內做越界的事
對話風格：自我辯解增多，邏輯鏈出現跳躍
關鍵轉折：開始失去上級信任，手段被檢討
卷末狀態：孤立，但還沒放棄

##### V5 — 崩潰者
核心焦慮：秩序代表的身份與混亂行為之間的裂縫撕裂全身
決策模式：「理性」決策但帶有明顯的非理性參數
對話風格：破碎、跳躍、自說自話
關鍵轉折：咖啡杯事件——瞳孔縮緊，完整的他碎在那一秒
卷末狀態：崩潰退場，顧衡這個層級已不足以構成威脅

##### V6 — 囚徒
核心焦慮：被自己維護的系統審判
決策模式：零決策權，只能反應
對話風格：沉默為主，偶爾說出沒有人要聽的真話
關鍵轉折：內部審判，成為系統的替罪羊
卷末狀態：被關、被軟禁、或被廢黜

##### V7 — 幽靈
核心焦慮：世界繼續運轉，沒有他反而更好
決策模式：逃亡，避開所有曾經的獵物
對話風格：低沉、沙啞、像從井底傳出來
關鍵轉折：逃亡路上遇到曾經被他迫害過的人
卷末狀態：苟活，失去了所有身份

##### V8 — 復仇者
核心焦慮：那些取代他的人憑什麼
決策模式：策劃復仇，但資源和能力都已萎縮
對話風格：空洞的威脅，連他自己都不信
關鍵轉折：試圖復仇，發現自己已經沒有執行力
卷末狀態：復仇失敗，最後一口氣洩掉

##### V9 — 空殼
核心焦慮：復仇成功了（或失敗了），然後呢
決策模式：模仿活人的決策，但沒有內驅力
對話風格：幾乎不說話，開口也是別人的句子
關鍵轉折：意識到內心是空的，秩序信仰沒了，恨也沒了
卷末狀態：行屍走肉

##### V10 — 旁觀者
核心焦慮：不再焦慮，這才是最可怕的
決策模式：不決策，純粹觀察
對話風格：偶爾冷評，像旁白，像V1的自己但沒了溫度
關鍵轉折：從參與者轉為純粹的紀錄者
卷末狀態：心如死灰，但眼睛還睜著

##### V11 — 紀錄者
核心焦慮：如果我也不記得了，誰來證明這一切發生過
決策模式：紀錄、保存、傳遞
對話風格：書面語氣，與V1呼應但多了滄桑
關鍵轉折：開始寫——不是報告，是紀錄
卷末狀態：從秩序獵人變成了歷史見證者

##### V12 — 被捲入者
核心焦慮：不想再參與，但世界不讓他置身事外
決策模式：被動回應，每一步都是被逼的
對話風格：被迫開口，詞不達意，舊傷被翻出來
關鍵轉折：被捲入最終衝突，無法逃避
卷末狀態：站在他不想要的位置上

##### V13 — 犧牲者
核心焦慮：最後一次選擇的機會
決策模式：終於有一次真正的主動選擇——救人或毀人
對話風格：最後一句話——可能是道歉、詛咒、或沉默
關鍵轉折：做出最終選擇，與自己的過去和解或徹底決裂
卷末狀態：弧線終點——死、殘、或終於自由

##### V14 — 句號
核心焦慮：已經沒有焦慮了
決策模式：無
對話風格：無（或一句話的迴響）
關鍵轉折：被提及/被記得/被遺忘
卷末狀態：存在不存在都不重要了——一個句號

| **不可替代資產** | 掌握黑箱斷供閥的配額調度權限；六艙段跨域物資流向改單權 |
| **行為邏輯** | 系統效率極致化，每一步決策都能用「降低配額成本」或「優化資源配置」合理化。不認為自己在做壞事——他是在遵循制度的數學邏輯 |
| **台詞風格** | 冷峻、術語堆砌、帶著行政命令的語調。沒有廢話，只有配額數字 |
| **台詞範例** | 「您的分類等級不支持此類配額請求」「刪除會優化系統效率」「這是制度評估，不是個人意見」 |
| **常見誤判** | 1. 以為配額催收官只掌握後勤、實際掌握斷供權；2. 以為他有道德底線、實際他把道德當作「低效率變數」；3. 以為他懼怕威脅、實際威脅觸發他的「防異常流程」 |
| **崩潰表現** | 不是手停在半空——是配額調度邏輯陷入死循環、他的眼神從機械性轉為反覆掃描（尋找系統漏洞、尋找新的優化方案），最後發現自己成為了「不可優化的異常元素」 |
| **與主角衝突** | 林燼通過反向投放製造的邏輯死鎖，讓配額體系無法判斷誰該活誰該死——這對祁闕而言是比死亡更恐怖的事：制度失效 |
| **第五卷弧線** | 獵犬(140-144)：無差別執行白名單清洗 → 困獸(145-148)：被反向投放困在決策悖論 → 系統崩壞者(149-150)：執行最後的「非人道式決策」來恢復秩序 |

| 項目 | 內容 |
|---|---|
| **身份** | 地下信仰組織領袖（約70人規模的神祕派系） |

| **致命弱點** | 相信「制度本身就是神聖的」，因此相信只要向制度靠攏，自己就能被利用而獲利。這是他最大的盲點；組織擴張速度超過他的掌控範圍，內部已有外部滲透者 |
| **台詞風格** | 宗教性、引經據典、帶著對「秩序」的變態崇拜 |
| **台詞範例** | 「母艦的呼吸就是神聖的節奏」「被選中的人不需要反抗，只需要獻祭」「配額就是真理的量化」 |

| **與主角衝突** | 林燼的反抗本質上是「否定制度的神聖性」。祁老者會被激怒（林燼褻瀆了他的信仰），同時也會威脅要向高層舉報（他仍然相信高層會保護「有用的工具」） |
| **黑化條件** | 當高層無視他的獻忠誠（如白名單制將祁老者的追隨者同樣劃入待清洗名單）時，他會經歷信仰瓦解。黑化不是反抗——而是從「親近權力的聖職」轉變為「複製權力的冷酷」，成為第二個祁闕 |
| **第五卷動向** | 第122-125章：試圖透過宗教動員為白名單制招募「自願犧牲者」；第126-130章：發現自己的信徒也被納入清洗名單，信仰開始動搖；第131-135章：尋求與林燼的交易（以組織忠誠換取豁免权），被拒絕後陷入精神崩壞 |
