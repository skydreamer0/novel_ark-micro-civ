## 三、第一幕：產線重構與邊界偵查（Ch.241-260，20章）

### 3.1 內世界三線結構（本卷新加入）

內世界在本卷有獨立的三條子線，不是產能管理的背景板：

**子線1：派系政治（擴張 vs 保守的矛盾）**
- 戰派（燕九）：爆兵探索，犧牲內世界福利換產能
- 守派（工匠）：先鞏固再出去，怕擴張太快內世界崩潰
- 尋派（蘇嵐）：平衡派，但1.5s延遲讓她的判斷開始被質疑
- 衝突點 Ch.257：戰/守在共識區公開辯論 — 內世界第一次民主制度壓力測試
- 衝突點 Ch.278：尋派分裂（部分認為蘇嵐1.5s影響決策，轉向支持戰派）

**子線2：內世界基礎建設**
- 共識區灰白色拱型隨擴張出現裂縫（象徵內世界成長太快）
- 淨化塔#7啟動 → 孵化腔錨點在內世界深處顯現
- 黑盒子 → 儲存設計者原始協議拷貝，需內世界共識解鎖

**子線3：內世界居民視角（I線，每10章插入一次）**
- Ch.250：普通居民第一次看到穿規則層單位列隊出發 — 街道兩旁全是人
- Ch.260：老居民站在共識區外圍旁聽戰守辯論 — 「我們以前連說話都不敢」
- Ch.270：工匠派學徒聽老師傅講「以前我們只能躲在母艦角落」
- Ch.275：供配所職員看到外部燃料送達通知時手在發抖 — 聖城第一次有外部補給
- Ch.290：年輕製造人口聽說產能回到20萬/週，在街道上哭了
- Ch.295：第一代製造人口站在節點化儀式現場 — 「我以為聖城只是個庇護所，原來它是真的」

### 3.3 爆兵三階段詳細（每階段有代價+戰果+爽點）

**階段一：重構期（Ch.241-254）**
- 產能：8→12萬/週
- 單位：穿規則層偵查型（裸結構網適配，輕武裝，極速快）
- 代價：強行無權限製造消耗內世界規則燃料儲備40%
- 戰果：首批偵查型成功跨越邊界並回傳數據
- 內世界震盪：規則燃料儲備見底 → 民生配給縮減 → 戰/守首次對立
- **爽點**：無權限爆兵首次成功（Ch.242）、偵查型穿越邊界（Ch.258）

**階段二：投射期（Ch.255-274）**
- 產能：12→16萬/週
- 單位：戰鬥型（蟲族基底融合，近戰專精）+ 通訊型（跨邊界中繼）
- 代價：戰鬥型使用孵化腔蟲族基底模板 → 每次製造消耗孵化腔能量 → 文明記憶讀取變慢
- 戰果：生存派交易後獲得外部規則燃料補給 → 儲備回升；戰鬥型在Ch.270伏擊戰中首秀
- 內世界代價：孵化腔能量降低 → 部分文明記憶片段永久丟失（陳硯感知到）
- **爽點**：燕九率3艘戰鬥型1:3反殺黑市派伏擊（Ch.270）、林燼V7重塑規則域立威（Ch.267）

**階段三：節點期（Ch.275-300）**
- 產能：16→20萬/週（50%穿規則層）
- 單位：新增節點護衛型（規則域防禦）+ 遠程探索型（長程巡航+自給30天）+ 探險者型（陳硯混合意識模板）
- 代價：探險者型需要陳硯分裂意識碎片 → 每次分裂他的27%殘留下降 → 自我認知模糊
- 戰果：聖城可同時支援內世界+邊界探索+外圍艦隊交易三條補給線；邊界探索隊抵達灰燼聯邦前哨
- 內世界代價：第一代穿規則層單位開始出現「規則疲勞」→ 工匠派必須研發第二代
- **爽點**：V10解鎖時蘇嵐看到因果線（Ch.282）、產能20萬/週震撼外圍艦隊（Ch.286）、祁闕翻轉（Ch.292）
