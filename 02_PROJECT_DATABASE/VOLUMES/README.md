## 與 STATE.md / FUTURE.md 的分工

- **VOLUMES/volNN.md**：該卷的**寫作藍圖**（章節節拍 + 重大轉折 + 伏筆埋點），通常凍結
- **STATE.md §3**：該卷**完稿後的狀態回顧** + 對 STATE.md §1 當前快照的貢獻
- **FUTURE.md §11–§13**：尚未寫的卷次的**長線大綱**

寫第 N 章時三檔互補：vol06.md 看「這章該推什麼」、STATE.md §1 看「上一章留了什麼鉤子」、FUTURE.md 看「這章不能洩漏哪些 V8+ 真相」。
