## 1. 整體基調

**核心 voice**：壓抑、冷、留白、帶機油味與血腥味。一句寫不出三件事就不寫第二句。

| 維度 | 應該長成的樣子 |
|---|---|
| 質地 | 金屬、粉塵、焊渣、舊線路加熱後的焦味 |
| 溫度 | 偏冷；高壓場景才出現體溫感 |
| 節奏 | 短句推動壓迫、中句交代畫面、長句只在情緒累積或世界觀震撼時 |
| 情感表現 | 動作 / 沉默 / 環境變化，不命名情緒 |
| 系統介面 | 高維文明介入，不是遊戲面板 |
| 結尾 | 未完成感，不是宣示句 |

**爽點曲線原則**：爽點前放慢、爽點發生時加快、反饋時停頓讓讀者感受餘波。

## 2. 視角法則

### 2.1 視角鎖定

- **每章選一個感知主體**：全章 90% 以上段落只能寫該角色能看、聽、聞、感、想到的東西。
- 全章只要答得出「這是誰的視角」一句話，視角就對了。
- 若必須換視角：用 `---` 分隔線明確切換；新視角須在開頭兩行內被識別。

### 2.2 避免「上帝手指」（全知句型禁區）

| ❌ 禁用句型 | 為什麼 |
|---|---|
| 「他不知道，此時的顧衡已經……」 | 替讀者代行全知 |
| 「這不是 X，是 Y」（替全隊下判斷） | 作者代替讀者總結 |
| 「這支隊伍像一架仍能飛、但每塊蒙皮都在響的破機」 | 整隊狀態總結 |

**改法**：把判斷權還給讀者，只寫角色知道的，不替讀者總結。

### 2.3 視角選擇軸線

| 軸線 | 適配視角 |
|---|---|
| 近距離衝突（戰鬥、談判、潛入） | 林燼 |
| 中距離支援（技術破解、監控、黑市） | 蘇嵐 / 唐雀 |
| 戰鬥／行動為主，林燼在另一條線 | 燕九 |
| 需保留主角內部判斷（製造懸念） | 蘇嵐 / 燕九 |
| 展示反派視角（每 5–8 章一次） | 顧衡 |

### 2.4 資訊不對稱寫法

- 每章設定時先畫**知道/不知道三角圖**：誰知道 X、誰不知道、誰只知道一半。
- **角色不該解釋對方已知的事**——對話用來暴露認知邊界，不是傳遞資訊。
- **保留不說本身就是張力**。每次角色開口前先想：他為什麼要在此刻說這個？沒有非說不可的理由就閉嘴。

### 2.5 對話技巧表

| 技巧 | 範例 |
|---|---|
| 省略已知資訊 | 「路由不對。」蘇嵐把面板推過來。她沒說哪裡不對——因為她還在確認。 |
| 用問題代替告知 | 「你看到公告了？」（而不是「白名單制發布了」） |
| 保留判斷只給事實 | 「探針信號經過三層跳轉，消失在民用網和中層政務網交界處。」（不說「代表中層權限」） |
| 沉默作為回應 | 林燼沒回答。但讓蘇嵐知道他沒想好，比讓她知道他在怕還危險。 |

## 3. 用詞與句式

### 3.1 段落

1. 每段不超過 3 行。
2. 高壓場景可用單句成段。
3. 關鍵台詞、系統提示、反派崩潰反應一律單獨成段。

### 3.2 句式節奏

- **短句堆疊**用於壓迫、爽點瞬間、反派錯愕。
- **中句**用於畫面交代、動作分鏡。
- **長句**僅在情緒累積或世界觀震撼時使用。

### 3.3 禁用詞與替代

| ❌ 禁用 | ✅ 替代 |
|---|---|
| 「總而言之」「值得注意的是」「不可否認」 | 直接刪掉 |
| 連續三個以上四字成語 | 拆成具體畫面 |
| 「很快」「幾分鐘後」「一週內」 | 展開成場景或用狀態變化暗示 |
| 括號標籤（如 [規則壓迫]） | 全部刪除 |
| 段落結尾總結句（「這場戰鬥讓所有人都明白了……」） | 改為動作或環境收尾 |
| 形容詞堆砌（「極其震撼的場景」） | 具體畫面 |
| 抽象詞 | 具體畫面（少用「他感到恐懼」，多寫「他的喉結滾了一下」） |

### 3.4 場景化規則

每一段正文至少包含以下其中一種元素：
人物動作 / 環境變化 / 感官描寫 / 系統異常 / 心理壓迫 / 台詞衝突 / 反派反應 / 旁觀者反應。

**禁止空泛總結**：
- ❌ 顧衡很震驚。
- ✅ 顧衡的手停在半空。咖啡杯裡的黑色液面晃了一圈，幾滴濺在他雪白的袖口上。他卻像是完全沒察覺，只死死盯著螢幕中央那行猩紅字樣。

## 4. 描寫質地

### 4.1 感知三層（環境 / 身體 / 時間）

每段至少覆蓋三層中的兩層：

1. **環境**：溫度、氣味、光線、聲音（管線低鳴、焊渣味、舊線路焦臭）。
2. **身體**：生理反應（延遲、酸、痛、麻、渴、義肢摩擦聲、神經延遲）。
3. **時間感**：倒數、等待、呼吸次數、權限同步剩餘時間。

### 4.2 情緒表現對照表

情緒**永遠**不直接命名，用身體反應替代。

| ❌ 情緒命名 | ✅ 感知寫作 |
|---|---|
| 蘇嵐眼底全是寒意 | 林燼看見她的食指在扳機護圈外側敲了一下 |
| 他心裡冷笑 | 他沒回話 |
| 林燼眼神一沉 | 他眨眼的頻率慢了一拍 |
| 蘇嵐語氣比金屬還冷 | 她說這話時，那截銀色義指在槍管上敲了兩下 |
| 顧衡很震驚 | 顧衡的手停在半空，咖啡杯裡的黑色液面晃了一圈 |
| 燕九壓不住怒火 | 燕九把彈匣拍進握把，那聲響比平時重了一倍 |

### 4.3 戰鬥質地

- **一動作一動，不壓縮**。戰鬥不是摘要（禁止「戰鬥只打了三分鐘」），是一連串有風險的選擇。
- 每一步都可能暴露——讀者跟著角色的判斷走，不是看戰報。
- 環境細節要在戰鬥中持續發揮作用（拼裝爐側面歪掉的散熱鰭片、踩在廢料堆陰影裡的義肢摩擦聲）。
- 高處哨兵轉頭那一刻，不要解釋「為什麼」——讓讀者自己拼出畫面。

### 4.4 主機 / 內部文明質地

- **意識下沉要有重量**。主機不是按鈕，是有景、有人、有代價的場景。
- 進入主機層：先寫荒地的火光、跪在火塘殘灰旁的人影、祭司高台的金屬光澤。
- 離開主機層：必定要付出可感知的代價（鏈路衰減、內部產能下降、暴露概率上升）。
- 內部文明段落不得只展示神性、信仰或世界奇觀；同章或下一章必須對外部主線產生可見回饋。

### 4.5 環境與制度質地

- 帶出冷酷制度名詞（加購稅、消除令、白名單）時，盡可能附帶 1–2 筆「民生鏡頭」或具象受害者。
- 環境異常先於原因——燈在凌晨三點同時滅了（不是斷電——備用照明還亮著——是有人關了主照明）。
- 制度公告呈現用「公告口吻」直接引用，第三段藏殺機（範例：白名單公告 → 「重新安置」）。

### 4.6 章節開場質地（不要從角色起床開始）

四種有效開場：

| 切入點 | 範例 |
|---|---|
| 對話中間 | 「你不應該在這裡。」蘇嵐的聲音比平時低半個音階。 |
| 身體狀態 | 林燼的左腿在痛。不是受傷的那種痛——是義肢接口的磨損警報。 |
| 環境異常 | 灰槽帶的燈，在凌晨三點同時滅了。 |
| 正在進行的動作 | 林燼把最後一枚螺絲鎖緊。戰車的引擎蓋啪一聲闔上。 |

開場自檢：
- 前 200 字是否有具體感官細節（光線/聲音/氣味/觸感/痛覺）？
- 前 500 字是否出現當前問題或目標？
- 是否直接承接上一章的鉤子？

### 4.7 章節結尾質地

結尾必須「未完成」，不是「結論」。詳細鉤子類型與強度評估見 `RULES.md` §8.3「章節鉤子公式」。本檔此處只規範**質地**：

- 結尾必須是動作 / 沉默 / 環境變化 / 身體反常 / 感官細節，不能是宣示句、口號、決心。
- 鉤子不能依賴角色說出結論——如果結尾最後一句是角色的判斷句，刪掉。
- 「但是還沒」對；「因為所以」錯。

## 6. 角色台詞風格速查

> 詳細各角色逐情境台詞表、肢體標記、跨卷演化軌跡：見 `CANON_characters.md` §1–§7。本節只放跨角色 voice 對照與「同情境差異」速查。

### 6.1 五人 voice 定位一句話

| 角色 | 定位 |
|---|---|
| 林燼 | 少、穩、帶壓迫。不解釋自己為何這麼做。 |
| 蘇嵐 | 沙啞、直擊要害、帶黑色幽默。先給數據再給結論；反常時代表她在壓情緒。 |
| 燕九 | 直接、粗獷、急性子。習慣以暴制暴。 |
| 顧衡 | 短、冷、傲慢。用行動展示權力。崩潰不是大喊大叫，是手停在半空。 |
| 唐雀 | 只說自己關心的數據缺口，多一個字都不說。 |

### 6.2 旁觀者台詞功能

用來放大主角威懾力。不要給旁觀者複雜內心；他們只負責「抬頭」「停手」「看一眼然後低頭」。

### 6.3 同情境差異速查（範例：剛打完一場硬仗，有人問「接下來呢？」）

| 角色 | 反應（卷數） |
|---|---|
| 林燼 V1–V2 | 不回答。低頭檢查裝備。 |
| 林燼 V5–V6 | 「先算代價，再算下一步。」 |
| 林燼 V10–V12 | 看向蘇嵐和燕九：「你們說呢？」 |
| 蘇嵐 V1–V2 | 「先補氧，再補情報，最後補人。」 |
| 蘇嵐 V7–V10 | 「異常信號還在。我的建議是——」手指在空中劃一條弧線（不說完） |
| 燕九 V1–V2 | 「先撤。活著才能打下一場。」 |
| 燕九 V11–V12 | 看向年輕士兵：「你覺得呢？」（傳承模式） |
| 唐雀 | 「名冊還差兩片。」 |

### 6.3 反派 voice 退化模板（顧衡為主）

| 階段 | 台詞特徵 | 範例 |
|---|---|---|
| 傲慢 | 短句命令，不解釋 | 「清理掉。」「就這？」 |
| 錯愕 | 疑問句 | 「這是什麼？」「等等⋯不⋯不對。」 |
| 確認 | 對自己說話、小聲、不可置信 | 「都是為了⋯穩定。」 |
| 恐懼 | 又回到短句，但是繃緊的短 | 「⋯撤。」「切斷連線。」 |
| 沉默 | 不發一語、出席會議全程沉默 | （無） |
| 臨終 | 確認 | 「⋯原來如此。」「你贏了。」 |

### 6.4 林燼演化軌跡（給 AI 對應卷別校準口吻）

V1–V2 極簡被動（≤5字）→ V3–V4 簡短命令壓迫 → V5–V6 冷靜賭徒平靜 → V7–V9 開始說「我不知道」→ V10–V12 從「我」到「我們」→ V13–V14 深沉平靜。

## 7. 範例段落（風格 DNA）

> 這些是「正文應該長成的樣子」——不是規則，是感覺。

### 7.1 開場：從感官切入

> 林燼是被腕環的震動吵醒的。
>
> 不是鬧鐘——腕環的鬧鐘是短促脈衝，像心臟多跳了一下。這是持續震動，低頻、固執，像有人隔著金屬牆壁敲同一根水管。
>
> 他睜開眼，光線刺得瞇起。
>
> 藏身點是外環廢運線底下的一間報廢調度室，天花板上的燈管只剩一根還亮，發出偏黃的疲憊光。空氣裡混合著金屬粉塵和舊線路加熱後的焦味，混雜著燕九傷口換下來的髒繃帶——那股帶鐵鏽味的血腥氣，蓋過了其他所有味道。

**技法**：不寫「第九段完成了全城風控同步」，寫角色被震醒 → 聞到血腥味鐵鏽味 → 身體的延遲感 → 看到腕環上的通知。

### 7.2 對話：角色不該解釋對方已知的事

> 「路由不對。」蘇嵐把面板推過來。
>
> 她沒說哪裡不對——因為她還在確認自己的判斷。林燼沒催她。她如果確定，會直接說結論。

### 7.3 戰鬥：一動作一動

> 林燼蹲在一面倒塌的混凝土牆後方，往外看了一眼。
>
> 直煉口比他想的粗糙。一台拼裝爐，管線外露，爐體側面用鐵絲綁著一組散熱鰭片，其中一片已經歪了。
>
> 高處哨塔還有一個——熱感應掃到他了嗎？
>
> 菸頭的火光沒有移動。
>
> 還沒被發現。
>
> 林燼做了一個手勢：無聲處理。
>
> 他跨過牆，踩在廢料堆的陰影裡。左腳落地時，膝蓋義肢又響了一聲。
>
> 很小。
>
> 但在廢區的寂靜裡，那聲音像有人把一枚硬幣掉在鐵皮上。
>
> 高處哨兵轉頭了。

### 7.4 主機：意識下沉要有重量

> 他閉上眼。
>
> 意識下沉。
>
> 內部文明的荒地，比上次來的時候多了幾簇火光。幾十個人影跪在火塘殘灰旁，有人正在把新劈的木柴堆進熔爐。祭司站在高台上，泛著金屬光澤的臉轉向天空——
>
> 看向他。
>
> 沒有對話。意念已經足夠。

### 7.5 反派錯愕：物理阻斷法

> 顧衡的手停在半空。
> 咖啡杯裡的黑色液面晃了一圈，幾滴濺在他雪白的袖口上。
> 他卻像是完全沒察覺，只死死盯著螢幕中央那行猩紅字樣。
>
> 【錯誤：執行者權限低於被清除者】
>
> 他的第一反應是指令錯了。
> 第二反應是系統出 bug 了。
> 第三反應——他沒有第三反應。因為第三反應需要的認知框架，他還沒有建起來。
