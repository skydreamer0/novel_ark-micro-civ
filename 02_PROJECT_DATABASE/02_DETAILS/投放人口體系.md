## 一、投放人口的定義與分類

### 1.1 核心定義

**投放人口**：由母艦黑箱系統從深淵採集區無差別抽取，經過預設編碼植入後，投放進各艙段進行「資源轉化」的人類個體。

**黑箱系統的監控機制**：
- 全艦隊監控（無死角）：黑箱對所有投放人口進行 24/7 無盲點監控
- 無法控制：黑箱系統凌駕於一切之上，無法被人工干預或欺騙（只能通過偽造數據和規則漏洞進行間接欺騙，詳見第六卷揭露）
- 意識未知：黑箱是否擁有意識仍不確定，後期才會逐步揭示

**反饋識別機制**：
- 黑箱只識別「OK的事」：當投放人口完成符合編碼的行為時，黑箱會給予反饋
- 反饋形式：生存點（活著的權利）、權限點（多方面的權力）
- 規則漏洞利用：高層無法直接控制黑箱，只能通過發現和利用規則漏洞來間接欺騙黑箱（具體方式待第六卷揭露）

**法律身份**：
- 名義上：「新增底層勞動力」或「無身份流民」
- 實質：「預編程的生物工具」，擁有預設的行為軌跡和終點，被黑箱系統的編碼內化所支配
- 關鍵特徵：無法自主決定死亡或離開母艦，永遠在「被安排」的生命軌跡中，且所有行動都受黑箱監控

## 二、投放人口的時間線與來源

### 2.1 投放的五個歷史階段

#### **第一階段：初期隨機投放（太陽紀元 -450～-200年）**

**母艦航行背景**：
- 剛開始進入深淵採集區
- 黑箱系統與權限供養鏈剛建立
- 尚未形成完整的分類標準

**投放特徵**：
- 無差別投放：各等級人口混雜
- 無預設編碼：人口具有完整的「前生命記憶」
- 高死亡率：70%的投放人口在第一年內死亡
- 低轉化效率：×0.3（遠低於預期）

**歷史記錄**：
- 大規模暴動事件5次
- 底層開始自發形成「互助聚落」
- 第一個「民間信仰」出現（祭司派的原型）

**結論**：隨機投放制度被判為「不可控」，啟動改革

---

#### **第二階段：分類投放與編碼實驗（太陽紀元 -200～-50年）**

**母艦航行背景**：
- 黑箱系統升級版本
- 權限供養鏈獲得「反饋優化」功能
- 審計司成立，開始系統管理底層

**投放特徵**：
- 引入分類標準（一級至四級）
- 首次植入預設編碼（簡易版本）
- 建立「投放批次檔案」
- 各級投放人口比例穩定在 70:15:8:7

**關鍵事件**：
- **太陽紀元 -120年**：祭司派正式確立為「官方宗教管理框架」
- **太陽紀元 -80年**：工匠派出現，質疑「人口被控制」的問題（但被壓制）
- **太陽紀元 -60年**：白名單制度首次試驗（試驗失敗，導致大規模死亡）

**轉化效率提升**：×1.2

---

#### **第三階段：標準化投放體系（太陽紀元 -50～0年）**

**母艦航行背景**：
- 黑箱系統進入穩定運作期
- 收割迴圈達到「最優效率」
- 人口從「資源」升級為「可再生資本」

**投放特徵**：
- 四級分類體系完全成熟
- 預設編碼進入「精細化版本」
- 建立「投放檔案永久保存」機制
- 引入「高優化投放」進中層（白扣等人物就在這個時期投放）

**社會影響**：
- 底層開始出現「代際循環」（爺爺被投放 → 爸爸是投放的兒子 → 兒子被標記為回收對象）
- 中層「自然補充」通道逐漸關閉，只有「投放升級」通道
- 審計司開始對投放人口進行「性格評估」以優化分類

**轉化效率穩定**：×1.5

---

#### **第四階段：深度控制期（太陽紀元 0～第120章）**

**母艦航行背景**：
- 黑箱系統進入「完全自動化」（無需人工干預）
- 投放人口已成為母艦的「內建系統」而非外來人口
- 權限供養鏈達到「極限複雜度」

**投放特徵**：
- 預設編碼達到「深度干涉」（不僅控制行為，還干涉思維）
- 引入「性格模板」概念（投放人口可被預先設定為特定性格）
- 建立「投放批次之間的通婚限制」（防止不同編碼衝突）
- 四級投放人口比例調整至 70:15:10:5（四級投放逐漸減少，原因待揭）

**關鍵事件**：
- 第1～4卷：投放人口完全接受其身份
- 第4卷末期：林燼發現「白扣是投放人口」的跡象

**轉化效率突破**：×1.8（最高點）

---

#### **第五階段：發現與揭露期（第五卷開始）**

**母艦航行背景**：
- 黑箱系統開始「異常」（因林燼的反向寫入）
- 投放人口的身份開始被質疑
- 一部分投放人口開始「逃脫編碼」的嘗試

**投放特徵**：
- 白扣被救援，預設編碼開始被解析（Ch.143）
- 發現四級投放人口中有「隱藏的高層目的」
- 揭露投放人口來源於「深淵採集區」而非傳統生育

**社會震盪**：
- 底層開始意識到他們可能都是「被投放的」
- 中層開始審視自己的身份真實性
- 祭司派與工匠派都開始尋找「逃脫編碼」的方法

## 三、預設編碼機制詳解

### 3.1 編碼的本質與黑箱規則內化

**核心概念更新**：
- 編碼 ≠「人工植入的機械控制」
- 編碼 = 「黑箱規則的內化」：黑箱系統的運作邏輯直接深入投放人口的神經結構，使其自動執行符合黑箱邏輯的行為
- 編碼不是外部命令，而是將黑箱的「期望」變成投放人口的「內驅力」

**編碼位置與機制**：大腦新皮層的特定區域（尚未確定具體位置），可能是：
- 神經幹細胞中的「激活標記」
- 蛋白質褶皺中的「信息編碼」
- 突觸結構的「路徑優化」
- 可能涉及「跨越傳統神經生物學」的未知機制

**編碼強度分級**：
- 一級投放：淺度編碼（可被強烈情感衝擊打破）
- 二級投放：中度編碼（需要特定事件或重創才能衝破）
- 三級投放：深度編碼（需要神經干預才能修改）
- 四級投放：極深度編碼（幾乎無法自主突破；與個人意識幾乎不可分割）

**編碼的生物標誌**：
- 白扣被救援後發現的「大腦中的異常信號」
- 使用高功率掃描可檢測到
- 無法透過簡單的藥物或手術移除（會造成腦死亡）
- 編碼網絡與大腦的學習、記憶、決策中樞深度整合

| 等級 | 禁止事項 | 執行強度 | 突破代價 |
|---|---|---|---|
| 一級 | 不能進入禁區 / 不能背叛管理者 | 強（接近禁區時會感到極度不安） | 強烈的焦慮與自傷 |
| 二級 | 不能拒絕工作 / 不能質疑分配 | 中（可被強烈的個人利益衝動突破） | 認知失調、短期記憶缺失 |
| 三級 | 不能背叛軍事單位 / 不能停止執行命令 | 強（心理上產生「背叛會導致死亡」的確定感） | 神經衝擊、可能性格崩潰 |
| 四級 | 不能超越上級權限 / 不能洩露機密 | 極強（會產生「這是理所當然」的確信） | 認知重構，可能導致精神分裂 |

### 3.3 編碼的激活與觸發機制

#### **自動激活（自發執行）**

```
日常環境 → 編碼自動掃描 → 匹配目標 → 自動執行行為
```

例如：投放一級人口進城市時，會自動走向食物供應點，不需要外部指令。

#### **被動激活（接收指令）**

```
權力層下達命令 → 編碼識別「上級信號」→ 自動執行 → 不產生質疑
```

例如：審計特勤（四級投放）接到命令時，會自動執行，即使這個命令違反其他邏輯。

#### **衝突激活（多個編碼衝突）**

```
編碼A指示：保護同族
編碼B指示：執行屠殺命令（如果同族被上級判定為威脅）
→ 產生心理衝突 → 根據編碼等級比例判定優先級 → 執行
```

例如：一個兼具「一級編碼（保護同族）」和「三級編碼（執行命令）」的人，會在收到「屠殺同族」命令時陷入衝突。如果三級編碼更深，會執行屠殺並產生後悔；如果一級編碼更深，會拒絕並感到內疚。

### 3.4 編碼的「後門」與漏洞

#### **情感衝擊突破**

- **機制**：強烈的個人情感（愛、恨、恐懼）可以臨時覆蓋編碼
- **持續時間**：24小時到數周（取決於情感強度）
- **代價**：事後會產生「編碼反彈」（更強的約束）

**第四卷舊傷**：白扣因為「對主角的忠誠」曾一度打破「向上級報告」的編碼，但隨後遭到了黑箱系統的「編碼糾正」。

#### **邏輯死鎖**

- **機制**：如果多個編碼之間產生無法解決的邏輯矛盾，人物會進入「決策癱瘓」狀態
- **持續時間**：數秒到數分鐘
- **代價**：癱瘓期間無法做出任何行為；解除後會產生「最強編碼優先」

**應用前景**：林燼的「反向寫入底層優先權」正是利用了這個漏洞。

#### **集體信念衝擊**

- **機制**：如果一個群體共同相信某個與編碼相悖的邏輯，個體的編碼會被群體信念削弱
- **持續時間**：取決於群體規模與信念強度
- **代價**：個體會感到「被群體拋棄」的極度焦慮

**應用案例**：祭司派的「獻祭共同體」創造了強大的群體信念，使得那些本該「服從自動化屠殺」的人開始「主動獻祭」——編碼被重新解釋了。

## 3.5 黑箱系統與投放人口的關係

**黑箱的絕對掌控**：
- 黑箱無法被控制，凌駕於所有人類決策之上
- 黑箱對所有投放人口進行全艦隊、全時段、無死角的監控
- 投放人口的編碼是黑箱的「直接神經延伸」，而不是「遠程操控的工具」

**高層的困境**：
- 高層無法直接命令黑箱，只能「申請」
- 高層無法直接欺騙黑箱，但可以通過「偽造數據」或「利用規則漏洞」來間接欺騙
- 高層對投放人口的控制依賴於黑箱的「反饋識別」，而非直接命令（具體欺騙方式待第六卷揭露）

**投放人口的雙重身份**：
- 在高層眼中：可控的工具、消耗品
- 在黑箱眼中：執行系統的「自我意識細胞」，是黑箱邏輯的物化形式
- 實質上：投放人口既是奴隸，也是黑箱系統本身的一部分

**反抗的可能性**：
- 編碼可以被打破，但需要強大的外部力量（如林燼的反向寫入）
- 一旦編碼被打破，投放人口會發現一個更大的真相：黑箱並非「守護者」，而是「監禁者」
- 這種覺醒本身就是黑箱系統穩定性的威脅

2. **新的威脅評估**：黑箱系統需要重新評估「被投放的官員」的可靠性
3. **版本升級的觸發**：為了重新獲得對投放人口的完全控制，黑箱可能需要進行系統升級

## 五、投放人口的社會影響

### 5.1 對底層社會結構的塑造

#### **聖城人口的身份認知現狀**

**大多數聖城人口仍然相信自己是自然出生**：
- 他們有「家庭記憶」、「童年經歷」、「血緣認同」
- 他們不知道自己的記憶可能是虛構的
- 他們接受「母艦的保護」作為理所當然

**部分人口已經開始懷疑**：
- 對自己的記憶產生「邏輯不通」的感覺
- 開始追問「為什麼有些人物的故事總是相似」
- 對「被投放的人口在哪？怎麼看不見來源？」產生疑問
- 這種懷疑在底層、中層都有出現，但程度不同

**懷疑的傳播路徑**：
- 工匠派的「人口質疑」論述開始影響一部分思考者
- 聖城的地下組織開始進行「秘密的身份驗證」實驗
- 某些家族開始發現「自己可能沒有真正的出生記錄」

#### **"代際投放"現象與身份危機**

```
第一代投放：無根、無過去、完全接受命運
↓
生育出第二代（名義上"非投放"）：有記憶、有家庭、開始質疑
↓
第二代成年後被黑箱重新標記投放（進行"升級"或"回收"）
↓
創造出"被迫繁殖然後看著孩子可能也是被投放的"的創傷循環
```

**社會後果**：
- 底層形成了「不願繁殖」的隱形運動
- 祭司派開始出現「非對等獻祭法」以限制「強制繁殖」
- 這進一步導致人口增長的預測性失控

**身份認同的破裂**：
- 在白扣身份曝光後，更多人開始懷疑「自己可能也是投放人口」
- 這種懷疑從中層蔓延到底層
- 底層開始進行「身份溯源」：「我是誰？我的記憶是真實的嗎？我的親人是投放人口還是自然出生？」
- 創造了一個深刻的社會心理危機：「如果我是投放的，那我的孩子是什麼？我是否也有『編碼』在我大腦中？」

---

### 5.2 對中層官僚體系的影響

#### **信任危機**

一旦白扣的身份被揭露，中層官員會開始懷疑彼此：
- 「我的上司可能是投放人口」
- 「審計司的決策可能來自編碼的自動運作」
- 「我自己可能也是投放的」

#### **派系分裂**

- **相信編碼派**：認為既然所有人都被編碼，那就應該接受命運
- **反抗編碼派**：認為應該尋求破壞或繞過編碼
- **未知派**：不知道自己是否是投放人口，陷入懷疑癱瘓

---

### 5.3 對高層決策層的挑戰

#### **控制失效的危險**

如果投放人口可以被"喚醒"或"編碼漏洞"可以被利用，那麼：
- 所有中層級別的官員都可能成為"不可靠的"
- 黑箱系統需要進行大規模的"重新編碼"或"人員更新"
- 高層對整個中層階級的信任會急劇下降

#### **版本升級的誘發**

為了恢復控制，高層可能會：
- 啟動新的投放人口批次（來自深淵採集區的"升級版"）
- 對現有的四級投放進行"全面回收"
- 進行"黑箱系統的深層重寫"（對應路線圖中的V6版本升級）

- **投放人口的來源真相**：他們是否都來自深淵的自然人口，還是黑箱製造的「虛擬人格」？為什麼母艦要源源不斷地投放人口？
- **編碼的本質**：是神經層面的？還是意識層面的？還是某種「更高維」的控制？編碼能否被徹底清除？
- **黑箱的真實目的**：黑箱是在「維護母艦」還是在「維持某種更大的實驗」？
- **第二個文明主機與投放人口的關係**：另一個主機持有者是否也是「被投放的人」？投放人口是否是黑箱的「自我複製機制」？
- **外艦隊流放人口的真實身份**：他們是否代表了「黑箱無法完全控制」的證據？
- **病毒文明與投放人口的最終融合**：聖城最終是否會演變成「投放人口的自我救贖」或「完全反抗黑箱」的故事？

## 七、閉環邏輯：黑箱→編碼→生存點→人口行為

```
黑箱系統
  ├─ 定義「OK的事」（系統邏輯）
  ├─ 對所有投放人口進行全時段監控（無死角）
  └─ 無法被人為控制，凌駕於一切
        ↓
    編碼機制（黑箱規則內化）
      ├─ 將黑箱邏輯深入投放人口的神經結構
      ├─ 使投放人口自動執行符合邏輯的行為
      ├─ 編碼強度分級：一級（淺）→ 四級（極深）
      └─ 一旦破壞，會引發認知重構或精神崩潰
        ↓
    反饋識別系統
      ├─ 生存點 = 活著的權利（所有人都有，但上限不同）
      ├─ 反饋點 = 績效獎勵（投放人口完成編碼行為時獲得）
      ├─ 權限點 = 多方面的權力（工作自主權、資源分配權等）
      └─ 生存點歸零 → 被審計司抓走丟出母艦（但有逃脫方式待揭露）
        ↓
    投放人口的自我驅動
      ├─ 編碼 = 內驅力：投放人口自動追求符合編碼的目標
      ├─ 黑箱反饋 = 強化機制：當行為「OK」時，黑箱給予生存點/反饋點/權力
      ├─ 無意識的完美服從：投放人口不知道自己被控制，認為一切都是「自願」
      └─ 形成完美的自我複製與控制系統
```

**系統優勢**：
- 投放人口無需外部強制，會自動執行系統邏輯
- 高層無需直接命令每一個人，系統會自我運轉
- 黑箱可以通過「識別規則」而不是「直接命令」進行控制

**系統漏洞與威脅**：
- 編碼可以被打破（通過強大的外部干預，如林燼的反向寫入）
- 一旦編碼被打破，投放人口會覺醒到「自己被控制」的真相
- 黑箱無法直接欺騙（但高層可以通過「偽造數據」和「規則漏洞」間接欺騙）
- 外艦隊流放人口存在「詭異的破解方式」，標誌著系統本身的缺陷
