# 規則霧與清霧技術

> 根據 2026-05-24 討論確立。
> 正式入庫版。

---

## 一、規則霧基礎設定

**規則霧**是深淵中瀰漫的極稀薄規則密度殘粒，濃度約 `0.01～0.5 r/m³`，本質是遠古文明崩潰後逸散出的規則碎片。深淵本身不是傳統太空，而是規則密度趨近於零的維度夾層，因此這些殘粒會在低規則環境中長期漂浮。

母艦航行時，艦殼維持高規則密度，會像靜電吸附灰塵一樣吸附規則霧。霧附著在艦殼表面後，逐漸累積成一層「規則垢」。

規則垢不是普通污垢，而是一層錯亂的微規則膜。它會干擾艦殼的規則密度調製，使航向穩定陣列與殼體調製器出現延遲、偏移、誤判，最後導致母艦或小型艦轉向失靈。

母艦的躍遷引擎本來就是靠規則密度梯度驅動，並透過局部調整殼體規則密度分布來改變航向；因此規則垢一旦干擾殼體調製，就會直接影響航行安全。

---

## 二、規則霧危害

規則霧本身濃度低時危害不明顯，但累積成規則垢後會造成四類問題。

第一，**轉向失靈**。規則垢會干擾殼體外層的規則密度分布，使母艦或小型艦無法準確改變航向。

第二，**燃料消耗增加**。目前 Ark 的主要處理方式是加速躍遷，靠高溫摩擦燒掉霧垢。但加速躍遷本身需要大量規則燃料，長期使用會加劇母艦資源壓力。

第三，**引來深淵生物**。規則垢會在艦體外形成混亂但可感知的規則殘渣層，對某些深淵生物而言類似食物、訊號或獵物標記。

第四，**外殼污染與航跡暴露**。規則垢太厚會危害航行，但完全清乾淨也不是好事。薄霧層在某種程度上能遮蔽艦體高規則密度訊號，一旦清得過亮，反而可能讓艦體在深淵中更醒目。

舊航民對此有一句常用語：

> **「殼不能亮，亮了就有人看見。」**

---

## 三、現有官方解法：加速躍遷燒垢

Ark 官方目前最常用的清垢方式是**加速躍遷燒垢**。

做法是短時間提升航行速度，讓艦殼外層的規則垢在劇烈規則摩擦中被燒掉。這種方式粗暴、直接、可靠，但燃料成本很高。

推進設定中，母艦躍遷模式本來就分為巡航、加速躍遷、深層躍遷與緊急跳躍，其中加速躍遷一次約需 200 單位規則燃料，深層躍遷與緊急跳躍消耗更高。

因此，加速燒垢只適合 Ark 這種大型母艦偶爾使用，不適合外圍艦隊、小型先遣艦、深淵移民艦長期依賴。

## 四、清霧技術總定義

**清霧技術**不是消滅深淵中的規則霧，而是處理已附著在艦殼上的規則垢。

正式名稱可定為：

> **剝律清垢技術**

核心定義：

> 透過逆相規則脈衝、外殼刮霧裝置、生物啃食、共振節拍等方式，將附著於艦殼表面的規則垢剝離、削薄、導流、封存或低效回收，以恢復殼體規則密度調製能力，降低加速躍遷燒垢的燃料消耗。

設定邊界：

> 清霧技術不解釋規則霧的終極來源。
> 規則霧真正來源、異常霧垢本質、深層霧訊號，全部保留到 V14。

## 五、清霧技術三代體系

### 第一代：加速躍遷燒垢

**使用者：** Ark 官方、高層航路調度中心。

**原理：**
透過加速躍遷讓艦殼外層產生高強度規則摩擦，把霧垢燒掉。

**優點：**
- 技術成熟。
- 執行簡單。
- 對大型母艦有效。
- 不依賴外部生物或人工外殼作業。

**缺點：**
- 規則燃料消耗高。
- 不適合頻繁使用。
- 小型艦燃料儲備不足，難以承受。
- 只能粗暴清除，無法精準保留安全灰膜。
- 容易暴露航跡。

**敘事定位：**
這是舊文明的高耗能解法，象徵 Ark 用燃料與權限硬穿深淵。

---

### 第二代：刮霧艇／逆相脈衝清垢

**使用者：** 外圍艦隊、清障艦、部分先遣艦殘存者。

**原理：**
小型清障艇貼近艦殼，用刮霧臂、逆相脈衝釘、局部規則場震動，把霧垢從艦殼上剝下來。

流程：

```text
偵測霧垢厚度
→ 小艇貼近艦殼
→ 逆相脈衝鬆動霧垢
→ 刮霧臂剝離表層
→ 霧垢收入隔離罐或拋棄
```

**優點：**
- 比加速躍遷省燃料。
- 可以局部清理。
- 適合外圍艦、小型艦、受損艦。
- 不需要母艦完整高權限。

**缺點：**
- 極度危險。
- 作業者需要貼近外殼。
- 清下來的霧垢會形成霧尾，可能引來深淵生物。
- 清障員死亡率高。
- 無法大規模穩定覆蓋 Ark 全艦。

**相關職業：**

> **刮殼人**
> 外圍艦隊或先遣艦後代中的危險職業，專門在艦殼外清霧、補殼、回收霧垢罐。

**敘事定位：**
這是外圍艦隊與舊航民在資源不足下發展出的生存手藝。

---

### 第三代：霧食螅馴化清垢法

**使用者：** 舊航民、深淵移民、後期聖城。

**核心概念：**
深淵中存在一類小型群落生物，會啃食規則霧與規則垢。舊航民長期漂流時發現，部分艦殼如果附著這類生物，反而不容易積垢失控。於是他們逐漸發展出馴化、餵養、引導、回收這類生物的清霧方法。

這種生物正式設定為：

> **霧食螅**

舊航民俗稱：

> **灰螅、吃灰的、殼絨、霧毛**

聖城整理後名稱：

> **霧食螅馴化清垢法**

Ark 官方稱呼：

> **非授權深淵生物附殼行為**

---

## 六、霧食螅設定

### 1. 基礎定義

**霧食螅**是深淵表層常見的微型群落生物，通常附著於高規則密度物體表面，以低密度游離規則殘粒與規則垢為食。

它們肉眼看起來像灰白色絨毛、薄膜或潮濕的灰斑。大量聚集時，會像一層緩慢流動的灰色苔衣覆蓋在艦殼外。

### 2. 生物特性

霧食螅有四個核心特性。

第一，**吃霧垢，不吃艦殼**。
正常馴化狀態下，霧食螅只啃食低密度、游離態的規則垢，不會攻擊穩定艦殼規則鍍膜。

第二，**群體同步**。
螅群不是單體行動，而是整片灰膜一起移動，可以被特定低頻規則節拍引導。

第三，**過飢會失控**。
如果霧垢不足、誘餌不足，或清霧過度，螅群可能開始啃咬艦殼外層的規則鍍膜。

第四，**會被異常霧垢污染**。
普通霧垢可食，但某些異常霧垢會讓霧食螅變異。變異後的螅群可能啃殼、自燃、狂奔、形成假航跡，甚至發出吸引深淵生物的訊號。

## 七、舊航民的「餵殼」技術

舊航民不稱其為工程技術，而稱為：

> **餵殼**

含義是：不是清理艦殼，而是讓艦殼「吃一層活灰」，用霧食螅把危險霧垢吃到安全厚度。

完整流程：

```text
捕捉野生霧食螅
→ 用規則垢殘渣餵養
→ 篩掉會啃殼的危險個體
→ 培養溫馴螅群
→ 清霧時釋放到艦殼外
→ 用低頻節拍引導螅群移動
→ 保留安全灰膜
→ 用誘餌垢罐把螅群收回
→ 封存、分株、休眠或再次餵養
```

這套技術不能單靠圖紙學會，真正核心是經驗。

舊航民知道：
- 哪種霧食螅能養。
- 哪種灰色代表溫馴。
- 哪種白斑代表過飢。
- 哪種黑線代表吃到異常霧垢。
- 清到什麼厚度必須停。
- 什麼霧紋不能讓螅群靠近。
- 如何用誘餌垢罐把螅群召回。
- 螅群收不回來時該燒哪一段，保哪一段。

這就是移民知識轉移的價值。

## 八、舊航民設定

### 1. 身份來源

**舊航民**是早期失聯先遣艦隊、漂流小艦、深淵殖民點與外圍艦隊逃亡者的後代。他們不是完整艦隊文明，而是長期在深淵中靠小艦、破損艙、外殼維修艙、畸變區庇護點求生的漂流社群。

先遣艦隊本來就是主宇宙早期派出調查深淵、封鎖污染、修補規則漏洞的艦隊，後來因深淵吞噬太快而被困在外部；其中有些漂流求生，有些建立小型殖民區，有些與艦體、AI 或深淵素材半融合。

### 2. 社會特徵

舊航民世代生活在艦殼、外層維修艙、半廢棄小艦中。

他們的文化特徵：
- 不信任大型母艦。
- 不相信 Ark 的「聖航」敘事。
- 將深淵視為危險環境，而非純粹敵人。
- 兒童從小學習辨識霧紋。
- 成年禮是第一次外殼清霧。
- 每個家族都保存一罐自己的霧食螅株。
- 他們不說「清霧」，而說「餵殼」。
- 他們不說「工程師」，而說「看霧的人」。

### 3. 對 Ark 的態度

舊航民知道 Ark 不是單純救世方舟，而是會吞噬遇到的一切資源的文明級捕食者。這與深淵遺族設定中 Ark 的外部形象一致：Ark 會把遺族、殘民、先遣艦、深淵原生種、漂流艦等轉化為燃料、零件、情報、兵力或樣本。

因此舊航民不願把霧食螅技術交給 Ark。他們相信一旦高層掌握這技術，就會把養螅民變成耗材，把霧食螅工業化，甚至武器化。

## 九、為什麼 Ark 沒有普及霧食螅清霧？

這點需要明確，避免設定漏洞。

1. **Ark 規模過大**：霧食螅適合小艦、局部艦殼、清障艇，不適合直接覆蓋四百公里母艦。大規模投放會造成外殼生物污染。

2. **Ark 不信任活體技術**：Ark 的黑箱與高層偏好可審計、可控制、可編碼、可量化的技術。霧食螅會繁殖、突變、逃逸，不符合官方控制邏輯。

3. **霧食螅會干擾監控**：螅群附著外殼時會形成微弱生物規則噪音，遮蔽局部外殼讀數。對高層而言這是風險，對舊航民而言卻是隱蔽優勢。

4. **舊航民刻意保密**：霧食螅不是設備，而是族群知識。舊航民寧可失去艦，也不願把完整馴化口訣交給 Ark。

5. **高層有更粗暴的燃料解法**：只要燃料還夠，高層更願意加速燒垢，而不是冒生物污染風險。

## 十、聖城如何取得清霧技術

主角方不是撿到圖紙，而是透過**移民、庇護、互信、共同改良**取得。

### 第一階段：接納舊航民移民

V6 左右，主角遇到一批被 Ark 採集艦或清障艦追捕的舊航民。他們可能來自失聯先遣艦殖民點、漂流清霧艇群，或被母艦標記為可回收資源的小型聚落。

事件流程：

```text
舊航民聚落被 Ark 外圍艦隊盯上
→ Ark 判定其為高價值回收目標
→ 主角／聖城介入救下一部分人
→ 舊航民被迫進入聖城庇護區
→ 他們帶來灰罐與霧食螅株
```

舊航民一開始不信任聖城，因為在他們看來，所有大型文明最後都會吃掉小文明。

### 第二階段：灰罐誤會

舊航民帶入聖城的「灰罐」一開始被認為是污染物或深淵生物樣本，差點被銷毀。

後來聖城發現，灰罐中保存的是休眠霧食螅株。

這一幕可以用來建立衝突：

> 聖城想消毒。
> 舊航民拼死阻止。
> 雙方第一次爆發信任危機。

### 第三階段：第一次救命

聖城小型規則艇、外部作戰艙或反熵登陸艙第一次出現規則垢過厚，轉向失靈。

官方方案是加速躍遷燒垢，但燃料不足，或環境不允許。

舊航民提出：

> **「不要燒，餵牠們。」**

他們釋放少量霧食螅，清掉局部霧垢，使小艇恢復轉向。

這是聖城第一次承認：舊航民不是難民，而是深淵航行知識的持有者。

### 第四階段：聖城理論化

聖城審計軌與後勤軌開始研究霧食螅行為，發現螅群能被低頻規則節拍引導。

聖城將舊航民經驗與自身共振井技術融合：

```text
舊航民霧紋判讀
+ 霧食螅馴化
+ 聖城共振節拍
+ 審計軌異常判讀
= 可控生物清霧系統
```

主角的內部文明本來就有共振井與信仰回流機制，後續 V3 聖城時代也會進入軍團、後勤、審計三軌治理；這讓聖城有能力把舊航民的口傳手藝整理成制度化技術。

### 第五階段：共同改良

舊航民教聖城「怎麼養、怎麼看、怎麼停」。
聖城教舊航民「怎麼控、怎麼隔離、怎麼降低死亡率」。
深淵遺族或其他移民提供抗污染飼料、穩定殼材、生物隔離材料。

最後形成聖城版本：

> **信仰共振控螅清霧系統**

## 十一、聖城版清霧系統

正式名稱：

> **信仰共振控螅清霧系統**

簡稱：

> **控螅清霧**

核心組成：

```text
霧食螅吃垢
+ 舊航民霧紋判讀
+ 共振井節拍控群
+ 刮霧艇回收螅群
+ 審計軌判讀異常霧垢
+ 後勤軌養殖與隔離
= 聖城清霧系統
```

### 職能分工

| 角色 | 職責 |
|------|------|
| 舊航民 | 霧紋判讀、螅群馴養、外殼作業、危險預警 |
| 聖城後勤軌 | 螅群繁殖、休眠罐、誘餌垢、隔離艙、飼料配比 |
| 聖城審計軌 | 判斷哪些霧垢能吃，哪些霧垢不能碰；監控螅群是否變異 |
| 聖城軍團軌 | 清霧作業時護航、回收失控螅群、處理深淵生物襲擊 |
| 主角／文明主機 | 提供共振節拍、H-Compute 支援、局部規則覆寫、危機時中斷清霧程序 |

## 十二、清霧副產物：規則垢殘渣

霧食螅吃掉規則垢後，會產生三種副產物。

### 1. 灰渣

普通排泄物，低純度規則殘渣。

可低效轉化為規則燃料，但效率很低，約 `5%～12%`。不能用來取代外域規則碎片，只能作為補助燃料或誘餌材料。

### 2. 誘餌垢

經過處理的規則垢殘渣，可用來引導螅群返回，或製成誘霧彈吸引深淵生物。

戰術用途：
- 偽造航跡。
- 引開追擊者。
- 誘導深淵生物攻擊敵方。
- 清霧後收回螅群。

### 3. 異常垢核

少數霧垢中會凝出難以解析的異常節點。

> 設定邊界：不解釋來源。不說它是否與母巢、病毒、終極深淵有關。只標記為「不可餵食、不可轉化、不可長期保存」。

異常垢核可以作為 V14 伏筆。

## 十三、技術限制與風險

### 1. 不能清得太乾淨

霧食螅只能把霧垢吃到安全厚度，必須保留一層薄灰膜。

清太乾淨會造成：
- 艦體高規則密度訊號暴露。
- 深淵生物更容易感知。
- 航跡更明顯。
- 艦殼直接承受深淵污染。

### 2. 不能餵異常霧垢

異常霧垢會讓螅群變異。

變異表現：
- 螅群顏色轉黑或透明。
- 行動不再聽節拍。
- 開始啃咬艦殼。
- 形成假航跡。
- 發出吸引深淵生物的規則震盪。
- 在艦殼縫隙中繁殖失控。

### 3. 螅群需要回收

清霧後必須用誘餌垢罐收回螅群。若殘留在艦殼縫隙中，短期可能繼續清垢，長期會形成生物污染。

可能後果：
- 感測器延遲。
- 局部艦殼讀數異常。
- 維修員感染。
- 艙壁內側出現灰絨。
- 黑箱把區域標記為深淵污染。

### 4. 清霧會產生規則聲音

這不是物理聲音，而是規則層震盪。深淵生物可能感知到。

所以清霧不是安全維修，而是一次高風險外殼行動。

### 5. 需要活飼料

霧食螅要靠規則垢殘渣維持活性。長期沒有霧垢吃，就會休眠、互食或退化。

因此聖城不能無限養殖，必須維持小規模種群。

## 十六、正式入庫版摘要

```text
規則霧是深淵中極稀薄的規則密度殘粒，航行時會附著艦殼並累積成規則垢，干擾艦殼規則密度調製，造成轉向失靈並吸引深淵生物。Ark 官方目前主要依靠加速躍遷燒垢，但燃料消耗過高。

清霧技術正式名為「剝律清垢技術」，其目的不是消滅深淵中的規則霧，而是處理已附著在艦殼上的規則垢。清霧技術分為三代：第一代為 Ark 官方的加速躍遷燒垢；第二代為外圍艦隊與先遣艦殘存者使用的刮霧艇／逆相脈衝清垢；第三代為舊航民掌握的霧食螅馴化清垢法。

霧食螅是一種深淵微型群落生物，能啃食艦殼表面的規則垢。舊航民稱其馴化與使用方式為「餵殼」。他們透過捕捉、餵養、篩選、釋放、引導與回收螅群，使艦殼保持在不影響轉向但仍具遮蔽效果的安全灰膜狀態。

聖城在 V6 接納一批舊航民移民後，透過庇護、醫療、自治與共同改良取得此技術。舊航民提供霧紋判讀與霧食螅馴化經驗，聖城以共振井、後勤軌與審計軌將其理論化，形成「信仰共振控螅清霧系統」。該系統後期成為子文明艦隊脫離 Ark、在深淵中獨立航行的前置技術。

限制：霧食螅不能餵食異常霧垢，不能清得太乾淨，清霧後必須回收螅群，否則可能造成艦殼生物污染、感測延遲、深淵生物追蹤或螅群變異。規則霧終極來源、異常霧垢本質與霧食螅深層生態意義保留至 V14 揭露。
```
