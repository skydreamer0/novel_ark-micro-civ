# 母艦推進系統與深淵環境

> 補充細節：躍遷引擎原理、深淵環境物理性質、導航系統、裂縫生成與投放機制。
> 初稿，待打磨。

## 一、躍遷引擎

### 1.1 核心原理：規則密度梯度驅動

母艦的躍遷引擎並非傳統推進器——它不產生推力，而是利用**規則密度梯度差**讓母艦在深淵中「滑行」。原理如下：

```
高規則密度區（母艦周圍）→ 低規則密度區（深淵深處）
                   ↓
        母艦被「推」向低密度方向
                   ↓
       就如同氣球在高壓氣體中向低壓區移動
```

- 母艦自身維持一個高規則密度氣泡（殼體規則密度約 1000 r/m³）。
- 深淵環境的規則密度極低（通常 < 50 r/m³，深處趨近於 0）。
- 梯度差產生的「規則壓力」推動母艦前進。
- 方向控制：透過局部調整殼體規則密度分布，改變壓力梯度方向。

### 1.2 躍遷引擎的物理實現

| 組件 | 位置 | 功能 |
|------|------|------|
| 主反應爐 | 內核區核心 | 產生維持全艦規則密度的基礎能量 |
| 規則密度調製器 | 龍骨全長分布 | 控制殼體各部位的規則密度，製造梯度方向 |
| 躍遷噴口（×4） | 艦尾四角 | 規則密度洩壓口——在低密度區「噴出」高密度規則燃料產生反推 |
| 航向穩定陣列 | 艦體外殼 | 16 組規則密度天線陣，微調局部梯度向量 |

**躍遷噴口的工作機制**：
- 平時：維持艦內循環，不向外噴射。
- 躍遷時：將過剩的規則燃料從噴口定向釋放，在艦尾形成高密度區，推著母艦向低密度方向前進。
- 這意味著：**每次躍遷都在消耗規則燃料儲備**。躍遷越頻繁、距離越長，燃料消耗越大。

### 1.3 躍遷速度與模式

| 模式 | 速度 | 燃料消耗 | 持續時間 | 用途 |
|------|------|----------|----------|------|
| 巡航 | ~0.3 c（相對深淵介質） | 低（5 單位/天） | 無限（補給充足時） | 常規航行 |
| 加速躍遷 | ~3 c | 中（200 單位/次） | 最多 4 小時 | 短距離移動、躲避威脅 |
| 深層躍遷 | ~30 c | 高（5,000 單位/次） | 約 15 分鐘 | 跨域長距離跳躍 |
| 緊急跳躍 | ~100 c | 極高（20,000 單位/次） | 約 5 分鐘 | 逃生——可能導致結構受損 |

> **注意**：加速躍遷和深層躍遷必須在規則密度穩定區域進行。在低規則密度區（< 30 r/m³）進行深層躍遷，可能導致殼體撕裂。

### 1.4 航路調度中心

位於中層的核心導航設施，負責：

- **規劃躍遷路徑**：計算各區域的規則密度梯度分布，規劃最節省燃料的航行路線。
- **投放時窗管理**：當母艦靠近裂縫密集區時，協調各艙段的投放作業時間。
- **避障導航**：監測深淵中的規則密度異常點（可能的威脅或障礙物），提前調整航向。
- **結構完整性監控**：每次躍遷後對艦體進行全結構掃描，確認無規則密度疲勞裂痕。

航路調度中心的名義隸屬於技術行會，但實際運作受審計司與董事會直接控制。其內部設有**三級授權制度**：
- 一級：日常巡航路徑規劃（中層操作員可執行）
- 二級：躍遷模式切換與投放時窗（需技術行會主管簽核）
- 三級：緊急跳躍與航向重大變更（需審計司或董事會直接批准）

## 二、深淵環境

### 2.1 深淵的物理性質

深淵不是傳統意義上的「太空」——它是規則密度趨近於零的維度夾層。

| 性質 | 數值 / 描述 |
|------|------------|
| 規則密度 | 10~50 r/m³（表層）；< 5 r/m³（深層） |
| 溫度 | 接近絕對零度，但母艦殼體維持約 15°C 恆溫 |
| 介質 | 非真空——充滿極稀薄的「規則霧」（殘留規則密度微粒） |
| 可見光 | 無——深淵本身不發光。母艦光學傳感器視野為純黑 |
| 時間流速 | 與母艦內部一致（無顯著時間膨脹） |
| 空間拓撲 | 非歐幾里得——距離概念模糊，「近」和「遠」由規則密度梯度決定而非物理距離 |
| 重力 | 無外部重力場。母艦內部重力由旋轉艙段與人工重力系統提供 |

**規則霧**：深淵中瀰漫著極稀薄的規則密度殘粒，濃度約 0.01~0.5 r/m³。這些殘粒是遠古文明崩潰後逸散的規則碎片。當母艦航行時，規則霧會附著在殼體表面，隨著時間積累成「規則垢」——需要用規則燃料定期清洗，否則會干擾殼體的規則密度調製。

### 2.2 深淵的分層結構

從母艦的視角，深淵可分為三層：

```
表層（0~10,000 km 範圍）
  │  規則密度：30~50 r/m³
  │  特徵：裂縫密集區。母艦在此層進行資源採集。
  │  威脅：低～中（裂縫不穩定、熵行者滲漏）
  ▼
中層（10,000~100,000 km 範圍）
  │  規則密度：5~30 r/m³
  │  特徵：規則霧濃度降低，裂縫稀少。
  │  威脅：中～高（空間畸變、導航偏移）
  ▼
深層（> 100,000 km 範圍）
    規則密度：< 5 r/m³，趨近於零
    特徵：絕對黑暗，規則密度極低。據說深淵母巢的本體在此層。
    威脅：極高（規則崩解、母巢接觸風險）
```

母艦從不主動進入深層。所有已知的深層資訊來自遠古航行紀錄與第二載體（0xF1_0xXE_0_0xLL）的廣播。

### 2.3 深淵中的威脅

除了熵潮，深淵中還存在以下已知威脅：

#### 規則風暴
- 深淵中規則密度分布的突發劇烈波動。
- 表徵：母艦殼體規則讀數在數分鐘內波動 ±200%，所有導航儀器失效。
- 持續時間：數小時至數天不等。
- 應對：關閉非必要系統，啟動殼體穩定陣列，等待風暴過去。
- 代價：每次規則風暴會消耗 1,000~3,000 單位規則燃料儲備。

#### 畸變區
- 深淵中規則密度異常高的「斑點」，通常直徑數百至數千公里。
- 成因不明——可能是遠古文明廢墟，也可能是母巢的規則排遺。
- 靠近畸變區時，母艦內部會出現短暫的物理常數漂移（重力波動 ±5%、電子設備邏輯錯誤）。
- 已知的高價值資源點（裂縫密集），但也是高風險區。

#### 深淵生物（未確認）
- 第二載體的廣播中暗示深淵中存在「不是我們理解的生物」。
- 母艦的遠古航行紀錄中曾記錄「殼體撞擊事件」——有東西在深淵中撞擊了母艦外殼，撞擊點留下規則密度異常烙印，至今未消退。
- 是否存在主動獵食者？當前設定未定論。

#### 規則真空泡
- 深淵中規則密度完全歸零的球形區域，直徑從數公尺到數公里不等。
- 若母艦誤入規則真空泡，殼體規則密度會迅速向外擴散，導致艦內各艙段規則密度急降——相當於艦體在「漏氣」。
- 探測方式：航路調度中心透過監測規則霧流動方向來預判真空泡位置。

### 2.4 裂縫生成與投放機制

裂縫是深淵表層最常見的規則密度異常現象，也是母艦的主要資源來源。

#### 生成機制

```
熵潮侵蝕時空 → 高規則密度（母艦殼體）與低規則密度（深淵）之間形成急遽梯度
         ↓
梯度差在特定共振頻率下導致時空撕裂 → 裂縫形成
         ↓
裂縫內部暴露高規則密度殘留區（200%~800% r/m³）→ 資源可開採
         ↓
裂縫維持數分鐘至數小時後自行閉合 → 等待下一次生成
```

#### 裂縫類型

| 類型 | 規則密度 | 維持時間 | 資源類型 | 風險 |
|------|----------|----------|----------|------|
| 微型裂縫 | 200%~300% | 5~15 分鐘 | 能量晶核（少量） | 低 |
| 標準裂縫 | 300%~500% | 30 分鐘~2 小時 | 規則碎片、能量晶核 | 中 |
| 大型裂縫 | 500%~700% | 2~8 小時 | 高純度規則碎片、活性樣本 | 高 |
| 畸變裂縫 | 800%+ | 不固定（極不穩定） | 未知（可能有遠古文明遺物） | 極高—不建議進入 |

#### 投放帶與投放時窗

母艦會調整航向使特定艙段靠近裂縫密集區，這個區域稱為「投放帶」。

投放流程：
1. **航路調度中心預測**：根據規則霧流動方向與歷史數據，預測 24~72 小時內的裂縫生成位置。
2. **投放時窗分配**：各隊伍根據合約等級分配到不同的投放時間窗口。
3. **隊伍部署**：隊伍從指定艙段（通常是第十三段冷凝層附近的投放口）進入外域。
4. **採集作業**：隊伍必須在裂縫維持時間內完成採集並返回，否則裂縫閉合後被困在深淵中。
5. **回收**：返回隊伍經公轉化區進行樣本登記與污染讀數。

投放時窗由航路調度中心集中管理，企業與承包軍可購買優先時窗，底層隊伍則分配到剩餘窗口——通常是裂縫快閉合時的低收益時段。

## 三、導航系統

### 3.1 常規導航

母艦在深淵中導航依賴以下系統：

| 系統 | 原理 | 精度 | 限制 |
|------|------|------|------|
| 規則密度測繪儀 | 掃描周圍規則密度分布，建構三維梯度地圖 | 中等 | 受規則風暴干擾 |
| 壓差節點定位法 | 利用各艙段氧壓差異推算回母艦的相對位置 | 低—中 | 僅在母艦附近有效 |
| 慣性導航系統 | 追蹤母艦躍遷歷史推算位置 | 低（累積誤差大） | 長時間航行後需校準 |
| 信仰錨點（主角） | 主機投放信仰信標作為固定座標 | 高（24 小時內） | 僅限主角已知區域 |
| 深層信標（理論） | 母巢巨網中的節點信號，可作為絕對座標參考 | 未知 | 需接入規則層裸結構網才能接收 |

### 3.2 深淵中的位置概念

在深淵中，傳統的 XYZ 三維座標系無效。位置由以下參數定義：

```
位置 = f(規則密度梯度向量, 規則霧流向, 距離最近畸變區距離, 距離母艦距離)
```

這意味著：
- 同一個「位置」在不同時間可能有不同座標（因為規則霧流向會變）。
- 距離感模糊——A 到 B 的躍遷耗時可能因規則密度梯度變化而大幅波動。
- **迷航是真正的威脅**：若導航系統全面失效，母艦可能永遠找不到回已知區域的路。

### 3.3 盲舵會與舊航道圖

盲舵會掌握的舊航道圖是母艦上最可靠的導航資料之一。
- 來源：數百年來累積的壓差節點觀測記錄與艙外維修日誌。
- 優勢：不依賴電子導航——純粹靠「感知」規則密度變化的經驗導航。
- 限制：有效期短（底層艙段拓撲偏移頻繁），且盲舵會對核心資料嚴格保密。

## 四、深淵與深淵母巢的關係

### 4.1 當前認知（V5-V6）

主角目前所知：
- 深淵是一個規則密度極低的維度夾層。
- 母艦在此航行，靠規則密度梯度驅動。
- 裂縫是資源來源，也是威脅入口。
- 深淵中存在「巨網」——母巢的規則網絡，範圍遠超母艦尺度。

### 4.2 真相（V8+逐步揭露）

深淵不是自然存在的空間——
- 它是原初病毒文明崩潰後，其規則守護系統（深淵母巢）的「代謝廢棄物」累積形成的。
- 母巢存在於深淵深層，其規則網絡延伸至表層，形成裂縫與畸變區。
- 母艦航行於深淵中，本質上是在母巢的「體表」移動。
- 每一條裂縫，都是母巢的「毛細孔」——規則密度透過這些孔洞向外擴散，同時也是母巢感知外界的觸角。

> **終極隱喻**：深淵不是海洋，母巢不是海洋中的島嶼。深淵是皮膚，母巢是身體。母艦是一隻在皮膚表面爬行的寄生蟲。

## 五、禁忌（不可違反的設定邊界）

1. **不要讓母艦在深淵中「停下來」**——停下來意味著規則密度梯度消失，母艦會被困在該區域難以重新啟動躍遷。
2. **不要讓深淵生物在 V7 前正式現身**——可以暗示、可以撞擊、但不能看見或接觸。
3. **不要讓主角獲得完整的深淵地圖**——深淵的地圖永遠是局部、暫態、不可靠的。
4. **不要解釋規則霧的終極來源**——留給 V14 的終局真相。
5. **不要讓深層躍遷成為常規手段**——每次深層躍遷都應該帶來明顯代價。
