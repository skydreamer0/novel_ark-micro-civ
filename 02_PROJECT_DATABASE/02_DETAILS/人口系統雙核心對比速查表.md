## 核心對立四大柱

### 柱1：製造源頭

```
投放人口：
  來源 → 深淵採集區
  過程 → 無差別採集 + 預設編碼植入
  控制 → 黑箱（神經層面內化）

製造人口：
  來源 → 聖城自主生成（複製/孕育/融合）
  過程 → 有目標製造 + 思想教育
  控制 → 聖城（理念層面認同）
```

### 柱2：監控能力

```
投放人口：
  監控方式 → 全艦隊、全時段、無死角
  監控強度 → 深度（編碼 = 神經層面的實時監控）
  監控漏洞 → 編碼可被「強大的外部干預」破壞，但會被黑箱實時發現

製造人口：
  監控方式 → 屏蔽區內無法監控
  監控強度 → 零（黑箱看不到）
  監控漏洞 → 屏蔽一旦破裂，完全暴露；但在屏蔽內完全安全
```

### 柱3：忠誠度維持

```
投放人口：
  機制 → 編碼內化 + 反饋點系統
  成本 → 零（編碼自動運行）
  穩定性 → 極高（除非編碼破壞）

製造人口：
  機制 → 理念教育 + 共同使命感
  成本 → 持續教育投入
  穩定性 → 中等（信仰破裂時會快速倒戈）
```

### 柱4：戰術靈活性

```
投放人口：
  優勢 → 高度紀律性、低失誤率、執行力強
  劣勢 → 編碼限制創意、無法進行「出格」的戰術
  特徵 → 「完美的棋子」

製造人口：
  優勢 → 高度自主性、創意戰術強、適應多環境
  劣勢 → 紀律性不足、會出現「臨陣脫逃」、新手傷亡高
  特徵 → 「自由的戰士」
```

| Day 1-2 | 部署第一波人口 | 標記為「未授權異常體」 | 開始「懷疑」自己身份 | 「黑箱的控制可被打破」的驚喜 |
| Day 2-3 | 秘密宣傳 | 進入「觀察期」判為虛警 | 積極「尋求聯絡聖城」 | 「身份認同危機」的蔓延 |

### 第3-5天：聖城防衛 vs 黑箱回收

|---|---|---|---|---|
| Day 3 | 激活第二波部署 | 派遣回收小隊 | 開始「公開選邊」 | 「第一場關鍵戰役」 |
| Day 3-4 | 武裝對抗 + 秘密談判 | 試圖「活體回收」 | 部分向聖城靠近，部分保持中立 | 「投放人口內部分裂」 |

|---|---|---|---|---|

| Day 6 | 激烈武裝對抗 | 進行「地毯式清除」 | 與聖城進行「聯合對抗」 | 「三方對立升級為全面內戰」 |
| Day 7+ | 最終選擇：反攻/逃脫/投降 | 全面屠殺或妥協談判 | 投放人口「開始主動覺醒」與「自我解放」 | 「文明的未來將被決定」 |

## 快速查詢：某某人物應該屬於哪個系統？

### 判斷標準

```
Q: 這個人物是「投放人口」還是「製造人口」？

Step 1: 檢查「預設編碼」
  ├─ 有編碼 → 投放人口（來自黑箱）
  └─ 無編碼 → 製造人口（來自聖城）

Step 2: 檢查「出生記憶」
  ├─ 有完整的家庭/成長記憶 → 可能是投放人口（記憶是虛構的）
  └─ 無出生記憶（直接「被製造出來」）→ 製造人口

Step 3: 檢查「所在位置」
  ├─ 在聖城屏蔽區內 → 製造人口（很少有投放人口能進來）
  └─ 在屏蔽區外 → 投放人口（或逃脫的製造人口）

Step 4: 檢查「忠誠對象」
  ├─ 對黑箱/高層有無法抗拒的服從感 → 投放人口
  └─ 對聖城/自由有主動的信仰 → 製造人口
```

|---|---|---|---|

  └─ 「人口系統」本身開始瓦解

## 黑箱「三段式反應」的觸發與緩解

### 第0-3天觀察期：如何延長？

**聖城可以做的**：
- 進行「屏蔽系統最大化」，隱蔽人口製造的進度
- 製造「虛假編碼更新數據」，讓黑箱認為這是「正常的人口變化」
- 避免「公開宣佈」，保持「低調運作」

**黑箱可以做的**：
- 無法直接「升級」，只能「持續監控」
- 試圖「破除屏蔽」以確認異常體的真實身份
- 向高層報告，等待「人工決策」

**結論**：觀察期通常是「最安全的窗口」，聖城應該在此期間完成「至少15,000-20,000人的部署」。

---

### 第3-5天回收期：如何度過？

**聖城的策略**：
1. **武裝對抗**：以「新製造人口」對抗「黑箱回收小隊」
2. **秘密談判**：試圖「說服回收小隊指揮官背叛黑箱」（利用白扣案例）
3. **全速部署**：在回收期內完成「第二波人口部署」

**黑箱的困境**：
- 無法「快速升級」（需要時間派遣回收小隊）
- 無法「直接摧毀」（試圖「活體回收」以檢測異常體）
- 無法「預測聖城的行動」（因為黑箱無法進入屏蔽區）

**結論**：回收期是「關鍵的對抗窗口」，戰鬥的勝負將決定黑箱是否進入第三階段。

---

### 第5-7天清除期：如何生存？

**聖城的極限選擇**：
1. **投降**：接受被清除（失敗）
2. **逃脫**：利用「鼠人族通道」進入深淵（冒險）
3. **反攻**：進入黑箱中樞進行「終極反抗」（絕望賭注）
4. **談判**：激發「派系衝突」以爭取「妥協空間」（高風險）

**黑箱的執行計劃**：
- 第一步：切斷能源（聖城進入「停電狀態」）
- 第二步：派遣高階部隊（進行「地毯式清除」）
- 第三步：局部封鎖（阻止逃脫）
- 第四步：全面屠殺（無條件清除所有異常體）

**結論**：清除期是「全面對抗」的時期，聖城必須在此期間「做出終極決定」。

## 「投放vs製造」的根本衝突

### 衝突的本質

```
投放人口的困境：
  「我是被製造的，但我以為我是自然出生的」
  「我的所有行為都被編碼控制，但我以為這是我的選擇」
  「我無法脫離黑箱，因為黑箱就在我的神經裡」

製造人口的困境：
  「我知道我是被製造的，但我被告知這代表自由」
  「我的所有行為都是我的選擇，但這個選擇會被黑箱追殺」
  「我無法躲避黑箱，因為屏蔽遲早會被破壞」

根本問題：
  「什麼是真實的自由？」
  「被迫的自決 vs 內化的奴役，哪個更可怕？」
  「如果記憶都可以被虛構，身份還有意義嗎？」
```

### 衝突的解決可能性

| 可能性 | 結果 | 機率 |
|---|---|---|
| **聖城全勝** | 黑箱被摧毀，製造人口與覺醒的投放人口重建文明 | 20% |
| **黑箱全勝** | 聖城被消滅，投放人口體系繼續統治 | 30% |
| **妥協結局** | 黑箱停止監控聖城，聖城獲得有限自治空間 | 30% |
| **多方僵持** | 聖城、黑箱、投放人口形成三角對立 | 15% |
| **文明融合** | 製造人口與投放人口的融合，形成「新人類文明」 | 5% |

### 當寫到某個場景時，快速檢查

**Q：這個場景應該顯示哪個系統的特性？**

| 場景 | 應該展現的系統 | 檢查點 |

| 人物在對抗黑箱 | 製造人口（或覺醒的投放人口） | 抗拒的動機是什麼？ |
| 人物在執行秘密任務 | 製造人口（或被策反的投放人口） | 是否有編碼的約束感？ |
| 人物在經歷痛苦選擇 | 兩者都可能（性質不同） | 痛苦的根源是什麼？ |

## 最後的提醒

### 核心宣言的力量

> **「我們不是被投放的奴隸，我們是自己造出來的自由人！」**

這句話在第五卷成為「改變局面的關鍵武器」，不是因為它是「真理」，而是因為：
1. 它觸碰了每個人內心最深的「自由渴望」
2. 它提供了「身份認同的新可能性」
3. 它激發了「對黑箱秩序的質疑」

但同時，讀者應該在故事推進中開始懷疑：
- **這句話本身是否也是一種「新的編碼」？**
- **聖城是否也在「操縱」投放人口的身份認同？**
- **「自由」是否只是「另一種形式的奴役」？**

這些問題將成為第六卷及以後的核心議題。
