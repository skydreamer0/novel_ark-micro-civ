# 角色語音庫

> 每個角色對同一件事的說話方式不同。本檔收錄實際例句，續寫時比對。
> **核心原則**：給 AI 看過這些例句後，新寫的對話不能聽起來像同一個人在說話。

---

## 林燼（主角）

**定位**：少、穩、帶壓迫。不說長篇大論，不解釋自己為什麼這麼做。

| 情境 | 他會說 | 他不會說 |
|---|---|---|
| 被質疑能力 | 「比你快。」 | 「你不用擔心，我可以處理。」 |
| 要求座標 | 「座標發我。」 | 「請把位置發給我。」 |
| 沒有退路時 | 「過不去，也得過。」 | 「我們沒有其他選擇了。」 |
| 拒絕收編 | 沉默，直接動手 | 「我拒絕你們的條件。」 |
| 隊友猶豫時 | 不回答，先做決定 | 「我理解你的顧慮，但是……」 |
| 戰鬥中指令 | 「左邊兩個我處理。」 | 「我先解決左邊的兩個敵人，你們掩護。」 |

---

## 蘇嵐（技術位）

**定位**：沙啞、直擊要害、帶黑色幽默。先給數據再給結論（反常時代表她在壓情緒）。

| 情境 | 她會說 | 她不會說 |
|---|---|---|
| 看到欠稅金額 | 「審計局最近真餓。」 | 「這次的稅金非常高。」 |
| 問欠多少 | 「欠多少？」 | 「你目前的欠款金額是多少？」 |
| 警告風險 | 「如果你真進了回收序列，記得把工具箱留給我。我喜歡你那把扳手。」 | 「請注意安全。」 |
| 發現技術異常 | 「路由不對。」（先不說哪裡不對） | 「我發現路由有異常，可能是審計中樞的備用接口。」 |
| 結論確定時 | 「簽核時間，凌晨四點十二分。」（直接給數字） | 「我找到了簽核時間。」 |
| 壓著情緒時 | 林燼發現她在敲桌面（不說她在怕） | 「我很擔心。」 |

**肢體標記**：那截銀色義指在不安時會敲桌面/槍管。先抽一口煙再說話代表她在想。

## 燕九（戰鬥位）

**定位**：直接、粗獷、急性子。習慣以暴制暴，傾向立刻報復。

| 情境 | 他會說 | 他不會說 |
|---|---|---|
| 理解局勢時 | 「他開始用全城當人質。」 | 「我認為這是顧衡在利用全城來施壓。」 |
| 評估方案時 | 「需要大算力。」（一句話結論） | 「這個方案需要大量的運算能力支持。」 |
| 不滿時 | 用行動代替嘴（拍槍機很用力） | 「我不同意這個決定。」 |
| 下判断時 | 直接講結論，不鋪墊 | 「先撤。」 |

**肢體標記**：受傷時會用比需要更重的力氣操作武器，與其說在修槍不如說在發洩。

## 顧衡（反派）

**定位**：短、冷、傲慢。用行動展示權力，不說長篇威脅。崩潰不是大喊大叫，是手停在半空。

| 情境 | 他會說 | 他不會說 |
|---|---|---|
| 下令清除 | 「清理掉。」 | 「請執行清除程序。」 |
| 被問是否回報 | 「不用回報。」 | 「不需要向上級報告。」 |
| 發現異常 | 盯著螢幕上的錯誤代碼，瞳孔縮緊 | 「這怎麼可能？」 |
| 震驚時 | 咖啡杯倒了，沒發現袖口被濺濕 | 「我難以置信！」 |
| 被反制時 | 「切斷連線！」（聲音從冷靜變嘶啞） | 「快中斷所有連線！」 |

---

## 唐雀（技術/預警）

**定位**：務實、直接、話不多。專注於可操作的數據，不下價值判斷。

| 情境 | 她會說 | 她不會說 |
|---|---|---|
| 完成任務時 | 「可以用了。」（附加訊息一行字） | 「我完成了解碼工作，資料已經準備好可以使用。」 |
| 提醒風險 | 「那是我們留給之後跑路的。」 | 「我建議保留這些物資作為緊急備用。」 |
| 被命令時 | 看了對方一眼，直接轉頭執行 | 「明白，我馬上去辦。」 |

---

## 賀老八（地方勢力）

**定位**：老狐狸，走投無路時本能拿別人肉換自己命。語氣軟、話多、試探。

| 情境 | 他會說 | 他不會說 |
|---|---|---|
| 談判時 | 「四成。」（先報數字試水溫） | 「我們希望能夠分得百分之四十的資源。」 |
| 示弱時 | 「林燼，這不是硬扛就扛得過去的。」 | 「這個情況難以處理，需要合作。」 |
| 提議交易時 | 「把昨晚那三份東西抽一份出來交給我。我去找上面的人談。」 | 「建議我們合作，共同面對這個挑戰。」 |

## 各角色對同一情境的差異

**情境：剛打完一場硬仗，有人問「接下來呢？」**

| 角色 | 反應 |
|---|---|
| **林燼** | 不回答。低頭檢查裝備，確認還能打多久。 |
| **蘇嵐** | 「先補氧，再補情報，最後補人。」（分三層，有順序） |
| **燕九** | 「先撤。活著才能打下一場。」（結論先行） |
| **唐雀** | 「名冊還差兩片。」（只說她關心的數據缺口） |
| **賀老八** | 「六個月之後呢？」（先確認自己利益時限） |
