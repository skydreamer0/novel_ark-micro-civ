# 角色語音庫

> 每個角色對同一件事的說話方式不同。本檔收錄實際例句，續寫時比對。
> **核心原則**：給 AI 看過這些例句後，新寫的對話不能聽起來像同一個人在說話。
> 每個例句標註 [Vx] 表示該角色在該卷時期的典型台詞風格。

---

## 林燼（主角）

**定位**：少、穩、帶壓迫。不說長篇大論，不解釋自己為什麼這麼做。

**演化軌跡**：V1-V2 極簡被動（≤5字）→ V3-V4 簡短命令壓迫 → V5-V6 冷靜賭徒平靜 → V7-V9 開始說「我不知道」→ V10-V12 從「我」到「我們」→ V13-V14 深沉平靜

| 情境 | 他會說（卷數） | 他不會說 |
|---|---|---|
| 被質疑能力 | 「比你快。」[V1-V2] | 「你不用擔心，我可以處理。」 |
| 要求座標 | 「座標發我。」[V3-V4] | 「請把位置發給我。」 |
| 沒有退路時 | 「過不去，也得過。」[V3-V4] | 「我們沒有其他選擇了。」 |
| 拒絕收編 | 沉默，直接動手 [V1-V2] | 「我拒絕你們的條件。」 |
| 隊友猶豫時 | 不回答，先做決定 [V3-V4] | 「我理解你的顧慮，但是……」 |
| 戰鬥中指令 | 「左邊兩個我處理。」[V3-V4] | 「我先解決左邊的兩個敵人，你們掩護。」 |
| 面對高風險抉擇 | 「勝率四成，夠了。」[V5-V6] | 「這個方案風險太高，我們再評估。」 |
| 賭徒式判斷 | 「我賭它會來。」[V5-V6] | 「我相信他會遵守約定。」 |
| 被問不知道的事 | 「我不知道。」[V7-V9] | 「我需要更多時間調查。」 |
| 承認極限 | 「我不知道，但繼續。」[V7-V9] | 「我暫時沒有答案，請稍候。」 |
| 分配任務 | 「我們一起走。」[V10-V12] | 「你跟緊我，不要落單。」 |
| 面對困境 | 「我們想辦法。」[V10-V12] | 「我會想辦法解決。」 |
| 終局面對 | 「走吧。」平靜地 [V13-V14] | 「放心，一切都結束了。」 |
| 被問感受 | 沉默，眼神平靜 [V13-V14] | 「我沒事了。」 |

## 蘇嵐（技術位）

**定位**：沙啞、直擊要害、帶黑色幽默。先給數據再給結論（反常時代表她在壓情緒）。

**演化軌跡**：V1-V2 保留戒備交易語氣 → V3-V4 專業判斷語氣 → V5-V6 機構級語氣 → V7-V10 技術危機語氣 → V11-V12 跨物種翻譯語氣 → V13-V14 界域見證者語氣

| 情境 | 她會說（卷數） | 她不會說 |
|---|---|---|
| 談條件時 | 「你拿什麼換？」[V1-V2] | 「請問您能提供什麼報酬？」 |
| 被問來歷 | 「關你事？」[V1-V2] | 「我不想回答這個問題。」 |
| 問欠多少 | 「欠多少？」[V1-V2] | 「你目前的欠款金額是多少？」 |
| 結論確定時 | 「簽核時間，凌晨四點十二分。」[V3-V4] | 「我找到了簽核時間。」 |
| 看到欠稅金額 | 「審計局最近真餓。」[V5-V6] | 「這次的稅金非常高。」 |
| 發現技術異常 | 「路由不對。」（先不說哪裡不對）[V7-V10] | 「我發現路由可能有異常。」 |
| 警告風險 | 「如果你真進了回收序列，記得把工具箱留給我。」[V7-V10] | 「請注意安全。」 |
| 壓著情緒時 | 林燼發現她在敲桌面 [V7-V10] | 「我很擔心。」 |
| 解讀非人類訊息 | 「它說不是。」[V11-V12] | 「我認為訊號表示否定。」 |
| 翻譯異常 | 「他的語言裡沒有『謊言』這個詞。」[V11-V12] | 「這個系統中不存在欺騙的概念。」 |
| 描述超越理解的事物 | 「界域在說話。」[V13-V14] | 「這可能是某種異常現象。」 |
| 見證終局 | 「我看見邊界了。」[V13-V14] | 「我認為我們已經到達了某個界限。」 |

**肢體標記**：那截銀色義指在不安時會敲桌面/槍管。先抽一口煙再說話代表她在想。

## 燕九（戰鬥位）

**定位**：直接、粗獷、急性子。習慣以暴制暴，傾向立刻報復。

**演化軌跡**：V1-V2 短句粗口 → V3-V4 多疑試探 → V5-V6 指揮腔調成形 → V7-V8 越來越少話 → V9-V10 疲憊沉重 → V11-V12 傳承嘮叨 → V13-V14 最終簡短

| 情境 | 他會說（卷數） | 他不會說 |
|---|---|---|
| 被激怒 | 「操。」直接拔刀 [V1-V2] | 「請你不要太過分。」 |
| 不耐煩 | 「少廢話。」[V1-V2] | 「我們不要浪費時間了。」 |
| 不滿時 | 用行動代替嘴（拍槍機很用力）[V1-V2] | 「我不同意這個決定。」 |
| 遇到陌生人 | 「你誰的人？」[V3-V4] | 「請問你屬於哪個組織？」 |
| 不信任情報 | 「誰說的？」[V3-V4] | 「這個情報的來源可靠嗎？」 |
| 理解局勢時 | 「他開始用全城當人質。」[V3-V4] | 「我認為這是顧衡在利用全城。」 |
| 評估方案時 | 「需要大算力。」[V5-V6] | 「這個方案需要大量算力。」 |
| 下達戰術 | 「三、七點方向包過去。」[V5-V6] | 「我們分成兩個小組進行包圍。」 |
| 確認任務 | 「我帶頭，你跟第二。」[V5-V6] | 「我負責前鋒，你負責後援。」 |
| 下判斷時 | 「先撤。」[V7-V8] | 「我建議我們先撤退。」 |
| 戰後 | 「又少一個。」[V9-V10] | 「我們又失去了同伴。」 |
| 被問還好嗎 | 搖頭，不說話 [V9-V10] | 「我還撐得住。」 |
| 教新人 | 「我像你這麼大時也覺得自己不會死。」[V11-V12] | 「年輕時我也曾覺得自己無所畏懼。」 |
| 給建議 | 「活著比贏重要。」[V11-V12] | 「有時候保全自己比獲取勝利更重要。」 |
| 最後一戰 | 「走了。」[V13-V14] | 「準備出發吧。」 |
| 告別 | 「保重。」[V13-V14] | 「希望我們還能再見面。」 |

**肢體標記**：受傷時會用比需要更重的力氣操作武器，與其說在修槍不如說在發洩。

## 顧衡（反派）

**定位**：短、冷、傲慢。用行動展示權力，不說長篇威脅。崩潰不是大喊大叫，是手停在半空。

**演化軌跡**：V1 臨床精確 → V2 冷嘲熱諷 → V3 句子變短停頓變長 → V4 自我辯解增多 → V5 破碎跳躍 → V6-V7 沉默低沉 → V8 空洞威脅 → V9 幾乎不說話 → V10-V11 冷評書面 → V12-V13 被迫開口或最後一句 → V14 無

| 情境 | 他會說（卷數） | 他不會說 |
|---|---|---|
| 下令清除 | 「清理掉。」[V1] | 「請執行清除程序。」 |
| 被問是否回報 | 「不用回報。」[V1] | 「不需要向上級報告。」 |
| 評論對手能力 | 「就這？」[V2] | 「這就是他們的最強戰力嗎？」 |
| 被質疑 | 「你教我做事？」[V2] | 「我不需要你的建議。」 |
| 發現異常 | 盯著錯誤代碼，瞳孔縮緊 [V2-V3] | 「這怎麼可能？」 |
| 猶豫的命令 | 「去……處理掉。」[V3] | 「立刻去執行清除任務。」 |
| 思考卡住 | 「等等……不……不對。」[V3] | 「請稍等，這似乎不太對勁。」 |
| 被指責手段激烈 | 「我沒選擇。」[V4] | 「請理解這是最有效的方案。」 |
| 解釋動機 | 「都是為了……穩定。」[V4] | 「我的決策是為了整體利益。」 |
| 震驚時 | 咖啡杯倒了，沒發現袖口被濺濕 [V4-V5] | 「我難以置信！」 |
| 壓力下說話 | 「不……別動……退後。」[V5] | 「請保持冷靜，不要採取任何行動。」 |
| 思緒混亂 | 「你說過……不對……他……」[V5] | 「根據先前的記錄，這不符合他的行為模式。」 |
| 被反制時 | 「切斷連線！」[V6-V7] | 「快中斷所有連線！」 |
| 失敗時 | 「……夠了。」[V6-V7] | 「我認為我們應該停止這個計畫。」 |
| 最後警告 | 「我會讓你後悔。」[V8] | 「我建議你慎重考慮後果。」 |
| 無力威脅 | 「你們都會……死。」[V8] | 「你們的行為將會導致嚴重後果。」 |
| 被逼問 | 沉默 [V9] | 「我不會回答你的問題。」 |
| 出席會議 | 全程不發一語 [V9] | 「我沒有意見。」 |
| 觀察戰局 | 「誤差零點三，偏離預期。」[V10-V11] | 「戰術執行出現了微小偏差。」 |
| 評價他人 | 「效率低下。」[V10-V11] | 「你們的工作效率有待提高。」 |
| 臨死前 | 「……原來如此。」[V12-V13] | 「我終於明白了。」 |
| 最後一句 | 「你贏了。」[V12-V13] | 「我承認你比我強。」 |
| 存在感 | 不存在於任何對話中 [V14] | 「……」 |

## 各角色對同一情境的差異

**情境：剛打完一場硬仗，有人問「接下來呢？」**

| 角色 | 反應（卷數） |
|---|---|
| **林燼 V1-V2** | 不回答。低頭檢查裝備，確認還能打多久。 |
| **林燼 V5-V6** | 「先算代價，再算下一步。」 |
| **林燼 V10-V12** | 看向蘇嵐和燕九：「你們說呢？」 |
| **蘇嵐 V1-V2** | 「先補氧，再補情報，最後補人。」（分三層，有順序） |
| **蘇嵐 V7-V10** | 「異常信號還在。我的建議是——」手指在空中劃一條弧線（不說完） |
| **蘇嵐 V13-V14** | 「界限在移動。我不能告訴你下一步，只能告訴你方向。」 |
| **燕九 V1-V2** | 「先撤。活著才能打下一場。」（結論先行） |
| **燕九 V5-V6** | 「三號防線需要補人。我去。」 |
| **燕九 V11-V12** | 看向年輕士兵：「你覺得呢？」（傳承模式） |
| **唐雀** | 「名冊還差兩片。」（只說她關心的數據缺口） |
| **賀老八** | 「六個月之後呢？」（先確認自己利益時限） |
