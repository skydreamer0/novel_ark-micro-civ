## 一、核心衝突：新舊框架對比

### 1.1 舊框架（V8-V14 outline 版本）

| 層面 | 內容 |
|------|------|
| **衝突本質** | 母艦意識甦醒 → 內部幻覺恐慌 → 暴露 → 對抗 → 融合 |
| **反派身份** | 母艦本身（規則層清道夫）+ 審計司 |
| **覺醒模型** | 五層梯度：做夢→察覺→驚醒→反擊→對峙 |
| **終局目標** | 新天道誕生，文明與母艦意識融合 |
| **內部危機** | 幻覺（集體做夢）、派系分裂、暴露風險 |
| **關鍵隱喻** | 「整個母艦是一場正在做的夢」 |

### 1.2 新框架（5/24 世界觀重構）

| 層面 | 內容 |
|------|------|
| **衝突本質** | 病毒三層指令（求生→掠食→歸巢）逐步揭露，外律者債務系統 vs 文明主機自主權 |
| **反派身份** | 外律者（第一債主）殘留規則 → 母巢（收割節點）→ 舊宇宙回收系統 |
| **覺醒模型** | 三階段揭露：戰利品(V5) → 原初戰爭(V6) → 病毒返航指令(V8+) |
| **終局目標** | 斷債→消化→新天道（完成主宇宙未完成的規則消化） |
| **內部危機** | 規則燃料倫理（人當柴燒）、權限供養鏈道德代價、派系路線鬥爭 |
| **關鍵隱喻** | 「主宇宙只贏了戰爭；林燼贏的是戰後清算」 |

### 1.3 根本差異

| 面向 | 舊框架問題 | 新框架解決 |
|------|-----------|-----------|
| **衝突驅動力** | 模糊的「母艦甦醒」，讀者難以直觀理解 | 明確的「外律者債務系統」，每次衝突都有經濟+道德雙重重量 |
| **反派層次** | 單一（母艦變敵人） | 三層（外律者殘留→母巢→舊宇宙回收系統），有階梯感 |
| **主角成長命題** | 「變強→逃脫」 | 「變強→拆解債權→建立新秩序」更具使命感 |
| **伏筆承載力** | 前期設定難承接後期揭露 | 外律者線可追溯至黑箱/裂縫/投放編碼，全書統一 |

## 二、各版本命題對照與修改建議

### 2.1 V8（Ch.181-210）—— 舊：「做夢階段」／新：「偏航逃亡階段」

**舊命題**：母巢無意識干擾 → 內部幻覺恐慌 → 派系分裂
**新命題**：偏航逃亡 + 舊宇宙回收系統追殺 + 病毒歸巢指令起始揭露

| 要素 | 舊 V8 | 新 V8（建議） |
|:----:|-------|--------------|
| 核心衝突 | 內部幻覺恐慌，文明集體做夢 | 偏航後被舊宇宙追殺，同時病毒歸巢指令開始干擾導航 |
| 反派 | 規則層清道夫（抽象） | 舊宇宙回收系統 + 母巢收割信標（具體） |
| 內部危機 | 幻覺（不知真假） | 歸巢指令 vs 林燼偏航意志的精神拉鋸 |
| 揭露層級 | — | 第三層揭露開始：病毒 = 外律者返航指令 |
| 如何保留 | 幻覺現象可保留為歸巢指令的副作用 | 改為「偏航後導航系統被病毒反向牽引，產生集體方向錯亂」 |
| 爽點 | 打規則清道夫 | 首次正面對抗舊宇宙回收系統（具象化敵人） |

**關鍵改動**：
- 保留 V8 原有的「30 章」結構和「集體異常感」
- 但異常感的來源從「母艦做夢」改為「偏航後病毒歸巢指令干擾導航系統」
- 幻覺內容從「模糊的夢境」改為「往母巢方向走的強烈衝動」
- 第七卷後段（Ch.176-180）需先建立偏航事件 → V8 自然承接

### 2.2 V9（Ch.211-240）—— 舊：「察覺階段」／新：「債務覺醒階段」

**舊命題**：暴露風險爆漲，派系徹底分裂
**新命題**：林燼意識到自己不是唯一欠債者——整艘 Ark 都是外律者的抵押品

| 要素 | 舊 V9 | 新 V9（建議） |

| 核心衝突 | 暴露風險 vs 隱藏成本 | 斷債代價 vs 繼續負債的生存壓力 |

| 反派升級 | 審計司全面獵殺 | 舊宇宙回收系統 + 母巢引力場 + 黑箱債務催收 |
| 終局定位 | 從對抗到適應 | 從「躲債」到「認清債務來源」 |

### 2.3 V10（Ch.241-270）—— 舊：「驚醒階段」／新：「斷債準備階段」

**舊命題**：權限被封，被迫反擊

2. **「無聲異常感」不能消失**：需轉化為歸巢指令干擾的具體表現（導航錯亂、方向感知扭曲、對母巢方向的強烈吸引力），保留 Ch.148-175 建立的情緒基底
3. **需確認第六卷第二三幕規劃**：`03_ARCHITECTURE/08_第六卷架構_Ch161-210.md` 第二三幕目前為待定，若 V8 命題改變，第二三幕的過渡設計也需調整
