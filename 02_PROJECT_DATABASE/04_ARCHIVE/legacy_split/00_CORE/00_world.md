# 世界規則：母艦 OS、力量體系、資源循環

> 本作為本作最高邏輯準則。所有世界觀、力量體系、經濟稅制均統一在此框架下。

---

## 1. 核心世界邏輯：母艦即「作業系統」

末日母艦（Ark）是一個封閉的**「物理作業系統 (Physical OS)」**。所有物理資源（氧氣、空間、重力）都是系統分配的硬體資源。

- **生存的本質**：底層勞工不是在生活，而是在**「掛機運行」**
- **存活點 (Survival Credits)** = 低頻算力 (L-Compute)，維持肉體運行的貨幣
- **文明主機** = 高維度新 OS，寄生在母艦系統中運行
- **審計司** = 系統的防火牆/防毒軟體

### 成長雙線
- **外線（權限竊取）**：盜取管理員權限 → 身分提升、戰術模組、物理權限
- **內線（代碼覆寫）**：內部文明用信仰（H-Compute）計算 → 強行覆寫母艦局部物理規則

### 統一資源表

| 概念 | 統一邏輯定義 | 故事中的表現 |
|---|---|---|
| 存活點 | 低頻算力 (L-Compute) | 維持肉體硬體運行的基礎貨幣 |
| 信仰/信念回流 | 高頻算力 (H-Compute) | 主機專用能量，執行規則覆寫 |
| 規則碎片 | API 指令集 | 直接調用母艦底層代碼的快捷鍵 |
| 文明階段 | OS 內核版本 | 主機版本號（神殿時代 = V2.0） |

### 衝突邏輯：異常進程處理
1. **收編**：將非法進程轉化為系統插件，剝奪主動權
2. **去敏**：刪除進程中的異常代碼（抹除意志或記憶）
3. **清洗**：徹底格式化某一區域的硬體資源

## 2. 母艦結構與社會

### 層級分明的資源區塊
- **上層（內核區）**：董事會與審計司，最高系統修改權限
- **中層（應用層）**：企業聯盟與技術行會，半管理員權限
- **下層（數據冗餘區）**：維修工、契約勞工，最低硬體維護費

### 外域與裂縫
- 母艦航行中接觸到的異質宇域，提供原始能量與規則樣本
- **裂縫類別**：資源型（硬件材料）、規則型（外部代碼片段）、文明遺骸（古代知識）

### 人口控制：進程管理
- **起源池**：進程初始化中心，人口通過基因模板批量生成
- **去敏處理**：抹除異常進程的屬性
- **收編**：將高算力進程納入官方權限表

### 熵潮（宇宙級威脅）
- **定義**：宇宙正在加速衰亡，規則密度持續下降
- **規則密度**：維持物理定律所需的底層信息量
- **熵化**：規則密度跌破臨界值後的不可逆崩壞
- **死艙**：人口或規則密度跌破門檻後被封閉的艙段

## 3. 力量系統：主機覆寫與權限階梯

### 力量核心：高頻算力 (H-Compute)
- **反哺**：內部文明將運算結果回傳給主角，注入母艦環境
- **覆寫**：當反哺發生時，身邊物理規則被強行改寫

### 主機版本

| 階段 | 名稱 | 核心能力 |
|---|---|---|
| 0 | 開機狀態（種火殘片） | 主機初始化，建立基礎物理聯繫 |
| 1 | 種火時代 | 信仰雛形，基礎防禦與感知權限 |
| 2 | 神殿時代 | 文明分層化，群體協同與初階物質具現 |
| 3 | 聖城時代（第五卷） | 軍團+後勤+審計三軌治理 |

### 戰術手段
- **規則碎片**：病毒腳本，置入節點後更改運行效率
- **權限鏡像**：短暫模擬高級官員生理特徵
- **邏輯炸彈**：製造矛盾指令導致系統節點當機

### 代價與風險
- **反哺失真**：信仰偏差導致能力產生 Bug
- **算力溢出**：過度調用導致大腦過熱
- **系統報警**：大規模覆寫吸引防火牆注意，100% 時啟動區域格式化

### 系統提示風格
- ✅ 【權限判定中】【代碼覆寫啟動】【檢測到未授權進程】
- ❌ 等級、HP/MP、經驗值、技能點
