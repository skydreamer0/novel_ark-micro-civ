# 末日母艦：我的體內有個微縮文明 (Mothership: Inner Micro-Civilization)

> **一句話概念**
> 穿越成底層維修工與拾荒者的主角，在掛載於末日母艦上的巨型賽博都市中，獲得了能養殖文明、吸收信仰、加速煉化外域資源的「文明主機」。當所有人都被逼著掠奪外域以維持生存時，他逐步發現，整座城市、外域裂縫與高層統治，本質上都是更高層收割體系的一部分。

## 🧠 長線核心命題

故事的**主骨架為「掠食文明線」**，**暗線為「黑箱革命線」**。前期主打掠奪、升級、養文明、反殺與打臉；中後期逐步揭露母艦、高層、裂縫與收割體系的真正關係。

主角面臨的終極問題是：
- 他能不能在成為掠食者的過程中，**不完全變成**和頂層一樣的怪物？
- 他養出的文明，會不會最終也變成另一個剝削機器？
- 他是否能用同樣的系統，反過來吞掉整個牧場？

## 🤝 共創指南 (Contributing)

歡迎參與本宇宙的擴寫與設定補充！所有共同創作者請遵循以下原則：

1. **爽感優先**：劇情設定優先保證爽感、節奏與連續性。
2. **邏輯自洽**：世界規則要能自洽，確保高層壓迫系統在設定上能合理運轉。
3. **三線並行**：每次重要升級都要同時推動「戰力成長」、「文明演進」與「陰謀揭露」三條線。
4. **主角區隔**：主角的行事風格與底層邏輯，必須始終和普通掠奪者（或高層剝削者）拉開本質差異。

## ⚙️ 閱讀器設定擴充建議

目前網頁閱讀器已支援透過 `reader.config.json` 進行外部設定，方便後續擴張而不必改動 `app.js`：

- `githubOwner` / `githubRepo` / `githubBranch`：切換資料來源。
- `includeExtensions`：可擴充讀取副檔名（例如 `".md"`、`".txt"`）。
- `includeFolders`：可擴充章節來源資料夾（例如加入 `docs/特別篇`）。

此外也可用網址參數做臨時覆蓋：

- `?owner=xxx&repo=yyy&branch=main`
- `&folders=docs/01_正文,章稿`
- `&ext=.md,.txt`

範例設定檔請參考專案根目錄的 `reader.config.json`。
