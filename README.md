## 🌌 世界觀與舞台

### 世界主舞台
- 主場景是一座依附在**末日級流亡母艦**上的巨型賽博都市。
- **城內生態**：公司、幫派、義體醫療、黑市、工業區、下層艙段與資源轉化區。
- **城外區域**：外域裂縫、拾荒帶、投放帶與高風險遠征區。
- **都市本質**：都市看似繁榮，實際上是高度壓榨、配給化、持續縮圈的生存節點。

### 世界底層規則
1. 母艦所在宇宙正在衰亡，可居住空間會持續被熵蝕侵吞。
2. 若沒有新的外域資源注入，艙段會封閉、能源下降、氧氣與居住配額縮減。
3. **外域資源**不是普通礦物，而是帶有規則、生命、能量與異常性的高價值樣本。
4. 所有人都必須參與吸收、煉化、轉化外域資源的生存鏈條。
5. 上層不只是維持秩序，而是在故意放大危機，讓整個城市變成**吞噬機器**。

### 高層壓迫機制
高層掌握城市秩序、資源入口與轉化口，並用制度阻止底層真正爬升：
- **複製人補充機制**：高層能大量投放低成本複製人、契約勞工與債務人口。
- **內宇宙投放機制**：真正的高層會在自己的內宇宙裡養人，成熟後再投放到母艦外圍與都市底層。
- **官方轉化區壟斷**：所有外域收穫理論上都要進入官方轉化區，由官方抽成與分配。
- **極致稅制壓榨**：生存稅、空間稅、煉化稅、戰備稅、債務稅。
- **監控與收編**：任何成長過快的人，都可能被審計、徵召、抽編或直接奪權。

## 👤 核心人物設定

### 主角定位
- **身分**：穿越者、底層維修工、拾荒者
- **起點**：活在都市最底層，熟悉殘骸、裝備修補、風險區生存
- **主線張力**：主角發現自己走上的道路，與真正頂層「養殖文明」的方式高度相似，但他仍保有人類記憶與道德殘響。

### 主角最大優勢：雙層煉化權限
主角的核心機遇是獲得了「文明主機」級核心。
- **表面上**：和其他人一樣要把收穫送進官方體系。
- **實際上**：文明主機能暗中截流高價值部分，送入自己的內部宇宙。**別人是在替上層打工，主角則是在偷上層的燃料養自己的文明**。

**後續可解鎖優勢**：
- **時間壓縮權限**：內部文明的研究、演化與煉化速度遠快於外界。
- **文明鏡像權限**：能從敵對樣本、外域殘骸與高層投放物中抽取文明模板。

## ⚙️ 核心系統與雙線成長

本專案最重要的設計為**雙線成長結構**，讓每次冒險都能同時推進個人戰力、文明成長與世界真相。

### 1. 外域掠奪循環
主角在外面戰鬥、拾荒、拆遺跡、吞外域樣本，獲得高風險高收益的原始資源。

### 2. 文明煉化循環（內部文明系統）
主角把資源投進「文明主機」的內部空間，交給其中的文明去理解、研究、崇拜與轉譯。
內部文明不是單純信徒，而是主角的解析引擎、神性放大器與規則試驗場。

**五時代制**：
1. **種火時代**：建立最初信仰與認知錨點
2. **神殿時代**：形成祭司、工匠與戰士階層，開始主動解析力量
3. **鋼鐵時代**：吸收賽博技術與外域碎片，走向神權科技混合
4. **聖城時代**：形成大型信仰網路與文明軍團，集體參與煉化
5. **星環時代**：能向外擴張、分封、殖民，逐步成為完整小型文明體系

**信仰機制**：
信仰不是單純供奉，而是集體參與煉化。居民越理解主角的神性、越參與儀式、戰爭、研究與敘事，整個文明就越像主機的分散式處理器，能把外域資源更快轉成主角可用的力量。

### 3. 主機反哺循環
內部文明完成吸收後，主機再把能力、模組、科技或規則權限反饋給主角。

## 🧠 長線核心命題

故事的**主骨架為「掠食文明線」**，**暗線為「黑箱革命線」**。前期主打掠奪、升級、養文明、反殺與打臉；中後期逐步揭露母艦、高層、裂縫與收割體系的真正關係。

這不是單純的末日求生，而是一個被設計成必須不停吞噬外域、壓榨人口、養殖文明才能延續的**宇宙牧場**。
主角面臨的終極問題是：
- 他能不能在成為掠食者的過程中，**不完全變成**和頂層一樣的怪物？
- 他養出的文明，會不會最終也變成另一個剝削機器？
- 他是否能用同樣的系統，反過來吞掉整個牧場？

## 🤝 共創指南 (Contributing)

歡迎參與本宇宙的擴寫與設定補充！所有共同創作者請遵循以下原則：

1. **爽感優先**：劇情設定優先保證爽感、節奏與連續性。
2. **邏輯自洽**：世界規則要能自洽，確保高層壓迫系統在設定上能合理運轉。
3. **三線並行**：每次重要升級都要同時推動「戰力成長」、「文明演進」與「陰謀揭露」三條線。
4. **主角區隔**：主角的行事風格與底層邏輯，必須始終和普通掠奪者（或高層剝削者）拉開本質差異。
