# 熵潮與存在級威脅

> 本文件定義「為什麼所有人——包括高層——都必須瘋狂吸收外域資源」的宇宙級根因。

---

## 1. 熵潮（Entropy Tide）

### 1.1 定義

宇宙正在加速衰亡。這不是傳統物理學中遙遠的「熱寂」，而是一種**可感知、可量測、正在蠶食一切物質與規則**的活性退化現象。

- **規則密度**：空間中維持物理定律正常運作所需的「底層信息量」。
- **熵潮**：規則密度持續下降的宏觀過程——彷彿有某種外力正在從宇宙中「抽取」基礎規則。
- **熵化**：當某一區域的規則密度跌破臨界值後，該區域發生不可逆崩壞——物質分解、因果失序、時間流向紊亂，生命不可存續。

### 1.2 可觀測表現

| 熵化階段 | 觀測特徵 | 對應危機 |
|---|---|---|
| **潛伏期** | 設備效能每月微跌 0.8～2.1%；精密儀器偶發讀數漂移 | 只有技術部門注意到，大眾不知情 |
| **加速期** | 金屬出現無外力裂痕；能源電池自放電率異常；弱 AI 出現邏輯迴圈 | 維修成本暴增，底層首先感受到物資緊縮 |
| **臨界期** | 物理常數可觀測漂移（重力、光速、電磁力）；有機體出現無徵兆細胞崩解 | 區域不再適合人類居住，必須撤離或注入大量規則燃料 |
| **熵死** | 空間本身「塌陷」——物質還原為無結構粒子，時間流凝滯，信息歸零 | 不可逆。該區域永久失能。 |

### 1.3 熵潮的速率與壓力

- 母艦所在的宇域，熵化速率約為每標準年流失 **4.7%** 區域規則密度。
- 不進行任何規則補給的情況下，母艦核心區域將在 **12～15 年** 內進入臨界期。
- 邊緣艙段（維護等級最低）的熵化速率約為核心區域的 **2.6 倍**——這就是底層永遠首先承受壓力的根本物理原因。

## 2. 裂縫與熵潮的關係

### 2.1 裂縫的真正成因

裂縫並非母艦躍遷的「副產品」——它們是**熵潮侵蝕時空時撕開的破口**。

- 高規則密度區域與低規則密度區域之間形成劇烈梯度差，在引力、電磁等場的共振下產生時空撕裂。
- 裂縫連接的「異質宇域」，其實是**尚未被熵潮完全抽乾的殘餘高密度區**。
- 這就是「裡面既危險又充滿資源」的原因——那些規則碎片、能量晶核、知識矩陣，都是高密度區域的殘留物。

### 2.2 裂縫作為採集窗口

| 裂縫收益 | 實質 |
|---|---|
| 能量晶核 | 凝結態規則密度，可直接注入母艦系統對沖熵化 |
| 規則碎片 | 局部物理定律的「快照」，可被轉化爐解析並釋放為通用規則補給 |
| 知識矩陣 | 已滅亡文明保存的高階規則封裝，解析後效率極高但風險極大 |
| 活性生物樣本 | 在高密度環境演化的有機體，體內天然含有規則密度——提取需要「二次轉化」 |

### 2.3 第三卷揭密預埋

> 裂縫不是自然現象。數據表明裂縫的生成頻率與位置分布呈現高度規律性——它們可能是被某種「更高層存在」刻意製造的。目的未知。

## 3. 所有階層為何都逃不掉

### 3.1 底層

- 居住艙段的維生系統（氧循環、淨水、溫控）全部依賴規則燃料運轉。
- 斷供 = 斷氧。底層沒有任何緩衝儲備。

### 3.2 中層（企業聯盟 / 承包軍 / 技術行會）

- 武器、義體、轉化爐、醫療設備的效能隨熵化持續下降。
- 必須用外域資源定期「校準」設備，否則精度與可靠度會跌至不堪用。
- 承包軍每次出征的損耗率中，約 **23%** 直接來自設備熵化失效而非敵對戰損。

### 3.3 高層（董事會 / 審計司 / 轉化委員會）

- **高層也擁有「私有內宇宙」** ——他們的內宇宙同樣受到熵潮侵蝕。
- 高層的內宇宙靠外域資源供養，停止供給 = 養殖鏈崩塌 = 反哺能力歸零。
- 高層的「永生方案」（意識備份、身體替換）也依賴規則密度維持——熵化會讓備份數據損毀。

### 3.4 母艦整體

- 躍遷引擎靠**規則密度梯度差**驅動——沒有足夠儲備就無法跳躍。
- 停止躍遷 = 停在原地被熵潮追上 = 全艦緩慢熵死。
- 因此母艦的航行方向實質是**逃離熵潮前鋒**。

## 5. 主角的「反熵」地位

### 5.1 文明主機的獨特性

- 主角的文明主機是**唯一已知能自行產生規則密度的系統**。
- 其他所有實體（高層內宇宙、母艦系統、轉化爐）都只能「消耗」規則密度，無法「創造」。
- 文明主機的規則密度自生長來源 = **信仰**。內部文明的信仰行為產生的認知共振，能自發結晶出新的規則密度。

### 5.2 為何是主角而非高層

- 高層的「內宇宙養殖」是**工業化模式**——他們投放人口、壓榨信仰、高效提取。
- 但工業化養殖的信仰是「恐懼驅動」的，產出的規則密度品質低、衰減快。
- 主角的文明主機採用**共生模式**——文明自主發展，信仰出自認知而非恐懼。
- 這使得主角內宇宙的規則密度**純度更高、半衰期更長**，甚至可能達到「永續自增長」。

### 5.3 長線終局命題

> 如果主角真的建成了「反熵引擎」，他是否願意對所有人開放？
> 還是他也會像高層一樣，把這個能力變成壟斷工具？
> 「不依賴掠奪的新秩序」是否真的可能，還是只是另一種更精緻的牧場？

## 6. 擴寫規範

- 任何涉及「為什麼必須掠奪」的劇情描寫，都應回溯到本文件的熵潮機制——不得以單純的「高層邪惡」作為唯一解釋。
- 前兩卷角色對話中可以用模糊說法（「城市在老化」「設備又掉效能了」），不需直呼「熵潮」。
- 第三卷起才正式引入「熵潮」「規則密度」等術語。
- 每次提到熵潮相關數值（衰退率、時限），需與本文件一致或附註偏差原因。
