## 1. 末日母艦

- 規模：跨世代流亡艦，含多圈層艙段。
- 現狀：能源老化、氧循環衰退、艙段逐步封閉。
- 真相：對外宣稱「文明保種計畫」，實際是高層維持權力的永續收割模型，懼怕AI覺醒，一直運用人工去維持系統運轉，

## 2. 附掛都市（主舞台）

- 上層：董事會、審計司、轉化委員會。
- 中層：企業聯盟、承包軍、技術行會。
- 下層：維修工、拾荒隊、契約勞工、債務人口。
- 斷層規則：階層可流動但成本極高，多數上升通道是收編陷阱。

## 3. 外域裂縫群

- 定義：母艦躍遷過程中可短時連結的異質宇域窗口。
- 特徵：高價值、高污染、高不確定死亡率。
- 功能：城市經濟燃料來源，也是高層挑選可用人才的屠宰場。

## 4. 母艦生存方程式

高層必須維持「外域養殖 → 轉化供給 → 艙段維穩」的閉環。

### 三大硬約束

1. **能源衰退**：主反應爐每月衰退 3.2%，72 小時無補給即二級限電。
2. **最低人口**：每艙段需 62% 運作人口，否則失去自維持能力。
3. **污染上限**：公轉化區日淨化量固定，超量回灌氧水循環致輻射病。

### 寫作意義
- 高層殘酷政策非單純邪惡，而是「延後崩潰時間」。
- 主角提出的替代秩序必須同時回答能源、人口、污染三條硬約束。

## 5. 社會控制與稅制

### 身份稅制

- **存活點 (Survival Credits)**：底層通用貨幣。
- 每日固定生存稅（最低生存線 = 20 存活點）：
  - 基礎氧氣稅：10 點/天
  - 床位租金：5 點/天
  - 抗輻射廢水配額：5 點/天

### 控制手段

1. 人口補充：高層持續投放複製人、債務人口，底層無法透過怠工談判。
2. 上升管道：綁定終身契約或不可逆賽博改造。
3. 審計監控：演算法與審計官雙軌執法。
4. 收編流程：招募 → 試忠誠 → 奪權限 → 替換人格。
