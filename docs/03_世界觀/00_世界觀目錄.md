# 世界觀設定資料夾

此資料夾用於集中管理《末日母艦：我的體內有個微縮文明》的世界觀設定，與章綱/企劃文件分離，方便後續增補與查閱。

## 檔案導覽

- `03_世界觀總綱.md`：世界觀總綱（時間線、舞台結構、資源規則、權力機制）
- `01_陣營與角色.md`：勢力關係與關鍵角色定位
- `02_場景與生態危機.md`：場景分區、探索風險與劇情用途
- `04_專有名詞辭典.md`：名詞表，統一後續寫作語彙

## 維護規範

1. 新增設定時先更新 `03_世界觀總綱.md` 的對應段落，再補到細目檔案。
2. 名詞以 `04_專有名詞辭典.md` 為唯一命名基準，避免同義詞混用。
3. 每次修改建議在檔案末尾附上「變更註記」。
