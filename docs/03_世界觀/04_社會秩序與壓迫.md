# 社會秩序與壓迫機制

> 來源：從 `03_世界觀總綱.md` §5 獨立出來。

---

## 1. 身份稅制與底層生存壓力

- **存活點 (Survival Credits)**：底層區域的通用貨幣，一單位等同於維持最底限維生的耗材（又稱「標準氧時 SOH」）。
- **每日固定生存稅（最低生存線 = 20存活點）**：
  - 基礎氧氣稅：10 點/天
  - 床位租金（底層膠囊艙）：5 點/天
  - 抗輻射廢水配額：5 點/天
- **開局極致壓榨**：主角餘額不足 20 點，若無法湊齊點數，隔日將面臨斷氧或被「清理工」強制抽血抵債。

## 2. 社會控制手段

1. **人口補充**：高層持續投放低成本的複製人、債務人口，使底層無法透過怠工來談判。
2. **上升管道**：獎勵看似開放，實際綁定終身契約（成為「企業狗」或進行無法逆轉的賽博改造）。
3. **審計監控**：由演算法與審計官雙軌執法，任何不正常的物資流通或過快的實力躍昇都會被標記。
4. **收編流程**：對於高價值個體的標準SOP：招募—試忠誠—奪權限—替換人格。

## 3. 經濟運行細則

### 固定六項日常扣費指標（以「存活點 / 日」計）

1. **氧稅**：依艙段空氣循環負荷浮動，基準 8～14 點。
2. **居住稅**：床位、艙段通行與基礎能源配給，基準 4～10 點。
3. **維修耗材**：工具折舊、濾芯、密封膠與外骨骼零件，基準 3～12 點。
4. **醫療風險金**：工傷保命包、抗污染針劑與急救押金，基準 2～9 點。
5. **債務利息**：欠稅與黑市短債日息，基準 3～15 點。
6. **黑市抽成**：洗貨、寄售、情報中介手續費，基準 2～11 點。

### 三角色日均收支範圍（一般日，未含重大戰損）

| 角色                      | 日均收入範圍 | 日均支出範圍（六指標合計） | 日均淨額範圍 | 生存判讀                                       |
| ------------------------- | ------------ | -------------------------- | ------------ | ---------------------------------------------- |
| 底層（維修工/拾荒工）     | 18～42 點    | 24～58 點                  | -20～+8 點   | 長期位於負債區，需靠接高風險單或灰色套利續命。 |
| 中層（技術行會/小承包商） | 55～110 點   | 38～92 點                  | -12～+52 點  | 可積累緩衝金，但一旦被調升稅級仍可能迅速失血。 |
| 承包軍（企業武裝外勤）    | 80～180 點   | 62～148 點                 | -28～+72 點  | 收入高但損耗與風險金極高，實際破產率並不低。   |

### 制度效果

六項指標將「生存成本」拆解為可監控、可追繳、可差異化調升的壓迫旋鈕；角色越接近上升通道，越容易被精準抽血。

## 4. 新增制度節點：第十三段「躍階氧稅」

- **制度定義**：在鐵鏽帶冷凝層試行的動態氧稅模型。個體在高風險區停留越久，氧稅倍率越高（90 分鐘後啟動 1.8 倍，180 分鐘後升至 2.4 倍）。
- **官方說法**：用於抑制污染外溢與降低醫療急救負載。
- **真實用途**：
  1. 與稅務風控分綁定，能精準挑出「高收益低服從」對象進行抽血。
  2. 迫使底層隊伍縮短作業窗口，增加黑市借貸與違規通關依賴。
  3. 為審計司提供即時行為資料，建立可預測的抓捕節奏。
- **劇情後果模板（1～3 章內）**：
  - 當章：主角為避稅被迫改道，錯過合法出貨窗口。
  - 次章：因押金不足，隊伍必須與灰色中介簽短期高息契約。
  - 第三章：審計端藉由通行異常標記發起定向盤查。

## 5. 新增制度節點：污染清算網與「延淨排序」

- **制度定義**：董事會將高污染貨、降級艙段往返人員與封艙前資產估值外包給清譜會。任何進出高風險區的隊伍，都可能被要求繳納污染押金，離場時再接受二段式污染評級與清艙估值；若未通過，就會進入延淨排序並凍結押金 24～72 小時。
- **官方說法**：用於保護潔淨區、提升淨化資源利用率，避免污染貨與高風險人員直接回灌主都。
- **真實用途**：
  1. 用押金凍結與延淨排序抽乾底層現金流，逼迫隊伍在最缺錢時折價出貨。
  2. 用污染評級把高價樣本打成低價污染貨，再由清譜會指定買家抄底。
  3. 把反覆被判高污染的個體納入「低優先生存名單」，為末批產出、危險作業與強制收編預先篩人。
- **制度補刀效果**：
  - 與躍階氧稅串聯後，隊伍在進場前被抽一次，離場後再被抽一次，形成完整的「風險雙收」。
  - 與污染風險押金並行時，表面上是醫療與潔淨機制，實際上是把污染處理能力變成壓迫性金融工具。
- **劇情後果模板（1～3 章內）**：
  - 當章：主角帶出的高價樣本被清譜會重判為污染貨，合法出貨窗口當場失效。
  - 次章：隊友因延後淨化出現惡化症狀，隊伍被迫向黑市借高息周轉以維持治療與押金。
  - 第三章：清譜會把評級資料同步給審計端，主角被標成污染收益異常者，迎來定向盤查與封帳風險。
