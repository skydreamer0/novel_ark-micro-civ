# 都市信仰與意識形態

> 本文件定義附掛都市（主舞台）中的信仰生態與意識形態光譜。與內部文明的「信仰—反哺」機制形成鏡像對照。

---

## 1. 信仰地圖總覽

都市社會的信仰不是單一教派，而是一個**多層光譜**——從官方意識形態到地下異端，每一種信仰都是對「我們為什麼必須掠奪」這個問題的不同回答。

| 信仰 | 核心教義 | 主要信眾 | 對掠奪的態度 |
|---|---|---|---|
| 恆航教（官方） | 母艦是聖方舟，航行是神聖使命 | 上層、中層管理階級 | 掠奪 = 聖戰，犧牲理所當然 |
| 焚數派 | 純粹信息才是永恆，肉體是累贅 | 技術行會邊緣成員 | 掠奪只為獲取「純資訊」 |
| 回歸派（地下） | 母星仍在，折返才是救贖 | 底層老一輩居民 | 反對掠奪，主張節制 |
| 前行派（地下） | 前方有「應許宇域」，到達即解脫 | 底層年輕人、冒險家 | 支持掠奪，但反對高層壟斷收益 |
| 靜寂信仰 | 熵化即涅槃，不應抵抗 | 極少數邊緣艙段殘民 | 反對一切掠奪，求「與宇宙同歸」 |

## 2. 恆航教（官方意識形態）

### 2.1 教義核心

- 母艦的航行是「聖航」，是人類文明延續的唯一方式。
- 所有為航行做出的犧牲都是「崇高的」——包括底層的生存壓榨。
- 反應爐被稱為「聖火」，維持聖火燃燒是所有人的義務。

### 2.2 實際功能

- **不是一個組織化的教會**，而是滲透在官方敘事、教育系統、公共廣播中的隱性意識形態。
- 審計司在執行抓捕時會引用恆航教的詞彙：「淨化」「審判」「對航行的背叛」。
- 公轉化區的回收儀式被包裝為「為聖航獻禮」。

### 2.3 裂痕

- 底層幾乎沒人真信——恆航教在底層的信用已破產。
- 但中層管理者需要它作為執法的道德外衣。
- 高層本身也不信——他們知道母艦航行的真實原因是逃離熵潮，但不會公開。

### 2.4 續焰團（恆航教的底層動員變體）

- **教義包裝**：續焰團不直接談神學，而是把恆航教的「聖航」「獻禮」「延續火種」翻譯成底層能聽懂的交易話術：今天吃苦，是在替家人換氧、替自己換階級、替街區換明天。
- **組織狀態**：它不是正式教會，而是遊走在救濟站、低價氧吧、轉運前哨與封艙倒數區之間的動員網。表面上提供食物、藥包與臨時庇護，實際上用「奉航志願書」把人導向高風險外勤、降級艙段作業與末批產出。
- **對掠奪的態度**：掠奪不是被迫，而是「贖罪與晉升的機會」；只要願意奉獻身體與工時，就有資格被看見、被記名、被分配更好的命。
- **高層態度**：官方不完全直管，卻默許它存在。因為續焰團能把原本需要強徵的人口，轉化成看似自願的勞動批次，遠比赤裸暴力更省成本。
- **劇情價值**：續焰團會搶走主角與灰鏈原本可能吸納的底層信任，把希望本身變成競爭資源；當主角開始在底層建立新秩序雛形時，它會是最早出手抹黑與奪民心的組織之一。

## 3. 焚數派（技術異端）

### 3.1 教義核心

- 物質世界注定衰亡（這一判斷暗合熵潮真相），真正值得保存的只有「純粹信息」。
- 人類應逐步「上傳」意識至數據載體，擺脫肉身的規則密度依賴。
- 裂縫中的「知識矩陣」是已滅亡文明嘗試「信息化永生」的遺產。

### 3.2 組織狀態

- 不是公開教派，而是技術行會與駭客社群中的小圈子文化。
- 蘇岚所活動的灰色駭客圈與焚數派有交集。
- 擁有少量自行改裝的「意識快照」設備——粗糙且高風險。

### 3.3 劇情價值

- 提供「非掠奪式生存路線」的另一種假說——但「上傳」技術實際上也需要大量規則密度，構成自相矛盾。
- 焚數派的技術可能成為中後期探索「知識矩陣」的工具。

## 4. 回歸派 vs 前行派（底層分裂）

### 4.1 回歸派

- **核心主張**：母艦應該折返出發點（母星），而非永無止境地前進。
- **信眾組成**：底層中年長者、曾聽過「祖輩口傳歷史」的人群。
- **實際行動**：消極抵抗——拖延工作、偶爾破壞通訊設備、散佈「母星仍在」的謠言。
- **高層態度**：不視為威脅，但會利用回歸派的存在製造「底層分裂」。
- **真相伏筆**：回歸派可能掌握了母艦出發點的部分真實座標——但母星區域的規則密度可能已經歸零。

### 4.2 前行派

- **核心主張**：前方的某個宇域擁有「無限規則密度」，只要到達就能終止掠奪循環。
- **信眾組成**：底層年輕人、冒險性格的拾荒者、部分承包軍退伍者。
- **實際行動**：積極投入裂縫探索，相信每次深入都在「靠近應許之地」。
- **高層態度**：暗中扶植——前行派的狂熱讓底層自願投入高風險作業，降低動員成本。
- **真相伏筆**：「應許宇域」的概念可能是高層刻意散播的——用希望取代鞭子。

### 4.3 兩派衝突

- 回歸與前行的分裂，讓底層無法形成統一的反抗力量——這正是高層樂見的。
- 主角可以利用兩派的衝突做資訊套利，也可以在合適時機嘗試整合。

## 5. 內部信仰 vs 外部信仰：鏡像衝突

### 5.1 對照關係

| 維度 | 主角內宇宙的信仰 | 都市外部的信仰 |
|---|---|---|
| 驅動力 | 認知與敬畏（自發產生） | 恐懼與需求（被動驅動） |
| 對「神」的態度 | 崇拜主角 = 真實可交互的存在 | 崇拜母艦/聖火 = 無法驗證的抽象概念 |
| 規則密度效應 | 信仰產生高純度規則密度 | 不產生規則密度（純意識形態控制） |
| 失真風險 | 信仰偏差 → 反哺失真 | 信仰偏差 → 社會動亂 |

### 5.2 劇情交叉點

- 當主角的內部文明發展到**神殿時代後期**，內部祭司可能嘗試與外部信仰建立映射——這會帶來認知污染風險。
- 若外部的某種信仰教義滲透進內部文明，可能引發嚴重的反哺失真事件。
- 反過來，若主角將內部文明的「真信仰產物」流出，可能在外部引發新的信仰革命。

## 6. 擴寫規範

- 每新增一個信仰/意識形態，需定義：**教義核心、信眾組成、對掠奪的態度、高層態度、劇情價值**。
- 信仰名稱以 `05_專有名詞辭典.md` 為準，避免同義詞混用。
- 底層角色的對話中可自然引用回歸派/前行派的說法，作為文化厚度而非說教。
- 恆航教的用語（聖航、聖火、淨化、獻禮）可作為審計司和官方廣播的常用語。
