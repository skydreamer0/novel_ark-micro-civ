# 資源與轉化規則

> 來源：從 `03_世界觀總綱.md` §3 獨立出來。

---

## 1. 三層資源類型

1. **物質資源**：晶礦、殘骸、可拆解器件。
2. **規則資源**：法則碎片、場域核心、特殊現象模組。
3. **生命資源**：活樣本、群落胚種、文明記憶體。

## 2. 官方轉化流程（對外版本）

1. 採集隊提交外域樣本。
2. 公轉化區檢定價值。
3. 系統抽稅與配給。
4. 餘額轉為個人配額與債務抵扣。

## 3. 黑箱轉化流程（真實版本）

1. 前端採集資料即被標記。
2. 一次抽稅後進入「二次黑箱抽取」。
3. 高價值部分轉入高層私有內宇宙。
4. 底層拿到經稀釋後的生存份額。
