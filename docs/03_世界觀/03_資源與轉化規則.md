# 資源與轉化規則

> 來源：從 `03_世界觀總綱.md` §3 獨立出來。

---

## 1. 三層資源類型

1. **物質資源**：晶礦、殘骸、可拆解器件。
2. **規則資源**：法則碎片、場域核心、特殊現象模組。
3. **生命資源**：活樣本、群落胚種、文明記憶體。

## 2. 官方轉化流程（對外版本）

1. 採集隊提交外域樣本。
2. 公轉化區檢定價值。
3. 系統抽稅與配給。
4. 餘額轉為個人配額與債務抵扣。

## 3. 黑箱轉化流程（真實版本）

1. 前端採集資料即被標記。
2. 一次抽稅後進入「二次黑箱抽取」。
3. 高價值部分轉入高層私有內宇宙。
4. 底層拿到經稀釋後的生存份額。

---

## 4. 內宇宙資源輸出機制

> 主角的文明主機在發展到一定階段後，可以將內部產出**反向輸出**至外部世界。這是主角從「純消費者」轉為「生產者」的關鍵能力。

### 4.1 階段性解鎖

| 文明階段 | 可輸出內容 | 限制條件 |
|---|---|---|
| 種火時代 | 無輸出能力 | 主機能量僅夠維持內部運轉 |
| 神殿時代（中期） | 少量純化規則燃料 | 每日產量 ≈ 底層工人 3 天收入，需手動提取 |
| 鋼鐵時代 | 規則燃料批量輸出 + 初階戰鬥生物投放 | 生物離開內宇宙後壽命有限（≈72 小時），規則燃料純度受信仰品質影響 |
| 聖城時代 | 高純度規則燃料 + 軍團級生物投放 + 工藝品具現 | 大規模投放會短暫降低內部規則密度，需冷卻時間 |
| 星環時代 | 持續性輸出流 + 永續生物部署 + 規則改寫工具 | 依賴完整的內部信仰網路穩定度 |

### 4.2 資源販售規則

- **規則燃料**：主角的內宇宙產出的規則燃料，純度遠高於外域採集品（因為來自信仰而非掠奪）。
- 在黑市中，高純度規則燃料是**硬通貨**——比存活點更可靠。
- 主角必須控制販售量，否則會引起審計司注意——「一個底層拾荒者怎麼會有高純度燃料？」

### 4.3 生物投放規則

- 內部文明培育的戰鬥生物可被投放至外部世界。
- **限制**：
  - 離開內宇宙後，生物與內部信仰網路斷開，規則密度持續流失。
  - 鋼鐵時代前的生物，外部存活時間 ≈ **72 小時**。
  - 聖城時代後可建立「信仰錨點」延長存活時間。
- **戰術價值**：等同於擁有私人軍隊，不依賴傭兵或承包軍。

## 5. 黑暗經濟學

### 5.1 債務衍生品

- 底層的欠稅和黑市短債可以被**打包交易**——類似現實中的資產證券化。
- 中層承包商購買底層債務包，獲得對應人口的「勞動支配權」。
- 極端情況下，整個底層小隊的債務可以被一次轉讓——買方獲得該隊所有勞動產出的優先分配權。
- **劇情價值**：主角可以利用債務市場反向操作——買下敵對勢力的債務來癱瘓其運作。

### 5.2 信仰貨幣化

- 如果信仰可以產生規則密度（如主角的內宇宙所證明），那麼**虔誠人口本身就是一種資源**。
- 高層可能早已在暗中進行「信仰收割」——從大規模宗教活動中提取微量規則密度。
- 恆航教的公共儀式、公轉化區的「獻禮」儀式，可能不只是意識形態工具，而是規則密度的隱性採集站。
- **第五卷伏筆**：主角發現公共儀式的場地下方埋有規則密度收集器。

### 5.3 污染配額交易

- 污染不是純粹的負面事物——它是一個**被精確控制的壓迫旋鈕**。
- 高層設定每日污染淨化上限，再將「淨化配額」分配（出售）給各級勢力。
- 配額持有者可以選擇：自用（保護自己區域）或轉售（賺取溢價）。
- 有些承包商**故意在底層製造污染**，再高價出售淨化配額——製造需求再販售解方。

## 6. 高層私有內宇宙的養殖收益模型

> 高層並非個個都有內宇宙——只有董事會核心成員與部分高階審計官擁有。

### 6.1 與主角的差異

| 維度 | 高層內宇宙 | 主角的文明主機 |
|---|---|---|
| 運作模式 | 工業化養殖——投放人口、壓榨信仰、高效提取 | 共生模式——文明自主發展，信仰出自認知 |
| 信仰品質 | 恐懼驅動，規則密度純度低、衰減快 | 認知驅動，純度高、半衰期長 |
| 規模 | 多個，每個專精單一資源類型 | 單一但全能型 |
| 擴張方式 | 持續投放外部人口（複製人、債務人口） | 內部人口自然繁衍 |
| 瓶頸 | 需要持續外域資源維持 = 仍然是消費者 | 理論上可達到自給自足 = 唯一的生產者 |

### 6.2 高層養殖鏈

1. 高層用外域資源建立私有內宇宙的基底規則環境。
2. 投放底層人口（常為複製人或被「收編」的高價值個體）。
3. 強制培養信仰（通常靠恐懼與獎懲制度）。
4. 提取信仰產生的規則密度。
5. 用提取的規則密度維持自身永生方案與權力基礎設施。
6. **瓶頸**：因為信仰品質低，需要不斷加大投放量——這就是高層瘋狂壓迫底層的另一個原因。
