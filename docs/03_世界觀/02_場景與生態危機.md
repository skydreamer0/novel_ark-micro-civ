# 場景與生態危機 (Loot & Map Design)

> *「別去看頭頂那些虛偽的霓虹燈，往下看。看看你靴子底下踩著的鏽鐵與血污，那才是這座城市真正的地基。」*

本文件將世界觀中的場景轉換為**附帶極限生存壓力的網遊地圖**。主角的移動軌跡並非出自探索的浪漫，而是被高昂的「存活點」債務與「文明主機」貪婪的升級需求驅趕著前進。

---

## 1. 絞肉機的底層：微光區（艙底維修帶）
*新手村 / 絕對高壓的生存試煉場*

*   **感官調性 (Atmosphere)**：
    *   **視覺**：永遠昏暗的閃爍紅燈。狹窄的管線間塞滿了如同蜂巢般的膠囊睡眠艙。牆壁上流淌著不知名的黏稠液體。
    *   **嗅覺**：刺鼻的機油味、劣質抗輻射藥劑的甜膩味、以及排風扇故障帶來的強烈體臭與血腥味。
    *   **聽覺**：沉重的巨型金屬氣閥開合聲、遠處幫派火拼的零星槍火、和鄰居因為無法繳納氧氣稅而在半夜被強制抽血的絕望慘叫。
*   **准入條件**：主角開局的出生地。不需要任何簽證，但一旦走進這裡，就等同於在自己的生命倒數計時器上按下加速鍵。
*   **區域勢力 (怪區)**：鐵鏽幫的收水費打手、因輻射而大腦退化的拾荒瘋子、失控暴走的工業搬運機甲。
*   **可獲取資源 (Loot Table)**：
    *   **基礎存活點**：一天高達十幾個小時的修補破鐵管或倒賣廢水，勉強能賺 5~10 點（剛好夠付一天的氧氣稅，毫無盈餘）。
    *   **劣質義體殘片**：可以用主機的【絕對消化】稍微提煉出微量金屬能量，勉強維持主機的待機消耗。
*   **劇情用途**：極限生存壓力的具象化。黑市情報與走私氣閘門的接頭處。

## 2. 死亡的星空墓場：近域拾荒帶
*前期主要副本 / 母艦外殼緩衝區*

*   **感官調性 (Atmosphere)**：
    *   **視覺**：幾乎沒有重力。外面懸浮著母艦過去無情拋棄的上千個巨型艙段殘骸，宛如一片寂靜的鋼鐵亂葬崗。遠處的恆星光芒被母艦巨大的陰影遮蔽，只剩下死寂的深空冷光。
    *   **嗅覺**：氣密服內循環的塑膠味與自己緊張的冷汗味。一旦聞到除了這兩種以外的味道，代表氣密服已經破裂，即將死於真空或輻射。
    *   **聽覺**：除了自己急促的呼吸聲外，絕對死寂。
*   **准入條件**：必須繳納 30% 生命收益給幫派，或擁有走私黑客提供的氣閘門密碼。沒有基礎氣密服就是在找死。
*   **區域勢力 (怪區)**：
    *   **同行黑吃黑陷阱**：比怪物更致命的是同樣來拾荒的人類。
    *   **屍潮型異變體 (R1級怪)**：曾經被拋棄的底層人口，在被微量外域輻射感染後，肉體發生了扭曲融合，變成的無意識游蕩怪物。
*   **可獲取資源 (Loot Table)**：
    *   **R1級異變體核心**：文明星火期（階段一）升級必備祭品。
    *   **古代艦船電子板與廢棄模組**：洗淨血跡後帶回公轉化區，可換取 50~200 不等的厚實存活點，是一夜暴富的最佳途徑，也是被強盜盯上的催命符。
*   **劇情用途**：第一卷初期累積原始資金、獲得主機第一波反哺升級、上演「極限撿漏」與「反殺掠奪者」的主要戰場。

## 3. 瘋狂的規則熔爐：裂縫投放帶 (外域接口)
*中期高危核心副本 / 扭曲常理的屠宰場*

*   **感官調性 (Atmosphere)**：
    *   **視覺**：色彩斑斕卻令人作嘔的異度空間。天空可能呈現出碎玻璃般的折射狀，地面可能是柔軟跳動的血肉，或是會自我生長的半透明晶體。在這裡，物理學定律如同廢紙。
    *   **嗅覺**：充滿了燒焦的臭氧味，有時候會聞到不可名狀的奇異香氣，那是能引發大腦幻覺的高維神經毒素。
    *   **聽覺**：空氣在尖叫。重力場被撕裂時會發出如同指甲刮黑板的銳利尖嘯。
*   **准入條件**：必須擁有企業聯盟重金簽發的「特許開採證」，或者冒死偷偷混入企業的空投登陸艇。
*   **區域勢力 (怪區)**：
    *   **規則系異變體 (R2~R3級大怪)**：完全不符合碳基生物邏輯的怪物。牠們的攻擊方式可能不是利爪，而是「瞬間改變你周圍十公尺的重力方向」、「讓你的血液沸騰」或「發射能切除部分記憶的光束」。
    *   **企業高階黑手雇傭兵**：這裡是法外之地，也是企業互相殘殺的修羅場。
*   **可獲取資源 (Loot Table)**：
    *   **體系完整的知識矩陣**：主機晉升【神殿時代】的必備高階材料。
    *   **高級規則碎片**：能被內部文明的工匠解析，並大幅強化主角本身的戰鬥模組（例如解鎖瞬間位移、光學隱形或力場護盾）。
*   **劇情用途**：中後期大場面的史詩級戰鬥舞台。主角在這裡截流最頂級的資源，正面衝突企業力量，並逐漸觸碰到宇宙衰亡的真相。

## 4. 虛偽的天堂：公轉化區 (企業與官方中繼站)
*極致繁榮與極致壓榨的交匯處 / 特殊劇情點*

*   **感官調性 (Atmosphere)**：
    *   **視覺**：刺眼的純白燈光、一塵不染的金屬地板。巨大的全息投影看板上輪播著虛假的招募廣告。全副武裝的機械守衛以絕對精準的步伐巡邏。
    *   **嗅覺**：令人安心的、經過多重過濾的純淨氧氣香味，甚至帶有極其昂貴的人工植物清新劑味道。
    *   **聽覺**：優雅的輕音樂，掩蓋了設施深處提煉物資的沉悶轟鳴。
*   **准入條件**：擁有合法市民身分ID，並且當日生存稅未欠費。
*   **劇情用途 (非主要戰鬥區)**：
    *   **揭露黑箱的嘲諷感**：主角在此繳交「已經被體內主機連皮帶骨偷吃過核心好處」的殘渣，看著傲慢的審計官將其當作寶貝般收走，並換取高額的合法存活點（極強的打臉與反向收割爽感）。
    *   **情報與市場樞紐**：高階懸賞榜、官方任務發布地與黑市情報洗白的中轉站。

## 5. 風險與場景切換節奏 (Pacing Guide for Auto-Writing)

1. **日常壓迫循環 (Low-Risk/High-Tension)**：在「微光區」被幫派或債主壓迫打臉 -> 為了湊出隔天的氧氣錢，不得已前往黑市接取高風險黑單 -> 前往「公轉化區」交任務，遭遇上層的蔑視與剝削。
2. **升級突破循環 (High-Risk/High-Reward)**：當體內主機提示需要特殊祭品 -> 鋌而走險強行進入「裂縫投放帶」拼命 -> 歷經慘烈反殺獲得高級掉落物 -> 隱蔽在安全屋內完成【文明時代】的大跨度升級，戰力暴增。
3. **場景切換頻率**：寫作時務必保持「內外場景對比」。每 3 章至少切換一次主要場景（例如：微光區的髒亂 -> 裂縫的迷幻危險 -> 內部文明莊嚴的神殿與狂熱祈禱），利用強烈的視覺落差帶來持續的閱讀刺激。
