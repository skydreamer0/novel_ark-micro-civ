# 世界觀名詞表

## 核心系統

- **文明主機**：主角持有的內部文明培育與煉化核心。
- **反哺**：內部文明將解析結果回傳給主角的過程。
- **雙層煉化**：官方流程之外，主角私有截流再煉化機制。

## 場景與制度

- **末日母艦**：承載都市與艙段的流亡級巨艦。
- **附掛都市**：主故事發生地，依附在母艦外層結構。
- **公轉化區**：官方唯一合法外域資源轉化機構。
- **二次黑箱抽取**：高層在官方流程後進行的隱性抽成。

## 人口與統治

- **投放批次人口**：由高層系統性投入底層勞動市場的人口。
- **收編流程**：對高潛力個體進行控制、奪權與替代的制度。
- **未授權文明載體**：高層對主角此類變數個體的風險標籤。

## 財務與稅戰

- **稅務風控分**：審計系統依個體收益波動、來源乾淨度、任務死亡率推算的風險分數；分數越高，稅率與抽查頻率越高。
- **洗白通道**：將外域與黑市收益轉成「合法存活點」的流程與節點集合，通常由公轉化區外包商、行會帳戶或人頭合約構成。
- **灰色節點直煉**：繞過公轉化區主流程，在半合法維修節點完成樣本預處理與初級煉化以降低官方抽成的高風險作法。

## 外域相關

- **外域裂縫**：母艦航道中短時出現的異質空間窗口。
- **外域裂縫等級標記**：統一記為 `R1~R5 + 詞綴`（例：`R2-鏡折`、`R4-蟻潮`）；`R` 代表裂縫風險與治理成本基準，詞綴只描述同類裂縫的行為差異。
- **規則碎片**：可改寫局部效率或法則的小型權限載體。
- **污染超標**：樣本異常影響超過人體/文明承載閾值。
