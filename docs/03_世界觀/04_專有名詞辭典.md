# 世界觀專有名詞與核心定義辭典

> 本文件為《末日母艦：我的體內有個微縮文明》的官方詞彙表。在所有正文撰寫中，請嚴格使用此處的命名與定義，確保世界觀的高冷距、賽博龐克感與史詩氣質。

---

## 1. 核心力量系統

- **文明主機 (The Genesis Engine / The Inner Engine)**
主角穿越後體內融合的神秘造物。它不僅是一個儲物箱，更是一個能培育、進化微縮文明的「神載處理器」。它可以將各種危險的外域物質轉化為信仰與規則力量。
- **文明反哺 (Civilization Feedback / Divine Echo)**
內部微縮文明通過集體研究、膜拜與獻祭，將外域物品徹底解析後，反饋給主角（神明）的戰術模組、規則權限與防禦護盾。
- **雙層煉化 (Dual-Layer Refining)**
本作最大的外掛與爽點機制。表面上主角遵守官方的資源上繳流程；實際上，最有價值、最核心的規則精華早已被直接截流，投餵給了體內的微縮文明進行無損煉化。

## 2. 統治制度與經濟體系

- **存活點 (Survival Credits / SCs)**
底層唯一承認的貨幣。它不是普通的錢，本質是「生存配額」。1 點約等於 1 小時的標準維生消耗（標準氧時 SOH）。
- **氧氣稅 / 生存稅 (Respiration Tax / Survival Levy)**
高壓統治的核心手段。系統每日強制扣除最低 20 存活點。一旦帳戶歸零，所在區域將切斷供氣，或直接派出「清理工」抽血抵債。
- **公轉化區 (Official Conversion Facility)**
由統治階級把持、母艦上唯一擁有合法外域資源轉化資格的巨大機構。它們用極高的苛捐雜稅剝削底下出生入死的傭兵與拾荒者。
- **二次黑箱抽取 (Secondary Blackbox Extraction)**
核心陰謀點。公轉化區收走物資後，對外宣稱用於全艦維修，實際上最純粹的「規則精華」會在中控室被秘密抽取，直接供給金字塔尖端的權貴們續命使用。
- **收編流程 (Assimilation Protocol)**
系統對待有潛力底層者的標準化抹殺流程：用高薪招募卸除防備 → 替換致命義體掌握生殺大權 → 強制洗腦重置人格，使其成為完美的暴力機器。
- **審計官 (The Auditors)**
母艦系統直轄的執法獵犬，屏棄了笨重義體、僅使用高階規則遺物武裝的生化人類。專門獵殺那些「成長曲線超出底層極限」的不安定變數。

## 3. 世界舞台與地理名詞

- **末日母艦 (The Doomsday Mothership)**
一艘在衰亡宇宙中航行了數千年的行星級流亡艦。船體外層正因為缺乏能源而不斷封閉、枯萎、剝落。
- **附掛都市 (The Parasitic Metro)**
依附在母艦外層鋼鐵骨架上建成的巨型賽博都市。由上到下分為：掌握陽光的「天光區」、充斥消費主義的「霓虹區」、以及主角所在的法外廢土「微光區」。
- **外域裂縫 (The Outer Rifts)**
母艦在躍遷期間短暫捕獲的異維度空間裂口。雖然充滿了違反認知的物理規則與極度危險的異變體，但卻是這艘船唯一能獲取全新資源、抵禦宇宙「熵蝕」的燃料庫。
- **近域拾荒帶 (Near-Orbit Scavenging Belt)**
母艦外圍剝落的廢棄艙段形成的無重力隕石帶。是底層拾荒者們用命換取銅牆鐵壁與廢土物資的第一線戰場。

## 4. 宇宙與怪物現象

- **熵蝕 (Entropy Erosion)**
整個大宇宙正在不可逆地邁向熱寂的過程。可居住的空間與可用的能量正在坍縮，逼迫所有文明必須採取極端手段（例如母艦的掠奪牧場機制）來苟延殘喘。
- **規則碎片 (Rule Fragments)**
外域裂縫中極有價值的產物。它們是高度凝結的高維資訊體，能夠短暫覆寫、扭曲現實的局部物理法則。
- **污染超標 (Over-Pollution Limit)**
當接觸過多外域資源且無法有效轉化時，碳基生物會發生嚴重的細胞崩潰與精神異變；即使是主角體內的微縮文明，一旦吸入超過承受上限的污染，也會導致信徒發瘋、社會失序的毀滅性災難。
