# 世界觀總綱（World Bible）

## 1. 核心命題

- 這是一個「文明吞噬文明」的世界，生存依賴持續掠奪外域。
- 母艦都市不是避難所，而是高層設計的轉化牧場。
- 主角的文明主機，讓他從被收割者轉向「反向收割規則利差」。

## 2. 主舞台結構

### 2.1 末日母艦

- 規模：跨世代流亡艦，含多圈層艙段。
- 現狀：能源老化、氧循環衰退、艙段逐步封閉。
- 真相：對外宣稱「文明保種計畫」，實際是高層維持權力的永續收割模型。

### 2.2 附掛都市（主故事舞台）

- 上層：董事會、審計司、轉化委員會。
- 中層：企業聯盟、承包軍、技術行會。
- 下層：維修工、拾荒隊、契約勞工、債務人口。
- 斷層規則：階層可流動但成本極高，多數上升通道是收編陷阱。

### 2.3 外域裂縫群

- 定義：母艦躍遷過程中可短時連結的異質宇域窗口。
- 特徵：高價值、高污染、高不確定死亡率。
- 功能：城市經濟燃料來源，也是高層挑選可用人才的屠宰場。

## 3. 資源與轉化規則

### 3.1 三層資源類型

1. **物質資源**：晶礦、殘骸、可拆解器件。
2. **規則資源**：法則碎片、場域核心、特殊現象模組。
3. **生命資源**：活樣本、群落胚種、文明記憶體。

### 3.2 官方轉化流程（對外版本）

1. 採集隊提交外域樣本。
2. 公轉化區檢定價值。
3. 系統抽稅與配給。
4. 餘額轉為個人配額與債務抵扣。

### 3.3 黑箱轉化流程（真實版本）

1. 前端採集資料即被標記。
2. 一次抽稅後進入「二次黑箱抽取」。
3. 高價值部分轉入高層私有內宇宙。
4. 底層拿到經稀釋後的生存份額。

## 4. 力量系統與雙線成長

### 4.1 外線：主角個體成長

- 來源：戰鬥、拾荒、黑市套利、規則碎片應用。
- 指標：生存權限、戰術模組、感知/算力增幅。
- 風險：成長過快會進入審計追蹤名單。

### 4.2 內線：文明主機成長與反哺 (外掛升級樹)

**階段 0：開機狀態（殘缺的種火）**
- **升級條件**：初始狀態。
- **內部生態**：荒蕪陸地與初火信徒（小泥人/數據幽靈）。
- **初始反哺**：
  - **【絕對消化】**：無害吞噬輻射垃圾，轉化率提升 200%。
  - **【微型儲物空間】**：1 立方公尺的安全走私空間。

**階段 1：種火時代的完整重燃**
- **升級條件**：吞噬 3 顆 R1 級異變體核心，或 1 件稀有規則遺物。
- **內部變化**：建立村落，打磨石器，產生微弱的信仰。
- **反哺能力**：
  - **【石核護甲】**：消耗信仰形成抵擋致命一擊的硬化立場。
  - **【危險感知】**：提前 3 秒預警惡意。

**階段 2：晉升神殿時代的門檻**
- **升級條件**：吞噬「體系完整的知識矩陣」與 10 單位純粹能量源。
- **內部變化**：形成祭司（解析）、工匠（打造）、戰士（戰鬥）三階級。
- **反哺能力**：
  - **【神權推演】**：戰鬥時 10 倍速推演敵人弱點（會導致大腦超載）。
  - **【物質具現 (初階)】**：短暫具現化工匠打造的內部武裝至現實世界。

*(後續時代如：鋼鐵時代、聖城時代、星環時代將於主線推進時陸續解鎖)*

### 4.3 反哺規則

- 內部文明每次完成「集體理解—集體實驗—集體儀式」，可產生反哺。
- 反哺型態：能力模組、裝備藍圖、規則權限、信仰護盾。
- 限制條件：若外部樣本污染超標，會觸發文明失序事件。

## 5. 社會秩序與壓迫機制

### 5.1 身份稅制與底層生存壓力
- **存活點 (Survival Credits)**：底層區域的通用貨幣，一單位等同於維持最底限維生的耗材（又稱「標準氧時 SOH」）。
- **每日固定生存稅（最低生存線 = 20存活點）**：
  - 基礎氧氣稅：10 點/天
  - 床位租金（底層膠囊艙）：5 點/天
  - 抗輻射廢水配額：5 點/天
- **開局極致壓榨**：主角餘額不足 20 點，若無法湊齊點數，隔日將面臨斷氧或被「清理工」強制抽血抵債。

### 5.2 社會控制手段
1. **人口補充**：高層持續投放低成本的複製人、債務人口，使底層無法透過怠工來談判。
2. **上升管道**：獎勵看似開放，實際綁定終身契約（成為「企業狗」或進行無法逆轉的賽博改造）。
3. **審計監控**：由演算法與審計官雙軌執法，任何不正常的物資流通或過快的實力躍昇都會被標記。
4. **收編流程**：對於高價值個體的標準SOP：招募—試忠誠—奪權限—替換人格。

## 6. 母艦生存方程式

> 定義：高層必須維持「外域養殖 → 轉化供給 → 艙段維穩」的閉環，否則母艦會在可計算時間內進入不可逆崩潰。

### 6.1 硬約束一：能源衰退速率

- **約束定義**：主反應爐每月自然衰退約 3.2%，必須靠外域能量晶核與規則燃料補齊，否則艙段供能會連鎖降載。
- **若停止養殖（停止外域能量輸入）**：
  - **72 小時**：進入二級限電，非核心艙段照明、電梯、淨水改為輪供，底層治安與醫療崩壞。
  - **30 天**：主維生系統降至危險閾值，氧循環效率跌破安全線，出現艙段級缺氧死傷與暴動。
  - **180 天**：主反應爐無法維持最小躍遷與保溫負載，母艦大量艙段永久失能，文明退化為局部據點求生。

### 6.2 硬約束二：艙段維持最低人口

- **約束定義**：每一艙段需維持最低運作人口（工程、醫護、安保、回收）約 62%，低於門檻將失去自維持能力。
- **若停止養殖（停止外域資源與人口補充）**：
  - **72 小時**：排班立即斷層，維修與安保同時缺員，事故率上升並迫使高層啟用緊急徵召。
  - **30 天**：關鍵工種過勞與死亡造成技能斷代，多個艙段進入「名義開放、實際封艙」狀態。
  - **180 天**：低於最低人口門檻的艙段全面功能退化，形成不可回收死艙，母艦社會分裂為資源堡壘與棄民帶。

### 6.3 硬約束三：外域污染處理上限

- **約束定義**：公轉化區每日污染淨化上限固定，超量污染會累積至內循環；若無持續養殖收益，無法支付高成本淨化流程。
- **若停止養殖（停止外域樣本篩選與污染對沖）**：
  - **72 小時**：污染緩衝池達上限，低階淨化設備超載，底層首先出現輻射病與突變症狀。
  - **30 天**：污染回灌氧水循環，醫療體系進入分級放棄模式，非生產人口被系統性剔除。
  - **180 天**：艙內生態鏈崩潰，母艦轉為長期高毒環境，高層只能在少數潔淨區維持統治。

### 6.4 推演意義

- 高層的殘酷政策並非單純邪惡，而是以「延後崩潰時間」為核心目標。
- 任何主角提出的替代秩序，都必須同時回答能源、人口、污染三條硬約束。
- 因此政治鬥爭本質是：誰能提出更長期、代價更低的生存方程式。

## 7. 故事階段對應世界揭密

- 第一卷：揭露「黑箱二次抽取」與「人口批次投放」。
- 第二卷：揭露「高層內宇宙養殖鏈」與母艦艙段淘汰機制。
- 第三卷：揭露母艦航行真正目的與外域裂縫的人為性。
- 長線終局：主角文明是否複製同樣壓迫，或建立新秩序。

## 8. 可持續擴寫接口

- 每新增一個外域地圖，需定義：資源、污染、文明遺留、劇情鉤子。
- 每新增一個勢力，需定義：收入來源、壟斷手段、對主角態度。
- 每新增一項能力，需標註來源（外線/內線）與代價。
