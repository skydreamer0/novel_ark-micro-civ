# 世界觀總綱（World Bible）

> *「在這艘船上，每一口乾淨的氧氣，都散發著另一個文明燃燒殆盡的焦味。而我們，不過是負責添柴的柴薪。」*
> —— 底層微光區，某個死於輻射病的拾荒者遺言

## 1. 核心命題與基調

- **文明吞噬文明的宇宙悖論**：這個宇宙正在經歷不可逆的「熵蝕」衰亡。唯一延續存活的方法，就是撕裂空間、掠奪其他維度的外域樣本，並將其榨乾。
- **母艦都市的謊言**：對外宣稱是「文明保種計畫最後的方舟」，實際上卻是高層精心打造的**轉化牧場**。底層人類的掙扎與死亡，只是這台巨大絞肉機運轉時的潤滑油。
- **主角的反噬權限**：憑藉體內的「文明主機」，主角從一個被系統隨意收割的底層耗材，轉變為**「反向收割系統利差」**的寄生神明。

## 2. 主舞台結構與階級斷層

### 2.1 終焉方舟：末日母艦
- **規模與現況**：一艘體積堪比月球的跨世代流亡巨艦。由於長達數千年的躍遷與能量老化，母艦的外圍艙段正逐漸被封閉、拋棄。空氣循環機組發出沉重的悲鳴，每一秒都在提醒所有人：末日並未結束，只是被拉長了。
- **絕對隔離**：核心區由最高議會把持，底層人終其一生連核心區的一束光都見不到。

### 2.2 附掛都市（主故事舞台）
依附在母艦中後段外殼上的金屬違建群，猶如寄生在巨鯨身上的藤壺。
- **上層（天光區）**：董事會、審計司、轉化委員會的所在地。享受著模擬人造陽光、純淨的植物園與無限的生命配額。
- **中層（霓虹區）**：企業聯盟、承包軍、技術行會。這裡是裝滿無人生產線與高階義體醫院的消費煉獄，充滿著虛假的繁榮。
- **下層（微光區與艙底）**：維修工、拾荒隊、契約勞工、債務人口的聚集地。充斥著機油味、漏電閃爍的燈管、以及無處不在的幫派火拼。
- **階級流動的騙局**：看似有上升通道（如立功受封、受雇企業），實際上所有通道盡頭都是「收編陷阱」：摘除原生器官、植入控制晶片，成為徹底的企業走狗。

### 2.3 外域裂縫群（高危燃料庫）
- **定義**：母艦在深空躍遷過程中，短暫捕獲的異質宇域窗口。
- **特徵**：充滿致命的空間扭曲、違反物理常識的異變體，以及高強度的未知污染。
- **本質**：這是驅動母艦運轉的唯一燃料來源。高層故意將裂縫座標開放給底層，把下層傭兵與拾荒者當作「人肉過濾器」與「探雷器」。

## 3. 資源與黑箱轉化規則

### 3.1 三層資源維度
外域的產物不僅僅是礦物，它們帶有瘋狂的資訊與能量：
1. **物質資源**：高維礦石、外域戰艦殘骸、可拆解的奇異器件。
2. **規則資源（高價值）**：法則碎片、場域核心、能扭曲局部物理的異常模組。
3. **生命資源（極危險）**：活體樣本、異星群落胚種、甚至是破碎的文明記憶體。

### 3.2 剝削機制的極致：二次黑箱抽取
- **表層流程（官方版本）**：底層拾荒者九死一生帶回樣本 → 進入「公轉化區」鑑定價值 → 系統抽取高達 80% 的「設施維護稅」 → 剩下的殘渣折算成微薄的【存活點】發給拾荒者。
- **裡層真相（黑箱流程）**：系統收走的 80% 並沒有用於維持底層生存。它們會進入「二次黑箱」，將最核心的「規則與生命精華」提純，直接注入高層大人物們私有的「內宇宙」中，作為他們續命與封神的養料。

## 4. 力量系統與雙線成長機制

本作最大的特點在於主角在「現實肉體」與「體內神明」兩個身份間的雙軌成長。

### 4.1 外線：肉體進化與裝備迭代（賽博拾荒者）

- **成長指標**：義體/生化改造度、戰術模組庫、神經反應速度、防禦力場階級。
- **達摩克利斯之劍**：一旦主角的表現「超出了窮人該有的極限」，就會觸發系統的異常警報，引來「審計司」的無情抹殺。因此主角必須學會**藏拙與偽裝**。

  - 膠囊床位租賃稅：5 點
