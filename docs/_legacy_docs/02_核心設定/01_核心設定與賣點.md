# 核心賣點與商業鉤子 (Pitch & Hooks)

這份文件是在寫作卡關、或者不知道接下來該寫什麼的時候，用來「校準大方向」的指北針。
任何偏離以下三大賣點的劇情，都應該被砍掉或縮減。

---

## 1. 一句話簡介 (Elevator Pitch)
_（請確保任何介紹這本書的文案都能涵蓋這些元素）_

> **在吃人的末日母艦底層，穿越成負債維修工的林燼，靠著體內一台能「養殖微縮文明」的神秘主機，瘋狂掠奪宇宙裂縫資源，一步步從底層螻蟻反向吞噬整個統治階級的賽博修仙/升級大作。**

## 2. 核心三大爽點 (The Big 3 Hooks)

1. **【降維打擊的資源轉化率】(外掛爽點)**
   *   別人拿命換回來的污染資源，只能被官方抽走 80% 換一點點氧氣。
   *   主角的「微縮文明」能百分之百吸收污染，並把裡面的規則碎片直接變成主角的專屬超能力/裝備。**這種「別人眼裡的垃圾是我眼裡的極品」的撿漏感，是前期最大的爽點。**

2. **【從「求生壓迫」到「極致反殺」】(情緒爽點)**
   *   前一秒：被 40 點生存稅逼到走頭無路，被幫派踩在腳底下。
   *   後一秒：在裂縫中借主機之力反殺所有算計他的人，帶著滿滿一背包的隱藏資源回來，看著終端機上的存款數字暴漲。**先抑後揚的打臉感。**

3. **【群星級的文明養成】(長線養成爽點)**
   *   看著體內的文明從「泥人鑽木取火」，一路進化到「賽博神權」、「星環戰艦」。
   *   **信徒的狂熱崇拜與主角在現實中的冷酷無情形成強烈對比。** 體內文明的每次大階級突破，都必定對應主角在現實中的一次大規模屠殺或階級躍昇。

## 3. 核心衝突 (Core Conflict)

*   **表層衝突**：主角缺氧氣、缺錢、缺裝備，必須不斷進入高危險的「外域裂縫」與其他勢力爭搶資源。
*   **深層衝突**：母艦高層本質上也在玩「養殖文明」，主角越強大，就越容易被高層這張巨大的「捕蚊網」給盯上。主角必須在「苟住發育」和「必須掠奪才能升級」之間走鋼索。

## 4. 定位標籤與對標作品
*   **類型標籤**：賽博龐克、宇宙生存、外域掠奪、文明養成、殺伐果斷。
*   **對比風格**：帶有《詭秘之主》的升級神秘感 ＋《夜的命名術》的底層賽博群像 ＋《虧成首富》那種「每次結算收穫都爆表」的數值爽感。
