# 母艦與都市結構

> 來源：從 `03_世界觀總綱.md` §2 獨立出來。

---

## 1. 末日母艦

- 規模：跨世代流亡艦，含多圈層艙段。
- 現狀：能源老化、氧循環衰退、艙段逐步封閉。
- 真相：對外宣稱「文明保種計畫」，實際是高層維持權力的永續收割模型。

## 2. 附掛都市（主故事舞台）

- 上層：董事會、審計司、轉化委員會。
- 中層：企業聯盟、承包軍、技術行會。
- 下層：維修工、拾荒隊、契約勞工、債務人口。
- 斷層規則：階層可流動但成本極高，多數上升通道是收編陷阱。

## 3. 外域裂縫群

- 定義：母艦躍遷過程中可短時連結的異質宇域窗口。
- 特徵：高價值、高污染、高不確定死亡率。
- 功能：城市經濟燃料來源，也是高層挑選可用人才的屠宰場。

> 裂縫分類標準與詞綴化規則詳見 `02_場景與生態危機.md`。

## 4. 母艦生存方程式

> 定義：高層必須維持「外域養殖 → 轉化供給 → 艙段維穩」的閉環，否則母艦會在可計算時間內進入不可逆崩潰。

### 4.1 硬約束一：能源衰退速率

- **約束定義**：主反應爐每月自然衰退約 3.2%，必須靠外域能量晶核與規則燃料補齊，否則艙段供能會連鎖降載。
- **若停止養殖（停止外域能量輸入）**：
  - **72 小時**：進入二級限電，非核心艙段照明、電梯、淨水改為輪供，底層治安與醫療崩壞。
  - **30 天**：主維生系統降至危險閾值，氧循環效率跌破安全線，出現艙段級缺氧死傷與暴動。
  - **180 天**：主反應爐無法維持最小躍遷與保溫負載，母艦大量艙段永久失能，文明退化為局部據點求生。

### 4.2 硬約束二：艙段維持最低人口

- **約束定義**：每一艙段需維持最低運作人口（工程、醫護、安保、回收）約 62%，低於門檻將失去自維持能力。
- **若停止養殖（停止外域資源與人口補充）**：
  - **72 小時**：排班立即斷層，維修與安保同時缺員，事故率上升並迫使高層啟用緊急徵召。
  - **30 天**：關鍵工種過勞與死亡造成技能斷代，多個艙段進入「名義開放、實際封艙」狀態。
  - **180 天**：低於最低人口門檻的艙段全面功能退化，形成不可回收死艙，母艦社會分裂為資源堡壘與棄民帶。

### 4.3 硬約束三：外域污染處理上限

- **約束定義**：公轉化區每日污染淨化上限固定，超量污染會累積至內循環；若無持續養殖收益，無法支付高成本淨化流程。
- **若停止養殖（停止外域樣本篩選與污染對沖）**：
  - **72 小時**：污染緩衝池達上限，低階淨化設備超載，底層首先出現輻射病與突變症狀。
  - **30 天**：污染回灌氧水循環，醫療體系進入分級放棄模式，非生產人口被系統性剔除。
  - **180 天**：艙內生態鏈崩潰，母艦轉為長期高毒環境，高層只能在少數潔淨區維持統治。

### 4.4 推演意義

- 高層的殘酷政策並非單純邪惡，而是以「延後崩潰時間」為核心目標。
- 任何主角提出的替代秩序，都必須同時回答能源、人口、污染三條硬約束。
- 因此政治鬥爭本質是：誰能提出更長期、代價更低的生存方程式。
