- 來源：戰鬥、拾荒、黑市套利、規則碎片應用。
- 指標：生存權限、戰術模組、感知/算力增幅。
- 風險：成長過快會進入審計追蹤名單。

## 3. 反哺規則

- 內部文明完成「集體理解 → 集體實驗 → 集體儀式」，可產生反哺。
- 反哺型態：能力模組、裝備藍圖、規則權限、信仰護盾。
- 限制：外部樣本污染超標，會觸發文明失序事件。

## 4. 反哺失真機制

1. **失真來源**：當內部信仰被錯誤敘事主導。
2. **外部偏差表現**：
   - 防禦型：過度僵硬，降低機動。
   - 感知型：敵我不分的威脅標記。
   - 具現型：高耗能炫技，無法持久。
3. **累積後果**：主機過載率提升，「能力可用但戰術失靈」的假性成長。
4. **修正路徑**：校準神諭、公開驗證儀式、替換失職祭司、重建教育文本。
