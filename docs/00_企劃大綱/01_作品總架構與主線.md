# 賽博宇宙小說架構設計

## 文件目的

這份文件整理目前已確認的世界觀與故事架構，作為後續共創的主參考。原則是只寫已定下來的內容，不強行補完還沒決定的細節。

## 一句話定位

在一個持續衰亡、居住空間不斷縮小的宇宙裡，人類被迫依靠掠奪外域資源維持生存。主角作為穿越而來的底層維修工與拾荒者，得到能養殖文明、吸收信仰、加速煉化的文明主機，從此走上一條和頂層牧場主同源卻不完全相同的崛起之路。



## 故事總架構

故事主骨架是「掠食文明線」，負責主要爽感：掠奪、偷資源、快速升級、反殺對手、養大文明、踩碎高層安排。暗線是「黑箱革命線」，負責逐步揭露母艦、高層、裂縫與宇宙牧場的真正結構。前期以升級和掠奪為主，中後期逐步打開更大的真相與格局。

## 連載續航機制

為了保證故事不會寫幾十章就失速，後續每一卷都應該盡量維持同一套推進節奏：

1. 城市生存壓力上升。
2. 新的外域機會或投放事件出現。
3. 主角被迫或主動進場掠奪。
4. 戰利品投入內部文明完成煉化。
5. 主角用新能力反打當前敵人或制度壓制。
6. 再揭露更高一層的統治真相或新的危機。

只要每次大事件都能同步推進戰力線、文明線與真相線，這本書就能維持穩定的爽感和連續性。

## 目前已確定的重要設定

- 主角是穿越者。
- 主角開局身分是底層維修工與拾荒者。
- 主舞台是依附末日母艦的巨型賽博都市。
- 世界存在真正的生存危機，所有人都被逼著吸收外域資源維生。
- 高層會大量投放複製人與內宇宙人口，製造過飽和競爭。
- 高層靠公轉化區與多重稅制鎖住所有人的上升空間。
- 主角最大的不同點是擁有雙層煉化權限。
- 主角的內部文明以信仰參與煉化，並走五時代成長。
- 故事主線是掠食文明，暗線是黑箱革命與宇宙牧場真相。
