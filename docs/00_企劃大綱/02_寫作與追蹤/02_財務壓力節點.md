# 每 5 章財務壓力節點（第一卷）

> 用途：寫作時快速查閱主角的財務壓力線，確保經濟張力持續上升。

| 章段      | 破產線（低於即觸發強制清算）       | 翻身線（達標可主動擴張）          | 暴露線（超過即進高風控）  |
| --------- | ---------------------------------- | --------------------------------- | ------------------------- |
| 1～5 章   | ≤ -60 存活點（含欠稅）             | ≥ +20 存活點（可續租與補裝）      | 單日異常淨利 ≥ 35 點      |
| 6～10 章  | ≤ -80 存活點（含醫療欠款）         | ≥ +60 存活點（可建立首個安全庫）  | 3 日累計淨利 ≥ 120 點     |
| 11～15 章 | ≤ -100 存活點（連帶隊伍債）        | ≥ +120 存活點（可擴編兩人小隊）   | 單次任務毛利率 ≥ 220%     |
| 16～20 章 | ≤ -140 存活點（含封鎖期罰息）      | ≥ +180 存活點（可承受封鎖 1 輪）  | 被標記收益來源 ≥ 2 條     |
| 21～25 章 | ≤ -180 存活點（據點損失入帳）      | ≥ +260 存活點（可重建據點）       | 高風險樣本占收益 ≥ 55%    |
| 26～30 章 | ≤ -220 存活點（進入收編/清除名單） | ≥ +360 存活點（具備主動開戰本錢） | 7 日累計異常收益 ≥ 480 點 |
