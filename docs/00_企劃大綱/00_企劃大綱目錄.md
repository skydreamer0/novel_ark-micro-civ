# 企劃大綱目錄

此資料夾用於集中管理《末日母艦：我的體內有個微縮文明》的專案架構與大綱。

## 檔案導覽

- `01_故事主線與卷軸規劃.md`：故事整體架構、核心衝突與各卷大目標。
- `02_系列階段發展.md`：定義各卷的升級與揭密節奏，長線規劃。
- `03_第一卷大綱.md`：第一卷的詳細劇情規劃、引爆事件與節奏設計。
- `06_既有章節不改_增量擴寫方案.md`：針對舊稿件的擴寫策略與修改指南。
- `07_伏筆追蹤表.md`：1～42 章已埋伏筆的回收狀態追蹤。

## 維護規範

- **大綱與架構**：若長線劇情有變動，請優先更新 `01_故事主線與卷軸規劃.md` 與 `02_系列階段發展.md`。
- **卷首規劃**：每開啟新的一卷，應建立對應的卷大綱（如 `04_第二卷大綱.md`）。
- **寫作指南**：參與共創或生成正文時，務必遵循專案根目錄的 `NOVEL_WRITING_RULES.md`，以維持統一的賽博龐克廢土調性。
