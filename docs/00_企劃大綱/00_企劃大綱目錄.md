# 企劃大綱目錄

此資料夾用於集中管理《末日母艦：我的體內有個微縮文明》的專案架構與大綱。

## 檔案導覽

### 核心架構 (根目錄)
- `01_作品總架構與主線.md`：故事整體架構、核心衝突與各卷大目標。
- `02_全書階段與揭密藍圖.md`：定義各卷的升級與揭密節奏，長線規劃。

### 01_分卷大綱
負責存放各卷的詳細劇情規劃與章節節奏設計：
- `01_第一卷_艙底種火.md`：1～30 章的詳細劇情規劃、引爆事件與節奏設計。
- `02_第二卷_裂縫稅戰.md`：31～60 章的章回大綱與衝突設計。
- `03_第三卷_聖城試煉.md`：61～90 章的跨艙段流亡、鋼鐵時代啟動與高層養殖實證。

每份卷綱統一採用 **§1~§7** 七段式結構：
> §1 卷定位 → §2 幕次劇情 → §3 角色弧 → §4 能力路線 → §5 揭密管理 → §6 一句話總結 → §7 章回落地卡

### 02_寫作與追蹤
存放與實際寫作執行相關的文件與規定：
- `01_伏筆追蹤表.md`：已埋伏筆的回收狀態追蹤。
- `02_財務壓力節點.md`：第一卷的每 5 章財務壓力參數。

## 維護規範

- **大綱與架構**：若長線劇情有變動，請優先更新根目錄的「核心架構」兩份文件。
- **卷首規劃**：每開啟新的一卷，應在 `01_分卷大綱/` 建立對應的卷大綱（如 `03_第三卷_聖城試煉.md`），使用 §1~§7 的統一結構。
- **寫作指南**：參與共創或生成正文時，務必遵循專案根目錄的 `NOVEL_WRITING_RULES.md`，以維持統一的賽博龐克廢土調性。
