# 世界觀總綱（World Bible）

## 1. 核心命題

- 這是一個「文明吞噬文明」的世界，生存依賴持續掠奪外域。
- 母艦都市不是避難所，而是高層設計的轉化牧場。
- 主角的文明主機，讓他從被收割者轉向「反向收割規則利差」。

## 2. 主舞台結構

### 2.1 末日母艦

- 規模：跨世代流亡艦，含多圈層艙段。
- 現狀：能源老化、氧循環衰退、艙段逐步封閉。
- 真相：對外宣稱「文明保種計畫」，實際是高層維持權力的永續收割模型。

### 2.2 附掛都市（主故事舞台）

- 上層：董事會、審計司、轉化委員會。
- 中層：企業聯盟、承包軍、技術行會。
- 下層：維修工、拾荒隊、契約勞工、債務人口。
- 斷層規則：階層可流動但成本極高，多數上升通道是收編陷阱。

### 2.3 外域裂縫群

- 定義：母艦躍遷過程中可短時連結的異質宇域窗口。
- 特徵：高價值、高污染、高不確定死亡率。
- 功能：城市經濟燃料來源，也是高層挑選可用人才的屠宰場。

## 3. 資源與轉化規則

### 3.1 三層資源類型

1. **物質資源**：晶礦、殘骸、可拆解器件。
2. **規則資源**：法則碎片、場域核心、特殊現象模組。
3. **生命資源**：活樣本、群落胚種、文明記憶體。

### 3.2 官方轉化流程（對外版本）

1. 採集隊提交外域樣本。
2. 公轉化區檢定價值。
3. 系統抽稅與配給。
4. 餘額轉為個人配額與債務抵扣。

### 3.3 黑箱轉化流程（真實版本）

1. 前端採集資料即被標記。
2. 一次抽稅後進入「二次黑箱抽取」。
3. 高價值部分轉入高層私有內宇宙。
4. 底層拿到經稀釋後的生存份額。

## 4. 力量系統與雙線成長

### 4.1 外線：主角個體成長

- 來源：戰鬥、拾荒、黑市套利、規則碎片應用。
- 指標：生存權限、戰術模組、感知/算力增幅。
- 風險：成長過快會進入審計追蹤名單。

### 4.2 內線：文明主機成長

- 種火時代：建立神話與共同敘事。
- 神殿時代：形成祭司—工匠—戰士分工。
- 鋼鐵時代：神權與技術融合，擴大煉化效率。
- 聖城時代：跨區域秩序與軍團化。
- 星環時代：具備對外殖民與分封能力。

### 4.3 反哺規則

- 內部文明每次完成「集體理解—集體實驗—集體儀式」，可產生反哺。
- 反哺型態：能力模組、裝備藍圖、規則權限、信仰護盾。
- 限制條件：若外部樣本污染超標，會觸發文明失序事件。

## 5. 社會秩序與壓迫機制

1. **身份稅制**：生存稅、艙段稅、氧配額稅、戰備稅。
2. **人口補充**：複製人、債務人口、投放批次人口。
3. **上升管道**：獎勵看似開放，實際綁定終身契約。
4. **審計監控**：由演算法與審計官雙軌執法。
5. **收編流程**：招募—試忠誠—奪權限—替換人格。

## 6. 故事階段對應世界揭密

- 第一卷：揭露「黑箱二次抽取」與「人口批次投放」。
- 第二卷：揭露「高層內宇宙養殖鏈」與母艦艙段淘汰機制。
- 第三卷：揭露母艦航行真正目的與外域裂縫的人為性。
- 長線終局：主角文明是否複製同樣壓迫，或建立新秩序。

## 7. 可持續擴寫接口

- 每新增一個外域地圖，需定義：資源、污染、文明遺留、劇情鉤子。
- 每新增一個勢力，需定義：收入來源、壟斷手段、對主角態度。
- 每新增一項能力，需標註來源（外線/內線）與代價。
