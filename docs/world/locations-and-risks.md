# 場景分區與風險配置

## 1. 艙內場景

### 1.1 艙底維修帶
- 調性：潮濕、漏電、擁擠、治安失控。
- 劇情用途：開場生存壓力、債務倒數、黑市接單。
- 典型風險：幫派抽成、設備事故、假藥流通。

### 1.2 公轉化區
- 調性：秩序森嚴、程序化、半軍管。
- 劇情用途：提交戰利品、數據作假、揭露黑箱抽取。
- 典型風險：審計抽查、樣本失控、資產凍結。

### 1.3 中層企業圈
- 調性：霓虹奢華外皮下的契約剝削。
- 劇情用途：談判、背叛、資源置換。
- 典型風險：條款陷阱、雇傭兵清算、信用封鎖。

## 2. 艙外場景

### 2.1 近域拾荒帶
- 調性：低至中風險、資訊透明度低。
- 劇情用途：初期累積資源、建立隊伍默契。
- 典型風險：輻射殘留、屍潮型異變體、同行搶奪。

### 2.2 裂縫投放帶
- 調性：高風險高回報，時間窗口短。
- 劇情用途：關鍵升級、主機截流、反派追獵。
- 典型風險：時空錯位、規則污染、定位暴露。

### 2.3 深層外域遺跡
- 調性：極高風險，常伴文明殘響。
- 劇情用途：卷末大戰、真相碎片、能力躍遷。
- 典型風險：認知污染、地圖失效、團隊精神崩潰。

## 3. 風險等級定義

- R1（可控）：常規裝備可處理。
- R2（警戒）：需隊伍分工與撤離預案。
- R3（致命）：需消耗底牌，傷亡率高。
- R4（禁入）：通常僅高層授權隊伍可進入。

## 4. 場景使用節奏建議

1. 每 3 章至少切換一次場景層級（艙內/艙外）。
2. 每 5 章至少出現一次 R3 以上事件，兌現壓迫感。
3. 卷末場景建議同時滿足：高風險 + 高揭密 + 高收益。
