# 勢力與關鍵角色設定

## 1. 權力金字塔

### 1.1 高層：母艦董事會
- 目標：維持能源與人口供應，延長統治壽命。
- 手段：法規壟斷、轉化壟斷、資訊黑箱。
- 與主角關係：潛在敵對，視主角為未授權載體。

### 1.2 執行層：審計司
- 目標：篩查異常成長者並完成收編。
- 手段：資料追蹤、資產凍結、合法清除。
- 與主角關係：第一個中長期對抗勢力。

### 1.3 經濟層：企業聯盟
- 目標：承包外域作業，壟斷裝備與藥劑。
- 手段：債務綁定、合約陷阱、黑市投機。
- 與主角關係：可交易也可背刺，屬灰色敵友。

### 1.4 基層武裝：艙底幫派
- 目標：控制下層物流與暴力秩序。
- 手段：保護費、走私、情報販售。
- 與主角關係：前期主要壓迫來源與升級踏板。

## 2. 主角陣營（第一卷可用）

### 2.1 主角：林燼（暫名）
- 起點：維修工兼拾荒者。
- 稀缺能力：文明主機雙層煉化權限。
- 主要矛盾：要活下去就得吞噬規則，但吞噬越多越接近高層。

### 2.2 戰鬥位盟友
- 功能：正面火力、護衛撤離、關鍵時刻替主角扛壓。
- 人物弧：從拿錢辦事轉為共同生存。

### 2.3 技術位盟友
- 功能：拆解、入侵、偽造資料、關閉監控。
- 人物弧：由功利合作走向價值綁定。

### 2.4 可選醫療位盟友
- 功能：外傷處置、污染抑制、心理緩衝。
- 人物弧：在救人與資源不足之間承擔道德衝突。

## 3. 反派階梯（第一卷）

1. 幫派頭目：壓迫主角生存空間，製造第一層血仇。
2. 企業代理人：提供高回報陷阱，逼主角站隊。
3. 審計官：代表制度暴力，啟動收編或清除流程。

## 4. 勢力互動模板（供寫作時套用）

- 幫派 × 企業：以暴力維持灰色供應鏈。
- 企業 × 審計司：以合規名義完成利益輸送。
- 審計司 × 董事會：上呈「風險指標」決定清除優先級。
- 主角團 × 各方：以資訊差與機動性撕開縫隙。
