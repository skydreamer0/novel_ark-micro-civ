# 第34章：保證金陷阱

影子配額跑通第二票後，顧衡沒有立刻收網，反而故意放慢追擊。

這是更危險的信號。

林燼盯著監控牆上的稅流曲線，越看越不對。新稅契上線本該讓小隊大量破產，但數據顯示破產速度被刻意壓平，像有人在控制「死得不要太快」。

燕九很快抓到一條內部推送：審計署將啟用「保證金互保池」。表面上是保護小隊，實際上是把多支小隊綁在同一風險池，一隊出事、全池補差。這意味著顧衡可以只打最弱一支，抽乾整池，逼剩下的人自願投靠代理。

「他在做金融版的人質鏈。」燕九說。

林燼把白板上的灰鏈流程劃掉一半重畫：「那我們就先劫他的池。」

目標定在 B-11 互保池結算節點，一座掛在空中軌道下方的半封閉金流站。每天午夜，該站會把三十六支配額隊伍的保證金波動彙總上傳。只要能改寫一次結算標記，就能把「事故責任」從整池分攤改成局部凍結，替至少六支小隊爭出一週喘息。

問題是，這種結算節點不是拿槍能解。

燕禾提出一個冒險方案：假冒「池內家屬申訴團」。互保池最近連續扣款引發民怨，只要帶著足夠真實的求助名單和醫療帳單，能逼值班審核開人工窗口。人工窗口一開，燕九就有機會把補丁塞進內網。

蘇岚聽完皺眉：「你要我們演弱者？」

「不是演，」燕禾說，「那些名單是真的。只是我們把真的痛，拿來撞開假的門。」

行動當晚，林燼換上破舊工服，背著一袋氧片空殼；燕禾披著醫療站外套，手裡抱滿病歷本。兩人站在結算站外，和十幾名真正來申訴的居民混在一起。蘇岚與燕九潛伏在高架陰影，等窗口開啟。

「你緊張嗎？」燕禾小聲問。

「比在裂縫裡還緊張。」林燼回答，「在裂縫裡知道誰要殺你，這裡不知道。」

零點十分，窗口終於開。值班審核員是個年輕女人，眼下青黑，說話機械：「按序遞件，無關人員後退。」

燕禾把第一份病歷推上去，聲音不高，卻字字清楚：「這孩子的抗凝療程被扣到斷藥，理由是‘互保池事故補差’。事故不是他家的，藥卻是他停。」

後面人群開始躁動，更多帳單被舉起。窗口瞬間從審核點變成壓力鍋。

值班員頂不住，只能打開人工複核通道。那一刻，燕九在暗頻低聲：「門開了，給我九十秒。」

九十秒裡，蘇岚在外圍製造一場小規模配電故障，讓監控畫面短暫降幀；林燼在窗口前故意與保全爭執，把注意力全拉到自己身上；燕禾持續遞件，逼審核員不敢關窗。燕九則像外科手術，把一段「事故責任延遲凍結」補丁打進結算腳本。

第八十七秒，補丁寫入成功。

第八十八秒，警報燈亮。

有人發現內網異常。

場面立刻失控，保全開始驅離。林燼拉著燕禾衝出人群，蘇岚從高架上打斷兩盞追照燈，燕九關閉了自己留下的最後一個連線痕跡。四人分三路撤離，最後在廢橋下會合。

撤回藏點後，燕九手還在抖：「補丁只能撐七天，而且只能凍結六支隊伍。」

「七天就夠。」林燼說，「夠我們去談更多影子配額，夠六支隊伍不在今晚破產，夠顧衡多花七天找不到到底哪裡被改。」

天亮前，結果回傳。

B-11 池內六支小隊的扣款被標記「待複核」，對應家屬醫療款暫停追繳。三個社區很快傳來消息：原本要被停掉的兩處臨時藥點恢復供應，民間互助網開始主動幫忙轉運。

這是灰鏈第一次在「不搶物資」的情況下直接改變制度後果。

但顧衡當天下午就反制：

> 「互保池申訴窗口改為遠端排號，現場申訴即刻取消。」

燕禾看著公告，眼裡有怒，也有一點疲倦：「他每次都學得很快。」

林燼點頭：「我們也得更快。下一步不是再撞窗口，是要找出誰在寫他的結算腳本。」

章末，燕九把一段被忽略的程式簽名放大：`QW-Delta-17`。

這個簽名曾在第30章二級收割預案附件裡出現過。

意味著，顧衡的稅戰與收割預案，背後是同一支手。

灰鏈不再只是在城市裡偷生，它開始摸到對手真正的骨頭。
