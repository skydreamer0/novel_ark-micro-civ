# 第38章：托養池外

托養池不在地圖上。

它在官方系統裡被叫「青苗照護中心」，實際是分流前的暫存倉。孩子在裡面按分數被分配：高分進教育線，中分進維護線，低分直接進低耗工序。低耗工序的另一個名字，是慢性消失。

卓汐女兒叫小岑，九歲，檔案評語只有一句：`注意力偏移，配合度不足。`

燕禾看到這句話時眼眶發紅：「這句話可以寫在任何不肯乖乖耗掉自己的人身上。」

托養池防線不重，但程序鐵。出入都需雙簽：照護員簽章 + 分流系統即時核驗。硬搶能成功一次，後果是整池封鎖、其他孩子分流加速。

林燼不接受這種勝利。

他們設計的是「合法撤離偽裝」：以醫療轉診名義帶出小岑，同時替另外兩名高危評估孩子做延遲標記，爭取後續營救窗口。

燕九負責核驗端，燕禾負責照護流程對話，林燼與蘇岚做外圍保護。真正關鍵在卓汐——她必須在館內值班時把一段轉診授權插進上游，否則任何偽裝都會被秒拆。

行動下午開始。托養池大門外排著來看孩子的家屬，卻只有少數能真的見到人。更多人只是交費、簽字、再等通知。

燕禾穿白外套走進接待區，聲音平穩：「青苗中心第七區，三名兒童轉急性凝血風險評估，需送西環醫療點。」

接待員皺眉：「沒收到預約。」

燕禾把終端推過去，裡面是卓汐剛塞進去的加急單。接待員掃碼，綠燈。

第一關過。

第二關在內層核驗閘。系統突發抽查，要求照護員本人二次生物識別。燕九在外面看到跳碼，臉色變了：「有人盯著這單。抽查不是隨機。」

林燼低聲：「顧衡？」

「不像，」燕九說，「更像上游有人在看卓汐。」

時間被壓縮到窒息。若二次識別過不了，整單作廢，卓汐也會暴露。

燕禾突然做了個誰也沒預料的動作——她把自己的手按在識別板上，報出一串鎖定池內部值班暗語。那是她在被收容那段日子裡記下的流程漏洞：某些時段，系統會接受「輪值照護代理」識別，只要暗語與當日值班碼匹配。

三秒後，閘門亮綠。

蘇岚在耳機裡爆了句粗口：「你怎麼知道這還能用？」

燕禾回答很淡：「因為他們懶，懶得改那些不影響抽成的舊規。」

小岑被帶出來時，比照片更瘦，眼睛卻很亮。她看見卓汐沒在場，先是警惕，直到燕禾蹲下來把一枚小螺帽放到她掌心——那是卓汐說過的母女暗號。小岑才點頭，乖乖跟走。

離場最後一步，外圍突然響起警報。不是托養池內警報，是街區風控警報。顧衡剛下發臨時抽查，封鎖半徑擴到三條街。

「提前了。」林燼咬牙。

蘇岚帶著孩子們走下水道備線，林燼留下斷後。他在巷口連續切掉兩台巡防無人機，聲響把追兵拉向自己。燕九趁亂把兩名未能帶出的孩子標記成「需隔離觀察」，硬生生把她們的分流時間往後推了七十二小時。

撤回藏點後，小岑第一句話是：「媽媽說你們會很兇，可你們看起來比較累。」

蘇岚笑了一下，笑得發苦：「兇也是很累的一種。」

晚上，卓汐在北弧館收到女兒安全抵達的暗碼，回傳只有一個字：`謝`。

但同一時間，顧衡發布追查令：

> 「青苗中心轉診單遭篡改，涉案團隊與`灰鏈`相關。」

這道追查令把灰鏈從「稅流干擾者」升級成「人口流程破壞者」。

林燼看完，沒有後悔。他只在板上把原本的下一步改掉：

`39：入口前置清場` 被劃掉，改成 `39：先守人，再奪入口`。

因為他突然意識到，對方真正恐懼的不只是物資被搶，而是有人開始把「被分流的人」重新當作人。
