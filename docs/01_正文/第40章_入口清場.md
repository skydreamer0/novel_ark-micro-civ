# 第二卷《裂縫稅戰》
## 第40章：入口清場

軍管試營運前七天，R-17 入口周邊像被刀刮過。

代理勢力拆掉所有民間補給攤，把常年盤踞的黑市中介驅到兩公里外，連廢棄維修井都被封焊。顧衡顯然吸收了前面幾章教訓：不給灰鏈任何「低成本切入」空間。

林燼把整張區域圖攤開，先圈出三個必拆點：

1. 東側高架狙擊位（封死撤離）。
2. 南側稅契核驗棚（控制合法通道）。
3. 地下回壓閥（可一鍵封鎖所有維修井）。

「我們不是去奪入口，」他對隊伍說，「我們先讓入口周邊‘不能被他一鍵關門’。」

清場不是一晚打完，而是連續三夜削弱。

第一夜，蘇岚帶兩名鹽釘隊員摸掉東側狙擊位。她沒選硬攻，而是把對方電源切到備援，等狙擊手下樓檢修時在樓梯間放倒。整晚零槍聲，只留下一段被改寫的值勤記錄：`設備自檢失敗，建議停用48小時`。

第二夜，燕九與燕禾做南側核驗棚。兩人偽造一批「上游測試稅契」，讓系統在凌晨進入高負載回歸測試，核驗棚被迫切到人工模式。人工模式意味著可以被拖、被騙、被人情突破。

第三夜最硬，地下回壓閥。

這裡由顧衡直屬灰甲守，且配有反主機干擾器。主機預警在該區會延遲半拍，林燼等於要在最不利環境下近身作業。

行動前，燕禾把一枚小岑做的紙折火種塞給林燼：「帶著，別迷路。」

林燼笑了笑，把紙折火種放進胸袋。

地下閥室潮濕狹窄，鋼梯每走一步都響。林燼與蘇岚從上方進，燕九在外面接管壓力數據。灰甲巡邏比預期密，兩人連續三次貼牆避過熱掃，直到閥室核心。

核心面板是老式機械+新式電子混搭。要讓它失去「一鍵封鎖」能力，不是炸掉，而是把其中兩個互鎖齒輪互換，讓指令變成「延遲執行」。延遲三十秒，足夠隊伍逃命。

蘇岚負責警戒，林燼拆板。就在齒輪快換完時，反主機干擾器突然增幅，主機畫面整片雪花。

「有人發現我們了。」燕九在外頻喊。

林燼不敢停手，靠裸眼對位完成最後一扣。灰甲腳步聲已到門外，蘇岚連開兩槍打碎照明，黑暗中兩人翻過排水槽撤離。子彈在背後掠過，蘇岚大腿擦傷，血沿靴跟滴一路。

兩人衝出閥室時，燕九立刻下達測試指令：`封鎖演練`。

面板回應：`指令接收，30秒後執行`。

成功。

這三夜清場換來一個關鍵結果——入口仍在顧衡手裡，但他不再能「即按即鎖」。

唐雀看完測試回報，吐出一口長氣：「這才像能打仗的地形。」

章末戰術會議，林燼宣布下一步：

`41：假配額大車隊佯攻。`
`42：真隊進入口取控制模組。`

蘇岚把繃帶繞緊，抬頭問：「這次要打多大？」

林燼看向 R-17 的紅色標記：「打到顧衡以為我們只想搶貨，然後在他眨眼那一瞬間，搶他的門。」
