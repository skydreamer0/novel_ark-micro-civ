# 第7章：第十一段會面

第十一段比第九段亮。

不是因為希望更多，而是因為監控更多。

林燼穿著燕九改過的灰色工務外套，胸口貼著假識別片，跟著補給車隊混進中層通道。每經過一道閘門，耳邊都會響起掃描聲，像刀刃在骨縫上輕刮。

蘇岚走在他左後，低聲道：「顧衡選會面地在‘白格咖啡艙’，看著像談生意，實際是審計局的半公開場。」

「他想讓我知道，拒絕會有多少雙眼睛看著。」林燼說。

燕九在隊頻補一句：「你只要拖十五分鐘，我就能從外圍偷進他們的本地節點。」

林燼嗯了一聲，推門進入白格咖啡艙。

艙內沒有音樂，只有磨豆機單調的嗡鳴。靠窗位置坐著一個西裝筆挺的男人，手套是純白色，袖口有細銀邊。

顧衡。

他沒抬頭，只把對面座椅推開半寸。

「D9-4417，坐。」

林燼坐下，沒碰桌上的杯子。

顧衡終於抬眼，目光像秤。

「你最近很忙。裂縫任務、黑市搶奪、匿名投放影片。」他笑了笑，「底層難得出一個會算帳的人。」

「所以你想買我？」

「是保護你。」顧衡語氣溫和，「你交出資料來源與技術細節，我讓你進第十一段技術序列，清空舊債，配額翻倍。」

林燼看著他：「代價呢？」

「簽十年服務契約。」

「十年太短吧？你們不是習慣簽到死？」

顧衡笑意淡了些。

「你可以把它當成合作邀請。」

「我比較像在看勒索單。」

兩人對視數秒。

顧衡把一枚微型晶片放到桌面，推過來。

「這裡有你的體檢記錄、前科標註、還有你在第九段所有接觸人的名單。」

林燼沒拿。

顧衡繼續道：「我不是來問你願不願意，是來告訴你，你已經沒有別的路。」

隊頻裡，燕九聲音響起，極輕：

「我進去了。三分鐘。」

林燼壓住呼吸，故意拖話。

「如果我現在答應，刀犬幫的追殺就會停？」

「會。」

「第九段的懸賞撤掉？」

「會。」

「那個叫N-74的批次，會停嗎？」

顧衡眼神第一次出現冷硬。

「那不是你該問的層級問題。」

林燼笑了。

「那也不是你該拿來談條件的命。」

顧衡把手套拉平，聲音降到最低：「別把自己當英雄。你只是剛好拿到一點邊角料。」

隊頻裡，燕九：「完成。拷出一份‘觀察組週報’，內容很髒。我們撤嗎？」

林燼站起身。

「我考慮一下。」

顧衡沒攔，只說了一句：「今晚零點前給答覆。過時，我按清除流程走。」

林燼走到門口時停了一下，回頭道：

「顧觀察員，下次別在咖啡裡下鎮靜劑。味道太明顯。」

顧衡瞳孔微縮。

林燼推門離開。

第十一段的燈依然很亮。

但他知道，這亮光底下全是刀。
