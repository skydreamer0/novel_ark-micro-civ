# 第二卷《裂縫稅戰》
## 第42章：十七分鐘

十七分鐘能做什麼？

對顧衡來說，十七分鐘只是一次會議遲到；對灰鏈來說，十七分鐘是整條鏈能不能活過冬季配給壓縮的分水嶺。

R-17 局部優先權令牌只能用一次，啟用後不可中止。林燼把所有人拉進最後一次戰前會議，規則只有三條：

1. 令牌一開，進場隊不回頭。
2. 外圍隊不追求殲敵，只保通道。
3. 若失敗，立即執行灰鏈分散備案，不做二次硬闖。

蘇岚聽完翻白眼：「每次你說‘不回頭’，最後都會有人回頭。」

林燼沒笑：「這次不行。」

進場隊由燕九領隊，成員包含鹽釘兩人與一名新加入的配額拆解師。林燼本想自己帶隊，被燕禾和蘇岚同時否決——外圍總控必須有人能在壓力下拍板，林燼若進裂縫，外線會亂。

凌晨一點五十九分，令牌上電。

> 【R-17 局部優先權接管，倒數 17:00。】

第一分鐘，入口開啟成功。燕九小隊入場。

第三分鐘，顧衡系統偵測異常，啟動緊急封鎖。

第五分鐘，封鎖命令被令牌壓制一次。

第七分鐘，灰甲增援抵達外圍。蘇岚帶著志工槍線在南側高架撐住第一波。

第九分鐘，裂縫內傳回第一批樣本識別：高純度殘核、稀有導電菌膜、兩枚可用規則碎片。

第十一分鐘，顧衡改走人工權限覆蓋，嘗試手動奪回開關。燕禾在中繼台瘋狂丟假警報，迫使三名人工覆核員分散處理。

第十三分鐘，燕九回報：「裝滿了，撤離中。」

第十四分鐘，外圍最壞情況發生——東側臨時炮位恢復，第一發砲彈砸在北橋，撤離路線被切斷一半。

林燼立刻改令：「走排熱主井，不走橋。」

排熱主井狹窄，高溫，平時只容單人通行。帶樣本通過幾乎不可能，但此刻沒有別的門。燕九當機立斷，把兩箱低值樣本丟棄減重，保核心殘核與規則碎片。

第十五分鐘，第一名隊員鑽出井口，手臂燙傷見骨。

第十六分鐘，第二、第三名隊員出井。

第十六分四十秒，燕九卡在井口，背包被碎鋼筋掛住。蘇岚衝過去拖人，子彈在她腳邊炸出火花。

第十六分五十五秒，林燼親手按下令牌終止，入口關閉。

第十七分鐘，顧衡奪回控制，但門後已空。

灰鏈拿到了它最需要的東西：

- 高純度殘核足夠支撐四週醫療與氧鏈成本。
- 導電菌膜可換取三支新隊伍的裝備升級。
- 兩枚規則碎片讓主機反哺模組進入下一階段。

更關鍵的是心理戰效果——第九段第一次有人在顧衡軍管前，正面奪走入口收益並全身而退。

但勝利不純。撤離中一名鹽釘隊員重傷、兩名志工失聯，蘇岚右手腕骨裂，燕九高熱不退。林燼在醫療點門外站了很久，才進去看人。

唐雀聽完結果，虛弱地笑：「你們把門搶到了。」

林燼搖頭：「只搶到十七分鐘。」

「十七分鐘也能養活很多明天。」唐雀說。

天亮前，顧衡在全段發佈最高級通報，首次公開把灰鏈定義為「制度級叛亂網」。同時，他宣告第九段將進入「分區稅戰管制期」，預示更大規模的封鎖與清洗。

林燼站在高架陰影，聽完通報，對隊伍說了第二卷前段的結語：

「我們證明了兩件事：他不是不可破，我們也不是只能躲。」

「下一段，不是偷一扇門。」

「我們要讓更多人有鑰匙。」

行動板最下方新增三行，為 43-50 章立下骨架：

`43-45：分區管制期生存與反封鎖。`
`46-48：入口收益分配戰。`
`49-50：逼出上游選樣模型。`

灰鏈火種記號在這一夜被畫上更多牆面。有人在旁邊補了一句話，字很歪，像孩子寫的：

`十七分鐘，夠我們記住你們。`

第二卷，進入中段。
