# 第89章 極限協同

排污道外的維修圈本該是撤離緩衝區，卻在十分鐘內變成第二戰場。熵行者群沿沉默艙外壁裂口成片湧出，數量比艙內還多，像被某個終止命令驅趕到出口。

灰槽帶接應隊剛到位就被衝散。阿原帶的A隊在東側架起電磁網，第一輪就被三個熵行者直接穿網而過。常規防禦對它們的有效性接近零。

林燼背著黑盒跳上維修台，直接接管全頻道指揮：A隊守東側燈塔做拖延，C隊帶民兵走南線煙幕，主隊走北線斷橋。所有單位按十秒節拍回報，不准超句。

這是多節點協同的極限版本。信息越短，延遲越低，錯誤也越少。

蘇岚把剩餘規則燃料全拆成微片，通過短波拋給各隊，讓每個火力點都能臨時做一發鎖定域。這等於把文明底倉最後儲備分散下注，失敗就是全線歸零。

燕九在斷橋口設置連鎖爆破，爆破目標不是熵行者本體，而是橋面影子重疊區。他先炸暗區再掃實體，成功把第一波追兵卡在橋中央。

可第二波立刻從橋下爬上來。熵行者開始模仿人類戰術，分成誘餌與突穿兩群。唐雀看出節奏後，命民兵把所有照明改成交錯閃頻，迫使它們影子失步。

交錯閃頻奏效，但電力撐不久。阿原報告東側燈塔只剩三分鐘。林燼做出最狠決定：放棄東側塔，把A隊撤回北線，讓塔自爆拖住追兵。

這個命令意味著原本答應守到最後的街區志工要被迫撤離，有些人可能來不及。阿原沉默一秒，仍照做。因為他知道黑盒與樣本一旦丟失，死的會是更多人。

東側燈塔爆炸時，整個維修圈被白火照亮。白火裡，熵行者群第一次出現明顯遲滯，像在同時接收衝突指令。林燼抓住窗口，下令主隊全速衝斷橋後端升降井。

升降井前站著一台重型維修機，原本報廢，唐雀硬把它啟動，當成臨時盾牆。熵行者撲上來時，維修機連同唐雀一起被撞退三米。

林燼把黑盒交給蘇岚，自己反身拉住維修機支架，和燕九一左一右把盾牆重新頂回井口。兩人像在推一扇會咬人的門，手臂肌肉都在發顫。

主機在這刻再次全開：祭司推演給出最短撤離序列，戰士投影只保留兩道，工匠端緊急量產一次性影子釘。影子釘被撒在井口邊緣，凡踩中者影子會被固定兩秒。兩秒足夠近距離切殺。

C隊終於帶著最後一批民兵抵達。林燼確認人數，少了七個。沒人有時間追問名字，因為熵行者第三波已經壓到五十米內。

升降井啟動需要四十五秒。這四十五秒成了生死倒數。每過十秒，林燼就報一次：三十五、二十五、十五。聲音越平，隊伍越不慌。

最後十秒，陌生女聲再次插入全頻道：『觀察序列記錄完成，樣本回收值上升。』同時，最前排熵行者突然停步，後排卻加速衝鋒，像被拿來做對照實驗。

林燼聽完只覺得胃裡發冷。有人在遠處看著他們流血，並把這場撤退當成可量化樣本。

升降井門終於開啟。林燼命所有非戰鬥員先下，自己和燕九、唐雀守最後。當井門剩半尺縫時，燕九把最後一捆爆藥塞進橋基，笑著說欠你的酒回去再喝。

爆炸把斷橋完全掀翻，熵行者群連同橋體掉進下層黑井。井裡傳來長長的回響，像整座沉默艙在磨牙。

井門閉合，世界終於短暫安靜。林燼靠著牆喘氣，手裡還攥著沾血的影子釘。這一戰之後，多節點協同不再是戰術展示，而是被鮮血驗證的唯一生路。

蘇岚在昏暗燈下讀出最終損耗：主機鏈路跌到臨界、規則燃料清空、外部三隊共傷二十六人失聯七人。

林燼點頭，沒有說節哀。他只說先封存證據、先救活的人，再討債。

他把當前行動的每一筆耗損都寫進清單，因為在這個時代，忘記代價就等於替敵人省子彈。
