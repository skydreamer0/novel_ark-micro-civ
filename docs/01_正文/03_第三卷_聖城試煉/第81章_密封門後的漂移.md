# 第81章 密封門後的漂移

林燼手背上的血痕還沒乾，密封門切割器已經咬進第二十一段的合金鎖環。火星沿著門縫向外噴，像一條被扯斷的紅線，正好接在上一夜沉默艙切割任務留下的傷口上。燕九靠在門側，肩上纏著止血帶，呼吸很重，卻仍把槍口頂在縫隙最先可能彈出的角度。

門板落地那一瞬，冷氣不是撲出來，而是被裡面的負壓猛地吸走。唐雀第一個進去，鞋底剛踩上格柵，耳機裡就傳來刺耳電噪。蘇岚迅速報數：環境規則密度低於灰槽帶三成，通訊延遲從零點三秒跳到兩秒以上，還在持續飄。

沉默艙不像廢墟，反而像被掏空的工廠。牆上每隔十米就有一組失效的導航燈，燈罩內殘留白霜，霜層下壓著舊編碼標牌。林燼用手電掃過去，看見『維生循環不保證』、『非授權工位禁止駐留』，字體是高層內部才會用的老式審計標準體。

第一段廊道走到盡頭，地面突然向下塌陷半層。燕九往前探腳，鋼格在他腳下無聲粉化，像被看不見的酸液啃掉結構。蘇岚低聲說不是腐蝕，而是材料內部鍵結失序，熵化已經把原本的金屬規則拆成鬆散顆粒。

林燼立刻改行進命令：不再走中央維修主道，改貼牆線走次級纜橋；每三十秒回報腳下硬度；任何人遇到無聲裂紋立即退兩步，不准賭運氣。他把決策權下放給前導位和技術位，自己只保留撤離與任務切換的最終指令。

隊伍推進到第三個轉角時，外層中繼忽然斷訊。阿原在另一端只留下半句『灰槽帶北塔有人在掃你們回傳識別』，隨後整條鏈路像被剪斷。沒有外部眼睛，沉默艙內每一步都等於盲走。

唐雀在舊控制櫃找到一塊還通電的路由板，硬拆出兩條備援線，蘇岚把它接進便攜分析儀，做成一個低頻脈衝中繼。訊號仍斷斷續續，但至少能讓短句傳回去。代價是回傳座標會延後四分鐘，若遭伏擊，外面收到時人可能已經換了位置。

第四節艙橋出現第一批異常記錄：牆面浮起像潮汐一樣的灰白紋，接觸時不會發熱，卻會讓照明頻率失真。燕九抬手示意停下，林燼注意到他瞳孔邊緣泛起細小紅點，那是低規則環境對義體神經的刺激反應。

林燼沒有說安慰話，只問一句還能不能帶路。燕九點頭，咬著牙把止痛針退了半格，保持清醒。他知道在這裡睡著，醒來的人可能已經不是原來的自己。

穿過灰紋區後，隊伍找到第一個可用節點：一座封存的航行資料旁站，門牌號『N-21/Drift-Archive』。這意味著黑盒子很可能就在更深一層，而不是被整批轉移。林燼當場改任務優先序：先鎖黑盒位置，再找養殖實物證據，避免在熵化區域長時間逗留。

就在唐雀嘗試開門時，遠處廊道傳來拖拽聲，像一串鐵鏈在地上慢慢拉行。聲音忽近忽遠，沒有固定節奏，和任何已知巡防機械都對不上。蘇岚把感測陣列開到最大，儀表卻只顯示一片空白。

空白本身就是警報。林燼下令熄主燈，只留低流明側燈，所有人貼著資料旁站外壁靜止。拖拽聲停在轉角外三秒，又突然出現在另一頭。沉默艙的空間像被人折了一道，方向感在這裡不再可靠。

他們終於切開旁站門，裡面保留著三年前最後一次值勤記錄。值勤員留下的語音短句只剩碎片：『漂移區...不回收...標記活體...』。最後一行文本更刺眼：『若見到人形，先確認影子是否延遲。』

林燼記下這句，心裡沒有發寒，只有更明確的計算。若連值勤規程都把『人形』視為可疑目標，接下來遇到的東西就不會按常理行動。

旁站底層地圖顯示前方有兩條路：一條通向黑盒庫，一條通向標註為『工業試驗圈』的封閉區。兩條路在中段可交會，但交會點標著高危漂移。時間與補給只夠他們賭一次。

林燼讓所有人投票不是為了民主，而是測壓。唐雀選黑盒先行，燕九選先探試驗圈，蘇岚給出折衷路徑：先用最短距離刺探試驗圈入口，確認有無實物鏈，再折返黑盒庫。

最終命令落下：按蘇岚方案走，四十分鐘封頂，超時直接切黑盒撤離。這不是完美選擇，是在可承受風險裡保留最大證據值。

隊伍離開旁站時，牆角多了一道剛剛不存在的腳印。腳印長度接近成人，深度卻比鋼靴輕得多，像一個沒有重量的東西在模仿人類步伐。燕九回頭看了很久，低聲說它在學我們怎麼走路。

林燼沒有回頭，他把計時器按下，帶隊朝工業試驗圈方向切入。耳機裡四分鐘延遲的外部回傳終於抵達：灰槽帶觀測塔捕捉到沉默艙外壁短暫亮起不明簽章，來源不屬於任何公開部門。

沉默艙的門已在他們身後自動鎖死。

他把當前行動的每一筆耗損都寫進清單，因為在這個時代，忘記代價就等於替敵人省子彈。

隊伍裡沒有人說勝利，只有人確認下一個班次、下一條撤離線、下一個還沒填平的洞。

灰槽帶的風從破裂管道吹過，帶著鐵屑聲，像在提醒每個還活著的人：機器還在轉，必須有人繼續把手伸進去。
