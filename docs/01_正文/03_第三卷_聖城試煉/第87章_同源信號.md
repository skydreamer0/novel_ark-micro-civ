# 第87章 同源信號

升降筒剛回到上層，林燼體內主機就像被重錘敲中。視野邊緣浮起成串金紋，耳邊不是噪音，而是有節奏的低語，語義不完整卻反覆出現同一組詞：『同源...分裂...聖城門限。』

蘇岚把他按到牆邊，迅速接上監測線。數據顯示主機正在主動對接外部某個殘留訊號源，頻段與原型艙拆解殼體完全一致。這不是污染，是識別。

唐雀提出立刻切斷。林燼否決。這類窗口一旦錯過，可能再也接不上。他命令所有人圍成半圈警戒，自己在可控範圍內放開主機吞吐。

訊號對接後，隊伍前方空氣出現短暫投影：一座比沉默艙更古老的圓城輪廓，城門上刻著多層同心紋。每一道紋都對應不同文明階段，鋼鐵紋只是最外圈。

祭司端回傳解讀：這是聖城試煉的前置驗證界面。要真正開門，必須同時滿足高密度規則殘留、同源訊號碎片、文明軍規協同三個條件。前兩項剛在原型艙達標，第三項需要在實戰中完成穩定閉環。

林燼把訊息壓進記憶，沒有宣揚。聖城不是眼前能吃下的果子，先活著出去才有資格談下一步。

同源信號還給了另一段碎片：三年前沉默艙封閉當日，確實有一台同型主機被拆解，拆解後殘片分送三個方向，其一留在此處，其二標記『審計深井』，其三標記『航行中樞禁庫』。

這讓母核碎片與主機起源線突然有了可追坐標。蘇岚抬頭看林燼，第一次在他眼裡看到近乎赤裸的貪婪與克制同時存在。那不是為錢，而是為活路與主權。

可訊號窗口只維持了九十秒。結束前，主機自動吞入一段高密度殘留，整體算力上限短暫躍升。代價是林燼鼻腔直接出血，體溫急降，像把整個人當導體硬扛過載。

燕九把他拽起來：活著出去再升級。

隊伍轉往黑盒庫路線時，熵行者再度出現，這次不是散兵，而是沿走廊兩側整齊列隊，像被同一指令驅動。它們沒有立刻進攻，只是同步抬頭看向林燼。

林燼讀懂那眼神裡的空洞——它們在等待更高優先序命令。沉默艙內還有一個指揮節點，可能就是剛才女聲來源。

他沒有浪費子彈，直接啟動戰士投影短脈衝，在隊伍前方打開一條『視線盲區』。熵行者群被短暫干擾，四人趁隙切進黑盒庫外環。

黑盒庫門口的鎖比想像古老，是雙盤機械鎖加單線審計簽。唐雀邊拆邊笑：越古老越好，古老系統不會臨時下載新規則。

蘇岚則抓緊時間整理同源訊號紀錄，做了三份異地加密。她很清楚，這段資料若只留一份，回灰槽帶前就可能被任何一方抹掉。

林燼在等待開鎖時給外部發出最短回傳：『證據到手，含同源訊號與養殖實物，準備撤。請做三層接應，優先醫療與封存。』

四分鐘延遲後，阿原回來一句：『灰槽帶外圈有陌生觀測舟靠近，旗標被抹。你們得比它們更快。』

黑盒庫終於打開。裡面只有一排舊式封箱，其中編號N-21-Flight的箱體仍亮著低電量指示。唐雀把箱體拔出來時，整條走廊警報同時響起。

熵行者群不再觀望，開始從兩端高速湧入。這不是遭遇戰，而是明確的堵截。

林燼把黑盒背到自己身上，聲音平得像冰：資料夠了，現在只做一件事——把人和證據一起帶出去。

他把當前行動的每一筆耗損都寫進清單，因為在這個時代，忘記代價就等於替敵人省子彈。

隊伍裡沒有人說勝利，只有人確認下一個班次、下一條撤離線、下一個還沒填平的洞。
