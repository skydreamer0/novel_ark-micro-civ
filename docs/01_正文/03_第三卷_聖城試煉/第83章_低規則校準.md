# 第83章 低規則校準

中庭白光熄滅前，林燼已把隊伍拉進西側維修縫。縫道只能容一人半側身通過，熵行者數量優勢在這裡被削到最低。燕九守後，唐雀在前拆舊管線，蘇岚蹲在兩人中間，把剛救下的分析儀核心插進手持終端。

她用不到三分鐘得出殘酷結論：初階生物投放模組在低規則環境下衰變速度是常態區的七倍，原本七十二小時的戰獸存活窗，進沉默艙後只剩十一到十五分鐘。

林燼讓她把數據直傳內部文明，不做過濾。很快，主機回傳祭司推演結果：若把生物骨架材料從標準鋼改為高碳陶鋼混編，再把神經節點改成雙冗餘回路，能把低規則衰變壓回二十五分鐘。

二十五分鐘仍短，但足夠打一場突圍或守一次門。問題是現場沒有完整工坊。唐雀把維修縫裡拆下來的彈性導管、碳纖片、舊伺服芯拿出來，說可以拼一套『臨時孵化籠』，前提是要有人撐住外面至少八分鐘。

燕九把槍托往地上一杵：八分鐘我撐。林燼沒說漂亮話，只把他肩上的止血帶再勒緊一格，然後把兩枚鎖定域彈交給他。這兩枚比任何鼓勵都實在。

外面熵行者開始試探維修縫入口。它們不急著衝，先往內丟斷裂金屬和舊工具，測隊伍的反應時間。林燼看懂後，故意讓第一件雜物砸進來不回應，第二件才開槍，打亂對方建立模型的節奏。

與此同時，蘇岚和唐雀幾乎把手燒穿。臨時孵化籠在低規則區每啟動一次就會回寫錯誤，兩人只能邊算邊改，把工匠班子回傳的參數一行行焊進控制板。

第九分鐘，第一批調校後的投放體終於落地。它們體型比原版小，外骨骼更薄，前肢改成鉤刃，背脊多了一排穩定鰭。看起來不像衝鋒獸，更像在崩壞地形裡專門撕開縫隙的『掘線犬』。

林燼把掘線犬放到入口外二十米，設置目標不是殺敵，而是『打斷影子連續性』。第一隻掘線犬撲中熵行者後，沒有咬軀幹，而是沿地面咬掉對方影子邊界。被咬中的熵行者像被拔掉骨頭，立刻坍縮。

策略成立。隊伍用三隻掘線犬換掉入口外七個目標，壓力第一次往下掉。

代價也立刻顯現：掘線犬存活平均二十二分鐘，超時後會自體粉化，連回收都做不到。這意味著每一次戰鬥都在燒主機底倉。

林燼在隊頻裡下了新的冷酷規則：投放體只用在打開路徑或保命關口，不准為了節省人力濫用。外部隊員受傷可以包紮，文明底倉一旦空掉，後面整卷都會斷血。

蘇岚聽到這句時手指停了半秒，最後還是點頭。她知道林燼把內外兩個世界當同一張資產表來算，這種算計殘忍，卻讓人活得更久。

調校完成後，四人重新進入中庭。熵行者果然改了戰法，不再直撲，而是分散站位，讓影子互相覆蓋，試圖降低被切影的風險。

唐雀立刻對應：用低頻閃光把地面分區，讓每一塊影子都被單獨打亮。燕九與林燼照亮區域逐格推進，掘線犬專門撕被標記的邊界。

十五分鐘後，中庭被打通。四人踩過滿地灰屑，走到中央失重平台。平台下方露出一條向下的螺旋梯，梯口標識『E-Seed/工業種子艙』。

這不是黑盒庫，卻是更接近真相的入口。林燼看了一眼任務計時，距離硬撤上限只剩二十七分鐘。任何理智指揮官都應先拿黑盒走人。

他沉默兩秒，最後下令：先下種子艙三層，找到實物鏈證據立即撤回。若三層內無決定性樣本，立刻轉黑盒庫。

燕九笑了一下，嘴角全是血味：你終於承認自己也會賭。

林燼回：我只賭值得付帳的局。

螺旋梯第一階踩下去時，整座中庭的灰光忽然同時熄滅，遠處傳來像多人同時吸氣的聲音。熵行者群沒有被清空，它們只是退到更深處，等隊伍把路走到最窄的位置。

他把當前行動的每一筆耗損都寫進清單，因為在這個時代，忘記代價就等於替敵人省子彈。

隊伍裡沒有人說勝利，只有人確認下一個班次、下一條撤離線、下一個還沒填平的洞。

灰槽帶的風從破裂管道吹過，帶著鐵屑聲，像在提醒每個還活著的人：機器還在轉，必須有人繼續把手伸進去。
