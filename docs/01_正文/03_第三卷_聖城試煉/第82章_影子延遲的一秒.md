# 第82章 影子延遲的一秒

工業試驗圈前的長廊像一條被抽乾血液的動脈。林燼帶隊貼牆前進，側燈照在地面，只看見零散工具和被拖走的防爆箱。燕九忽然抬手，示意所有人蹲下。前方三十米處站著一個穿維修服的人影，背對他們，一動不動。

唐雀用鏡片放大後倒吸一口氣：那件維修服是三年前封艙隊的款式，肩章編號早該在事故清冊裡註銷。更怪的是，那人影腳邊有影子，但影子的擺幅比本體慢半拍，像訊號延遲。

林燼想起旁站規程『先確認影子是否延遲』，立刻改射擊指令：不打軀幹，先打影子落點。燕九扣扳機，子彈擊中地面陰區，沒有火花，卻爆出一圈灰白碎屑。下一秒，人影整個翻轉過來，關節反向折疊，四肢貼地高速撲近。

常規彈對它幾乎無效。燕九連開三槍只打掉表層碎殼，內部像沒有固定器官，槍口動能進去就被散掉。唐雀改用切割脈衝，脈衝在它身上留下短暫亮斑，很快又被周圍空氣抹平。

蘇岚在隊頻裡大喊：它不是裝甲厚，是局部規則密度比環境更低，武器『命中』的定義被改寫。林燼立即把文明主機反哺的規則燃料片投進槍膛轉接器，讓彈頭帶一層可維持三秒的『鎖定域』。

第二輪射擊生效了。燕九的子彈打進鎖定域，熵行者胸口像被釘住一樣停頓半秒。這半秒夠林燼衝上去，把磁刃沿它的影子邊界一刀拉到底。實體沒有立刻倒下，但影子先裂成兩段，接著整個身體像失去支點一樣坍成灰片。

第一次遭遇戰只持續二十秒，卻讓每個人的呼吸節奏全亂。蘇岚快速記錄：熵行者可模仿人形，影子延遲約零點八至一點二秒；常規武器效果低；需用規則鎖定域固定局部判定。

問題緊跟著來。鎖定域彈藥每發都要消耗一片高純燃料，而他們只帶了四十片。照剛才消耗，遇到群體圍攻撐不到十分鐘。林燼當場切補給規則：非近距離威脅一律不開火，先用地形與誘餌拉開。

長廊盡頭，第二批熵行者從天花板檢修口掉下來，數量六。它們落地時像液體，站起時又變成人形，衣著混雜維修服、安保服與白袍實驗衣。每一張臉都沒有眼白，只有像燒過金屬的暗灰表面。

唐雀扔出脈衝誘餌，四個目標被拉偏，剩下兩個直接衝林燼。林燼不退反進，把它們引到長廊側面的失重井口。燕九在後方用鎖定域打斷其中一個影子，林燼一腳把另一個踢進井裡。井內沒有碰撞聲，只有長時間的下墜回音。

隊伍以最小代價清掉第二批，但蘇岚的便攜儀開始冒煙。低規則區讓芯片校時模組持續漂移，她只能每隔兩分鐘手動重置一次，不然敵我標記會顛倒。

林燼讓她把儀器核心交給唐雀，自己扛起外殼繼續走。這不是體貼，而是把最難替換的腦力資源留在最安全位置。

進入試驗圈外門前，牆上出現整排刻痕，像有人用指甲反覆刮同一句話：『不要看它的臉，看它影子先死。』字痕深淺不一，至少有十幾次重寫。

燕九低聲罵了一句，問林燼要不要撤。林燼看了眼剩餘燃料、任務時間和黑盒位置，回答只有四個字：再推十五分鐘。

外門開啟時，裡面不是實驗室，而是一座被切成多層的圓形中庭。中庭中央懸著失重平台，平台上有半截被燒蝕的標識牌：『工業化內宇宙環境壓力場』。

這行字讓所有人短暫沉默。那不是傳聞，不是估計，而是直接寫在牆上的用途。高層真的在這裡做過內宇宙工業化試驗。

然而更急迫的危機在腳下。中庭地板同時亮起十二個灰光點，熵行者從每個光點裡慢慢站起來，像被同一隻手從地板裡拉出。

林燼把最後一發照明彈打向穹頂，整個中庭被白光灌滿。白光裡，那些人形的影子全部朝反方向延伸，像在往隊伍身後靠攏。

他終於確定一件事：熵行者不只是獵食者，它們還在學習隊伍的站位、開火順序與撤退路徑。

這場戰鬥若拖成持久，沉默艙會比任何敵對派系都更快把他們吃乾淨。

他把當前行動的每一筆耗損都寫進清單，因為在這個時代，忘記代價就等於替敵人省子彈。

隊伍裡沒有人說勝利，只有人確認下一個班次、下一條撤離線、下一個還沒填平的洞。

灰槽帶的風從破裂管道吹過，帶著鐵屑聲，像在提醒每個還活著的人：機器還在轉，必須有人繼續把手伸進去。
