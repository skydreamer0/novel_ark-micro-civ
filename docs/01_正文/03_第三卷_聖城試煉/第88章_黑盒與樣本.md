# 第88章 黑盒與樣本

黑盒到手後的撤離沒有浪漫，只有重量。林燼背上的Flight箱體超過三十公斤，燕九扛著樣本冷封箱，唐雀抱著同源接口與硬盤，蘇岚則拖著幾乎要散架的分析儀。每一步都在跟時間和熵化速度賽跑。

他們選擇最短返程線，卻在第一個轉角遇到新問題：原本穩定的地面開始像水面一樣起伏，腳踩上去會延遲半拍回彈。蘇岚判定這是全域漂移前兆，空間拓撲正在鬆動。

林燼立刻把隊形改成繩索連結，四人腰間用磁纜互鎖，避免有人被局部漂移『吞位』。這套動作曾用在外艙維修，如今被拿來對抗像活物一樣的走廊。

第二個障礙來自黑盒本身。箱體在高漂移區自動啟動防拆協定，外殼釋放低頻脈衝，吸引周邊熵行者靠近。唐雀試著關閉失敗，只能反向加上一層噪聲罩，把吸引範圍壓到五十米內。

五十米仍太近。熵行者像聞到血味，從三條岔道同時靠攏。燕九提議分隊引開，林燼否決。此刻分隊等於送命，且證據鏈會斷。

他啟動備用方案：把兩支樣本管做成『影子誘餌』。蘇岚快速抽出微量殘液塗在廢棄機兵外殼，設定定時爆閃。誘餌被丟進北側管道後，熵行者群果然偏移一半。

代價是樣本完整性下降，可能影響後續法務驗證。林燼接受這個代價，因為活著送回去比完美樣本更重要。

撤退到中庭時，之前被清空的區域又被灰光點填滿。熵行者不斷從地板長出，像整座沉默艙都在排斥他們帶走證據。

唐雀把最後兩枚鎖定域彈交給林燼，自己改拿脈衝刀守側翼。她知道這兩枚彈只能留給關鍵門口。

關鍵門口很快到來——外層防線殘骸處。原本停擺的節點竟被遠端重啟三成，切割網重新亮起。蘇岚咬牙把之前抓到的主核映像打回去，強行製造權限衝突。

衝突窗口只開二十秒。林燼用一枚鎖定域彈釘住中央切割臂，燕九背著樣本衝第一個，唐雀推著蘇岚過第二個，林燼最後撤，腿側被網邊切出一道長口。

血順著褲管往下滴，林燼連看都不看，直接把第二枚鎖定域彈塞進後方走廊，炸出一片暫時封閉區，為隊伍爭到三十秒距離。

三十秒後，四人終於看到最初切進來的密封門。門外信號微弱恢復，阿原的聲音斷續傳來：『外圈觀測舟逼近...你們出門就會被標記...建議走廢棄排污道。』

排污道入口在門右下方，需要手動轉輪開啟。燕九手臂已經抖到幾乎握不住，還是把轉輪一圈圈扳開。門縫裡湧出黑水與冷霧，味道像鐵鏽和藥液混合。

林燼先把黑盒箱推進去，再讓樣本冷封箱滑下斜道。唐雀最後回頭看了一眼沉默艙主通道，灰光裡站著一排不再追擊的熵行者，像在目送。

她心裡發毛，卻也確認一件事：它們的任務不一定是殺死入侵者，更像確保某些東西不被完整帶走。

排污道通往沉默艙外壁維修圈。四人爬出來時，灰槽帶接應隊已在兩公里外點亮低頻燈標。可天空上方同時有三枚不明觀測探針降下，正在掃描撤離路線。

林燼把所有證據包綁在自己和燕九身上，命令接應隊不做火力迎擊，只做煙幕遮蔽。現在最怕的不是打不過，而是證據在交火中被打爛。

他們衝進煙幕那刻，身後沉默艙外壁傳來沉悶爆鳴。第二十一段開始局部自封，像要把還沒來得及抹掉的一切重新埋進黑暗。

林燼沒有回頭。黑盒在他背上發熱，像一顆將要開口說話的心臟。

他把當前行動的每一筆耗損都寫進清單，因為在這個時代，忘記代價就等於替敵人省子彈。

隊伍裡沒有人說勝利，只有人確認下一個班次、下一條撤離線、下一個還沒填平的洞。

灰槽帶的風從破裂管道吹過，帶著鐵屑聲，像在提醒每個還活著的人：機器還在轉，必須有人繼續把手伸進去。
