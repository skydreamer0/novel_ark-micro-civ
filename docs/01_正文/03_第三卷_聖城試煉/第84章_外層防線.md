# 第84章 外層防線

螺旋梯下方的空氣比上層更冷，冷到呼吸會刺喉。林燼把面罩濾芯切到重污染模式，仍能聞到淡淡甜腥味。蘇岚說那不是血，是規則燃料蒸散後殘留的金屬糖味，代表附近曾有高能設備長時間運轉。

第三層出口是一道環形閘門，門上刻著『工業種子艙-外層驗證帶』。唐雀剛把破解器貼上去，門內就亮起藍色掃描線，從四人頭頂一路掃到鞋底。掃描結果顯示：無授權，觸發隔離程序。

隔離程序不是鎖門，而是啟動外層防線。兩側牆板滑開，露出六組半人高安保節點，外形像縮小版轉運塔，每組節點頂部都有可旋轉的感測球。

第一波火力不是子彈，而是切割網。透明網格在空中展開，碰到管線就像切豆腐。燕九翻身把林燼壓進死角，自己後背被網邊刮出三道深口。

林燼看了一眼傷口深度，直接把止血噴劑丟給他，同時命唐雀關掉所有主動信號發射。外層防線依賴動態特徵鎖定，誰頻繁發訊誰先死。

蘇岚把分析儀改成被動監聽，抓到一條關鍵訊息：六組節點不是獨立單位，而是由中央『節律主核』同步。只要主核還在，打掉單個節點也會被旁邊接管。

問題是主核位置不在平面圖上。林燼讓唐雀去門楣上方找維修孔，自己和燕九負責牽制。掘線犬被投放到左側走廊，專門去咬節點投射出的切割網邊界，強行撕出射擊窗。

戰鬥節奏被拉成細碎的十秒單位：十秒探頭、十秒壓制、十秒換位。任何超過十秒的停留都會被網格覆蓋。林燼在這種節奏裡慢慢逼近中軸柱，終於在柱基看到一圈不同材質的固定環。

固定環後面藏著主核。它像一顆被嵌入牆體的黑色心臟，表面纏滿老式光纜。林燼沒有直接破壞，而是讓蘇岚先抓數據映像。若把它炸掉，可能連帶抹去防線日志，反而失去證據。

映像抓取過程中，防線開始切換模式。原本規律掃射變成不規則脈衝，切割網裡混入短距離熵化噴流。兩隻掘線犬當場粉化，燕九左腿義件也被噴流燒黑。

唐雀終於從門楣維修孔裡拉出一條備援纜，反向注入假授權封包。節點反應遲滯了零點七秒，這零點七秒被林燼抓成破口，一槍把主核外殼打裂。

裂縫裡噴出的不是火花，而是大量封存資料片，像被壓了很久的紙片突然炸開。蘇岚邊躲邊抓，抓到三個關鍵片段：環境壓力場維護記錄、樣本運輸簽收單、外層防線重啟權限名單。

名單上沒有完整姓名，只有代號與時間窗。其中一個代號是『CAT-Prime/奇數簽』，正對上他們在沉默艙資料層看見的習慣。

防線主核失去同步後，六組節點陸續停擺。閘門終於打開，後方是一條長達兩百米的觀察廊，廊兩側排列著破裂觀測窗。窗內可見大量圓柱形培養艙，但多數已空。

空艙比滿艙更讓人不安。它代表這裡不是剛開始，而是已經運作過、收割過、搬空過。

林燼在觀察廊盡頭看見一具被焊死在操作台上的白袍屍體。屍體胸前工牌還在，抬頭寫著『工業種子艙：樣本穩定組』。工牌背面有手寫字：『若你看到這裡，請別把它們帶回上層。』

燕九問這句是不是陷阱。林燼說不重要，重要的是有人在死前也把上層叫做『它們』，說明這個實驗艙後期已和高層本部產生斷裂。

他們沿觀察廊繼續推進，抵達一扇雙層氣密門。門後標識：『原型艙核心區，需雙重授權』。

唐雀把剛奪來的主核碎片接上破解器，蘇岚同步輸入CAT奇數簽習慣特徵，兩把『假鑰』拼成一把臨時真鑰。

氣密門在低鳴中開出一條縫，縫裡吹出的風帶著更濃的金屬甜味，還有微弱的人聲錄音在循環：『批次三十一，恐懼值不足，回收重投。』

林燼握緊槍柄，知道外層防線只是門檻。真正會改變全局的東西，正在門後等他們。

他把當前行動的每一筆耗損都寫進清單，因為在這個時代，忘記代價就等於替敵人省子彈。

隊伍裡沒有人說勝利，只有人確認下一個班次、下一條撤離線、下一個還沒填平的洞。

灰槽帶的風從破裂管道吹過，帶著鐵屑聲，像在提醒每個還活著的人：機器還在轉，必須有人繼續把手伸進去。
