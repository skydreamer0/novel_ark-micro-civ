# 第90章 灰槽帶卷末對峙

升降井把眾人送回灰槽帶北側維修倉時，天還沒亮。倉外卻早已站滿三層人：灰議會代表、維穩派使者、街區醫療隊、還有幾名未掛徽章的陌生觀察員。所有目光都落在林燼背上的Flight黑盒和燕九懷裡的樣本箱。

林燼第一句不是報功，而是報傷亡和封存需求：重傷優先進手術站，樣本進雙鎖冷庫，黑盒進三方見證解封室。誰先搶資料，誰就先滾出灰槽帶。

賀老八少見地沒抬槓，直接讓灰議會武裝拉起隔離線。維穩派使者祁序則遞上交易文件：按約，交黑盒可換『養殖清單索引』。

林燼沒立即交。他先要求把遺址實物證據做現場預驗。蘇岚當眾投影工業原型艙照片、壓榨曲線、樣本管編號與航向耦合記錄，整個倉庫一度安靜到只能聽見機械風扇聲。

當青苗中心代碼出現在投影上，醫療隊後排有人失控地砸了工具箱。這些年被說成『教育轉運』的人，原來都在某張效率曲線裡變成數字。

祁序臉色變了又穩住，聲稱索引只對授權方開放，要求立即履約。林燼把黑盒放到桌上，手掌壓著箱蓋：可以交，但先公開三點——索引真偽校驗方式、可追溯時間窗、禁止單方刪改條款。

這不是抬價，是防對方把索引做成一次性誘餌。祁序沉默片刻，最終同意在灰議會見證下簽附加條款。

黑盒解封後，唐雀用離線機讀出第一段航行摘要。數據顯示母艦多次航向變更並非尋找新資源，而是在避開某些高熵化區，且每次避讓都伴隨底層艙段批次壓縮。『逃離』二字沒有被寫明，方向卻已非常清楚。

祁序按約交出索引晶片。晶片裡不是完整名冊，而是一張可展開的路由圖：私有養殖節點、審計深井、航行中樞禁庫之間有固定轉運鏈。這張圖一旦和遺址實物對接，就形成首次可實證的高層養殖體系全景。

灰議會內部有人當場提議把證據只留灰槽帶，拿來談更高保護價碼。林燼當場否決：證據若只握在一手，下一秒就會被做價或做掉。必須分片上鏈，跨段同步。

賀老八看著他，終於承認一句：你不是在談生意，你在改遊戲規則。

與此同時，陌生觀察員中的一人走到前排，亮出一枚沒有部門名的黑色徽記，只說一句：『觀察序列更新，林燼列入核心監控。』

這句話沒有威脅語氣，卻比威脅更冷。核心監控意味著後續每一步都會被更高權限盯住，也意味著他的價值已高到不會被輕易抹除。

林燼回以同樣平靜：監控可以，別碰我的人。對方沒有答覆，只把一份密封令放在桌上離開。密封令封面標註『聖城試煉前置核驗：通過』。

蘇岚看到這行字時，手指微微發顫。她知道這不是獎章，而是一張強制入場券。進聖城，可能是更大權限，也可能是更精細的屠宰場。

夜裡，灰槽帶完成第一輪證據分片上鏈。L-雨棚原始工務序列被驗出時間戳早於審計署建立；同源接口與主機殘片匹配度超過九成；航向耦合檔與黑盒摘要互相校驗。多條老伏筆在這一夜被迫加速回收。

林燼把所有結果整理成三份包：一份給灰槽帶公共庫，一份給外段可信節點，一份鎖進主機私庫。任何一方背叛，剩下兩份都能讓真相繼續活。

他去醫療站看了燕九。燕九被固定在治療床上，還在笑，說下次別再讓我背玻璃管跑八公里。林燼回他一句：下次換你背黑盒。

天亮前，林燼獨自站到連接通道盡頭。遠處熵雲像鉛色海潮翻湧，灰槽帶的燈一盞盞亮起，像在黑水上釘住幾枚釘子。

他知道第三卷的帳暫時結了：實物證據到手，養殖體系被照亮一角，聖城前置條件達成，自己被推進更高監控層。

但更大的帳才剛開始。若母艦真在逃離，逃離什麼、要把誰丟下、誰在船頭選人，這些問題會在下一段航程裡變成刀。

林燼把密封令折進口袋，轉身走回仍帶血味的灰槽帶。路還很長，他沒有打算當樣本。

他把當前行動的每一筆耗損都寫進清單，因為在這個時代，忘記代價就等於替敵人省子彈。

隊伍裡沒有人說勝利，只有人確認下一個班次、下一條撤離線、下一個還沒填平的洞。

灰槽帶的風從破裂管道吹過，帶著鐵屑聲，像在提醒每個還活著的人：機器還在轉，必須有人繼續把手伸進去。
