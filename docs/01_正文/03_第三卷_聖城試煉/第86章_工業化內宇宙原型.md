# 第86章 工業化內宇宙原型

升降筒停止時，四人眼前是一個巨大半球艙。艙壁佈滿層層疊疊的導能管，像把一座城市倒扣在地下。半球中央懸著一顆黯淡晶核，四周繞著數百條傳輸臂，每條傳輸臂末端都連接空掉的培養艙位。

這就是高層的工業化內宇宙原型。它不是神祇，不是文明，而是一台把恐懼、痛覺和記憶榨成燃料的工廠。

蘇岚打開最近一台培養艙，裡面殘留的不是生物屍體，而是一疊被封膜保護的紀錄片。第一頁標題寫著：『低純度人口群壓榨曲線/批次31』。下面列著恐懼刺激、失眠時長、群體互疑程度對輸出效率的關聯係數。

唐雀翻到後頁，手都在抖。紀錄明確標註：若需短期增產，優先打斷底層互助網絡，提升舉報獎勵，製造「誰都不可信」的環境。這些手段和外部艙段近年的政策幾乎一模一樣。

林燼沒有停在震怒上。他要的是可驗證、可對外審核、可迫使對手失語的實物鏈。他命令三人分工：蘇岚掃參數原始檔，唐雀拆傳輸臂標號，燕九去找實體樣本容器。

燕九在東側找到冷封櫃，櫃門上有審計蠟封。砸開後，裡面整齊排著十二支樣本管，管壁印著不同艙段代碼。第九段、灰槽帶、青苗中心都在列。

青苗中心代碼出現那刻，隊頻裡短暫靜默。過去所有『轉運』『教育』『收編』的遮羞布在這些玻璃管前面全碎了。

蘇岚從主控台拉出更致命的資料：原型艙曾接入母艦主航行紀錄接口，並把每次航向調整與樣本產出綁定。簡單說，母艦去哪裡、放棄哪個艙段，部分取決於哪裡能榨出更穩定的輸出。

林燼把這段資料單獨封存，標記為『航向-養殖耦合證據』。這條線若公開，會直接撕開維穩派與收割派共同維持的合法性外衣。

核心區還有一座小型審訊台，台上固定著一具半拆解主機外殼。外殼內部的接口排列，和林燼體內文明主機的早期拓樸幾乎同源。唐雀忍不住問：這是不是另一台被養到一半的主機。

林燼沒回答。他只是伸手碰了一下外殼，主機內部立刻回傳劇烈干擾，像聽到久遠又破碎的求救。

干擾過後，半球艙上方投影突然被喚醒，播放一段停在37%的實驗報告。報告裡的高層研究員說：『共生式樣本不可控，收益低；建議全面轉工業壓榨路線。若有抵抗，先拆其主機，再回收群體。』

這句話讓林燼第一次在沉默艙裡產生近乎本能的殺意。他一直把主機當工具、當武器、當談判籌碼，但在這個半拆解殼體前，他清楚意識到那也是一個活文明的遺骨。

他把殺意壓成命令：全量備份報告、帶走十二支樣本、拆一段同源接口作實物。任何人不得私下刪改一字。

時間壓力已逼近極限。隊伍撤離前，林燼還做了一件事——把半球艙中剩餘的『恐懼刺激腳本』打上公開水印，並設置延時廣播。若他們出不去，灰槽帶至少能收到這些腳本真面目。

就在唐雀切割同源接口時，艙底傳來連續震動。蘇岚看儀表臉色發白：沉默艙低規則潮正在上湧，這裡很快會變成全域漂移帶。

林燼下令即刻撤離，但燕九在最後一個冷封櫃前停住。櫃內有一枚刻著『L-雨棚-原始工務序列』的硬盤。這是第一卷以來最老的疑點之一。

燕九問要不要拿。林燼看了眼重量與剩餘行動力，回答：拿，哪怕丟一把槍。

他們帶著樣本、接口、硬盤和大量原始檔衝向升降筒。背後半球艙的導能管開始逐段熄滅，像一頭巨獸在臨死前抽搐。

升降筒上升途中，隊頻突然插入一道陌生女聲，機械又平靜：『未授權取證完成，觀察序列升階。請攜證據返回，便於分類。』

沒人知道聲音來自哪裡，但四人都聽懂同一件事：他們已經不只是闖入者，而是被某個更高層級主動標記的實驗對象。

他把當前行動的每一筆耗損都寫進清單，因為在這個時代，忘記代價就等於替敵人省子彈。
