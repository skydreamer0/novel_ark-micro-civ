# 第二卷《裂縫稅戰》
## 第35章：鹽釘斷線

第三次影子配額行動前兩小時，鹽釘隊失聯。

唐雀的通訊碼從「低功率待機」直接變成「司法封存」，這種跳變只會出現在整隊被押進臨時審訊艙時。林燼看著終端上的灰色圖標，沒有立刻出發救人，而是先把所有配額資料拆分重加密。

蘇岚忍了三十秒，還是開口：「你不去？」

「去，」林燼說，「但不能帶著整條灰鏈去送。」

這是第二卷最難的一課：救人不能等於自殺，否則下一批人更快死。

燕九從黑市醫療端買到一段監視錄像。畫面裡，鹽釘四人被押進西工坊臨時審訊區，押送隊不是顧衡直屬灰甲，而是區域代理武裝。這意味著對手想把事情做成「地方違規」，避免引爆更大輿情。

「有縫。」燕禾說，「地方案子，流程比較鬆。」

林燼把作戰板分成兩半：

A 線：營救唐雀；
B 線：保住灰鏈不斷。

A 線由林燼、蘇岚執行，目標是把人從審訊區轉運途中截下；B 線由燕九、燕禾守，維持當晚物資分發與情報回收，確保三社區不因營救而斷供。

午夜，審訊區外下起金屬雨。押運車按固定路線走 C 型高架，最弱點在六號匝道下沉段。蘇岚提前三小時埋了電磁脈衝釘，能讓押運車在十五秒內斷電重啟。

「十五秒救四個人，夠嗎？」她問。

林燼檢查彈匣：「夠不夠都得做。」

車隊進入匝道時，脈衝釘同時啟動。前後兩車熄火，中車剛好卡在盲區。林燼從橋腹落下，一刀切斷後門鎖；蘇岚在上方連點兩名槍手。唐雀和隊員被鎖在拘束椅，手臂滿是神經探針痕，顯然剛做過高壓問訊。

唐雀抬頭看見林燼，第一句竟是笑：「我就知道你會來，通緝犯。」

「廢話等活著再說。」林燼把她扛起。

撤離並不順。第三名隊員因探針反噬抽搐，行動速度大幅下降。巡防無人機三十秒內就鎖到熱源，子彈打在橋柱像打鼓。蘇岚把最後一枚煙爆彈扔向上風口，煙霧被金屬雨壓得很低，剛好遮住撤離線。

四人拖著兩名傷員衝進排污涵管時，後方傳來擴音器：

> 「立即停止！你們正妨礙戰時稅流執法！」

唐雀邊咳血邊罵：「連抓人都能叫稅流，真他媽長見識。」

林燼沒回頭，只在隊頻裡丟下一句：「B 線狀況？」

燕九聲音傳來，冷靜得過分：「分發照常，但有人在社區散播消息，說燼站為了救鹽釘停了今晚供氧。人群開始慌。」

這就是對手真正的刀——不是把你打死，而是讓被你保的人先不信你。

燕禾接著匯報：「我把今晚分發清單公開上牆了，還附上時標與收件人簽收，能壓住一部分流言，但不夠。」

林燼邊跑邊下令：「開第二儲備，提前發一批應急氧片。告訴他們，今天缺口我用下次配額補。」

「那你手上的現金流會穿底。」燕九提醒。

「先別讓信任穿底。」林燼回答。

天亮後，鹽釘隊救出成功，但代價不輕：蘇岚左肩擦彈、林燼肋骨裂一根、灰鏈儲備少了四成。唐雀更糟，審訊留下神經後遺症，短期內無法再進裂縫。

她躺在臨時醫療床上，看著林燼：「你救了我，卻可能害自己斷鏈。你圖什麼？」

林燼把藥包放到床邊：「圖一件很笨但有用的事。讓人知道，跟我們合作不會被當耗材。」

當晚，三個社區牆面再次出現灰鏈記號，旁邊多了一句新字：

`斷線可接，人不丟。`

這句話很快傳開，甚至傳到兩支原本觀望的配額小隊耳裡。次日傍晚，他們主動來談影子配額條件。

顧衡的抓捕行動表面成功，實際卻幫燼站打了一則最貴的廣告。

章末，燕九把新接觸的兩支隊伍標在圖上，突然發現一個巧合：

這兩支隊伍出事前，都曾被同一個審計腳本 `QW-Delta-17` 標記「高事故傾向」。

腳本不只在算帳，它在挑誰先死。
