# 第99章 個人養殖回灌

真正把所有人壓得說不出話的，不是第一列武裝火車的時窗，而是那張只剩三成的資產表。

換熱站的冷牆上投著一整面清單。左邊是還能動用的東西：三成彈藥、兩組半壞感測列陣、七枚抑制釘、一批純度參差不齊的淨化包、從第十八段撬下來的算法碎片，以及灰槽帶還能擠出來的最後一批可煉化廢料。右邊則是所有不能不管的洞：燕九後續治傷、灰槽帶最低民生、唐雀佈防、追索遮蔽、下一輪轉進資金。

任何稍微正常一點的人看完，都會得出同一個結論。

這時候最應該做的是留底。

林燼卻把那份清單關了，直接說了一句：「除了最低民生線，剩下的全灌進主機。」

艙室裡瞬間安靜得只剩下蒸汽管的低鳴。

唐雀先炸了，「全灌？你是不是昨晚被那穿甲釘把腦子打歪了？我們就剩這點東西，留底還有機會跑，灌光了就是逼自己只能賭劫車！」

「本來就只能賭劫車。」林燼語氣平得近乎冷酷，「不拿到第一部分名冊，後面不管跑到哪一段，都只是換地方等著被挑。」

燕九坐在一旁，低頭重新綁緊右手的護腕，沒有開口勸。他知道這個決定有多狠，也知道林燼不是為了逞強。三成資源想同時保民生、保轉進、保戰力，看起來穩，其實哪一項都保不住。與其慢慢餓死，不如把還能燒的全燒成一場能咬穿中層的火。

蘇岚則看著那份被關掉的清單，問了唯一真正有用的問題。

「最低民生線留多少？」

「能讓灰槽帶撐過兩天。」

「兩天後呢？」

林燼抬眼看她，「兩天後要嘛我們把火車撕開，要嘛我們沒資格談第三天。」

蘇岚沉默了一瞬，然後點頭。

「那就做最狠的版本。」

主機回灌在半小時後開始。

林燼盤坐在熔爐核心前，把從第十八段帶回來的算法碎片、剩餘高純燃料、戰利品金屬和那三成資源一批批拖進私庫。體內微縮文明像整座城同時被戰鼓敲醒。工匠熔爐一座接一座亮起，戰士校場上成排長矛與重弩被重新掛上規則燃料，神殿高台則升起新的火光。

這一次，祭司沒有再把傷亡解讀成祭獻。

可校準過後的信仰，並不代表它就不猛烈。

相反，當整座文明都清楚下一場外部戰役將決定主機能否從「被挑的人」變成「奪名冊的人」，那股共振更像一股被導正方向的狂熱。祈禱聲、敲鐵聲、校閱口令彼此疊在一起，把規則密度一層層往上推。

工匠最先送出來的是三組通訊中繼骨架。那些東西像被剝了血肉的獸脊，外殼漆黑，節點處卻流著極細的暗銀紋。落到現實世界後，蘇岚帶人只花了十分鐘，就把它們架進伏擊區兩側山壁一樣的廢棄裝甲堆裡。原本一旦進入霧障就會互相失聯的死角，終於被硬生生串成一張能回話的網。

第二批出來的是重裝彈藥與抑制釘外掛模組。

唐雀原本還在心疼資源，看見那批彈藥的瞬間，整個人都安靜了。她蹲下去摸了一枚新鑄好的穿殼釘，指尖能感覺到裡頭壓著的規則燃料正在微微發熱。

「這要是還打不穿車甲，」她低聲說，「那就只能怪我們命賤。」

第三批最慢，也最吃主機。

那是戰獸骨架。

不是完整生物，而是三具能在外部存活數十小時的鋼鐵脊骨平台。工匠把它們做成四足形態，方便在廢軌和裝甲殘骸之間高速躍行；戰士則在背脊兩側加掛了短程火力與牽引鉤。一旦衝上車側，它們不是拿來撕人，是拿來替登車隊硬咬出第一道附著點。

當第一具戰獸骨架從主機投放口裡拖出來時，整個艙室的燈都暗了一瞬。林燼胸口像被人從裡頭狠狠干了一拳，喉間腥甜往上翻，差點當場吐血。

蘇岚立刻往前一步，「過載了？」

林燼抬手示意她別動，咬著牙把那口血壓回去。

「還沒。」

可只有他自己知道，主機的負載線已經被頂到什麼位置。那不是一般意義上的疲憊，而像有人把整座文明的震動都直接壓在他腦骨內側。每一次新產物成形，他都能清楚感到祭司祈禱聲更高、工匠熔錘更重、戰士呼吸更齊。那股共振在給他力量，也在提醒他，只要再多偏一步，狂熱就會重新變成失真。

所以他沒有讓祭司隨意擴張。

每一次新的投放完成，林燼都會親自把回傳到神殿的神諭改寫成同一句話。

不是勝利。

是借來的一次機會。

可就算如此，信徒眼底那股亮得近乎滾燙的東西，還是一天比一天更難壓住。工匠會在交付新武器時先朝神殿方向低頭，戰士則開始自發把校場勝負記成對神諭準確度的驗證。林燼吃得下這股紅利，也更清楚地知道，這股紅利如果哪天失手，就會翻成更大的治理炸點。

但現在，他沒有餘裕先處理未來。

因為現實裡，伏擊區已經成形。

廢軌兩側的裝甲堆裡埋好了通訊中繼，三具戰獸骨架各自匍匐在不同陰影裡，重裝彈藥一箱箱壓進預設火點。唐雀帶著火力位在最後校槍，燕九則拖著還沒好透的手臂，一點一點把登車繩索與鉤爪重新檢查過一遍。蘇岚坐在主控台前，把第一列火車的調度習慣與被標記過的物流節點疊成一條最薄、也最危險的進攻線。

外面霧障很重，遠處的鐵軌像浸在鐵灰色的水裡。

林燼最後一次看完所有人，正要開口，伏擊區最外圍的感測線忽然輕輕震了一下。

一下。

兩下。

第三下還沒來，前方霧裡已經亮起一點極冷的白光。

不是探燈，是車頭燈。

「來了。」蘇岚的聲音壓得極低。

整個伏擊區在那一瞬間安靜得近乎失真。

林燼把手按上主控終端，看著那列正在穿霧而來的光，緩慢吐出一口氣。

三成資源，全部燒光。

現在該看這場火，能不能把第一列車一起點著了。
