# 第109章 模型全相

冷庫裡，三片名冊終於拼上。投影牆上跳出完整模型，所有人都安靜下來。

戰爭需求模型不是單純統計，而是一套自動調節機制：外部戰區損耗升高，底層投放率與收割率同步上調；哪段人命便宜、哪段可承受輿情，就優先抽取哪段。

「我們就是他們的壓力閥。」唐雀聲音發啞。

蘇岚繼續往下翻，手突然停住。高優化投放清單裡，林燼、燕九、她本人，連陶青都被列入不同批次。備註欄寫著：

> 可塑高價值樣本，必要時跨段投放。

燕九低聲咒罵，拳頭捏得發白：「公開吧，現在就公開，讓全艙段一起瘋。」

「公開可以引爆秩序，但系統很快會回復，然後先殺我們。」蘇岚抬頭看林燼，「你怎麼選？」

林燼盯著模型核心，說得很慢：「先寫毒，再公開。先把它鎖死，讓它沒法自我修復。」

所謂毒，不是摧毀資料，而是把「待消除名單」與「核心維護名單」做邏輯纏繞，讓每一次清除都會誤傷高層自己。這會讓系統進入長時間死鎖，也意味著一旦啟動，誰都退不回去。

陶青問：「那我們呢？」

「我們可能先死。」林燼答得很平靜，「但他們以後每次按下清場，都得先割自己一刀。」

決議在三十秒內完成。蘇岚開始搭寫入框架，唐雀負責外網煙幕，燕九帶人死守外環。可還沒等框架完成，審計中樞發出全段播報：

> 物理消除倒數300秒。核心區封艙。

五分鐘。

厚重艙門轟然落下，把他們和外界切成兩半。空氣循環被下調，冷得像墓室。

蘇岚抬起頭，眼眶通紅：「五分鐘不夠。」

「那就把每一秒都當兩秒用。」林燼把神經接口重新插回後頸，「我進去做最危險那段，你們在外面拉住鏈路。」

倒數開始跳：299、298、297……

沒有誰再說勸阻的話。所有人都知道，接下來不是贏不贏，而是能不能把失敗的代價變成敵人永遠的傷口。

### 本章總結（模式B）
1. **本章推進主線**：
   - 真相線：完整拼出戰爭需求模型，確認底層人口被作為動態耗材參數。
   - 戰力線：團隊確立「先寫毒再公開」策略，從證據戰升級為規則鎖死戰。
   - 文明線：核心成員全數被列入高優化投放清單，群體命運與模型直接綁定。
2. **伏筆觸發／推進／回收**：回收「盟友被列名冊」幕級鉤子，推進「反向投放病毒」行動伏筆。
3. **不可逆變化**：艙段級物理消除倒數300秒啟動，戰場進入不可回頭的終局段。
4. **下章可承接壓力**：封艙後僅剩最後時間窗完成寫入，外線需以高損佯攻換取林燼的十秒演算。
