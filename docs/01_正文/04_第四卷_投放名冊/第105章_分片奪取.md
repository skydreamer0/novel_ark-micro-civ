# 第105章 分片奪取

收割盛宴會場白得過分，像一座把血洗乾淨後的屠宰廠。每張桌面都嵌著即時曲線，人口流量、戰損預估、收割收益，在優雅燈光下跳動。

祁序站在主桌中央，笑得得體：「歡迎第十五段代理人。流程開始前，請交出原始密鑰作誠意驗證。」

這是臨時加碼。按既定流程，密鑰應在會後核驗。祁序提前索要，就是要把林燼當場繳械。

林燼把硬殼箱推上桌，打開。裡面不是密鑰，而是三份條款：戰損公開、配給公開、清場責任公開。每份都帶可驗證哈希。

「要誠意可以，先把你們怎麼分死人講清楚。」林燼語氣平穩，「我簽之前，大家一起簽。」

會場各派代表瞬間變臉。遷徙派低頭裝沒看見，技術行會兩名代表互看一眼，誰也不敢先伸手。因為誰先簽，誰就失去把黑鍋甩給底層的餘地。

祁序笑意薄了一層：「你在挑戰會場秩序。」

「錯。」林燼看著他，「我在補上你們故意挖掉的秩序。」

對峙開始，唐雀的活也開始。她趁安檢輪替四十秒空窗，從側台鑽進備援節點，把分發給各派的名冊子片做鏡像。蘇岚在外線以「例行備份」掩護流量，燕九在外環壓住巡邏隊。

「鏡像進度35…62…84…」唐雀額角冒汗，「還差主簽章！」

就在這時，第三層一致性掃描忽然升壓。會場AI把林燼的心像波形與發言內容高頻比對，意圖抓出「表演痕跡」。

林燼乾脆把底層傷亡名單直接拉上公屏，一筆一筆念名字、年齡、死因、所屬配給線。他沒有躲情緒，讓怒和痛都在語調裡。掃描結果竟顯示「高一致」。

他看向祁序：「你們要真誠？這就是真誠。」

同一時間，唐雀終於扣上主簽章。「兩片到手！」

警報立刻炸響。AI發現簽章異常，封鎖門降下。祁序終於不演，抬手下令拘捕。會場守衛向林燼包抄，燕九在外環引爆預埋震盪包，給內線撕開一道側門。

蘇岚在耳麥裡急促回報：「第三片定位完成——審計中樞核心審核層，綁在不可逆物理消除鏈！」

林燼心口一沉。那意味著第三片不靠偷、不靠搶，只能在「被系統完全接管」的狀態下觸碰。

撤離途中，唐雀肩膀中彈，仍死抱著鏡像模組不放。林燼拖著她衝進維修井時，會場後方已傳來群眾與代表對打的混亂聲。林燼那三份條款像火種，把原本體面分贓桌掀成一鍋爛戰。

回到外線臨時點，兩片分片被立刻送進冷庫。蘇岚把第三片路徑投上牆，線路像一根直刺心臟的針。

「只剩一條路。」林燼說。

「主動投案。」燕九接話，臉黑得像鐵。

「對。」

屋裡沒人歡呼。因為所有人都知道這不是奇謀，是把自己送進屠刀上。

林燼進觀測模式確認內世界回應。鏡廳裡，他看見每個人的心像都在震動：蘇岚冷室裡的錄音機噪聲加重；燕九長廊的燈泡忽明忽暗；唐雀機房牆上多了一道裂紋。祭司長說：「外線每一次選擇，都在改寫他們內世界的牆。牆若裂到一定程度，人就會先於肉身崩壞。」

林燼退出時，手心全是冷汗。他不怕死，他怕把所有人的內世界都拖成廢墟。

但戰局沒有給他第三種選擇。


撤出會場後，灰槽帶臨時據點像被捅破的蜂巢。有人主張立刻散開躲追捕，有人主張趁亂再搶一把。蘇岚把兩片分片投到牆上，逐字展示哈希完整性，逼所有人先看證據再吵。

唐雀在止血時仍堅持繼續解包。她發現兩片分片裡藏了一段「內世界評級規則」：系統會根據個體在極端事件中的情緒穩定與服從概率，計算其投放價值。也就是說，高層不只收割人口，還在收割「可塑內世界」。

「這就是高優化保留名單的底層邏輯。」蘇岚說，「他們挑的不是最強的人，是最容易被改造成工具的人。」

林燼立刻下令對全隊做一次反向校驗：任何與既往內世界快照差異過大的隊員，先保護隔離，不得上前線。這條規則及時攔下一名差點失控的新人。那人並非叛變，只是過度驚嚇後進入強迫服從狀態，極易被敵方指令劫持。

燕九看完流程，吐了口氣：「以前我們怕的是子彈，現在還得防心被改寫。」

「一直都要防。」林燼說，「只是以前我們不知道。」

夜裡，唐雀終於把第三片坐標的附帶條件解出來：核心審核層會在拘押對象進入後，先抽取其「內世界主軸」作為身份鹽值。這意味著林燼若進去，將被迫把最核心那部分暴露給核心AI。

蘇岚看著他，聲音很輕：「你會被看見。」

林燼點頭：「但也只有這樣，我才能看見它。」


補時階段裡，蘇岚要求所有值班員在每次交接前口述「三條釘」：
一、我現在最怕失去什麼；
二、我這一班的首要保護對象是誰；
三、若命令與民生帳衝突，我站哪邊。

這套口述來自早年灰鏈值班模板，平時被嫌麻煩，戰時卻像把人從麻木裡拽回來。唐雀說它像給腦子上扭力扳手，能防止人在連續高壓下滑成純執行器。燕九最初最不耐煩，後來每次都第一個說，因為他發現只要說出口，槍就不會亂指。

林燼不在外線時，這套流程更像某種集體內世界對齊。每個人都在提醒自己：我們不是為了證明誰更硬，而是為了讓更多人活到下一班。這些話不熱血，甚至有點笨，卻在最壞的夜裡穩住了隊形。

同時，主機端的鏡廳新增「交接刻痕」功能。每個小隊完成一班，就在鏡廳石柱上留一條刻痕，旁註當班遺失與保住的東西。刻痕越多，祭司派推演顯示群體失序概率越低。工匠派把這結果轉成外線建議：任何連續兩班未做交接刻痕的小隊，必須強制休整。

這些細碎規則沒有爽感，卻是戰爭能否長跑的底盤。高層模型擅長把人切成參數，而他們開始學會把參數再拼回人。


更關鍵的是，這些交接語句開始外溢到居民層。配給站志工也學著在開窗前互問三句：今天先護誰、先發哪類藥、若突發搶奪怎麼保孩子與老人。最初有人笑這是裝腔，兩天後笑聲少了，因為秩序確實變穩。連最焦躁的隊伍也發現，當你知道旁邊的人在守什麼，恐慌就不容易把隊形撕開。

蘇岚把這現象寫進戰報，命名為「內世界公開最小單位」：不是把所有秘密掀開，而是至少說出你此刻的守護對象。她說這是對抗收割模型的第一步。模型靠隱匿與拆分運作，他們就靠最小透明把人重新連成網。

唐雀隨後做了個小工具，讓每個節點能用三個符號標記狀態：可守、可換、不可棄。符號簡單到孩童都看得懂，卻讓臨場調度效率提升一截。燕九嘴上嫌麻煩，實戰裡卻第一個要求全隊貼符號，因為他再也不想在煙霧裡猜誰還有力、誰已崩潰。

林燼後來回看這些紀錄時，才發現真正讓灰槽帶撐住的，不只是幾次漂亮反殺，而是一次次把「每個人都有內世界」變成可執行流程。那讓他們在金屬與火裡，仍保住了人的形狀。


那晚還有一個不起眼卻關鍵的動作：蘇岚讓各節點把「內世界守護對象」匿名匯總，做成熱點圖。結果顯示，大家最常填的不是自己，而是孩子、醫療站、回家路。這份圖讓調度策略再度微調：先守能承載最多守護對象的節點，再談戰術漂亮。林燼後來評價這是本幕最貴的資料——它不產生火力，卻決定火力應該為誰而用。


這份熱點圖也被祭司派刻進鏡廳底座，作為下一輪決策的硬約束：任何高收益行動若無法保護至少一個核心守護對象節點，直接否決。這條看似保守，卻讓團隊在最亂的時候沒有滑向純賭命。
