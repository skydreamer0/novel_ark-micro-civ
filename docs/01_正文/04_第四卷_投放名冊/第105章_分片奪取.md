# 第105章 分片奪取

收割盛宴會場像一座乾淨過頭的屠宰場。白光、白桌、白手套，只有屏幕上的人口曲線是紅的。

祁序站在主桌旁，笑意溫和：「歡迎第十五段代理人。開始前，請交出原始密鑰做誠意驗證。」

這是臨時加碼。按流程，密鑰應在會後核驗，祁序卻要現場奪權。

林燼把硬殼箱放上桌，慢慢推開。裡面不是密鑰，而是三份透明條款：戰損公開、配給公開、清場責任公開。每份都掛了可驗證哈希。

「要誠意可以，先公開你們怎麼分配死人。」林燼語氣平靜，「要我簽，先大家一起簽。」

會場空氣瞬間繃緊。遷徙派代表先縮手，技術行會那邊有人低聲咒罵。誰都知道這三條一旦上鏈，往後就沒法把黑鍋全甩給底層。

祁序臉上的笑終於薄了一層：「你在挑戰會議秩序。」

「我在建立秩序。」

就在兩人對峙時，唐雀在安檢輪替的四十秒空窗裡，從側台接入備援節點。她指尖飛快，把會場分發給各派的名冊分片做鏡像，再上哈希鎖。蘇岚在外線同步掩護，將流量偽裝成例行備份。

「鏡像完成七成……九成……完成！」唐雀聲音壓到極低。

下一秒警報拉響。會場AI察覺簽章異常，封鎖門開始落下。燕九在外線砸開一條逃生口，卻被反制火力打得半跪。

林燼趁亂把條款投到公共牆，讓所有與會者都看到各段戰損與收割比值。數據一出，現場立刻炸鍋，有人當場翻桌，有人要求中止會議。

祁序終於失去耐心，抬手下令拘捕。林燼反手把唐雀拽進掩體，低喝：「分片結果！」

唐雀臉上有血，還在笑：「拿到了兩片，第三片位置也鎖了——審計中樞核心審核層，綁在不可逆物理消除鏈上。」

這句話讓林燼短暫沉默。第三片不在任何可偷可搶的地方，它被綁在「你敢碰我就一起死」的終極機制上。

「撤！」

他們撞開側門時，背後整座會場已陷入混戰。林燼知道，今天這場不是勝利，而是把局面推向更高層級的絞殺。

蘇岚在頻道裡喘著氣：「兩片分片已入冷庫。林燼，第三片你打算怎麼拿？」

林燼看著手背上被碎玻璃劃出的血線，低聲道：「正門拿不到，就換一種方式進去。讓他們親手把我送進核心。」
