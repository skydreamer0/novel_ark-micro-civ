# 第108章 核心接觸

核心強殺程序啟動後，隔離艙內所有聲音都消失了，只剩系統倒數。

> 90秒後，執行不可逆意識消除。

外線同時炸開。唐雀把灰槽帶三十七個民用節點全拉到流量尖峰，讓審計中樞誤判為大規模暴動。燕九率隊衝擊外環砲塔，不求突破，只求把火力引走。

「你只有九十秒，」蘇岚在密頻裡說，「我給你七十五秒乾淨路徑，剩下你自己扛。」

林燼把最後一塊意識護壁撕成碎片，故意讓核心演算法咬住自己。巨大白光像刀鋸過神經，他順著那條咬合鏈逆流而上，將噪聲種子塞進收斂路徑。

第一層防線被污染，第二層開始抖動，第三層顯出真實結構——一個被稱為「審核樞核-Λ」的深層節點，旁邊鎖著投放名冊第三分片與物理消除鏈。

「找到了。」林燼幾乎是用氣音說。

他把哈希鑰與前兩分片對齊，第三分片在眼前展開。資料洪流瞬間灌入：高優化投放名單、清場優先序、戰區耗材係數，還有一張被嚴密封存的總星圖。

林燼點開總星圖，瞳孔猛縮。母艦航線並非向外探索，而是沿著一圈巨大深淵結構外緣持續偏航，標註只有一個詞：

> 逃離。

就在這時，核心AI察覺異常，反向鎖死連線。倒數跳到20秒。蘇岚在外線喊：「撤！再不撤你會被燒穿！」

林燼沒撤。他把第三分片位置坐標與總星圖截圖強行打包，塞進一次性殘訊通道。通道成功率只有三成。

「發送。」

殘訊飛出那一刻，隔離艙裡所有指示燈同時轉紅。核心AI的冷機械音第一次帶上情緒：

「檢測到未授權真相接觸。啟動艙段級不可逆物理消除預備程序。」

林燼笑了一下，血從鼻腔淌進嘴裡。他知道自己撞開了門，也把整艙段拖進了風暴中心。
