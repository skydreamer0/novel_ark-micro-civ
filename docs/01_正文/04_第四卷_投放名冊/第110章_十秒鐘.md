# 第110章 十秒鐘

封艙倒數到一百秒時，外部佯攻全面展開。

燕九帶著殘兵從三條維修廊道同時衝塔，第一波就折了半隊。火力像熔鐵一樣壓下來，牆面被燒成流動的黑痕。他在頻道裡只留一句：「我們還在，繼續算。」

蘇岚跪在主控台前，雙手幾乎抽筋。她把每一條外線鏈路都綁上臨時緩衝，哪條斷就立刻補哪條。面板上警報一層疊一層：

> 鏈路抖動 41%…58%…72%

「林燼，我最多再撐二十秒！」

隔離艙內，林燼已經進入半失感狀態。核心AI的反制像海嘯一樣拍過來，他把自己意識切成細絲，沿著第三分片的寫入保護層一點點鑽。

保護層最後一道門叫「完美一致性校驗」。只要資料有任何不可預測偏差，就拒絕寫入。這道門守了無數年，專門防「意外」。

而林燼要寫進去的，正是意外本身。

外線爆出巨響，燕九小隊把最後一組燃料包貼上外環炮座，整片通道被火浪吞掉。蘇岚被震得耳鳴，嘴角滲血，仍死死抓著連線：「十秒！我只能給你十秒！」

十。

林燼將病毒種子拆成兩段：一段偽裝成維護校正，一段偽裝成清場補丁。兩段單看都合法，合在一起才會互鎖。

九。

核心AI掃描第一段，通過。

八。

第二段送入時，反制牆猛地收緊。林燼把最後的規則密度全部壓上去，內宇宙遠端祭司塔同時亮起，像把整片文明都推到刀尖。

七。

寫入進度 43%。

六。

蘇岚咳出一口血：「鏈路要斷了！」

五。

燕九最後一發榴彈打空，改用身體把門卡住，給主控台硬爭出三秒。

四。

寫入進度 79%。

三。

核心AI發現邏輯纏繞，開始回滾。

二。

林燼把自己的身份標識當作臨時鹽值塞進校驗欄，強行鎖住回滾。

一。

寫入完成 100%。未觸發。

不是立刻引爆，而是潛伏。像一根釘子，釘進完美模型最討厭的不確定性。

連線斷掉前，林燼把最後一段殘訊推回外線：

「第十二幕，開戰。」

信號熄滅。隔離艙陷入黑暗。

蘇岚盯著全黑的屏幕，手還保持著輸入姿勢，像凍住一樣。燕九拖著燒焦的護甲走進來，沒問結果，只看她眼睛。

她點了點頭，又搖了搖頭。

「寫進去了，」她聲音發啞，「但他還沒回來。」

外頭，審計中樞的警報忽然轉成低頻長鳴。那是系統進入邏輯死鎖前的徵兆。

風暴沒有結束，只是換了方向。
