# 第102章 稅軌截流

第一列稅軌列車還在兩公里外，公告屏先亮起血紅字體：

> 第十五段配給稅臨時上調18%，違者列入惡意斷供追責。

公告剛跳出，南側配給站就傳來玻璃碎裂聲。人群不是在暴動，是在恐慌。誰都知道只要連續兩天領不到濾芯，肺裡的黑塵會把人活活磨死。

林燼站在調度塔高窗前，耳麥裡三條頻道同時開啟。A隊在西側舊管道布爆點，B隊潛進主站下層維修井，C隊已混入居民節點準備替代指令。蘇岚負責把一切動作封裝成「戰時民生補救流程」，讓審計法務至少慢半拍。

「倒數六十秒。」唐雀聲線平直，「列車簽章正常，護送無人機兩編隊，車頭搭載審計遠控核。」

燕九吐掉嘴裡的血味，「給我七秒空窗，我把主站門掰開。」

「你有五秒。」林燼說。

爆點起火。A隊在西側炸出一條看似失控的火龍，審計武裝果然向西偏移。B隊趁機切進主站。燕九第一個撞門，護甲被電弧燙得焦黑，仍硬把插入器釘進控制槽。

「主站接管三成！還差稅軌簽章！」

蘇岚同時讓C隊在公共屏推送臨時分發圖，每一箱物資來源、用途、簽收人全部公開。她故意把字放大：

> 本次截流屬戰時民生優先條款第11-3款，歡迎審計署現場核對。

這行字像把刀。審計法務終端開始反覆調取條款，因為那條規則確實存在，只是從來沒人敢在槍口下啟用。

列車進站時，鏈軌摩擦的尖嘯蓋過了人聲。月台上的人都以為會先爆槍戰，林燼卻先對著公攝像頭宣讀條款，再讓陶青帶醫療組優先點收止血纖維和抗菌凝膠。不是搶貨，而是先「定義」這批貨屬於誰。

審計遠控核在第三十秒發起反制，命令無人機鎖定截流小隊。唐雀把一段早準備好的噪聲包塞進行為模型，讓AI把燕九誤判成官方維穩隊。短短四秒誤判，B隊完成了簽章翻轉。

「截流有效！車廂一到三改掛民生帳！」

人群先是愣住，隨後爆出壓抑已久的喊聲。蘇岚在高台上大吼：「按清單領，不插隊，不搶藥！誰搶就是幫收割派！」

秩序竟真被喊回來。老人牽著孩子，傷員讓位給重傷，工匠幫忙搬箱。灰槽帶第一次在槍口下展現出「不是牲口」的樣子。

代價也立刻來了。第二波無人機自上層俯衝，A隊撤退時被追射，兩人當場倒下。燕九回身想救，被林燼硬拉：「不回頭！你現在回去，今天這條線全斷！」

燕九眼白泛紅，最後只把戰術刀插回腿側，轉身掩護隊伍撤。

就在全隊後撤時，唐雀忽然在頻道裡壓低聲音：「我抓到一條幽靈軌，簽章標記M-清洗物資。下一列不走公開站，直達審計中樞外環。」

蘇岚把路徑拉上牆，所有人心口同時發冷。這不是補給線，是清場線。今天他們截下的醫療耗材，很可能是原本用來維持「可收割人口穩定度」的工具。

「也就是說，」陶青盯著路徑，「他們先讓我們半死，再送藥讓我們活到下一次收割。」

林燼看著月台上被血和油混成黑泥的軌道，說：「對。這就是模型的仁慈。」

夜裡回到調度塔，林燼接入主機觀測模式。內世界裡，祭司塔前站著一排剛被命名的「紀錄者」，他們把今天每一箱物資、每一個傷亡、每一個被迫放棄的人名刻進銅板。工匠派則根據外線影像，推演下一次截流的最佳燃爆位置。戰士派要求立刻去搶幽靈軌。

林燼問：「你們覺得最先要救哪裡？」

祭司長回答：「先救會說話的人。若人人只剩求生本能，內世界會塌，外世界也會塌。」

他退出觀測模式時，耳邊仍是月台的尖嘯。這一戰改變了配給流向，也改變了灰槽帶對自己的認知：他們不是被動等死，而是能在規則縫裡把刀插回去。

但幽靈軌像一根更長的刺，已經對準審計中樞。


撤回調度塔後，截流清單沒有立刻封存，而是被蘇岚拆成三層公開：底層看得到總量與分發；醫療組看得到批次與有效期；戰術組看得到來源路由與風險標記。這種透明讓黑市中間商一夜間失去抽成空間，幾個慣常倒貨的人直接被群眾堵在巷口。

林燼沒讓人動私刑，只讓他們在公共牆前逐條對帳。每核對出一筆「截流前被挪用」的耗材，就把責任人名字貼進紅框。到了清晨，紅框滿了三整面牆，所有人第一次直觀看見，灰槽帶不是突然窮，而是被穩定偷空。

與此同時，唐雀對幽靈軌做了逆向模擬，發現M-清洗物資裡混有心理穩定劑和群體鎮靜霧化包。這代表審計中樞不只要清場，還要在清場前「管理恐慌」，確保收割前情緒曲線處於最佳區間。陶青看到後差點把杯子捏碎：「連我們害怕到哪個程度都在他們計算裡。」

晚上，林燼進觀測模式開臨時議會。祭司派主張在外線公開「情緒收割」證據，工匠派反對，認為過早公開會引發全面失控。戰士派則要求先破壞鎮靜霧化包，讓對手失去節奏控制。三派吵了半刻鐘，最後共同提交折衷案：先對內公開、對外留證，並把下一次截流重點從「救急」轉為「搶奪對手節奏工具」。

林燼採納。這一決策直接改變了戰術目標：下一戰不再只搶箱子，而是搶「誰定義恐慌」。

他離開觀測模式時，月台上還有未洗乾淨的血。那血提醒他，任何漂亮的分發圖背後，都有被犧牲的人名。模型不記名字，他們得自己記。


補時階段裡，蘇岚要求所有值班員在每次交接前口述「三條釘」：
一、我現在最怕失去什麼；
二、我這一班的首要保護對象是誰；
三、若命令與民生帳衝突，我站哪邊。

這套口述來自早年灰鏈值班模板，平時被嫌麻煩，戰時卻像把人從麻木裡拽回來。唐雀說它像給腦子上扭力扳手，能防止人在連續高壓下滑成純執行器。燕九最初最不耐煩，後來每次都第一個說，因為他發現只要說出口，槍就不會亂指。

林燼不在外線時，這套流程更像某種集體內世界對齊。每個人都在提醒自己：我們不是為了證明誰更硬，而是為了讓更多人活到下一班。這些話不熱血，甚至有點笨，卻在最壞的夜裡穩住了隊形。

同時，主機端的鏡廳新增「交接刻痕」功能。每個小隊完成一班，就在鏡廳石柱上留一條刻痕，旁註當班遺失與保住的東西。刻痕越多，祭司派推演顯示群體失序概率越低。工匠派把這結果轉成外線建議：任何連續兩班未做交接刻痕的小隊，必須強制休整。

這些細碎規則沒有爽感，卻是戰爭能否長跑的底盤。高層模型擅長把人切成參數，而他們開始學會把參數再拼回人。


更關鍵的是，這些交接語句開始外溢到居民層。配給站志工也學著在開窗前互問三句：今天先護誰、先發哪類藥、若突發搶奪怎麼保孩子與老人。最初有人笑這是裝腔，兩天後笑聲少了，因為秩序確實變穩。連最焦躁的隊伍也發現，當你知道旁邊的人在守什麼，恐慌就不容易把隊形撕開。

蘇岚把這現象寫進戰報，命名為「內世界公開最小單位」：不是把所有秘密掀開，而是至少說出你此刻的守護對象。她說這是對抗收割模型的第一步。模型靠隱匿與拆分運作，他們就靠最小透明把人重新連成網。

唐雀隨後做了個小工具，讓每個節點能用三個符號標記狀態：可守、可換、不可棄。符號簡單到孩童都看得懂，卻讓臨場調度效率提升一截。燕九嘴上嫌麻煩，實戰裡卻第一個要求全隊貼符號，因為他再也不想在煙霧裡猜誰還有力、誰已崩潰。

林燼後來回看這些紀錄時，才發現真正讓灰槽帶撐住的，不只是幾次漂亮反殺，而是一次次把「每個人都有內世界」變成可執行流程。那讓他們在金屬與火裡，仍保住了人的形狀。
