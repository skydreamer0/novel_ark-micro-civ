# 第102章 稅軌截流

第一列稅軌列車還沒進站，公告屏就先亮起紅字：

> 第十五段配給稅臨時上調 18%，違者視同惡意斷供。

同一時間，南側居民區爆出搶糧，人潮衝向配給站。唐雀在耳麥裡罵了一句：「他們用公告先點火，再讓列車來收屍。」

林燼盯著倒數計時，冷聲下令：「A隊，假爆破；B隊，進主站；C隊，上民用節點，準備替代指令。」

轟的一聲，西側舊管道被A隊引爆出大片火光。審計武裝立刻轉向，主站安防空出七秒。七秒內，燕九帶B隊穿過維修井，硬切進調度層。防火門落下時，燕九的護甲被燒穿一條口子，他咬牙沒出聲，把插入器拍進主控槽。

「主站接管三成，還差稅軌簽章！」

另一頭，C隊在底層公共屏上同步推送臨時配給流向。蘇岚把每箱物資從哪來、去哪、誰簽收全部公開。她在頻道裡說：「讓他們看懂，這不是劫車，是把我們自己的血拿回來。」

列車進站，裝甲門打開，成排稅箱滑入轉運軌。林燼親自帶人上前，沒有第一時間搬貨，而是先對著鏡頭念出審計條款編號，宣告「戰時民生優先截流」。這一手讓祁序派來的法務終端短暫宕機，因為條款存在，只是從未有人敢用。

「截流開始！」

鏈鋸切開封條，稅箱裡不是糧食，是高純氧濾芯、抗菌凝膠、止血纖維。灰槽帶最近三周最缺的東西，全在這裡。人群先是沉默，接著爆出壓不住的吼聲。

審計反應很快，第二批無人機已從上層壓下。唐雀突然喊：「別戀戰！我在路由裡抓到一條幽靈軌！」

她把軌跡丟到全隊面板：下一列列車不再經公開站，將直接送往審計中樞外環，標記為「M-清洗物資」。

林燼看了兩秒，抹掉臉上的灰：「我們截到的只是皮，肉還在裡面。全隊撤，準備下一輪。」

燕九拖著受傷腿撤離時回頭看了一眼月台。民眾正按蘇岚發的清單分發物資，第一次有人拿著終端大聲核對簽收。這種秩序比槍更難建立，也更難摧毀。

「林燼，」蘇岚壓低聲音，「你知道那條幽靈軌意味著什麼。」

「意味著我們要去審計中樞門口搶東西。」林燼把槍機推回去，聲音像磨過的鋼片，「也意味著，他們開始把戰場往我們喉嚨裡塞。」

### 本章總結（模式B）
1. **本章推進主線**：
   - 戰力線：完成稅軌列車首次戰術截流，建立A/B/C三隊協同實戰模型。
   - 文明線：民生帳公開流落地，底層開始形成可核對的分發秩序。
   - 真相線：發現直通審計中樞外環的幽靈軌（M-清洗物資）。
2. **伏筆觸發／推進／回收**：推進「審計深井路由」與「戰時稅則反用」伏筆。
3. **不可逆變化**：灰槽帶群眾首次參與可審計分發，舊有黑箱配給模式被打穿。
4. **下章可承接壓力**：傷員激增且醫療資源瀕臨耗盡，若救治系統崩潰，戰術勝利將立刻轉為治理災難。
