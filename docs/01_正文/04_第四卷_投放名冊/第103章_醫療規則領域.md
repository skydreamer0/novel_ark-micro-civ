# 第103章 醫療規則領域

臨時醫療站外，擔架排到三個轉角。第一批截流的物資剛到，第二批傷員就淹進來，連陶青都被逼得把手術燈拆成四組輪用。

「可逆重傷優先，失血超閾值先壓，腦損超線的放後！」陶青吼到嗓子嘶啞，「我不是神，誰都救不了就等於誰都救不成！」

一名少年抓住林燼袖口，手上全是乾掉的血：「我媽在裡面，求你給她一張治療符。」

林燼看向蘇岚。蘇岚很快回報：「主機算力可用值只剩四成，若無上限開符，今晚就會過載。」

林燼沉默了一秒，對陶青說：「分三級。A級可逆重傷，給低階治療符；B級靠手術和藥；C級先鎮痛，等下一批資源。」

這句話像刀。可整個醫療站沒有第二種活法。

祭司派很快在內宇宙開啟同步儀式，工匠派把簡化符紋刻進可拋式金屬片，讓外線醫療員能直接啟動。第一批符片用下去，止血速度明顯提升，但蘇岚面板上的紅字也開始跳：

> 主機過載率：67%→74%→79%

「再拉就會燒掉推演核心。」蘇岚咬牙，「你得下停用令。」

林燼看著手術台上那排人，最後還是點頭：「治療符每日配額封頂，超額一律轉手術流程。」

消息一出，醫療站外立刻有人砸門，罵聲、哭聲、祈禱聲混在一起。燕九拖著傷腿出去維持秩序，胸口剛換的止血貼又滲紅。

深夜，一名被俘的審計外勤在陶青手裡醒來，嘴唇乾裂，第一句話是：「你們不知道嗎？收割盛宴七十二小時後開場。第十五段只是前菜。」

他供出的時間表和蘇岚抓到的幽靈軌完全對上。更糟的是，名單裡標了醫療站座標，備註是「可在盛宴前清場」。

「他們要先拆我們的救命點，再辦盛宴。」陶青把手術手套丟進桶裡，臉色比牆還白。

林燼走到醫療站門口，外面還有人排隊等救。他終於明白戰爭需求模型的殘酷不只在前線，而是在每一個誰先活、誰後死的決定上。

「蘇岚，」他說，「把醫療站列為最高保護節點。從現在起，誰敢碰這裡，我們就把他的帳單掛上全艙段。」
