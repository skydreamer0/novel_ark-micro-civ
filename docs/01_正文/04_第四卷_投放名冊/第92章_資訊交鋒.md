# 第92章 資訊交鋒

B-17氧氣黑市沒有天亮這回事。

林燼踩下最後一段鐵梯時，頭頂的呼吸計費牌正好翻跳到新的分鐘數。猩紅數字沿著整條地下長廊一格格往前跑，像一排替人讀秒的墳碑。每一座小隔間外都掛著不同價格的氧氣標籤，純度、流量、停留時長，甚至談話區的靜音權限，都明碼標價。

「普通呼吸區，一分鐘三十六點。」蘇岚看了眼牆上的費率，臉色很難看，「如果進核心計時區，翻四倍。」

「他們不是在賣氧氣。」燕九半邊臉藏在面罩後，聲音悶得發沉，「是在賣猶豫的資格。」

林燼沒有說話，只把灰槽帶最後那點能挪的流動額度刷進了入口終端。

刺耳的滴聲響起，三人的臂環同時亮起綠光。計時開始。

遷徙派的人比他預想得更謹慎。

計時區最深處沒有包廂，只有一張焊死在地面的合金桌。桌對面坐著一個穿灰白過濾外套的人，面罩全封，只露出一塊窄長的觀察屏。對方連坐姿都像量過，沒有一寸多餘動作，身後兩個看似搬運員的護衛則始終停在氣閥與槍線都最舒服的位置。

「證明你不是帶著假包來騙氧氣的。」對方開口，聲音經過變調，分不出男女。

林燼沒有坐下，只把一枚封裝片推到桌面中央。

「先證明你不是想空套路由。」

對方看都沒看封裝片，「路由前綴。」

林燼報了一串昨夜觀察員離場後殘留在公共管線裡的前綴碼。

對方指尖在桌面輕敲一下，又報出第二段校驗要求。林燼這次慢了一秒，才把黑盒路由圖裡截下來的尾碼補上。蘇岚站在他斜後方，像是隨時能替他補算，但她從頭到尾一個字都沒說。

第三輪驗證更狠。

「養殖實證。」對方道，「不用全部。給我一個足以讓我今晚睡不著的片段。」

林燼把第二枚封裝片放上去。

「這是死艙倉壁內測到的回灌配比。」

對方終於伸手，兩根手指捏起封裝片，接到腕端的小型驗證模組上。幾秒後，那塊狹長觀察屏裡閃過一抹極短的光。哪怕對方沒有表情，林燼也知道，這一刀已經切進去了。

「所以，」變調後的聲音重新響起，「你手上確實有能把很多人拖下水的東西。」

「我手上有的是會讓你們看清自己也只是備料的東西。」林燼坐下了，目光始終停在對方那片冷色屏幕上，「我需要一條中層窗口。你需要證據和下一步流量。我們誰都別裝得像做慈善。」

計費牌在他們頭頂又跳了一分鐘。

每多待一秒，臂環裡的扣款提示就會無聲震一下。這地方最狠的不是槍，而是它把談判本身做成了出血。

對方像是笑了一下，又像只是屏幕亮度跳動。

「窗口有，但你不值整條路。你先回答我一個問題。昨夜那兩名沒有徽章的觀察員，為什麼沒當場拿走你？」

林燼眸光微微一凝。

這個問題不是試探，是稱重。

「因為他們想看我能不能替他們把更難看的東西先掀出來。」他平靜地說，「拿我，比放著我走更沒價值。」

對方沉默了兩秒。

「好答案。」

就是那兩秒的空白，把殺意送進來了。

林燼先看見的不是人，而是氣閥數值。

左側第三道供氧管的流量忽然不正常地下滑了零點七，像有人在總控裡悄悄捏了一把。幾乎同一時間，燕九肩膀一緊，整個人已經側身擋到蘇岚前面。

「趴下！」

嘭！

不是子彈，是一枚極小的壓縮污染囊從側面通風槽打了進來。囊體落地即裂，裡面噴出的不是煙，而是一層帶細碎金屬粉的灰霧。這玩意不是為了大範圍殺人，而是專門用來毀呼吸模組、讓人死得像設備事故。

黑市裡瞬間炸開尖叫。

那兩個看似搬運員的護衛第一時間抬槍，但不是朝污染囊，而是朝林燼。他們不是來保護談判的，是來確保在最亂的時候，持證據的人先死。

林燼已經翻過桌面。

他落地的同時，主機算力像被一根高溫針直接插進腦髓，瘋狂往外鋪。眼前的黑市結構被瞬間拆成數百條流量線，氣閥、照明、靜音牆、費率閘門、面罩認證模組，全都在他腦海裡拉成了可供切換的節點。

刺客不只一個。

右後方收費台前那個縮著肩膀的老人，袖子裡藏著短距針槍；牆邊蹲著的清潔工則正把第二枚污染囊推向總控管口。

「蘇岚，左側總閥！」

「看到！」

蘇岚幾乎是在林燼開口同時竄出去，手裡短接器狠狠戳進牆上的維護口。她不是去關閥，而是用林燼剛剛送過去的算力窗口，硬把普通區與計時區的供氧權限對調了一瞬。下一秒，原本應該窒住林燼這一桌的閥門反而猛地回灌，灰霧被高流量氣柱打散，當場沖回左側走廊。

隱在那裡的老人悶哼一聲，面罩直接爆出裂紋。

燕九已經衝了上去。

他沒有開槍，因為黑市裡人太密。他直接把旁邊那台氧氣計費機掀起來，當頭砸在老人身上。金屬外殼裂開，紅色數字一陣亂跳，收費聲與慘叫聲混成一團。

另一頭，真正的主殺手終於動了。

那名「清潔工」猛地從地上彈起，整個人像一把壓扁後突然張開的彈刀，速度快得不像底層改造。他的護臂上亮起細密的反干擾紋路，刀鋒卻不是往林燼喉嚨去，而是直取桌上的聯絡人。

他們要的不是單純殺人。

他們要讓這場交易徹底變成一場找不到源頭的黑市事故，讓任何一條線都死在這裡。

林燼一腳把合金桌踹斜，桌面與刀鋒擦出一串尖銳火花。同時，他把主機算力灌進對方護臂的認證模組裡，硬把那上頭的防護回路當作黑市的普通面罩識別去讀。

錯配只持續了零點三秒。

但夠了。

刺客手臂上的反干擾紋瞬間亂了一拍。林燼抓住這一拍，脈衝刀直刺進對方腋下裝甲接縫。那名刺客身體明顯一僵，卻沒有退，反手就把一枚信標釘進林燼腳邊的地面。

燕九撲過來想踢開，第二枚污染囊卻在這時炸在旁邊。

他只能轉向，用身體把兩個嚇傻的黑市平民連同蘇岚一起壓到遮蔽板後。灰霧刮過他的護頸與左臂，金屬表面立刻浮出一層不正常的暗斑。

「燕九！」唐雀的聲音從外部通訊裡炸進來，「外面來人了，兩組，不像黑市保衛，是專門封口的！」

桌對面的聯絡人終於動了。

對方沒有幫忙，只是極快地後撤一步，避開交火中心，然後看著林燼用最短的時間處理掉那名刺客。那目光像在看一件剛剛被投進熔爐、還沒定價的貨。

林燼拔刀時，那名刺客已經在吐血。他面罩裡的呼吸聲破碎得像漏風的焊管，卻還是抬起兩根手指，點了點地面那枚開始閃紅的信標。

「坐標……確認。」

紅光一亮到底。

整個黑市的牆面都在同時跳出一條新的匿名通告。

【可追索目標已標定。】

蘇岚臉色一白。燕九則抬起左臂，暗斑已經沿著皮膚底下蔓出了一條細線。

聯絡人終於把一枚灰色權限片拋到桌上。

「你手上的東西，是真的。」對方說，「但你現在還不值一條完整通行權。想換中層窗口，先去動第十八段的通行競價系統。把那套按價買命的東西砸開，我再跟你談下一輪。」

「你就這樣看著我們被追索？」林燼盯著對方。

「不然呢？」變調聲平靜得近乎殘酷，「你現在之所以還有交易價值，就是因為有人急著想讓你死。」

對方說完，帶著人乾脆利落地從後側維護門撤了出去，像從來沒來過一樣。

唐雀在外部通訊裡急得快罵人，「再不走，封口隊就進計時區了！」

林燼一把抓起那枚灰色權限片，另一手把還在閃紅的信標狠狠踩碎。

「撤。」

他轉身時，黑市頂部的計費牌又翻了一分鐘。

每一筆跳動的數字，都像在提醒他，從這一刻起，他們的呼吸、位置和名字，都正式進入了更高一級的追索清單。
