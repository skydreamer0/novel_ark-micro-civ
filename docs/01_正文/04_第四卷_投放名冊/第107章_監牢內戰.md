# 第107章 監牢內戰

「交出密鑰，三十分鐘。」

審計官的聲音在隔離艙裡迴盪，緊接著投影亮起：蘇岚被押在刑架上、燕九跪在血泊裡、唐雀倒在冷庫門口。畫面逼真到連呼吸霧都對。

林燼盯著幻象，沒有動。

「你每沉默一分鐘，就有一名盟友被處置。」審計官敲下第一個指令。

痛覺衝擊像鐵釘扎進顱骨。林燼悶哼一聲，意識邊界卻在下一刻被另一股力量撐開——內宇宙祭司派點亮了信仰塔，工匠派把演算骨架拉成防線，戰士派在邊界築起第一層「拒絕寫入」護壁。

「主上，」祭司長聲音傳來，「我們撐得住一輪。」

林燼咬牙回應：「不是撐，要拖。拖到外線把噪聲種子打進來。」

外部世界，蘇岚正趴在醫療站底層管線裡。她把唐雀改好的噪聲種子塞進廢棄麻醉通道，手背被蒸汽燙出大片紅痕也不鬆手。「三、二、一，送入。」

種子沿著老舊管線反灌進審計子系統。隔離艙內，審訊算法突然抖動，幻象邊緣開始破碎。審計官怒喝：「加壓！提高收斂率！」

第二輪衝擊降下，內宇宙邊界大片塌陷。工匠派主事當場吐血，祭司塔熄了兩座。林燼把意識硬拉回核心，將殘存規則密度壓成一柄細刃，逆著算法插進審訊流。

這不是反擊，只是讓對方每一步都更慢一點。

三十分鐘到，密鑰仍為空白。審計官沉默數秒，轉向後方監審員：「常規洗檔失效，申請核心演算法強殺。」

命令上傳，整座審計中樞的燈同時暗了一瞬。林燼在黑暗裡聽見遠處的機械轟鳴，像某種更大的怪物正醒來。

蘇岚在外線收到同樣提示，指尖發冷：「核心強殺授權……他們要直接碾碎他。」

燕九把槍背上肩：「那就把外面炸到他們算不出收斂路徑。」

林燼躺在束縛椅上，嘴角滲血，卻低低笑了一聲。這一刻他等很久了——只有核心演算法親自下場，他才能沿著那條最高權限連線摸到第三分片。
