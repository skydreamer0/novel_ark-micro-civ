# 第101章 戰場開盤

灰槽帶北側會議艙的燈一盞接一盞熄滅，只留下中央投影台還在亮。傷亡名單被放大到整面牆，紅色數字像未冷的鐵屑，刺得人眼疼。

「你拿回黑盒，不代表你就能拿走指揮權。」賀老八把手杖敲在桌面上，「我們現在缺藥、缺電、缺可用人口，你的『英雄打法』只會把灰槽帶拖進下一輪收割。」

他身後十幾名派系代表同時起立，聲音壓過了通風機：「交出戰區調度權，改由議會接管。」

燕九按住腰側傷口，往前一步，卻被林燼抬手攔下。林燼沒看賀老八，他把黑盒拆出的摘要晶片插進投影台。畫面一跳，出現三組對照欄位：戰損率、配給稅、投放優先級。

「你們說我只會打仗，行。」林燼聲音很平，「那就看帳。」

蘇岚在旁同步開啟民生帳公開流，會議艙外走廊也被接入，傷兵、工匠、值班維修員都能看到。林燼把第一段曲線放大：「過去十二周，灰槽帶繳納的配給稅提升三成，官方回補醫療耗材卻降到一半。這不是管理失誤，是被刻意抽乾。」

賀老八冷笑：「誰不知道？知道又怎樣？」

林燼點開第二段證據，黑盒裡的轉運簽章對上第十五段稅軌排程，形成一條隱藏路由。那條路由繞開公示節點，直通審計深井外環。「怎樣？怎樣就是我們還有籌碼。」

他將一份加密包拋在桌上：「遷徙派要證據，我給。技術行會要可驗證鏈，我給。條件是四十八小時緩衝，灰槽帶由我方戰時調度。」

會議艙安靜了三秒。第三秒，外頭響起警報——配給倉被匿名程序鎖死，民眾開始往艙門口聚集。賀老八派系的人臉色瞬間變了，因為倉鎖不是林燼動的，是高層在逼他們立刻選邊。

「這就是你們要的議會接管？」林燼抬眼，眸底像冷金屬，「他們連你們站在哪一邊都替你們算好了。」

蘇岚趁勢把第三段欄位投上去：收割優先級。灰槽帶被標記為B+可消耗區，附註是「可在局部戰爭中追加調撥」。會議艙裡有人失手摔了杯子。

「從現在開始，」林燼把戰術面板切成三塊，「A隊奪軌、B隊截流、C隊做假簽，把官方運補線拆成我們的補血線。願意跟的，今天拿配給；不願意跟的，明天等下一份收割名單。」

賀老八盯著他很久，終於把手杖往地上一杵，沉聲道：「我給你四十八小時。四十八小時後，灰槽帶若還是這個死樣子，你自己去審計署報到。」

林燼點頭：「成交。」

會議散去時，蘇岚低聲說：「你把自己押上去了。」

「本來就該有人押。」林燼看著投影台上那個B+標記，「不先把戰場開盤，我們連上桌輸的資格都沒有。」

### 本章總結（模式B）
1. **本章推進主線**：
   - 戰力線：林燼從小隊突擊轉為戰區調度，正式取得四十八小時戰時指揮窗口。
   - 真相線：公開「收割優先級」欄位，確認灰槽帶在戰爭需求模型中的可消耗定位。
   - 文明線：民生帳被納入決策場，內外線資料第一次同步給底層群體。
2. **伏筆觸發／推進／回收**：推進「戰爭需求模型」與「投放名冊分片路由」，並觸發審計深井外環路徑伏筆。
3. **不可逆變化**：灰槽帶治理從議會扯皮轉為戰時調度機制，林燼以個人信用綁定全區生存時限。
4. **下章可承接壓力**：四十八小時倒數啟動，第一列稅軌列車即將進站，若截流失敗將直接引發配給崩盤。
