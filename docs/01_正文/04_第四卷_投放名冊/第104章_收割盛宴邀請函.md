# 第104章 收割盛宴邀請函

「峰會名義叫『戰時供應協調會』，實際是收割配額分贓。」

蘇岚把截來的議程投到牆上，三層安檢輪替時間被她用紅圈標滿。「第一層查身份，第二層查簽章，第三層查行為模型。任何一步不像官樣，都會被標記清除。」

唐雀吹了聲口哨：「那我們就演得比他們還像。」

醫療站隔壁的舊倉被臨時改成造假工坊。工匠派在內宇宙輸出一套「供應績效包」模板，把灰槽帶戰損包裝成效率優化報表：死亡率下降、單位配給產出上升、治療符折損可控。每一項都是真數據，但切法完全是高層語言。

燕九看完表，皺眉：「我們在把人命寫成公式。」

林燼沒躲開他的目光：「我們不是認同這套公式，是先拿它進門，把桌子掀了。」

隊內爭論在午夜爆開。幾個新人不肯參與，覺得這就是跟收割派同流。陶青把手上的血漬擦了又擦，最後只說一句：「你不進場，他們照樣分。你進場，至少還能改掉幾條。」

最終方案定下：蘇岚負責簽章鏈，唐雀負責安檢節點噪聲，林燼帶核心包進場，燕九守外線接應。

次日清晨，正式邀請函送到灰槽帶。黑色合金卡片薄得像刀片，背面刻著金字：

> 第十五段臨時代理人：林燼

蘇岚掃描時忽然停住：「還有小字。」

她放大底部微雕，整個房間都靜了。

> 附註：林燼，列入高優化保留名單（戰爭需求模型v7.3）。

保留名單。這意味著林燼不是下一輪直接清除目標，而是被視作「高價值可塑材料」，在必要時優先投放到更高損耗戰區。

「他們要活著用你。」燕九聲音發冷。

林燼把卡片翻回正面，像是在看一份早知會來的判決：「好消息是，他們短期不會殺我。壞消息是，我離屠宰台更近了。」

他把邀請函收進口袋，對所有人說：「我們進去，不是參與盛宴，是拆桌。誰怕，就現在退。我不會怪。」

沒有人退。

外頭警報忽然響起，顯示峰會安檢已提前六小時啟動。蘇岚迅速關閉投影：「他們改時程了，逼我們立刻出發。」

林燼提起裝著績效包的硬殼箱，推開門。門外風很冷，像金屬刀刮過喉嚨。

「走吧，去看他們怎麼把收割叫成治理。」

1. **本章推進主線**：
   - 真相線：確認收割盛宴本質是配額分贓機制，並出現戰爭需求模型版本號實證。
   - 戰力線：完成潛入峰會的偽裝方案與分工，從街區作戰升級到制度場滲透。
   - 文明線：隊伍承擔「用敵方語言對抗敵方系統」的價值衝突與心理代價。
2. **伏筆觸發／推進／回收**：觸發「高優化保留名單」伏筆，直接綁定林燼個體命運。
3. **不可逆變化**：林燼身份由灰槽帶戰鬥者轉為官方標記的高價值樣本。
4. **下章可承接壓力**：峰會安檢提前，若奪不到分片，將失去進入審計核心的唯一合法跳板。
