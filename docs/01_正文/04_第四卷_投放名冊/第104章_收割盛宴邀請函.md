# 第104章 收割盛宴邀請函

舊倉改成的作戰室裡，牆上貼滿安檢流程。第一層身份、第二層簽章、第三層行為模型，任何一步像底層人，都會被判定「異常排除」。

蘇岚把截到的議程放大。標題寫「戰時供應協調會」，附錄卻是各段收割配額。誰多拿、誰少拿、誰承擔輿情，全有價格。

唐雀吹了聲口哨：「收割盛宴，連餐刀都編好號了。」

林燼把硬殼箱打開，裡頭是蘇岚做的「供應績效包」：戰損率、物資周轉、醫療回收率，全是真數據，卻用高層最愛的效率語言切片。這不是偽造，是翻譯。

燕九看完沉聲道：「我們在把死人寫成KPI。」

「對。」林燼說，「不這麼寫，門都進不去。進不去，就只能在門外等他們把下一批人寫成耗材。」

爭論在夜半爆開。幾個新成員說寧可硬打，不願穿著官樣去會場。陶青把手上的消毒液瓶放下，盯著每個人：「我今天剛把兩個孩子從失血邊緣拉回來。你們硬打，能打掉會場幾座砲塔；你們進場，可能改掉的是下一季的清場規則。自己選。」

沒人再說話。

決策成形。蘇岚負責簽章鏈，唐雀負責安檢噪聲與輪替空窗，林燼攜績效包進場，燕九留外線作接應。為防被全滅，蘇岚加了「內世界遠端見證」：會場內每個關鍵決策都同步回主機，由祭司與工匠打標，確保外線能判斷林燼是否被精神操控。

「你連我都要審？」林燼挑眉。

「每個人都審。」蘇岚不退讓，「你說過每個人都有內世界。那就一視同仁。」

清晨，正式邀請函到了。黑色合金卡薄得像一片冷刃，正面是「第十五段臨時代理人 林燼」。蘇岚掃描後皺眉：「底部有微雕附註。」

放大後，房間裡像被潑了一盆冰水：

> 林燼：列入高優化保留名單（戰爭需求模型 v7.3）

保留，不是保護。是「別急著殺，先榨乾」。

燕九握拳，關節咔咔作響：「他們要把你活著搬去更貴的戰區。」

林燼收起卡片，神色反而平靜：「這張卡同時是通行證和項圈。好消息是短期不殺我，壞消息是他們準備把我做成模板。」

唐雀忽然把另一份截獲資料甩上桌：會場第三層安檢新增「內世界一致性掃描」。不是讀取內容，而是比對你說的話和你心像波形是否一致。換言之，裝得再像，心裡一抖也會被抓。

「每個人都有內世界，他們終於承認了。」蘇岚冷笑，「只是用來抓人。」

林燼沉思片刻，下令調整方案：不壓抑情緒，反而把恐懼、憤怒、求生全部寫進合法敘事，讓一致性掃描讀到「真實的代理人」。他們不是要扮演高層，而是要扮演被壓榨到還能談條件的底層。

「我們進去不是參與盛宴，是拆桌。」林燼看向眾人，「怕的人現在退，不欠任何人。」

依舊沒人退。

出發前一刻，林燼短暫接入觀測模式。內世界裡，祭司塔旁新增一座「鏡廳」，每一面鏡子都映著隊友的心像：蘇岚是一間佈滿線路的冷室，中央放著弟弟的舊錄音；燕九是一條滿是彈殼的長廊，盡頭亮著小小燈泡；唐雀是無窗機房，牆上只寫一句「活著就是破解」。

林燼自己那面鏡子裡，只有一扇半開門。

他退出觀測，拉開作戰室門。門外風裡有金屬鏽味，也有遠處會場傳來的樂聲。收割盛宴提前六小時啟動。

「走。」他說。


出發前的最後演練在地下維修井進行。第三層一致性掃描最難，因為它抓的是「你說的話是否真來自你的內世界」。唐雀寫了十幾套對答腳本，最後全被林燼否掉。

「太乾淨了。」林燼說，「高層不會相信底層人能這麼穩。」

於是蘇岚反過來設計「可控瑕疵」：語句裡保留停頓，眼神裡保留疲憊，情緒曲線保留壓抑怒意。這不是演技，而是真實生活本來就有裂口。幾輪測試後，一致性通過率反而提升。

同時，燕九把外線接應點從兩處加到五處，每處都配一名能讀內世界日誌的副隊。規矩很簡單：若主隊員出現明顯失認或過激，不准硬上，立即撤換。燕九以前最討厭這種「心理流程」，這次卻自己先簽名。

蘇岚把邀請函做了三重備份：實體卡、量子影像、內世界鏡廳刻印。她解釋說，若會場內外全部失聯，至少鏡廳刻印還能證明誰被標記、何時被標記。唐雀聽完嘖了一聲：「我們現在連命運都要做容災。」

出發前十分鐘，林燼召集全員做一次短觀測。每人只說一件「不能被收割的東西」。有人說是弟弟，有人說是欠下的命，有人說是還沒學會寫字的女兒。輪到燕九，他沉默很久，說：「我不能被收割的是——隊友不是耗材。」

這句話被祭司派刻進鏡廳頂部，成了臨時戰誓。林燼看著那行字，才真正覺得這群人不是臨時拼湊，而是能一起進火場的隊伍。


補時階段裡，蘇岚要求所有值班員在每次交接前口述「三條釘」：
一、我現在最怕失去什麼；
二、我這一班的首要保護對象是誰；
三、若命令與民生帳衝突，我站哪邊。

這套口述來自早年灰鏈值班模板，平時被嫌麻煩，戰時卻像把人從麻木裡拽回來。唐雀說它像給腦子上扭力扳手，能防止人在連續高壓下滑成純執行器。燕九最初最不耐煩，後來每次都第一個說，因為他發現只要說出口，槍就不會亂指。

林燼不在外線時，這套流程更像某種集體內世界對齊。每個人都在提醒自己：我們不是為了證明誰更硬，而是為了讓更多人活到下一班。這些話不熱血，甚至有點笨，卻在最壞的夜裡穩住了隊形。

同時，主機端的鏡廳新增「交接刻痕」功能。每個小隊完成一班，就在鏡廳石柱上留一條刻痕，旁註當班遺失與保住的東西。刻痕越多，祭司派推演顯示群體失序概率越低。工匠派把這結果轉成外線建議：任何連續兩班未做交接刻痕的小隊，必須強制休整。

這些細碎規則沒有爽感，卻是戰爭能否長跑的底盤。高層模型擅長把人切成參數，而他們開始學會把參數再拼回人。


更關鍵的是，這些交接語句開始外溢到居民層。配給站志工也學著在開窗前互問三句：今天先護誰、先發哪類藥、若突發搶奪怎麼保孩子與老人。最初有人笑這是裝腔，兩天後笑聲少了，因為秩序確實變穩。連最焦躁的隊伍也發現，當你知道旁邊的人在守什麼，恐慌就不容易把隊形撕開。

蘇岚把這現象寫進戰報，命名為「內世界公開最小單位」：不是把所有秘密掀開，而是至少說出你此刻的守護對象。她說這是對抗收割模型的第一步。模型靠隱匿與拆分運作，他們就靠最小透明把人重新連成網。

唐雀隨後做了個小工具，讓每個節點能用三個符號標記狀態：可守、可換、不可棄。符號簡單到孩童都看得懂，卻讓臨場調度效率提升一截。燕九嘴上嫌麻煩，實戰裡卻第一個要求全隊貼符號，因為他再也不想在煙霧裡猜誰還有力、誰已崩潰。

林燼後來回看這些紀錄時，才發現真正讓灰槽帶撐住的，不只是幾次漂亮反殺，而是一次次把「每個人都有內世界」變成可執行流程。那讓他們在金屬與火裡，仍保住了人的形狀。


那晚還有一個不起眼卻關鍵的動作：蘇岚讓各節點把「內世界守護對象」匿名匯總，做成熱點圖。結果顯示，大家最常填的不是自己，而是孩子、醫療站、回家路。這份圖讓調度策略再度微調：先守能承載最多守護對象的節點，再談戰術漂亮。林燼後來評價這是本幕最貴的資料——它不產生火力，卻決定火力應該為誰而用。
