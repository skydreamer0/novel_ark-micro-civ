# 第106章 主動投案

四十八小時窗口到點那一刻，灰槽帶上空同時亮起十二枚追捕信標。審計鏈收束成網，所有外線通道只剩最後一輪撤離時間。

燕九把主力分成三批往暗渠撤，唐雀抱著分片副本衝進離線冷庫。她在門口回頭：「你最好不是又要逞英雄。」

林燼沒回答，只把冷庫雙鎖代碼交給她。「從現在起，你只聽蘇岚。」

蘇岚在指揮台前盯著他，眼裡全是火：「你想走那條路？」

「第三分片在核心審核層，只有被官方完整接管的人能碰到。」林燼把腕端權限逐條轉交，「我主動投案，你留外線，守民生帳。」

她咬著牙不說話。林燼把最後一條準則寫進共用系統：

> 決策優先級第一條：民生帳優先於軍功帳。

這一條一旦上鏈，就算林燼不在，任何資源調度也不能繞開底層供給。燕九看見後只說了句：「你這是把自己從指揮席拆下來。」

「不拆不行。」

追捕隊進入外環時，林燼主動關閉了隱蔽程序，打開定位燈，獨自走向官方審查點。沿途有人罵他瘋了，也有人沉默地給他讓路。

審查點白得刺眼，金屬門一層層合上。祁序站在玻璃後方，像等一個早寫好的結局。

「林燼，你涉嫌會場入侵、資料竊取、煽動艙段叛亂。」祁序語調平穩，「你可以保持沉默，也可以交出密鑰爭取減刑。」

林燼抬起被束縛的手腕，笑了笑：「我來這裡，不是求你減刑。」

他被押入最高等級物理隔離艙。艙內沒有光，只有機械臂把神經接口一根根釘進後頸。系統提示在黑暗裡亮起：

> 意識洗檔程序，12小時倒數啟動。

同一時間，外線指揮權正式轉到蘇岚名下。她看著那條倒數，聲音發啞：「全隊轉防守。守住醫療站、守住冷庫、守住公開鏈。誰都不許死得沒價值。」

燕九回了一聲「收到」，然後把安全栓扣死。所有人都知道，林燼進去後，時間只剩十二小時。

### 本章總結（模式B）
1. **本章推進主線**：

   - 文明線：民生帳被寫入最高決策準則，治理底線制度化。
   - 真相線：第三分片獲取條件被確認為官方完整拘押流程。
2. **伏筆觸發／推進／回收**：推進「主動被捕」伏筆，對接審計中樞核心戰場。
3. **不可逆變化**：林燼交出外線最高權限並進入物理隔離艙，指揮中樞正式轉移。
4. **下章可承接壓力**：12小時洗檔倒數開始，若內外線協同失敗，林燼將被抹除且第三分片永失。
