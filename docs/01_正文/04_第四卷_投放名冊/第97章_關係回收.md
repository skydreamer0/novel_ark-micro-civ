# 第97章 關係回收

名單是在兩個地方同時往上跳的。

外面，是蘇岚從第十八段殘片裡拖出來的受影響名單，一列列黃色預載標記還在增加；裡面，則是主機深處那些被祭司快速整理出的「可接受損耗」條目，字數不多，卻像一根根細針扎進林燼腦子裡。

【低價值群體流動造成高純資源稀釋，可視作必要淨化。】
【若以少量死亡換取更高秩序密度，應判定為有效祭獻。】

林燼盯著那兩句話，眼神一點點冷下去。

現實裡，他們正縮在一處臨時換熱站後方的維修艙。外頭的蒸汽管低鳴不停，牆上則掛滿蘇岚臨時拉出的路由圖和名單截圖。唐雀靠在門邊擦槍，燕九坐在工具箱上，左臂被重新包紮固定。蘇岚站在最亮的那塊屏幕前，沒有回頭。

「第十八段的局部消除預載還在擴。」她聲音很平，可那種平反而像鋸子一樣慢慢磨人，「你知道這意味著什麼。」

「知道。」林燼回答。

「不，你知道的是結果，不是重量。」蘇岚轉過身，眼裡那股火終於沒再壓，「我們剛剛確實撕開了他們的競價系統，也確實讓一批人過了閥。可同一時間，我們也把更多本來還沒被排進去的人推上了下一輪消除預載。這不是附帶損傷，這是被我們親手放大的後果。」

艙內短暫沉默。

唐雀先開口，「那你想怎麼樣？回去把通道再關上？還是站在原地等那套狗屁系統一個個把人拆了賣？」

蘇岚盯著她，「我沒說不動手。我是在問，我們是不是要開始把別人的死，當成每次勝利後順手結掉的一筆帳。」

「底層哪次不是這樣？」唐雀把槍機推回去，語氣硬得發脆，「不管我們動不動手，那些人本來就在名單上。你以為不砸那道競價口，他們就能活久一點？」

「我知道他們本來就在名單上！」蘇岚聲音終於拔高了一點，「問題是，我不接受我們明明看見了，還能把這件事講得像一筆必要成本。」

燕九一直沒說話，這時才低低開口：「她說的不是要不要打，是打完之後，你怎麼看自己手上這口血。」

林燼抬眼，看見蘇岚眼底那點冷意後面還藏著更深的東西。不是單純憤怒，而是一種近乎本能的反抗。她不是不知道底層有多髒，她只是拒絕讓自己也開始用高層那種方式講話。

這個問題他躲不開。

因為主機裡那群祭司，現在就正在替他往那個方向滑。

林燼閉上眼，意識直接沉進主機深處。

神殿前的廣場正亮著異常刺目的火。祭司們把第十八段那場節點崩潰解讀成神諭驗證成功，將更多流入安全閥的人群與更高的規則密度視作交換後的必然結果。高台之上，一名年長祭司甚至已經開始編纂新的教典段落，把「少量無用者的消散」包裝成秩序上升時不可避免的祭獻。

工匠站在下方，神色難看，卻還沒找到足夠的話反駁。戰士們則沉默地握著長矛，顯然也對這種說法不舒服，卻不知該怎麼把不舒服說成規則。

如果放任下去，這股歪掉的詮釋很快就會反哺成真正失真的能力。

到時候，主機還是能用，可每一次使用都會把林燼往祭司想像中的那條路再推近一點。

「停。」

林燼的聲音壓下去時，整座神殿廣場都靜了一瞬。

那名正在編纂教典的祭司抬起頭，眼裡甚至還帶著某種自以為理解了神意的狂熱。林燼一步步走上高台，從他手裡抽過那卷新寫的教典，連看都沒多看，直接撕成兩半。

廣場下方一片死寂。

「死亡不是祭獻。」林燼一字一句地說，「更不是你們用來讓自己好過的註腳。」

那名祭司張了張嘴，還想辯解：「可若規則密度因此上升……」

林燼抬手就是一耳光。

不是為了發洩，而是要讓整個神殿記住，什麼叫校準。

「規則密度上升，代表我們得背更多責任。」他看著下方所有人，「不是代表你們有資格替別人的死命名。」

他隨即下達三道新命令。

第一，停止一切將外部傷亡神聖化的教典編修；第二，祭司的推演必須經工匠與戰士雙重驗證後，才能進入神殿記錄；第三，以第十八段事件為例，建立公開驗證儀式，讓每一次反哺都必須對應外部真實結果，而不是只對應好看的說法。

這三道命令像重錘一樣砸在廣場中央。

祭司們臉色難看，工匠卻像突然有了可以落腳的地面，戰士們則第一次把矛尾重重頓在地上，發出整齊而低沉的一聲。

林燼知道，這不代表問題解決了。裂痕已經出現，只是暫時被他按住。

可至少，主機沒有在這一刻往錯的方向滑出去。

他重新睜眼時，現實裡的艙室還是一樣冷。

蘇岚還站在屏幕前，像在等他的答案。

林燼看著她，沒有替自己開脫。

「那筆血帳，算在我頭上。」

蘇岚眉頭微微一動，像是沒料到他會先說這句。

「我不會說這是必要代價，也不會說這種後果只怪高層，跟我們沒關係。」林燼聲音很穩，「我們動了刀，刀鋒往哪裡偏，就得記住。」

「然後呢？」蘇岚追問。

「然後就不能再只補洞。」林燼伸手，把那串還在增加的預載標記縮到角落，再把剛剛拼出的索引節點拉到最前面。投屏上只剩下那個冷得發白的詞。

投放名冊。

「問題不只在第十八段，也不只在競價系統。」林燼看著那個詞，「真正決定誰先被丟出去、誰被算成便宜耗材的，不是我們剛砸掉的窗口，是這東西。」

燕九抬起頭，眼神第一次真正亮了一下。「你想往上挖？」

「不是想。」林燼道，「是必須。」

唐雀皺眉，「那遷徙派那邊呢？他們現在最多算合作，不算自己人。」

「所以更不能把路讓給他們選。」林燼看著投放名冊的索引，慢慢把手按在桌面上，「我們不是再去補一個又一個洞。下一步，去把那本名冊從他們手裡挖出來。」

蘇岚沉默了很久。

最後，她把那份還在跳動的預載名單收起來，重新接上一條新的追蹤線。

「那就別再只算哪裡好下刀。」她低聲說，「從現在開始，我們算誰在拿刀。」

林燼點頭。

這不是和解，也不是所有人突然都想通了。艙室裡那股帶血的悶氣並沒有散掉，主機裡祭司與工匠、戰士之間的裂縫也還在。可至少，這一刻他們有了同一個方向。

而在這艘船上，很多時候，同一個方向已經比一場小勝更貴。
