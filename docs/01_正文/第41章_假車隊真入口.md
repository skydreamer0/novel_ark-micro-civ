# 第二卷《裂縫稅戰》
## 第41章：假車隊真入口

黎明前兩小時，第九段的工業廢氣在低溫下凝結成一片骯髒的灰色濃霧。

就在這平靜得讓人窒息的時刻，西工坊外環的廢棄高架橋上，突然出現了一支誇張得近乎愚蠢的補給車隊。

十二輛重型運載卡車組成了一條發光的鋼鐵長蛇。車身不僅沒有進行任何低視度偽裝，反而用螢光塗料刷滿了巨大的「灰鏈火種」記號。車頭掛著明晃晃的、一看就是偽造的最高級別稅契核准牌。卡車的引擎發出撕裂空氣的暴躁轟鳴，車頂的探照燈像是挑釁般，直直掃向防區警戒線上的光學傳感器。

這種車隊，在平時的第九段生存法則裡，絕對活不過三分鐘。但今天，它是林燼親自點頭放出去的誘餌。

「太招搖了。」臨時指揮所裡，燕禾盯著螢幕上那片刺目的高熱源信號，眼角因為緊張而微微抽搐。

「要的就是招搖。」林燼的聲音沒有任何起伏，一邊將一枚穿甲彈壓入彈匣。

顧衡的注意力必須被完整、徹底地拉向這場「大規模劫運」的幻覺裡。林燼很清楚高層代理人的思維定勢：當一塊價值連城的肥肉大搖大擺地從眼前晃過，他們的第一反應絕不是有詐，而是絕對的武力碾壓。

為了讓這塊肉聞起來足夠真，假車隊裡確實混編了兩輛裝滿殘核與高價抗素的真物資車。至於其他十輛，全塞滿了空木箱與高溫電熱發射器，用來製造龐大且逼真的熱源與路徑噪音，以此迫使官方的巡防系統做出錯誤的兵力分配。

蘇嵐帶領的志願突擊隊就坐在第五輛車的車斗裡。她負責把這場戲演到最逼真的程度。

「老闆，」她在隊頻裡低聲笑著，聲音伴隨著卡車底盤碾過碎石的劇烈顛簸，「要不要我順便在車頂跳個鋼管舞，讓顧衡的無人機看清楚點是誰來砸場子？」

「活著回來再跳。」林燼切斷了外連通訊。

同一時間，真正的潛入小隊只有最核心的三個人：林燼、燕九、唐雀。

他們的目標根本不是這幾天的熱門搶奪對象——裂縫樣本或補給箱，而是隱藏在 R-17 核心維護艙地下深處的一樣東西：**「局部控制模組」**。

那是一排能決定入口短時間開關優先權的硬體密鑰。那是顧衡實施軍管化的最底層邏輯門。

假車隊啟動後僅僅兩分半鐘。

顧衡果然上勾了。他甚至沒有經過第二道人工確認，直接下達了震耳欲聾的全域廣播，封鎖了西側六條主幹道。成群結隊的武裝巡防無人機如同被驚動的殺人蜂群，帶著幽綠色的索敵光柱撲向車隊。

連原本架設在 R-17 入口上方、防守最為嚴密的兩座高架火炮位，也因為判定威脅等級過高，底座發出沉重的機械碾壓聲，炮口轉向了西側準備遠程火力覆蓋。

「魚咬鉤了。」燕九輕聲說道，手指在微型終端上飛速跳躍。

三人借著無人機群遠去的轟鳴掩護，像三道幽靈般從地下廢水渠鑽出，沿著前幾天破壞的延遲閥管線，悄無聲息地滑入了 R-17 的腹地。

昨夜拚死改寫的「三十秒延遲」在這裡發揮了救命的作用。當顧衡的系統因為局部衝突而自動下達兩次預防性封鎖指令時，沉重的鋼製防爆門都因為那微小的齒輪錯位，而在半空中詭異地停頓了三十秒。

就是這三十秒的夾縫，讓三人得以在閘門徹底咬死前，一個翻滾滑入下一層空間。

維護艙門口，空氣驟然降溫。

這裡由四名穿著全覆蔽式灰甲的精銳守衛把守，走廊盡頭還懸掛著一台散發著紅光的反入侵自動化砲塔。

唐雀主動請纓打出第一槍。她之前受的骨裂傷還沒全好，額頭上沁著冷汗，但握住那把重型自製狙擊步槍的手，卻穩得像是一根釘入水泥的鋼釘。

「噗——」

第一發子彈直接精准地熄滅了砲塔的光學感測眼，爆出一團刺目的電路火花。

就在守衛轉頭的瞬間，林燼已經從黑暗的通風管道正上方落下。他的膝蓋狠狠砸在第一名守衛的頸椎上，伴隨著令人牙酸的骨裂聲，守衛連警報都沒來得及按就軟倒在地。第二名守衛剛舉起槍，林燼的軍刀已經從下顎防護服的接縫處捅入，切斷了聲帶與發聲器。

燕九負責以最快速度駭入門鎖的生物識別板。

剩下的兩名遠處灰甲剛從掩體後探出身準備火力壓制，就被兩發從極遠處飛來的狙擊彈精準地逼了回去。

那是蘇嵐的支援。她在假車隊誘敵的側翼空檔，冒著被無人機鎖定的風險，徒手爬上了兩公里外的一座廢棄信號塔，硬是給潛入小隊撐開了一把保護傘。

「你不是在演戲嗎？」燕九一邊焦急地破解不斷變換的密碼，一邊在隊頻裡吼道。

蘇嵐在通訊端大口喘著氣，背景音全是劇烈的爆炸聲：「演員……也可以加戲啊！」

「滴——權限通過。」

厚重的合金艙門緩緩開啟，裡面比他們想像的還要乾淨、冰冷。沒有守衛，只有房間中央矗立著一根不斷閃爍著藍色螢光的銀灰色控制柱。底座上，三個宛如心臟般微微跳動的模組正發出低微的嗡鳴聲。

燕九撲上去接通掃描儀，螢幕上的數據如瀑布般刷下，他的臉色卻瞬間變得慘白：
「糟了，林燼！量子加密底層有鎖……我們只能帶走一枚。拔錯任何一個，控制柱就會觸發內部熔毀，永久鎖定！」

三枚模組上方的懸浮屏幕分別標記著：
`[模組 A：稅契與核驗數據流]`
`[模組 B：事故回寫與警報總控]`
`[模組 C：入口局部優先權]`

稅契與事故，是顧衡能夠精準打擊灰鏈、查封物資的「刀」。而優先權，是這間屋子、甚至整個區域的「門」。

如果不解決刀，灰鏈的人依舊會不斷流血。

林燼沒有猶豫，連千分之一秒的遲疑都沒有：「拿優先權。C 模組。」

沒有門，任何資源的反制與洗白都只是在敵人的籠子裡拖延時間。只有拿到開門的權限，才能從根本上打破這場被動防守的爛局。

當燕九顫抖著手，將第三枚模組從控制槽中硬生生拔出的那一刻——

「嗚——！！」

全區的警報從原本的黃色閃爍瞬間升級成刺耳到令人耳膜生疼的猩紅長鳴。顧衡的主機顯然收到了核心硬件被強行剝離的最高等級告警，立刻無條件命令 R-17 全域強制封鎖。

「走！」林燼厲聲喝道。

幸運的是，地下回壓閥的物理延遲仍在生效。第一輪毀滅性的鋼牆封鎖指令在傳導途中被強行慢了三十秒。這三十秒足夠三人狂奔衝過第二道井口。

但幸運不會永遠眷顧。

當他們衝到第三道，也就是最後一道路障門前時，一支趕來增援的快速反應小隊已經堵死了通道。

密集的交叉火力如同一張發光的網兜頭罩下。

「躲開！」

唐雀猛地推開了抱著模組的燕九。子彈從側後方以一種詭異的角度擊穿了她防護服的薄弱處，狠狠咬進了她的右側腹部。

血花在空中綻放的瞬間，時間彷彿被無限拉長。

唐雀重重地跪倒在地，卻沒有發出慘叫，而是在倒下的那一刻，將手中的掩護閃光彈用盡全力朝前方擲出。

強光爆發的同時，她把插滿管線的微型駭客平台模組推向林燼的腳邊，嘶啞地吼道：「別停！走啊！」

林燼的腳步猛地頓住。

大腦深處的主機在這一刻以極高的頻率運算著「救援存活率：2.4%」。理智告訴他，回頭就等於全軍覆沒，連帶模組與之前的所有犧牲全部白費。但人類的記憶殘響，讓他握槍的手背青筋暴起，本能地想要轉身壓制火力。

唐雀死死扯住他防彈衣的下擺，滿是鮮血的手指幾乎摳進他的肉裡。那雙原本因為長期在暗處操作而顯得灰暗的眼睛，此刻亮得嚇人。

「你現在回頭……」她嗆出一口血，「我這槍就白挨了！滾！」

遙遠的狙擊點上，蘇嵐的怒吼透過無線電劈頭蓋臉地砸下：「你們走！我壓三十秒！我他媽壓死他們！」

這是一個在鮮血與算計中淬煉出的決定。

林燼狠狠咬碎了口腔裡的一顆合成鎮痛劑，沒有再說一句廢話。他一把將失去行動能力的唐雀扛上肩膀，單手持槍盲射壓制，燕九死死抱著裝有模組的防護盒，兩人跌跌撞撞地翻進了最近的工業排熱井。

而後方，蘇嵐在高塔上把最後兩個彈匣如同瘋魔般傾瀉而出，生硬地在追兵的路線上撕開了一道火線。當她最終不得不撤入備用逃生線時，身側引爆的震撼彈幾乎震聾了她的左耳。

假車隊那邊付出了極其慘痛的代價：十二輛車只有兩輛逃脫，三名志工重傷被俘，但主要的核心人員總算在極限狀況下撤離成功。

更重要的是，顧衡直到這場瘋狂行動結束整整四分鐘後，才從混亂的報告中意識到——真正的目標不是外圍的假貨，而是維護艙裡的入口模組！

但四分鐘，太晚了。

回到隱蔽的地下藏點，空氣裡瀰漫著濃重的血腥味與汗味。

燕九將滿是刮痕的 C 模組接入隔離端口。所有人都屏住呼吸，盯著那塊滿佈裂痕的終端螢幕。

字元在一陣閃爍後，穩定了下來，吐出了一行冰冷的綠色字句：

> 【R-17 局部優先權令牌：破解可用。】
> 【有效時長：17 分鐘。】

十七分鐘。

不長。甚至在和平年代，這只夠煮一杯合成咖啡再抽根菸。

但在第九段，這十七分鐘，卻足夠林燼做一件徹底改變戰局的事。在顧衡無法及時遠程拉閘的情況下，這十七分鐘，能把一支全副武裝的灰鏈採集隊送進裂縫深處，帶出足以支撐整個網格度過嚴冬的高價值樣本。

章末。

林燼站在簡陋的手術台前。劣質的醫療光打在唐雀正在被縫合的腹部傷口上。她因為麻藥不足而疼得渾身發抖，但依舊死死咬著毛巾沒有出聲。

林燼低頭看著終端上那個冰冷的「17:00」倒數預備計時器。

他知道，接下來不能有任何失誤。

「下一章，」他轉過身，面對滿身傷痕卻目光灼灼的隊員們，聲音低沉如鐵，「打開這扇門。」
