# 第二卷《裂縫稅戰》
## 第41章：假車隊真入口

黎明前兩小時，西工坊外環突然出現一支誇張得近乎愚蠢的補給車隊。

十二輛車，車身刷滿灰鏈火種記號，還故意掛著明晃晃的假稅契牌。這種車隊在平時只能活三分鐘，今天卻是林燼親自點頭放出去的誘餌。

「太招搖了。」燕禾盯著畫面。

「要的就是招搖。」林燼回。

顧衡的注意力必須被完整拉向「大規模劫運」，真入口行動才有縫。假車隊裡只有兩車真物資，其餘全是空箱與電熱假目標，目的是製造熱源與路徑噪音，迫使巡防做錯資源分配。

蘇岚帶隊坐在第五車，負責把戲演到底。她在隊頻裡笑：「要不要我順便在車頂跳個舞，讓顧衡看清楚點？」

林燼：「活著回來再跳。」

同時，真正的入口小隊只有三人：林燼、燕九、唐雀。目標不是搶樣本，而是進 R-17 核心維護艙取走「局部控制模組」。這模組能決定入口短時開關優先權，是顧衡軍管化的鑰匙之一。

假車隊啟動後兩分鐘，顧衡果然下令封鎖西側六條街。巡防無人機成群壓去，連兩座高架炮位都被調離 R-17。

「魚咬鉤了。」燕九說。

三人從地下延遲閥線進入入口腹地。第40章改過的三十秒延遲在這裡救了命：兩次封鎖指令都被拖慢，讓他們能卡在閥門閉合前穿過。

維護艙門口有四名灰甲和一台反入侵砲塔。唐雀主動請纓打第一槍，她傷還沒全好，握槍手卻穩得像釘子。第一發直接熄掉砲塔光學，林燼趁黑貼近放倒兩人。燕九負責駭入門鎖，剩下兩名灰甲被蘇岚遠程狙擊支援壓住——她在假車隊側翼空檔偷到一個高點，剛好能看見入口門廊。

「你不是在演戲嗎？」燕九邊操作邊喊。

蘇岚喘著笑：「演員也可以加戲。」

艙門開，裡面比想像乾淨，中央是一根銀灰色控制柱，底座插著三枚模組。燕九掃描後臉色變：「糟糕，只能取一枚。拔錯會觸發永久鎖定。」

三枚分別標記：`稅契核驗`、`事故回寫`、`入口優先權`。

林燼只看一眼就說：「拿優先權。」

稅契與事故是顧衡的刀，優先權是門。沒有門，任何反制都只是拖時間。

燕九拔出第三枚模組那刻，全區警報升級成刺耳長鳴。顧衡顯然收到核心告警，立刻命令 R-17 全域封鎖。幸好延遲閥仍在，第一輪封鎖慢了三十秒，夠三人衝到第二道井口。

但第三道門前，唐雀中彈。

子彈從側後打進她腹部，她跪下時還把模組先推給林燼：「別停。」

林燼幾乎想回身硬拼，被唐雀一把扯住袖子：「你現在回頭，我白挨。」

蘇岚在高點怒吼：「你們走！我壓三十秒！」

最終，林燼背起唐雀，燕九抱模組，兩人翻進排熱井。蘇岚在後方把最後兩個彈匣打空，等她撤進備線時，左耳已被震到聽不清。

假車隊那邊付出同樣代價：兩車被扣、三名志工輕重傷，但主要人員撤離成功。更重要的是，顧衡直到行動結束四分鐘後才意識到真目標不是物資，而是入口模組。

太晚了。

回到藏點，燕九把模組接上隔離端口，系統吐出一行字：

> 【R-17 局部優先權令牌：可用。有效時長 17 分鐘。】

十七分鐘，不長，卻足夠做一件改變戰局的事——在顧衡無法及時關門的情況下，把一支屬於灰鏈的採集隊送進去，帶出足以支撐整月的高值樣本。

章末，林燼看著躺在醫療床上的唐雀，低聲說：「下一章，打開這扇門。」
