# 賽博宇宙小說架構設計

## 文件目的

這份文件整理目前已確認的世界觀與故事架構，作為後續共創的主參考。原則是只寫已定下來的內容，不強行補完還沒決定的細節。

## 一句話定位

在一個持續衰亡、居住空間不斷縮小的宇宙裡，人類被迫依靠掠奪外域資源維持生存。主角作為穿越而來的底層維修工與拾荒者，得到能養殖文明、吸收信仰、加速煉化的文明主機，從此走上一條和頂層牧場主同源卻不完全相同的崛起之路。

## 世界主舞台

故事主場景是一座依附在末日級流亡母艦上的巨型賽博都市。城內是公司、幫派、黑市、義體醫療與配給制度，城外則是外域裂縫、拾荒帶、投放帶與高風險遠征區。這座城市表面上仍有賽博繁華，實際上是一個被高壓配給、稅制和外域採集綁死的生存工廠。

## 世界底層規則

1. 母艦所在的宇宙正在衰亡，可居住空間會被熵蝕持續侵吞。
2. 如果沒有新的外域資源注入，艙段會封閉，氧氣、能源、醫療與居住配額都會下降。
3. 外域資源不是普通礦物，而是帶有規則、生命樣本、能量殘片與文明殘渣的高階物質。
4. 所有人都必須參與吸收、煉化與轉化外域資源的鏈條，不參與就失去生存資格。
5. 外部危機是真的，但高層也在刻意放大危機，讓整座城市變成吞噬機器。

## 高層壓迫系統

高層掌握人口來源、轉化口與稅制，因此能讓所有人永遠處於爬不上去的狀態。

- 複製人補充機制：高層能大量投放低成本複製人、契約勞工與債務人口，讓底層始終過飽和。
- 內宇宙投放機制：真正的高層會在自己的內宇宙裡養人口、戰兵、信徒與文明碎片，成熟後再投放到母艦外圍與都市底層。
- 公轉化區壟斷：所有高價值收穫理論上都要經過官方轉化口，由高層抽成、分級、監控。
- 多重稅制：生存稅、空間稅、煉化稅、戰備稅與債務稅共同組成長期榨取系統。
- 上升封鎖：任何成長過快的人都會被審計、徵召、抽編、升級配額，甚至被直接收編成資產。

## 主角定位

主角是穿越者，穿越後的身分是底層維修工與拾荒者。他的起點很低，但也因此熟悉都市下層規則、殘骸處理、低階裝備維修與危險區求生方式。主角的強項不是一開始就無敵，而是比一般底層更能活、更能看懂系統漏洞。

## 文明主機與主角優勢

文明主機不是普通外掛，而是高層養殖文明技術的縮小版與提前版。它兼具內宇宙雛形、規則運算平台與文明煉化核心三種功能。

主角的最大優勢是「雙層煉化權限」。表面上他和其他人一樣把收穫送進官方體系，實際上文明主機能暗中截留最核心、最有價值、最難被偵測的部分，送入自己的內部宇宙先行消化。這讓別人是在替上層打工，主角則是在偷上層的燃料養自己的文明。

後續可逐步解鎖的能力包括：時間壓縮權限、文明鏡像權限、空間擴張權限與模組反哺權限。

## 雙線成長結構

這本書的核心是雙線並進。

第一條線是外域掠奪。主角外出進入裂縫、拾荒帶、投放帶與遺跡區，透過戰鬥、偷取、回收與反殺獲取高風險高收益的外域資源。這一條線負責戰鬥爽感、危機感與快節奏升級。

第二條線是內部文明養成。主角把資源丟進文明主機內部空間，交給其中的文明去理解、崇拜、研究、適配與轉譯。這一條線負責經營感、神國感與長線擴張感。

當內部文明完成煉化後，文明主機就會反向把力量模組、技術藍圖、義體方案、戰爭單位或規則權限回饋給主角，形成完整的成長閉環。

## 內部文明機制

內部文明不是裝飾，而是主角真正能滾雪球的核心。

信仰在這個系統裡不是單純供奉，而是集體參與煉化。居民越理解主角的神性、越投入儀式、戰爭、研究與敘事，整個文明就越像文明主機的分散式處理器，能更穩定地轉化外域資源。

文明成長採五時代制：

1. 種火時代：建立最初信仰與認知錨點。
2. 神殿時代：形成祭司、工匠、戰士階層，開始有組織地參與解析。
3. 鋼鐵時代：吸收賽博技術與外域碎片，形成神權科技混合形態。
4. 聖城時代：形成大型信仰網路、城市與軍團，集體煉化能力大幅提升。
5. 星環時代：可向外殖民、分封、派生附屬文明，真正成為完整內宇宙體系。

## 高層與主角的鏡像關係

這是整本書最重要的長線張力之一。高層養文明，是為了收割、維持統治和延續自己的宇宙；主角養文明，一開始是為了活下去、變強與反擊。但他越來越強，就越可能走向和高層相同的位置。這讓主角的道路不只是升級，而是持續面對「自己會不會變成下一個牧場主」的問題。

## 故事總架構

故事主骨架是「掠食文明線」，負責主要爽感：掠奪、偷資源、快速升級、反殺對手、養大文明、踩碎高層安排。暗線是「黑箱革命線」，負責逐步揭露母艦、高層、裂縫與宇宙牧場的真正結構。前期以升級和掠奪為主，中後期逐步打開更大的真相與格局。

## 連載續航機制

為了保證故事不會寫幾十章就失速，後續每一卷都應該盡量維持同一套推進節奏：

1. 城市生存壓力上升。
2. 新的外域機會或投放事件出現。
3. 主角被迫或主動進場掠奪。
4. 戰利品投入內部文明完成煉化。
5. 主角用新能力反打當前敵人或制度壓制。
6. 再揭露更高一層的統治真相或新的危機。

只要每次大事件都能同步推進戰力線、文明線與真相線，這本書就能維持穩定的爽感和連續性。

## 目前已確定的重要設定

- 主角是穿越者。
- 主角開局身分是底層維修工與拾荒者。
- 主舞台是依附末日母艦的巨型賽博都市。
- 世界存在真正的生存危機，所有人都被逼著吸收外域資源維生。
- 高層會大量投放複製人與內宇宙人口，製造過飽和競爭。
- 高層靠公轉化區與多重稅制鎖住所有人的上升空間。
- 主角最大的不同點是擁有雙層煉化權限。
- 主角的內部文明以信仰參與煉化，並走五時代成長。
- 故事主線是掠食文明，暗線是黑箱革命與宇宙牧場真相。
