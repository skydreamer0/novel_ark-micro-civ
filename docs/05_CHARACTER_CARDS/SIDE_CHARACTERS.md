# 角色卡：配角群

---

## 蘇岚（技術位盟友）

### 基礎資料
- **身份**：駭客 / 後勤 / 黑市中介
- **首次出現**：第1章
- **外在形象**：瘦削、顴骨高、眼角舊傷。左手無名指缺一截，用銀色義指替代。叼螢藍色發光煙絲。

### 不可替代資產
掌握「轉化區稅流稽核 API」灰色後門，可即時改寫信用評分與物流權限。

### 致命弱點
- 對記憶完整性有病態執著。
- 弟弟在企業教育艙作為「優先培養樣本」，可被綁架。

### 台詞風格
沙啞、直擊要害、帶黑色幽默。
- 「欠多少？」
- 「審計局最近真餓。」
- 「如果你真進了回收序列，記得把工具箱留給我。」

### 與主角價值衝突
主角相信「必要時可操弄真相」；蘇岚堅持資料不可偽造到傷及無辜。

### 黑化條件
主角利用她的後門製造大規模民生事故，或犧牲她弟弟換戰略優勢。

## 燕九（戰鬥位盟友）

### 基礎資料
- **身份**：底層傭兵 / 肉盾
- **首次出現**：第3章

### 不可替代資產
熟悉艙段狹窄戰場與企業傭兵戰術，持有「報廢外骨骼軍規維修碼」。

### 致命弱點
- 神經抑制劑成癮，72 小時不補藥會戰鬥顫抖與暴怒失控。
- 早年「清場任務」錄像在審計司資料庫中。

### 台詞風格
直接、粗獷、急性子。

### 與主角價值衝突
主角傾向長期佈局與保密；燕九習慣立刻報復、以暴制暴。

### 黑化條件
主角為大局犧牲底層平民，或放棄救回燕九舊隊員。
