# 第三卷大綱：聖城試煉

## §1 卷定位

- **卷名**：第三卷・聖城試煉
- **章數**：30 章（61～90 章）
- **情緒曲線**：
  - 前段（61～70）：被二級收割預案收網後的流亡壓力 + 跨艙段求生 + 熵潮概念初次顯性化
  - 中段（71～80）：灰槽帶政治棋局 + 鋼鐵時代啟動 + 多節點戰術成形
  - 後段（81～90）：沉默艙探索 + 高層內宇宙實物證據獲取 + 聖城前置條件達成
- **卷終點**：主角取得高層內宇宙養殖鏈的實物證據（非傳聞），正式進入高層觀察名單核心層級；內部文明完成鋼鐵時代跨階，「聖城試煉」啟動條件達成。

## §2 三幕式推進（30 章節奏）

### 第七幕：流亡與熵潮（61～70 章）

#### 劇情任務
- 二級收割預案全面生效：主角所在艙段被局部封鎖，灰鏈資產遭凍結，隊伍被迫撤離第七段。
- 主角帶隊伍逃往第十五段「灰槽帶」——降級運作中的灰色地帶，暫借灰議會的保護。
- 在灰槽帶首次直面「設備效能持續衰退」與「無外力裂痕」等現象，蘇岚從技術層面推導出「不是高層在搞鬼，是宇宙本身在壞」，引出**熵潮**概念。

#### 爽點與鉤子
- 逃離第七段途中，主角利用規則碎片反向癱瘓追擊清理者小隊的通訊鏈。
- 灰槽帶的「灰議會」與主角做交易：提供庇護，但主角須替灰議會進入封鎖通道偷取技術行會舊數據核心。
- 內部文明反哺產出「批量規則燃料」，主角首次用內宇宙產出補貼外部維生——一石激起千層浪，灰議會開始覬覦。

#### 章節節點
61. 二級收割預案首日：全城封鎖令與資產凍結
62. 主角決定棄城，與灰鏈殘部進行最後一次轉運
63. 逃離路線遭清理者小隊截斷，狹道伏擊戰
64. 通過第十三段鐵鏽帶時，發現冷凝層設備出現「無外力裂痕」
65. 進入灰槽帶，灰議會開出庇護條件
66. 蘇岚整合設備衰退數據，首次用數學模型描述「熵潮」
67. 主角執行灰議會任務：潛入封鎖通道取回舊數據核心
68. 數據核心中發現：艙段淘汰排程——灰槽帶只剩 180 天
69. 內部文明突破：工匠階層完成首批規則燃料量產
70. 規則燃料外流消息洩露，灰議會內部出現爭奪主角的聲音

### 第八幕：鋼鐵覺醒與節點擴張（71～80 章）

#### 劇情任務
- 主角在灰槽帶站穩腳跟後，開始利用灰議會的舊實驗設備推動文明升級——鋼鐵時代啟動。
- 同時，主角將灰鏈殘部重組為多節點協同單位：分佈在灰槽帶三個不同區域的獨立小隊，以文明軍規 V2 統一指揮。
- 維穩派與極端收割派的高層代理人同時出現在灰槽帶——前者想收編灰議會，後者想借灰槽帶封閉前最大化「末批產出」。主角夾在兩派之間博弈。

#### 爽點與鉤子
- 鋼鐵時代啟動：工匠階層與戰士階層合流，初階生物投放（72h存活）首次用於實戰——召喚出內部文明的戰獸作為前鋒。
- 多節點協同首戰：三支分隊在灰槽帶三條通道同步行動，打出時間差戰術，以少勝多擊退極端收割派的清除隊。
- 主角在灰議會與維穩派的談判中掌握關鍵情報：維穩派手中有一份「高層內宇宙養殖清單」的索引文件——但條件是主角必須帶隊進入第二十一段「沉默艙」取回被封存的航行黑盒子。

#### 章節節點
71. 灰議會要求主角交出規則燃料量產技術——主角拒絕，提出「產出分潤」方案
72. 鋼鐵時代啟動儀式：工匠與戰士融合，首批初階生物成形
73. 極端收割派代理人抵達灰槽帶，宣布「末批產出」倒計時
74. 主角部署三支分隊，建立多節點協同框架（文明軍規 V2）
75. 維穩派使者秘密接觸主角，開出「航行黑盒子 ↔ 養殖清單索引」交易
76. 首次多節點協同實戰：三通道同步行動，擊退清除隊先鋒
77. 灰槽帶民眾恐慌：封閉倒計時被公開，難民潮向連接通道湧去
78. 主角利用規則燃料穩定灰槽帶關鍵維生系統，換取民心與灰議會正式合作
79. 維穩派與極端收割派在灰槽帶爆發代理人衝突，主角漁翁得利奪取通行密鑰
80. 主角決定帶精銳突入沉默艙——追真相也追證據

### 第九幕：沉默艙深潛與聖城試煉（81～90 章）

#### 劇情任務
- 主角帶隊進入第二十一段「沉默艙」——三年前封閉的死艙。內部的「熵行者」極度危險，環境持續熵化。
- 目標雙重：① 取回航行紀錄黑盒子（維穩派交易條件）；② 尋找高層內宇宙養殖的實物證據（全書藍圖硬指標）。
- 沉默艙深處發現被遺棄的「高層實驗遺址」——一個失敗的工業化內宇宙原型，內有養殖鏈的實體設備與樣本記錄。
- 卷末，主角拿到完整證據後返回灰槽帶，交付黑盒子獲取養殖清單索引——兩份證據合流，形成首次可實證的「高層內宇宙養殖體系」完整圖景。
- 文明主機在沉默艙的極端環境中完成「聖城前置條件」——吞噬了實驗遺址殘留的高純度規則密度與「文明主機同源信號」碎片。

#### 爽點與鉤子
- 沉默艙中的「熵行者」戰鬥：極端環境下鋼鐵時代的初階生物投放與多節點戰術被推向極限。
- 高層實驗遺址揭示：高層的「內宇宙」是工業化養殖——恐懼驅動、低純度、高損耗——與主角的「共生式」形成鮮明對照，讓讀者直觀看到兩條路線的差距。
- 卷末黑盒子的部分解譯暗示：母艦的航行方向從一開始就不是「探索」，而是**逃離**——對應真相線「母艦航行真正目的」伏筆的加深（只加深不揭穿，留給第四卷）。
- 文明主機檢測到「同源信號」，確認：曾有另一個同類主機存在於這艘母艦上——被高層拆解用於建設養殖鏈。這是「文明主機同源信號」伏筆的首次實質性回收。
- 高層觀察名單正式升級：當主角帶著實物證據返回時，觀察員將其從「二級收割目標」升格為「核心監控對象」。

#### 章節節點
81. 進入沉默艙：密封門切割 + 第一次接觸熵化環境（設備異常、通訊漂移）
82. 熵行者首次遭遇戰：非人類實體的戰鬥模式讓常規武器失效
83. 內部文明緊急反哺：初階生物投放在低規則環境中快速衰變，工匠班子即時調校
84. 深入沉默艙核心區：發現高層實驗遺址的外層防線
85. 遺址突破戰：主角親自帶隊清掃遺址內的異變安保系統
86. 實驗遺址核心揭露：被遺棄的工業化內宇宙原型 + 養殖記錄實物
87. 文明主機偵測到「同源信號」——另一台主機曾在此處被拆解
88. 取回航行黑盒子 + 養殖實物樣本，開始撤退
89. 撤退途中遭熵行者族群圍攻，多節點協同進入極限操作
90. 返回灰槽帶・卷末對峙：主角手持實物證據，被觀察員升格為核心監控對象

## §3 角色弧

### 主角：林燼
- **卷內目標**：從單艙段流亡者升級為跨艙段行動者，取得高層養殖體系的實物證據。
- **核心弧線**：從「掠食者」向「執棋者」過渡——開始意識到，打倒一個稅制不夠，必須理解整個系統為什麼存在。
- **對文明主機態度轉變**：從「高效工具」到「有生命的文明體」——在沉默艙看到高層是如何對待另一台主機後，首次產生保護欲。

### 盟友

1. **燕九（戰鬥位）**：本卷承擔多節點分隊指揮壓力，從「單兵猛將」成長為「小隊長」。弧線從「聽命衝鋒」轉向「理解全局」。沉默艙之戰中神經抑制劑斷藥，靠意志力硬撐完最後兩場戰鬥。
2. **蘇岚（技術位）**：本卷核心功能角色——推導熵潮模型、解析黑盒子數據、破解實驗遺址安保。內心線：焚數派圈內朋友提供的「意識快照」技術讓她動搖——是否應該為弟弟做一份備份？
3. **灰議會代表（新角色・賀老八）**：灰槽帶的實際掌權者，技術行會殘部出身。表面合作，實際想竊取文明主機的規則燃料量產技術。本卷前半是亦敵亦友，後半暴露野心被主角壓制。

### 反派梯度

1. **極端收割派代理人（本卷主要對手）**：執行「末批產出」指令，在灰槽帶進行最後的人口與資源榨取。冷酷、高效、視底層為耗材。
2. **維穩派使者（灰色棋手）**：不是直接敵人，但利用主角達成自己的目的（取回黑盒子的航行數據）。對主角的價值在於：他握有養殖清單索引。
3. **高層觀察員（延續卷二・升級威脅）**：不親自下場，但在卷末將主角升格為核心監控對象，為第四卷鋪路。

## §4 能力與裝備路線

### 主角能力迭代
- **V7（61～70章）**：規則碎片逆向應用（癱瘓通訊）+ 規則燃料量產（內宇宙首次大規模外銷）+ 熵化環境感知
- **V8（71～80章）**：鋼鐵時代初階生物投放（72h戰獸）+ 多節點協同指揮（文明軍規 V2）+ 分隊級反哺同步
- **V9（81～90章）**：低規則環境適應作戰 + 生物投放即時調校 + 文明級法則協同（祭司推演 + 戰士投影 + 工匠量產同時調度）

### 裝備迭代
1. 反追蹤電磁屏蔽組件（前段・逃離用）
2. 灰議會提供的舊實驗級規則分析儀（中段・推動鋼鐵時代升級）
3. 沉默艙遺址回收的高層級感知增幅模組（卷末・顯著提升算力上限）

### 內部文明里程碑
- 規則燃料量產（前段・種火→鋼鐵過渡產出）
- **鋼鐵時代正式啟動**（中段・工匠與戰士融合、初階生物投放）
- 文明軍規 V2 落地（中段・多節點協同指揮框架）

### 內部代價
- **異端分裂（成形）**：鋼鐵時代啟動後，戰士階層與工匠階層因資源分配爆發衝突。主角介入：建立「聯合磨坊」制度，雙方共同監督產出分配。
- **祭司壟斷（升壓）**：祭司開始宣稱只有祭司才能解讀「同源信號」的意義，試圖壟斷情報解讀權。主角反制：公開信號原始數據，允許工匠與戰士代表獨立驗證。

## §5 揭密與伏筆管理

### 本卷必揭 3 件事
1. **高層擁有「私有內宇宙」且以工業化養殖模式運作**——取得實物證據（實驗遺址設備 + 養殖記錄），非傳聞。
2. **熵潮的存在與基本機制**——設備衰退不是高層搞鬼，而是宇宙級退化現象。第三卷正式引入「熵潮」「規則密度」術語。
3. **裂縫是熵潮的破口**——不是躍遷副產品，對應 `06_熵潮與存在級威脅.md` §2.3 的預埋。

### 本卷只埋不揭 3 件事
1. **母艦航行的真正目的**——黑盒子暗示「逃離」非「探索」，但不揭穿最終目的地，留給第四卷。
2. **同源信號的完整含義**——確認曾有另一台主機，但不揭示它的製造者與用途。
3. **熵潮不是自然衰亡，可能被人為加速**——暗示裂縫生成頻率有規律性，但不在本卷揭穿。

## §6 一句話總結

主角從被收割預案驅逐的流亡者，成長為能跨艙段行動、指揮多節點協同的新型威脅——在死艙深處掘出了高層最不願被看見的實物證據：他們也在養殖文明，而且養得比他更殘忍。
