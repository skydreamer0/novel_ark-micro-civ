# 第五卷：黑箱斷流 (Volume 5: Blackbox Severance)

## §1 卷定位
- **卷名**：第五卷・黑箱斷流
- **章數**：30 章（121～150 章）
- **情緒曲線**：
  - 前段（121～130）：反向投放後遺症擴散 + 高層反制上線 + 底層加購壓迫全面升級
  - 中段（131～140）：主角切入黑箱補給鏈、建立短時斷流能力 + 內部文明「聖城時代」成形
  - 後段（141～150）：跨艙段斷流實戰 + 高層配額真相揭露 + 首次改寫跨艙段秩序
- **卷終點一句話**：林燼以「可隨時觀測他人內世界回響」的主機外掛，把每一次戰場爽點都接到內世界運算上，第一次讓高層收割鏈斷糧，並逼出母艦遠征其實在兌現更上層收割配額。

## §2 幕次劇情（30章三幕式）

### 第一幕：反向污染回潮（121～130）

#### 劇情任務
- 承接第四卷「反向投放」後果：高層審計中樞啟動新稽核，針對灰槽帶與第七段周邊展開「白名單生存制」。
- 新增加購壓迫機制快速落地，底層被迫在「活命」與「債務」之間二選一。
- 林燼團隊追查黑箱補給鏈入口，發現投放名冊只是表層，真正控制戰爭節奏的是「黑箱斷供閥」。

#### 爽點與鉤子
- 主角以名冊反查，當眾拆穿維穩派代理人的「人道轉運」實為分級耗材投放。
- 蘇岚在審計直播中逆向證明「白名單生存分」可被操縱，打出智商爽點。
- 鉤子：黑箱斷供閥的主密鑰，綁定一批「內宇宙樣本配額帳本」，而帳本抬頭不是母艦高層。

#### 每10章總主題
- **主題**：反向投放不是勝利結束，而是制度戰升級起點。

---

### 第二幕：黑箱切口與聖城成軍（131～140）

#### 劇情任務
- 林燼以灰鏈殘部與中層裂隙勢力組成「斷流小組」，連續打掉三個補給轉運節點。
- 內部文明正式跨入**聖城時代**：從派系拉扯走向「軍團 + 後勤 + 審計神殿」三軌治理，開始穩定反哺外部主線。
- 盟友底線衝突集中爆發：燕九要優先救人，蘇岚要優先保密，林燼必須做出次優解。

#### 爽點與鉤子
- 主角首次達成「12分鐘跨艙段補給斷流」，讓前線高層代理軍團直接停火換氧。
- 內部文明不再只是產能，直接提供「規則審計鏡像」，把高層黑箱帳本映射成可追溯證據鏈。
- 鉤子：斷流成功觸發上層回收程序，出現「配額催收官」的跨艙段授權訊號。

#### 每10章總主題
- **主題**：從偷資料到斷補給，主角完成戰術層向制度層的硬跨階。

---

### 第三幕：配額對決與秩序撬動（141～150）

#### 劇情任務
- 高層掀底牌：以「母艦存續」為名啟動全域斷氧演習，逼主角交出黑箱密鑰。
- 林燼反打「制度級破局」：公開黑箱配額帳本 + 反向投放證據，讓三派（維穩/遷徙/收割）互咬。
- 在跨艙段決戰中完成第二次反向寫入，將「底層生存優先權」臨時寫進一段關鍵艙段的調度規則。

#### 爽點與鉤子
- 高壓下的終局爽點：主角不是單挑勝利，而是讓敵方整套補給制度暫時失效。
- 內部文明聖城軍團首次整建制投影到外部戰場，形成「內外雙戰場共振」奇觀。
- 卷末鉤子：配額帳本指向「星環中轉層」，第六卷入口打開。

#### 每10章總主題
- **主題**：誰掌握補給與配額，誰就定義文明生死。

## §2.5 宏觀資源與壓迫規劃（新增硬指標）

### 四段式資源鏈（本卷主軸）
1. **壓榨觸發**：以「白名單生存分」和「斷氧演習」製造恐慌，逼底層購買生存資格。
2. **外域掠奪**：高層以戰時名義擴大外域回收任務，死亡率上升但配給不升。
3. **制度抽取**：黑箱斷供閥按「配額帳本」優先供給可盈利艙段，低效艙段被系統性抽乾。
4. **個人養殖回灌**：底層被迫簽署意識快照與生理稅貸，個體被做成可重複計價的資產包。

### 加購型壓迫（至少兩種）
- **氧氣訂閱 2.0（家庭綁定）**：單人可停訂，家庭不可拆分，導致家戶連坐式債務。
- **通行競價保命險**：進入低污染區需競價，失敗者被分配到高危維修班。
- **傷病延期稅**：急救可先做，但每延後一次稅款，未來工時扣抵翻倍。

### 內世界貫穿機制（本卷強制）
1. **全員內世界可見化**：每個核心角色都擁有可被觀測的「內世界回響」（恐懼模型／信念構形／執念映射），不得只寫主角有內世界。
2. **主角外掛固定觸發**：林燼可在高壓場景短時進入「主機觀測態」，讀取他人內世界回響中的破綻與需求，作為戰術決策依據。
3. **爽點綁定內世界**：每個 major 爽點都必須同章或下一章顯示「內世界→外部收益」鏈路，避免爽點脫鉤。
4. **關係線靠內世界推進**：盟友衝突不只靠對話，需透過內世界回響揭示其底線來源。

## §3 角色弧

- **林燼（主角）**：
  從「反向入侵者」進化為「供應鏈破壞者」。他學會把爽點從個體反殺升級為制度級斷流；其外掛升級為「內世界回響讀取 + 聖城審計鏡聯算」，可持續理解他人真正驅動力，但也承擔觀測過載與情感污染代價。

- **蘇岚（盟友）不可退讓底線**：
  不接受把活人樣本化為「可迭代耗材」。其內世界核心是「修復者誓約」：寧可慢，也不把人變零件。
  - **衝突**：若要維持斷流窗口，最有效方案是複製敵方樣本標記；這會踩到她底線。
  - **次優解**：改用聖城鏡像演算，成本更高、時間更慢，但不直接消耗活體。

- **燕九（盟友）不可退讓底線**：
  不放棄已承諾撤離的平民名單。其內世界核心是「守約戰紋」：承諾一旦出口，成本由自己吞。
  - **衝突**：戰術上應切斷撤離通道換取決戰優勢。
  - **次優解**：他帶小隊硬守次要通道，主角主線壓力大增但守住信任。

- **核心反派：配額催收官・祁闕**：
  不是單純戰鬥強者，而是擁有跨艙段配給優先權與政策臨時生效權的「制度執行者」。其內世界呈現「秩序潔癖」：把人視為配額噪音，威脅在於能把合法程序變成大規模收割刀口。

- **反派梯度**：
  代理審計官（地區級）→ 黑箱節點總管（艙段級）→ 配額催收官祁闕（跨艙段級）。

## §4 能力路線（戰力/裝備/內部文明）

- **戰力線**：
  從「局部戰場主導」升級到「短時跨艙段補給斷流」，可在 10～15 分鐘窗口改變戰區攻守；並可在戰鬥中即時讀取敵我內世界回響，做動態戰術分流。

- **裝備與權限迭代**：
  1. 名冊索引破解器 → 2. 黑箱斷供閥鏡像器 → 3. 聖城審計鏡（可驗證配額帳本真偽）。

- **文明線（內部世界核心）**：
  聖城時代完成，形成三個可穩定輸出外部主線的模組：
  1. **聖城軍團**：承接高壓戰場的兵力與策略投影。
  2. **神殿後勤院**：把樣本/信仰/資材做財報式轉化，確保每次戰鬥有結算。
  3. **審計祭司庭**：把敵方制度語言翻譯成可操作的破口（讓「內部世界」成為規則武器，而非背景）。

- **卷內跨階成果**：
  內部文明完成「派系治理 → 機構治理」轉變，外部線可直接調用內部制度能力。

## §5 揭密管理

### 本卷必揭 3 件事
1. **黑箱斷供閥如何決定誰先斷糧**（不是故障，而是配額排序）。
2. **母艦外域遠征服務於更高層配額結算**（母艦不是最終決策端）。
3. **每個人都有可被制度利用的內世界回響**，而林燼可把這件事反過來變成破局工具（外掛爽點根據）。

### 本卷只埋不揭 3 件事
1. 星環中轉層的真正治理者身份。
2. 文明主機同源體仍在運行的數量與位置。
3. 祁闕背後的「配額宗主協議」完整條款。

### 伏筆回收規劃（對接追蹤表）
- **白扣身份**：🟡→🟢，在黑箱帳本裡對上真實授權鏈。
- **L-雨棚建造時間異常**：🟡→🟢，補出其作為早期配額節點的建設目的。
- **母核碎片來源**：🔴→🟡，僅揭示來自配額中轉實驗，不揭最終來源。

## §6 一句話總結

當高層把人命寫進配額公式，林燼就把配額公式變成戰場：他用內部文明的秩序能力，第一次讓整艘母艦看到——真正可怕的不是斷糧，而是底層學會了誰在決定誰該斷糧。

## Step 4 連貫性護欄自檢

- **角色邏輯**：延續第四卷後「由破壞轉向建立替代秩序」的走向，未重置人物動機。
- **反派膨脹方式**：以配給權、授權權、程序權膨脹，不只戰力膨脹。
- **盟友主體性**：蘇岚與燕九皆有不可退讓底線，且與主線決策構成真衝突。
- **壓力具象化**：氧氣訂閱、通行競價、傷病延期稅對民生產生可見受害場景。
- **結算節奏**：每三章固定含 [算] 結算，避免連續高壓任務流失控。
- **內世界貫穿**：每 1~2 章至少一次內世界回響觸發，不讓內部世界退化成背景設定。
