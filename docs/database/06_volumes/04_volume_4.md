# 第四卷：投放名冊 (Volume 4: The Deployment Roster)

## §1 卷定位
- **卷名**：第四卷・投放名冊
- **章數**：30 章（91～120 章）
- **情緒曲線**：
  - 前段（91～100）：進入中層勢力博弈的情報戰與試探 + 內部文明派系化陣痛
  - 中段（101～110）：局部戰場主導權爭奪 + 戰爭需求模型初窺
  - 後段（111～120）：投放名冊奪取戰 + 反向投放行動 + 高層降維打擊的化解
- **卷終點**：主角成功揭露並奪取「投放人口名冊與戰爭需求模型」，並掌握一次「反向投放」機會，將內部文明的影響力逆向刻入高層系統。

## §2 幕次劇情

### 第十幕：中層棋局與派系之爭 (91-100章)
#### 劇情任務
- 主角帶著高層養殖實證，正式對接中層勢力（如技術行會或遷徙派的暗線），開始「非暴力層面」的資訊戰。
- 內部文明因鋼鐵時代資源分配不均，出現明顯派系化（祭司派 vs 工匠戰士聯合派），產生內政壓力。
- 尋找「投放名冊」的線索，發現其被分割儲存於多個受高度保護的數據節點。

#### 爽點與鉤子
- 資訊戰初體驗：利用文明主機的算力優勢，在數據流中碾壓高層的常規防火牆。
- 內部派系衝突爆發：主角必須以「神」的姿態介入調停，展現全新的統御手腕。
- 鉤子：發現名冊中記載的「戰爭需求」與目前的母艦航向存在致命矛盾（加深「逃離」伏筆）。

#### 幕級三線進度目標
- **戰力線**：完成從小隊硬拼到「主機算力干預 + 多節點誘導」的第一次中層級實戰升級，並奪得第一個可用的中層窗口。
- **文明線**：內部文明從單一求生共識進入派系化治理，林燼必須第一次以統御者而非單純資源供應者的姿態壓住內部撕裂。
- **真相線**：從加購令、通行競價與節點路由中，摸到投放名冊分片與戰爭需求模型的制度證據，讓「逃離」不再只是猜想。

### 第十一幕：局部戰爭與模型現世 (101-110章)
#### 劇情任務
- 為了奪取數據節點，主角在中層勢力的暗中支持下，主導一場局部戰爭（可能發生在某個資源中樞艙段）。
- 戰爭過程中，揭露「戰爭需求模型」：高層其實是根據外部戰爭的「損耗率」來反向決定底層人口的「投放率與收割率」。
- 內部文明的「神權科技混合」在局部戰爭中大放異彩，形成穩定的戰術體系。

#### 爽點與鉤子
- 主導戰場：不再是小隊游擊，而是指揮多節點部隊進行陣地戰與資源截斷戰。
- 模型揭露的震撼：底層人命只是高層報表上的一個動態參數。
- 鉤子：發現主角自己的人員（或重要盟友）被列入了下一批次的「高優化投放清單」。

#### 幕級三線進度目標
- **戰力線**：林燼從突擊者轉為局部戰場調度者，具備奪車、截流、規則偽裝與主動投案滲透的複合能力。
- **文明線**：神權科技混合完成第一次大規模戰場應用，並付出人口、算力與信仰的真實代價，讓內部文明不再只是後勤掛件。
- **真相線**：完整解出戰爭需求模型的運作邏輯，並把母艦航行目標從「探索」翻轉為「逃離某種深淵結構」。

### 第十二幕：反向投放與名冊奪回 (111-120章)
#### 劇情任務
- 發動總攻，駭入/攻陷儲存完整「投放名冊」的核心資料庫。
- 面對極端收割派的瘋狂反撲與高層決策層的直接干預。
- 主角利用奪取的權限與文明主機的能力，實施「反向投放」——不是將人投放出去，而是將內宇宙的規則/病毒/意識反向植入高層的高優化投放清單中。

#### 爽點與鉤子
- 規則級對抗：文明主機的規則引擎 vs 高層的審計體系核心演算法。
- 反向投放的成功：給高層系統埋下一顆定時炸彈，完成跨艙段的秩序影響前置。
- 卷終點：名冊到手，主角勢力正式成為中層博弈的關鍵變數；高層察覺「異物」入侵系統，但已無法輕易剔除。

#### 幕級三線進度目標
- **戰力線**：拿到完整名冊副本、完成一次反向投放，並形成可短時干擾高層系統的規則級手段。
- **文明線**：在大犧牲後重整內部秩序，從狂熱動員轉向「有底線的神權治理」，為下一卷的長期戰爭體系打底。
- **真相線**：讓投放名冊、戰爭需求模型、同源信號與高層審計中樞首次被串成同一條證據鏈，但仍保留最高 AI 與最終目的地不揭。

## §3 角色弧
- **林燼 (主角)**：從「執棋者」進化為「破局者」。學會利用中層矛盾，並開始思考如何建立能取代現有高層的「新秩序」，而不僅僅是破壞。
- **蘇岚 (盟友)**：「意識快照」技術的道德困境爆發，最終決定不依賴高層技術，轉而利用內部文明的機制來備份/救治弟弟。
- **燕九 (盟友)**：適應指揮官角色，在局部戰爭中學會冷酷的戰損計算，但也始終堅守不將隊友視為純耗材的底線。
- **核心反派 (極端收割派中層指揮官)**：冷酷的數據信徒，認為一切都可以用「戰爭需求模型」來最佳化，最終被主角的「不確定性（反向投放）」擊潰。

## §4 能力路線
- **戰力線**：中層勢力博弈主導局部戰場。獲得跨艙段的數據/情報干預能力。
- **文明線**：文明派系化（內政壓力考驗統御力）。神權科技混合落地，形成穩定的「聖城」前置後勤與部隊體系。
- **裝備迭代**：從物理武裝過渡到資訊/規則級武裝。利用中層資源大幅強化機動性與通訊穩定性。

### 第四卷進度檢查點
- **第95章前後**：確認中層壓迫不是單點行政命令，而是與淨化加購、通行競價與清場收割綁在一起的制度鏈。
- **第100章前後**：必須拿到第一部分名冊或等值解碼入口，證明主角已從追查者升級為奪取者。
- **第105章前後**：戰爭需求模型從概念曝光升級為可被逆用的系統規則，主角開始以自己作為入侵楔子。
- **第110章前後**：林燼成功把自身送進審計中樞或等值核心場域，讓真相線與戰力線完全合流。
- **第115章前後**：反向投放完成，但創傷、折損與盟友犧牲必須完整結算，不能只寫成純勝利。
- **第120章卷末**：主角取得自治籌碼與完整名冊副本，同時被高層升格為「系統級病毒」，為第五卷主動出擊建立必要理由。

## §5 揭密管理
### 必揭 3 件事
1. **投放名冊與戰爭需求模型**：完整揭露收割與投放的數學邏輯。
2. **中層政治生態**：遷徙派與維穩派的具體運作方式與妥協底線。
3. **同源信號的初步解析**：確認同類主機曾經嘗試過改變母艦軌跡（呼應L-雨棚與逃離伏筆）。

### 只埋不揭 3 件事
1. **母艦的最終目的地**。（繼續加深，將「逃離」具象化為某個特定座標的迴避）。
2. **文明主機創造者的真實身分**。
3. **高層核心（長老會/最高AI）的真面目**。

## §6 一句話總結
林燼帶著從死艙挖出的血淋淋證據踏入中層棋局，在冰冷的戰爭需求模型面前，他以「反向投放」將內宇宙的規則像病毒般釘入了高層的命脈。

### 第四卷三章微循環索引
- **91～93**：加購壓迫啟動 -> 接觸遷徙派 -> 財報結算，建立中層對接入口、概念偽裝碎片與燕九污染代價。
- **94～96**：競價系統踩點 -> 節點突襲 -> 民生結算反噬，讓制度壓迫與主角行動的外部後果第一次正面撞上。
- **97～99**：內部關係回收 -> 政治保護費抽取 -> 主機戰備回灌，完成從倫理困境到主動設局的轉向。
- **100～102**：第一列火車強攻 -> 戰損盤點 -> 地圖熵化截車，將局部勝利立即轉成更高難度的時間競賽。
- **103～105**：三方混戰截胡 -> 醫療與民生結算 -> 反向思考，讓名冊、救援與自我投案三條線扣在一起。
- **106～108**：投案滲透 -> 內宇宙監牢戰 -> 接觸審計核心，推進主角作為楔子的高風險潛入。
- **109～111**：總星圖真相揭露 -> 反向投放前奏 -> 蘇岚佯攻，讓真相線與外部戰線同步進入極限狀態。
- **112～114**：禁忌儀式燃燒 -> 反向投放完成 -> 大逃亡，實現規則級反殺並立刻承受物理層追捕。
- **115～117**：創傷盤點 -> 分贓重建 -> 民生喘息，讓勝利代價與短暫自治紅利同時落地。
- **118～120**：內部新法則立下 -> 結算升級 -> 卷末收束，將林燼從破局者推向準制度制定者，接出第五卷補給線總攻。
