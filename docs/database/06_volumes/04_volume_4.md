# 第四卷：投放名冊 (Volume 4: The Deployment Roster)

## §1 卷定位
- **卷名**：第四卷・投放名冊
- **章數**：30 章（91～120 章）
- **情緒曲線**：
  - 前段（91～100）：進入中層勢力博弈的情報戰與試探 + 內部文明派系化陣痛
  - 中段（101～110）：局部戰場主導權爭奪 + 降維打擊的碾壓局
  - 後段（111～120）：主動攻入審計中樞大殺四方 + 掠奪海量資源與系統最高全限
- **卷終點**：主角憑藉極致武力與內部文明的黃金法則，硬生生打穿高層防線，奪取「投放人口名冊與戰爭需求模型」，並讓高層系統徹底臣服，成為中下層的無冕之王。

## §2 幕次劇情

### 第十幕：中層棋局與派系之爭 (91-100章)
#### 劇情任務
- 主角帶著高層養殖實證，正式對接中層勢力（如技術行會或遷徙派的暗線），開始「非暴力層面」的資訊戰。
- 內部文明因鋼鐵時代資源分配不均，出現明顯派系化（祭司派 vs 工匠戰士聯合派），產生內政壓力。
- 尋找「投放名冊」的線索，發現其被分割儲存於多個受高度保護的數據節點。

#### 爽點與鉤子
- 資訊戰初體驗：利用文明主機的算力優勢，在數據流中碾壓高層的常規防火牆。
- 內部派系衝突爆發：主角必須以「神」的姿態介入調停，展現全新的統御手腕。
- 鉤子：發現名冊中記載的「戰爭需求」與目前的母艦航向存在致命矛盾（加深「逃離」伏筆）。

#### 幕級三線進度目標
- **戰力線**：完成從小隊硬拼到「主機算力干預 + 多節點誘導」的第一次中層級實戰升級，並奪得第一個可用的中層窗口。
- **文明線**：內部文明從單一求生共識進入派系化治理，林燼必須第一次以統御者而非單純資源供應者的姿態壓住內部撕裂。
- **真相線**：從加購令、通行競價與節點路由中，摸到投放名冊分片與戰爭需求模型的制度證據，讓「逃離」不再只是猜想。

### 第十一幕：降維打擊與收割盛宴 (101-110章)
#### 劇情任務
- 為了奪取數據節點，主角在中層勢力的暗中支持下，主動發起一場跨艙段掠奪戰。
- 戰爭過程中，揭露「戰爭需求模型」：高層其實是根據外部戰爭的「損耗率」來反向決定底層人口的「投放率與收割率」。
- 內部文明的「神權科技混合」迎來技術爆炸，在戰場上形成絕對的火力碾壓。

#### 爽點與鉤子
- 降維打擊：從游擊戰直接轉變成平推局，主機出產的規則武器把高層的護甲當紙撕。
- 模型揭露的震撼：底層人命看似螻蟻，而主角卻藉此找到了掌控高層命脈的密碼。
- 鉤子：發現主角被列入了「高優化投放清單」，主角冷笑決定將計就計，主動送上門去「打劫」審計中樞。

#### 幕級三線進度目標
- **戰力線**：林燼化身為主戰場的死神，展現徒手拆機甲、單人破陣的無敵姿態，強勢掠奪海量經驗值與裝備。
- **文明線**：神權科技混合大發神威，內部文明吞噬敵方裝備，迎來全民強化的黃金時代，狂熱信仰轉化為實質作戰增益。
- **真相線**：完整解出戰爭需求模型的運作邏輯，並把母艦航行目標從「探索」翻轉為「逃離某種深淵結構」。

### 第十二幕：中樞洗劫與霸主誕生 (111-120章)
#### 劇情任務
- 發動總攻，強行轟開儲存完整「投放名冊」的核心資料庫大門。
- 面對極端收割派的瘋狂反撲與高層決策層的直接干預，主角以絕對暴力將其全數鎮壓。
- 主角利用奪取的權限與文明主機的能力，實施「反向投放」——將內宇宙的規則像霸王條款一樣強行植入高層中樞，把對方洗劫一空。

#### 爽點與鉤子
- 單挑中樞：以一人之力對抗整個高層AI與護衛軍，打出「我即是天災」的壓制力。
- 反向投放的盛宴：不僅拿走名冊，還把高層庫房裡的頂級物資全數搬空，看高層想管又管不了的憋屈樣。
- 卷終點：滿載而歸，主角勢力強勢割據，逼迫中層勢力臣服納貢；高層眼睜睜看著霸主崛起卻無能為力。

#### 幕級三線進度目標
- **戰力線**：拿到完整名冊副本、搜刮高層終極神裝，戰略與單兵能力並立頂峰。
- **文明線**：無傷大勝，帶回的海量高階資源讓內部文明直接躍升，信徒尊奉主角為萬能真神，打下統治階級的鐵血基盤。
- **真相線**：讓投放名冊、戰爭需求模型、同源信號與高層審計中樞首次被串成同一條證據鏈，但仍保留最高 AI 與最終目的地不揭。

## §3 角色弧
- **林燼 (主角)**：從「破局者」進化為「霸主」。不僅打破規則，更自己制定規則，順我者昌、逆我者亡的極致爽文男主氣場。
- **蘇岚 (盟友)**：不再陷入道德困境，而是藉由主機的無上威能，毫無副作用地治好弟弟，並成為林燼麾下最可怕的海客女王。
- **燕九 (盟友)**：鳥槍換炮，穿上神級概念武裝，化身無敵推土機，享受暴打高層精英的快感。
- **核心反派 (極端收割派中層指揮官)**：自以為掌控一切的數據信徒，最終在林燼的降維打擊下精神崩潰，被林燼像踩死蟲子一樣碾碎。

## §4 能力路線
- **戰力線**：中層勢力博弈主導局部戰場。獲得跨艙段的數據/情報干預能力。
- **文明線**：文明派系化（內政壓力考驗統御力）。神權科技混合落地，形成穩定的「聖城」前置後勤與部隊體系。
- **裝備迭代**：從物理武裝過渡到資訊/規則級武裝。利用中層資源大幅強化機動性與通訊穩定性。

### 第四卷進度檢查點
- **第95章前後**：確認中層壓迫不是單點行政命令，而是與淨化加購、通行競價與清場收割綁在一起的制度鏈。
- **第100章前後**：必須拿到第一部分名冊或等值解碼入口，證明主角已從追查者升級為奪取者。
- **第105章前後**：主角把戰爭需求模型當成自己的提款機，開始主動籌劃對高層的洗劫方案。
- **第110章前後**：林燼偽裝被捕，實則以君王之姿踏入審計中樞，準備大開殺戒。
- **第115章前後**：反向寫入與搜刮同時進行，敵人死傷慘重而己方零戰損，爽感瘋狂飆升。
- **第120章卷末**：帶著無數頂級物資與名冊榮耀回歸，中下層集體顫抖臣服，為第五卷出征奠定王座。

## §5 揭密管理
### 必揭 3 件事
1. **投放名冊與戰爭需求模型**：完整揭露收割與投放的數學邏輯。
2. **中層政治生態**：遷徙派與維穩派的具體運作方式與妥協底線。
3. **同源信號的初步解析**：確認同類主機曾經嘗試過改變母艦軌跡（呼應L-雨棚與逃離伏筆）。

### 只埋不揭 3 件事
1. **母艦的最終目的地**。（繼續加深，將「逃離」具象化為某個特定座標的迴避）。
2. **文明主機創造者的真實身分**。
3. **高層核心（長老會/最高AI）的真面目**。

## §6 一句話總結
林燼帶著染血的證據踢翻棋盤，用絕對武力與降維打擊撕裂高層防線，不但強搶命脈名冊，更把高層中樞洗劫一空，在廢墟上加冕為王。

- **118～120**：內部文明神權暴漲 -> 戰力突破天際 -> 卷末收束，林燼坐穩無冕之王寶座，帶著滿級神裝準備向高層算總帳。
