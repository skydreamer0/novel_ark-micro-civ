## 第四卷使用提醒

- **每 3 章至少要有一次顯性結算**：把「投入 -> 折損 -> 產出」寫清楚，不能只寫打贏了。
- **民生成本必須具象化**：淨化包、氧氣、保命險、醫療補給要落在具體人群，而不是只寫成抽象稅名。
- **制度收益不能零代價**：拿到通行權、自治權或名冊片段時，必須同步出現監控升級、戰損或盟友折損。
