# 社會階級與制度壓迫 (Social Order)

## 1. 存活點 (Survival Credits)
- **唯一貨幣**：底層通用代幣，代表維持生存的最低資源消耗。
- **每日固定支出 (30點/天)**：
  - 基礎氧稅：10 點
  - 床位租金：10 點
  - 抗輻射廢水與營養膏：10 點
- **壓迫邊界**：餘額低於 100 點將受「黃卡」警報；低於 20 點強制扣押器官或勞役。

## 2. 社會控制與剝削機制
- **人口紅利**：持續投放複製人、債務人口，稀釋勞動價值。
- **認證鎖死**：獎勵綁定終身契約（企業狗）或不可逆改裝。
- **審計監控**：演算法與審計官雙軌監測異常資產增長與規則偏移。
- **收編 SOP**：招募 -> 測試 -> 權限剝奪 -> 人格重置。

## 3. 資源抽取鏈 (The Extraction Chain)
資源從裂縫到高層「內宇宙」的四級剝削環節：
1. **外域收割 (物理風險)**：拾荒者獲取原始樣本。
2. **通路估值 (制度打壓)**：清譜會以「污染超標」強行低價收購，抽成 30-50%。
3. **官方徵收 (法律搶奪)**：審計司徵收「淨化門檻稅」與「通行費」。
4. **內宇宙沉澱 (最終歸宿)**：剩餘能量轉為高層內宇宙規則燃料。

## 4. 附加壓迫制度
- **延淨排序**：未購買「加標服務」者，醫療與淨化優先順位永遠墊底。
- **戰時稅則**：區域衝突時，生存物資稅率可由官方單方面瞬間調漲 300%。
- **審計豁免權**：企業聯盟專屬，可規避物資管制，二次盤剝底層資源。
- **盲舵領航**：官方默許灰產引領底層探索死艙，預收高額開荒死亡稅。

## 5. 衝突與反擊邏輯
- **生存驅動**：每一次能力升級會引發更高額的維護費與審計關注。
- **漏洞套利**：主角反擊不靠合法交易，靠主機提供的「系統漏洞」進行降維打擊。
- **灰鏈建立**：建立私人走私通道對抗「清譜會」的估值壟斷。
