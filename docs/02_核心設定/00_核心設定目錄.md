# 核心設定目錄

此資料夾用於集中管理《末日母艦：我的體內有個微縮文明》最核心的作品定位、角色設定與力量系統，是整部作品的靈魂基表。

## 檔案導覽

- `01_核心設定與賣點.md`：提煉作品的爽點、核心命題與商業定位，供快速理解與推廣用。
- `02_角色與陣營/`：目錄下存放主角軌跡、配角與敵對勢力、以及角色關係圖。
- `03_力量與文明系統.md`：文明主機、雙線成長、力量階段、反哺規則與失真機制。

## 維護規範

1. 核心賣點與定位（`01_`）通常不輕易更動，除非作品方向有重大調整。
2. 主角的成長軌跡（`02_`）應隨劇情進度定期更新，以確保角色行為的連貫性。
3. 力量系統（`03_`）每當解鎖新時代或新反哺能力時，需同步更新。
