# 角色與陣營目錄

此資料夾用於集中管理《末日母艦：我的體內有個微縮文明》所有出場人物、勢力組織與彼此間的利益關係。

## 檔案導覽

- `01_主角成長軌跡.md`：林燼的心路歷程、戰鬥風格、以及內在文明階段的代價管理。
- `02_陣營與敵對勢力.md`：各個「卡脖子」的敵對組織（如鐵鏽幫、承包軍、審計司）、戰鬥兵種與盟友設計。
- `03_角色關係圖.md`：主角團與外部勢力的利益鏈、壓制鏈與人情鏈總覽。

## 維護規範

1. 每次新增重要配角或敵對組織，應更新 `02_陣營與敵對勢力.md`，並標註他們的【壟斷資源（卡脖子點）】。
2. 若有重大的背叛、結盟或生死債產生，應同步更新 `03_角色關係圖.md`。
