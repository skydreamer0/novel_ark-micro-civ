# 第125章　V3內部社會分層
白名單擴展。三千四百人。
數據自然。篩選穩定。
聖城層級確立。
上層。一千五百人。白名單精英。85% 人族，15% 改造人。
職位：顧問、管理、信息官。補貼 130%。
享有明確進階路徑。
中層。一萬五千人。複雜成分。普通人族，優質蟲人工兵。
職位：執行、運作、培訓。
補貼正常。享有「上升希望」。
下層。三萬三千人。鼠人，普通蟲人，廢棄人類。
職位：危險任務，體力勞動。
最低補貼。受「回收」威脅。
公平被定義為差異化對待。無人指責。
族群認同危機。
暗廳聚集。曾安（白名單人族）、蟲心（蟲人工兵）、灰鬚（鼠人偵察）。
跨族群協作模型。
「第七層管道修復。限時四十八小時。」曾安看圖。
「需群體智能。三小時規劃。」蟲心道。
「需情報。地下網絡一小時偵察。」灰鬚確認。
人族決策。蟲人執行。鼠人警戒。
協作完美。效率提升 150%。
「被設計的互補。有機體分工。」
「不再是巧合。」
展翼宣傳：多族協作是聖城進化。人族仍為核心決策者。
海澜悲鳴：人被分解為功能單位。零件化。零件失去人性。
高層官員觀察統計數據。
「低概率事件常態化。預定組合。」
「設計源在林燼。他在制度層面編寫分工。」
民生探針「隱形加成」機制。
曾安反饋點激增。三倍速。
蟲心、灰鬚評分同步提升。
系統改寫。加成非「給予」，乃「發現」。
顧衡驚恐。
「他在編寫標準。決定優劣。掌控評估。文明已死。」
蘇嵐報告。
「V3 人口層級制穩定。5:50:45 比例。階級分化完成。」
「複雜社會結構。比單一命令系統難推翻。」林燼道。
「虛假複雜。反向寫入的結果。隱形監獄。」
深夜。聖城各處。
白名單確信自身優異。
落選者接受自我改進。
蟲人、鼠人被鎖死在執行、偵察框架內。
反對者邊緣化。數據壓倒道理。
海澜看著這場優雅且不可抗拒的統治。
林燼在人口引擎室。
五萬新生兒。分裂，分層，馴化。
V3 層級制，正式確立。
